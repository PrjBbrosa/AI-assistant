# 预紧力（FM，FM,min / FM,max）

**一句话**：装配拧紧时螺栓施加在被夹件上的轴向夹紧力，是整个 VDI 2230 校核链的核心状态量；因工艺散差呈区间分布 [FM,min, FM,max]。

**怎么理解**：

预紧力是螺栓连接"活着"的前提。外力作用时，预紧力让被夹件保持压紧，只有**剩余的那部分**（ΦN·FA）会落到螺栓上。预紧力的存在让螺栓远离疲劳与松动失效。

装配时，同一颗螺栓、同一个扳手，实际产生的 FM 不是一个点，而是一个**区间**：

```
FM,max = αA · FM,min          [N]     # αA = 拧紧系数（装配散差）
FM,max − FM,min = (αA − 1)·FM,min     # 装配散差带
```

## 两种计算模式（本工具）

### 模式 A：设计模式（默认）

给定 **外载 FA,max**、**密封要求 FK,req**、**防滑要求 FK,req,slip**、**嵌入损失 FZ**、**热损失 Fth**，**反推** FM,min（从而反推 FM,max、推荐的拧紧扭矩 MA）。

```
R1 反推:  FM,min ≥ FK,req + FA,max·(1−ΦN) + FZ + Fth
         FM,max = αA · FM,min
         MA = FM,max · (几何+摩擦因子)
```

此模式下，R3 恒满足（设计保证），实际结果回答"需要设计多大预紧力"。

### 模式 B：校核模式

已经装配完毕，**知道 FM,min**（从扭矩扳手读数推算或直接测量），**校核**是否满足密封 / 防滑 / 服役要求。

```
FM,work = FM,min − FZ − Fth − (1−ΦN)·FA,max
R3 通过: FM,work ≥ FK,req
```

本工具用于事后校核或"套现有设计"场景。

## 典型值（作为量级感）

| 连接场景 | FM,min 量级 |
|---|---|
| 家电螺钉（M3-M5） | 1–3 kN |
| 通用机械（M8-M12） | 10–40 kN |
| 汽车底盘（M14-M18） | 50–120 kN |
| 法兰密封（M20-M24） | 100–200 kN |
| 重型机械 / 塔筒（M27-M36） | 200–500 kN |
| 压力容器 / 核电（M42+） | 500–1500 kN |

## FM,min 与螺栓尺寸的关系

选型时常用的"FM,min / As" 约束：

```
σ_ax,asm = FM,max / As = αA·FM,min / As ≤ ν·Rp0.2
→ FM,min ≤ ν·Rp0.2·As / αA           [N]
```

对 M10 8.8 级、αA = 1.6、ν = 0.9：
```
FM,min 上限 = 0.9 · 640 · 58 / 1.6 = 20.9 kN
```

若设计需求 FM,min > 20.9 kN，M10 8.8 级装配会过紧 —— 要么换更粗规格（M12）、要么换更高等级（10.9 级可到 29 kN）。

## 为什么 FM,min 远离 0

新手常问："为什么不能让 FM,min 很小以节约螺栓？" 答：

1. **密封需要持续压力**：密封垫片要保持 10–20 MPa 接触压强 → FM,min 不可低于垫圈面积乘以要求压强
2. **防滑需要摩擦力**：`FM,min ≥ FQ,max / (qF·μT)`，横向载荷越大要求越高
3. **防松需要残余夹紧**：FM,min 过低时螺栓相对位置易微动 → 螺纹磨损 → 预紧力进一步流失 → 松脱

工程经验：**FM,min 不应低于 FM,max 设计上限的 40%**。若低于此值，说明连接过度设计（浪费）或散差控制过严（不现实）。

## 常见误用

- **在校核模式下把 MA 当 FM,min**：MA 是扭矩（N·m），FM 是力（N），两者关系需要摩擦模型换算，不能直接代入。
- **"推荐 MA" 的数值就是实际装配 MA**：工具推荐的是"按 FM,max 反算"的理论值，实际工艺中要考虑扳手校准误差、工人手感偏差、反复拆装衰减等。
- **忽略 FZ 和 Fth 的损失**：装配完立刻测的 FM 是 "FM,装配完"，不等于 "FM,工作时" —— 嵌入和热差会吃掉 10–30%。
