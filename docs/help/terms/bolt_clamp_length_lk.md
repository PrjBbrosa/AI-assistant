# 总夹紧长度 lK

**一句话**：被夹件总厚度之和（毫米），螺栓柔度 δ_S 和被夹件柔度 δ_P 的共同基准长度，也是沉陷损失、热损失估算的关键输入。

**怎么理解**：

lK 是螺栓轴向预紧力作用的有效长度——从螺栓头支承面到螺母支承面（DSV），或到基体螺纹孔入口（ESV）之间的总被夹厚度：

```
lK = sum_i t_i                    # 各层被夹件厚度之和 [mm]
```

lK 同时是两个柔度的分子：

```
螺栓柔度估算：delta_S ~ lK / (E_S * A_S)            # [mm/N]（简化公式）
圆柱被夹件：  delta_P = lK / (E_P * pi/4*(D_A^2 - d_h^2))  # [mm/N]
```

lK 越大，delta_S 和 delta_P 的绝对值均增大；但 Φ_N = delta_P / (delta_S + delta_P) 主要由两者的**比值**决定，与 lK 耦合但不是单调关系。单纯增大 lK 不能直接判断 Φ_N 升降——需同时考虑螺栓截面、被夹件截面及材料模量的相对关系。

## 与其他长度的区别

| 长度符号 | 含义 | 典型取值基准 |
|---|---|---|
| lK | 被夹件总厚度 | sum_i t_i，不含螺栓头/螺母高度 |
| l_bolt（螺栓全长）| 螺栓总长，从头顶到杆末 | lK + 头高 + 螺纹余量 |
| m_eff（有效啮合长度）| ESV 有效螺纹旋合深度 | 基体孔深 - 1.5 牙（首尾折减） |
| l_h（头下夹紧长度）| 螺栓头/螺母承压区外的杆段 | 通常 = lK（单层无垫圈）|

混淆 l_bolt 与 lK 是工程师最常犯的错误——螺栓总长比 lK 大约 1.5–2d（头高 + 螺母高 + 螺纹余量）。

## ESV 的特殊修正

对 ESV（螺纹孔连接），螺栓受力段延伸至旋合螺纹内。本工具 `compliance_model.py` 使用简化两段叠加模型：螺栓头过渡附加长度取 `0.4·d`，螺纹旋合端附加长度取 `0.33·d`，合计有效长度：

```
l_eff,ESV = lK + 0.4*d + 0.33*d                     # [mm]
delta_S   = l_eff / (E_S * A_S)                      # [mm/N]
```

通孔连接（DSV）对应螺母端附加取 `0.4·d`，合计 `lK + 0.4·d + 0.4·d`。具体数值参见 `core/bolt/compliance_model.py` 的 `calculate_bolt_compliance` 函数。此修正仅影响柔度计算，不影响 lK 本身的定义（lK 仍为被夹件厚度和）。

## 典型值参考

| 连接场景 | 典型 lK | 说明 |
|---|---|---|
| 薄板紧固（2 mm 钢板 + 3 mm 铝板）| 5 mm | 通常 < 0.5d，需短螺栓 |
| 中等法兰连接（2×10 mm 钢法兰）| 20 mm | 常用工况 |
| 厚凸缘（3×20 mm 铸件）| 60 mm | 长螺栓，delta_S 大，Φ_N 由 delta_P/delta_S 比值决定 |
| 发动机缸盖螺栓（铝体）| 80–100 mm | 需考虑热膨胀差 |

## 常见误用

- **填螺栓总长而非被夹件厚度**：M10 螺栓如果总长 40 mm，螺栓头高约 8 mm、螺母高约 8 mm，实际 lK ≈ 24 mm，若填 40 mm 高估柔度 67%，Φ_N 偏差导致预紧力计算严重失误。
- **多层连接只填一层厚度**：有平垫圈时忘记加垫圈厚度（典型 M10 垫圈厚 2 mm），漏了就少 2–4 mm，对短 lK 比例影响不小。
- **ESV 时 lK 还要手动加螺纹段长度**：柔度计算内部的附加长度（0.33·d）已由工具自动叠加，lK 字段本身仍填被夹件厚度和，不要手动加入螺纹段；工具的 `calculate_bolt_compliance` 函数会在内部自动处理。

**出处**：VDI 2230:2015 §5.1.2.2 Eq. 5.1（螺栓柔度）；VDI 2230:2015 §5.1.2（被夹件柔度定义）；DIN ISO 898-1（螺栓几何尺寸）
