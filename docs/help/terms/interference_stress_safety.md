# 材料最小安全系数（S_σ,min）

**一句话**：过盈连接**轴 / 轮毂应力校核**共用的最小安全系数门槛；"材料屈服强度 ÷ 最大 von Mises 应力" 必须不小于它，校核才算 PASS。

**怎么理解**：

过盈连接的第二类失效是"最大过盈端压坏"：加工公差组合到**最不利**情形时，接触压力达到 p_max，轴外壁和轮毂内壁产生的 von Mises 应力可能超过材料屈服强度。一旦屈服，连接从弹性进入塑性、过盈量快速松弛、扭矩容量持续下降，属于"首次装配就埋雷"的失效。

S_σ,min 就是在 p_max 工况下，要求两侧都留出的屈服裕度。本工具对轴侧与轮毂侧**分开校核**，共用同一个 S_σ,min 门槛：

1. **轴侧**：`S_shaft = Re_shaft / σvm_shaft,max ≥ S_σ,min`
2. **轮毂侧**：`S_hub = Re_hub / σvm_hub,max ≥ S_σ,min`

两侧任一不满足都判 FAIL。

## 公式（本工具实现）

von Mises 应力按 Lamé 解二维应力合成（轴向应力 ≈ 0，忽略装配产生的轴向预应力）：

```
厚壁几何因子（均为无量纲比值）：
   K_hub   = (D² + d²) / (D² − d²)                  [D,d: mm → K_hub: 无量纲]
   K_shaft = 1（实心）
           或 (d² + d_i²) / (d² − d_i²)（空心）    [d,d_i: mm → K_shaft: 无量纲]

von Mises 系数（也都是无量纲）：
   轮毂: VM_hub   = sqrt(1 + K_hub   + K_hub²)     [无量纲]
   轴:   VM_shaft = sqrt(1 + K_shaft² − K_shaft)   [无量纲]

最大过盈端应力：
   σvm_shaft,max = p_max · VM_shaft   [p_max: MPa, VM: 无量纲 → σvm: MPa]
   σvm_hub,max   = p_max · VM_hub              [σ: MPa, p: MPa]

安全系数：
   S_shaft = Re_shaft / σvm_shaft,max          [无量纲]
   S_hub   = Re_hub   / σvm_hub,max            [无量纲]

PASS 判据：S_shaft ≥ S_σ,min 且 S_hub ≥ S_σ,min
```

`K_hub` 越大（薄壁轮毂），von Mises 越大、应力安全系数越低。轮毂加厚到 `D/d ≥ 2.5` 左右时 K_hub → 1.4 趋近极限。

## 典型值

| 场景 | S_σ,min 建议 | 说明 |
|---|---|---|
| **静态 / 稳态，延性材料（钢、球铁）** | 1.1 ~ 1.3 | 允许微量塑性余量 |
| **动载 / 循环，延性材料** | 1.3 ~ 1.5 | 避免循环塑性 |
| **脆性材料（灰铸铁）** | 1.8 ~ 2.2 | 缺少塑性余量 |
| **冲击 / 安全关键** | 1.5 ~ 2.0 | 结合后果严重度上调 |
| **高温 / 高压工况** | 1.8+ | 材料性能随温度退化 |

**本工具默认 1.20**，是"稳态延性材料"的中值；脆性材料 / 冲击工况请上调。

## S_σ 与 S_slip 的区别

容易搞混：
- **S_slip,min** → 最小过盈端 + 滑脱支线 + "能不能传得动"
- **S_σ,min**   → 最大过盈端 + 应力支线 + "会不会压坏"

两者作用于过盈窗口的**相反端**：过盈越大越容易压坏，过盈越小越容易滑脱。S_slip 与 S_σ 共同定义了**过盈窗口的工程可接受范围**；过盈配合设计的核心就是让这个范围不为空。

## 常见误用

- **用 Rm（抗拉强度）代替 Re（屈服）**：本工具以屈服强度为分母；换成 Rm 会让 S_σ 虚高 30~50%。
- **忽视脆性材料折减**：灰铸铁没有真正屈服点，本工具按 Re 字段处理；脆性材料需把 S_σ,min 进一步上调（1.8~2.2）。
- **轴和轮毂用同一 Re**：钢轴 + 铝轮毂时，两侧 Re 差 2~3 倍；分开填是必要的。
- **高温工况按常温 Re**：碳钢 350°C 以上 Re 快速下降，铝合金 150°C 以上已显著退化；高温场合需按热态 Re 输入。
