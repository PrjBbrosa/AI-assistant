# 屈服强度（Rp0.2）

**一句话**：螺栓材料在 0.2% 残余塑性变形时的应力值，是螺栓"开始永久变形"的门槛，也是装配与服役应力校核的 **allowable 值基准**。

**怎么理解**：

金属拉伸曲线上没有像石头那样明确的"断点"，而是平滑过渡。工程上约定：**当卸载后材料仍有 0.2% 永久变形时**对应的应力，称为 Rp0.2 或 σ0.2。超过这个值螺栓就"拉松了"，即使卸载也不会回到原长。

Rp0.2 直接进入本模块最核心的两个校核：

```
R4 装配等效应力  σ_vm,asm ≤ ν · Rp0.2            # ν 典型 0.9
R5 服役等效应力  σ_vm,work ≤ Rp0.2 / S_F          # S_F 典型 1.1
```

R8 螺纹脱扣也用 Rp0.2 作为 τ_BS 默认值：

```
τ_BS (外螺纹剪切强度) 默认 = 0.6 · Rp0.2    [τ: MPa]
```

## Rp0.2 从哪来

1. **从强度等级自动填入**（首选）：选"8.8" → Rp0.2 = 640 MPa 自动填入并锁定字段。
2. **选"自定义"手动输入**：适用于不锈钢（A2-70 Rp0.2 ≈ 450 MPa）、铝合金螺栓、特殊材质。
3. **复查工厂证书**：关键连接应取供应商实测值而非标称值；批次偏差 ±5% 常见。

## 与抗拉强度 Rm 的关系

Rp0.2 < Rm。两者之差称为"强化裕度"，反映材料到达屈服后还能继续承载多少：

- **中碳钢调质**：Rp0.2 / Rm ≈ 0.8（8.8 级定义值）
- **合金调质**：Rp0.2 / Rm ≈ 0.9（10.9 / 12.9）
- **延性铝合金**：Rp0.2 / Rm 可低至 0.5–0.7

强化裕度高 = 脆性风险高（延性差）。高强度螺栓虽然 Rp0.2 高，但**断裂前塑性变形少**，预警时间短。

## 温度影响

Rp0.2 随温度升高显著下降：

- **≤ 室温 25 °C**：使用表列名义值
- **100 °C**：≈ 0.95 × 室温值
- **200 °C**：≈ 0.85 × 室温值（8.8 级从 640 降到 ~540）
- **300 °C**：≈ 0.70 × 室温值
- **> 350 °C**：强度急剧下降，超出常规螺栓使用范围

高温连接请按工作温度下的 Rp0.2 修正（本工具尚未实现温度自动修正，需手动填入高温下的 Rp0.2）。

## 常见误用

- **把 Rm 填入 Rp0.2**：高估允许应力约 25%，所有安全系数虚高。
- **选了"自定义"又没改默认值**：下拉切到"自定义"后 Rp0.2 字段解锁，默认值仍是上次切之前的数；**务必检查并手动修改**。
- **用标称 Rp0.2 校核长期高温螺栓**：500 °C 下 8.8 级螺栓的 Rp0.2 可能只有 300 MPa，按 640 MPa 校核会严重低估应力裕度。

**出处**：GB/T 228.1-2010 / ISO 6892-1（金属拉伸试验方法 — Rp0.2 定义）；GB/T 3098.1-2010 / ISO 898-1（强度等级 → Rp0.2 映射）
