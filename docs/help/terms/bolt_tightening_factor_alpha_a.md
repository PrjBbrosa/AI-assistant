# 拧紧系数（αA）

**一句话**：装配最大预紧力与最小预紧力的比值 FM,max / FM,min，反映装配工艺的散差程度；由拧紧方式与工具精度共同决定。

**怎么理解**：

同一颗螺栓、同一工人、同样的扳手，**十颗螺栓得到的实际预紧力不会完全一样**。工艺散差让 FM 落在一个区间：[FM,min, FM,max]。αA 就是这个区间的比值：

```
αA = FM,max / FM,min        [无量纲, ≥ 1.0]
```

工程意义：
- **αA → 1.0**：零散差，完全可控（理论极限，实际不可达）
- **αA = 1.2**：最大预紧力比最小预紧力高 20%（转角法 / 液压拉伸可达）
- **αA = 1.6**（默认）：扭矩法典型值，高与低相差 60%
- **αA = 2.0**：粗糙工艺或涂层散差大

## 为什么要显式处理散差

VDI 2230 的核心安全思路："按最坏情况设计"：

- **R3 残余夹紧力校核**：必须按 **FM,min** 够不够用校核（"最糟的那颗"要能密封）
- **R4 装配应力校核**：必须按 **FM,max** 会不会屈服校核（"最紧的那颗"不能断）

两端都要过。装配时你只能控制 MA 在某个值，但 FM 会按 1/αA 到 1 的范围散布。αA 越大，为了保证 FM,min 够用你就得把 MA 调高 → FM,max 也跟着高 → 对 R4 压力越大。

连接公式：

```
FM,max = αA · FM,min
FM,min ≥ FK,req + (1−ΦN)·FA,max + FZ + Fth   # R1 设计条件
```

## 取值指南

| 拧紧方式 | αA 建议 | 理由 |
|---|---|---|
| 扭矩法（机械扳手） | 1.6–1.8 | 扳手精度 + 摩擦散差 |
| 扭矩法（数显扳手） | 1.4–1.6 | 扳手精度好，摩擦仍散 |
| 转角法 | 1.15–1.35 | 避开摩擦，仅贴合扭矩不确定 |
| 液压拉伸法 | 1.1–1.15 | 几乎不受摩擦影响 |
| 热装法 | 1.1–1.2 | 温度控制精度决定 |

**本工具默认 αA = 1.6**（扭矩法中值）。切换拧紧方式时 UI 会给建议区间，但不自动改；**必须手动按工艺实际调整**。

## αA 小会怎样

假设 FK,req = 10 kN，FA,max·(1−ΦN) + FZ + Fth = 5 kN，FM,min = 15 kN 刚好够用。

- **αA = 1.2 设计**：FM,max = 18 kN → R4 按 σ = FM,max/As 校核较宽松
- **αA = 1.6 设计**：FM,max = 24 kN → R4 压力大、可能需要换大规格螺栓
- **αA = 1.8 设计**：FM,max = 27 kN → R4 更严、可能换 10.9 级

### 反过来：αA 填小但工艺散差大的风险

若工人实际 αA = 1.6 但你设计时按 αA = 1.2：
- 按 FM,max,设计 = 1.2·15 = 18 kN 校核了 R4 → PASS
- 实际 FM,max,真实 = 1.6·15 = 24 kN → 超出屈服，**装配阶段已损坏**

**所以：αA 宁大勿小，不可乐观。**

## 常见误用

- **根据"自己"的熟练度调小 αA**：散差不只来自工人，还来自扳手磨损、润滑批次、温度、螺栓批次。单人手感不足以证明 αA < 1.4。
- **不同工人调不同 αA**：设计应按"最差工人"取 αA，不要按班组平均。
- **αA = 1.0 配合"保证装配"**：零散差工艺不存在。填 1.0 等于没做 R4 校核。
