# 等效弹性模量 E'

**一句话**：两接触体弹性变形能力的合成参数，赫兹公式把它看作"一个虚拟的更软 / 更硬材料"来算接触斑与 p0。

**怎么理解**：

当两个不同材料的弹性体相互压触时，接触斑大小取决于"两侧一起能变形多少"——软的一侧变形多，硬的一侧变形少，合成起来成为一个"等效弹簧"。数学上：

```
1 / E' = (1 - ν1²) / E1 + (1 - ν2²) / E2     [E: MPa → E': MPa]
```

符号说明：
- E1、E2 — 两接触体的杨氏模量（弹性模量）
- ν1、ν2 — 两接触体的泊松比
- (1 - ν²) / E — 单侧材料的"平面应变柔度"因子

为什么不是简单的 `E' = E1 × E2 / (E1 + E2)`（调和平均）？因为接触区附近处于近似平面应变状态（侧向受约束无法自由收缩），有效柔度要除以 `(1 - ν²)`。

**数值示例**（同样 R', F 下 E' 越大 p0 越高）：

| 材料配对 | E1 (MPa) | ν1 | E2 (MPa) | ν2 | E' (MPa) |
| -------- | -------- | ---- | -------- | ---- | -------- |
| 钢-钢 | 210000 | 0.30 | 210000 | 0.30 | ≈ 115385 |
| 钢-铝 | 210000 | 0.30 | 70000 | 0.33 | ≈ 58605 |
| 钢-铸铁 | 210000 | 0.30 | 120000 | 0.26 | ≈ 82620 |
| 钢-PA66 | 210000 | 0.30 | 3000 | 0.40 | ≈ 3517 |
| 钢-青铜 | 210000 | 0.30 | 110000 | 0.35 | ≈ 81233 |

可以看出：**当两材料 E 差距大时，E' 接近较软材料的 E / (1-ν²)**（柔度由软侧决定）。这解释了为什么塑料齿轮接触即使对手是钢齿，p0 也由塑料一侧主导。

**E' 在赫兹公式中的位置**：

```
线接触：b = √(4·q·R' / (π·E'))   → b ∝ 1 / √E'
        p0 = 2·q / (π·b)         → p0 ∝ √E'

点接触：a = ((3·F·R') / (4·E'))^(1/3)  → a ∝ E'^(-1/3)
        p0 = 3·F / (2·π·a²)            → p0 ∝ E'^(2/3)
```

所以：**E' 提高一倍，线接触 p0 增加 41%，点接触 p0 增加 59%**。这就是为什么"换更硬材料" 往往不是单纯好事——硬材料降低变形但抬高 p0，对接触疲劳不一定有利。

**本工具实现**：`core/hertz/calculator.py:118` 直接按上式计算：

```python
e_eq = 1.0 / (((1.0 - nu1*nu1) / e1) + ((1.0 - nu2*nu2) / e2))
```

结果回显在 `result["derived"]["e_eq_mpa"]`，UI 结果卡片显示"等效弹性模量 E' = XXX MPa"。

**单位约定**：全部用 MPa（等同 N/mm²）。若手册给 GPa，乘 1000 后填 E。

**常见坑**：

- **混用 E 和 E'**：赫兹公式里出现的是 E'（合成模量）；若用代码里的 E1、E2 直接代入原式会错；本工具对用户透明，只需正确填 E1/E2/ν1/ν2，E' 自动派生。
- **泊松比填 0**：部分陈旧资料忽略 ν 项写成 `1/E' = 1/E1 + 1/E2`，这是在 ν = 0 的特殊假设下；本工具严格要求 `0 < ν < 0.5`，不接受 ν = 0。
- **把 E' 当成"平均模量" 理解**：`E'` 在量纲上仍是 MPa，但不是 E1 与 E2 的算术/几何平均；它本质上是"串联柔度的倒数"。

**出处**：Hertz 接触理论（Johnson 《Contact Mechanics》整理）。
