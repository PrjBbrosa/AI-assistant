# 疲劳载荷循环次数（N_L）

**一句话**：螺栓在服役寿命内预计承受的应力循环次数 [次]；决定疲劳许用应力 σ_ASV 的寿命修正因子。循环数越少，许用越高。

**怎么理解**：

金属疲劳不是"一来应力就坏"，而是**累计多少次循环后坏**。VDI 2230 / ISO 898-1 给出螺栓高周疲劳极限 σ_ASV（对应约 2×10⁶ 次循环的"持久极限"平台）。当实际循环次数 N_L 小于 2×10⁶ 时，许用应力可以上修；高于 2×10⁶ 时取平台值。

本工具的寿命修正因子：

```
cycle_factor = (2×10⁶ / N_L) ** 0.08     [N_L: 次 → 无量纲]   当 N_L < 2×10⁶
cycle_factor = 1.0                        当 N_L >= 2×10⁶
σ_ASV_effective = σ_ASV_table · cycle_factor          [MPa]
```

实现位于 `core/bolt/tapped_axial_joint.py:279-280`。0.08 是基于 Basquin-type 高周斜率的经验系数。

**直观理解**：

| N_L | cycle_factor |
|---|---|
| 10³（1000 次） | 约 1.83（许用提 83%） |
| 10⁴（1 万） | 约 1.55 |
| 10⁵（10 万） | 约 1.31 |
| 10⁶（100 万） | 约 1.11 |
| 2×10⁶（200 万，平台起点） | 1.00 |
| 10⁷（1 千万） | 1.00（平台不再下降） |

所以"寿命越短 → 疲劳许用越高"，对短寿命有限次循环的连接是有意义的修正。

## 什么时候用较小值

**N_L 填较小值**的场景：

- 一次性装配 + 拆卸的螺栓（总循环 < 1000）
- 只做功能测试、寿命短的夹具螺栓（10³ – 10⁴）
- 明确有限寿命设计（例如"运行 5 年 × 每天 10 次"约 10⁴ 级）

此时许用应力 σ_a,allow 可合理上修，避免把短寿命连接按 2×10⁶ 无限寿命限制而设计过大。

## 什么时候用 2×10⁶ 或更大

**N_L >= 2×10⁶**（工具把 cycle_factor 钳到 1.0）的场景：

- 常规工业机械连接（每天几十次 × 几年 = 10⁶ 级）
- 压力容器常规运行（温度 / 压力循环）
- 不明确循环次数时的**默认保守取值**

超过 2×10⁶ 再填更大值（10⁷、10⁸）对许用应力**没有额外影响**——工具固定用平台值。

## UI 默认

默认 `1 000 000.0`（10⁶），处在平台前夕，cycle_factor 约 1.06，相当于比平台值略高一点。这对多数通用工况合理，但若工况明显为高周（≥ 2×10⁶）建议填 `2 000 000` 或更大以对齐平台。

## 边界检查

`core/bolt/tapped_axial_joint.py:227` 用 `_positive(...)`，不允许 0 或负值。低于 10³ 次工程上已属低周疲劳范畴，0.08 指数不再适用；超过 10⁸ 对结果无影响。**真实超低周（< 10³）应另做非线性低周分析**，不要靠本工具线性外推。

## 典型填法

| 工况类型 | N_L 建议 |
|---|---|
| 一次性装配（不拆） | 10³ |
| 设备外壳、面板（偶尔拆卸） | 10⁴ |
| 普通工业机械 | 10⁶ |
| 连续运行设备（泵、风机、压机） | 2×10⁶ 或更大 |
| 振动 / 动载主导（汽车底盘、风电） | 10⁷+ |
| 不确定 | 2×10⁶（保守默认） |

## 与表面处理的关系

疲劳许用 σ_ASV 的**基表**是轧制螺纹的 σ_ASV[d]（见 `bolt_tapped_axial_surface_treatment`）。cycle_factor 在 σ_ASV 基表后乘入：

```
σ_ASV_effective = σ_ASV_table[d] · surface_factor · cycle_factor
```

surface_factor 对切削螺纹乘 0.65，对轧制螺纹乘 1.0。**两个修正独立**——填切削螺纹同时填 10⁴ 次，σ_ASV 会被同时上下调。

## 常见误用

- **N_L 填"每秒循环频率"或"小时数"**：必须是**总循环次数**（不是频率、不是时长）。若振动频率 10 Hz、运行 10⁵ 秒 = 10⁶ 次。
- **低周疲劳（< 10³）硬填**：0.08 指数在低周不准确，工具仍按公式算会过度乐观。低周（例如 500 次大应变循环）需按 Manson-Coffin 单独分析。
- **把"设计寿命年数"直接填入**：年数 × 频率才是循环数。
- **不确定时填 0 或负数**：工具直接抛 `InputError`。请选 2×10⁶（保守默认）。
