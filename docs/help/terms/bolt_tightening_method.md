# 拧紧方式

**一句话**：装配时控制预紧力大小的手段（扭矩法 / 转角法 / 液压拉伸法 / 热装法），不同方式的控制精度截然不同，直接决定拧紧系数 αA 的取值。

**怎么理解**：

螺栓装配的最根本问题是：**我怎么知道预紧力到了多少？**你无法直接测量一颗拧紧中的螺栓的拉力（除非装传感器），只能通过"外部可测量"的物理量间接推算。不同拧紧方式对应不同的间接量，精度差异巨大。

## 四种主流方式

### 1. 扭矩法（MA 控制）

**原理**：用扭矩扳手拧到规定扭矩 MA，据此推算 FM。
```
MA = FM · (0.5·d2·(P/(π·d2) + 1.155·μG) + 0.5·DKm·μK)
→ FM ≈ MA / (摩擦与几何因子)
```

**优点**：工具简单（便宜的扭矩扳手即可），不需要预先加载屈服点。
**缺点**：**摩擦散差支配结果** —— μG、μK 的 ±25% 波动传导到 FM 的 ±20% 波动。

**典型 αA**：**1.4–1.8**（控制精度最差）。

**本工具特殊处理**：扭矩法下服役阶段扭转残余 k_τ = 0.5，即保留约 50% 装配扭转应力进入 R5；其他方式下 k_τ = 0。

### 2. 转角法（扭矩 + 角度控制）

**原理**：先拧到一个小"贴合扭矩" M_snug，然后再转一个规定角度 Δθ。按虎克定律（弹性段伸长 ∝ 转角），**不再依赖摩擦**。

**优点**：避开了摩擦散差；工人只要读角度，不需读扭矩精度。
**缺点**：需要先拧到贴合状态（去除间隙），工具需要角度编码器或视觉标记；对 M_snug 估算误差敏感。

**典型 αA**：**1.15–1.35**（中等精度）。

### 3. 液压拉伸法

**原理**：用液压缸从螺栓顶端直接拉伸（预紧），保持拉力时旋紧螺母到贴合，然后卸液压 —— 弹性回弹留下预紧力。

**优点**：几乎不受摩擦影响（螺母旋紧时几乎不受拉伸螺母的摩擦）；可在线测力。
**缺点**：需要专用液压工具、螺栓需要有足够伸出长度、成本高。

**典型 αA**：**≤ 1.15**（最高精度）。

**用场**：大型法兰（风电塔筒、压力容器、船用法兰）、关键连接。

### 4. 热装法

**原理**：加热螺栓（或被夹件 / 二者）到一定温度，旋紧螺母到贴合，冷却后热收缩差产生预紧力。

**优点**：不受摩擦影响；可在特殊几何下使用。
**缺点**：需加热设备、时间长、温度控制敏感。

**典型 αA**：**1.1–1.2**。

**用场**：大型轴承座、电机底座、特殊机械。

## 怎么选

| 连接规模 | 精度要求 | 推荐方式 |
|---|---|---|
| 一般工业（汽车、机械） | 中 | **扭矩法**（80% 工业连接） |
| 中等关键（设备法兰） | 中高 | **转角法** |
| 高精度 / 大型 / 密封关键 | 高 | **液压拉伸法** |
| 大型特殊（轴承座等） | 高 | **热装法** |

## 常见误用

- **用扭矩法设 αA = 1.2**：不现实；真实的扭矩法 αA 极少低于 1.4。设小 = 假设精度 = 虚高 FM,min 余量。
- **转角法没做贴合扭矩**：直接从零拧进会把"除间隙"的角度也算作有效变形，FM 严重偏低。
- **不清楚工具实际精度**：数显扭矩扳手 ±4%、机械扭矩扳手 ±10%、普通 T 扳手 ±20%+。αA 与工具精度强相关。
