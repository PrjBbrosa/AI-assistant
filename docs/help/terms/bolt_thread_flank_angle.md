# 牙型角

**一句话**：螺纹牙侧相对螺纹轴法面的夹角；公制螺纹标准 60°。

**怎么理解**：

牙型角进入扭矩系数 K 的 `cos(alpha/2)` 分母，影响螺纹摩擦扭矩 MG 的计算精度。偏差 5 度能让 K 差 3-5%。实际设计中 90% 以上紧固件为公制 M 螺纹，牙型角 = 60°，无需改动；但在传动螺纹（梯形、锯齿）或老式英制场合务必核对。

## 各螺纹体系的标准牙型角

| 螺纹类型 | 牙型角 | 用途 |
|---|---|---|
| 公制 M | 60° | 通用 |
| 美制 UN/UNC/UNF | 60° | 英美 |
| 惠氏 BSW | 55° | 老式英制 |
| 梯形 Tr | 30°（单边 15°）| 传动 |
| 方形 | 0° | 老式传动 |
| 锯齿 | 3°/30°（不对称）| 单向高载 |

## 进入的公式

螺纹摩擦扭矩由牙型角 alpha 通过当量摩擦角引入：

```
螺纹摩擦扭矩：MG = FM * (d2/2) * tan(phi + rho')
其中 rho' = arctan(mu_G / cos(alpha/2))
alpha = 牙型角（通常 60° = pi/3 rad）
```

牙型角越大，`cos(alpha/2)` 越小，当量摩擦角 rho' 越大，螺纹摩擦扭矩 MG 越高。公制 60° 与美制 UN 60° 计算方法相同，但惠氏 55° 会让 rho' 偏大约 5%。

## 常见误用

- **把牙型半角 30° 填成牙型角**：VDI 2230 公式使用全角 alpha；若误填 30° 会导致 rho' 偏低，高估 FM 约 1-2%。
- **用梯形螺纹公式校核普通紧固件**：梯形螺纹 alpha = 30°，直接套入紧固件 VDI 公式，MG 将被低估约 15%，装配扭矩偏低。
- **自定义螺纹忘验证牙型角**：非标螺纹应从图纸/标准直接读取 alpha，不可凭记忆或通用值推算。
