# 服役摩擦系数（μ_T、μ_Ax）

**一句话**：过盈连接服役阶段**扭矩方向 μ_T** 与**轴向方向 μ_Ax** 的滑动摩擦系数；直接进入扭矩容量 T_cap 与轴向容量 F_cap 的乘法项。

**怎么理解**：

过盈连接的全部传递能力都来自"压力 × 摩擦 × 面积"。摩擦系数的精度直接决定容量估算精度。本工具把它分成两个方向：

- **μ_T**：扭矩方向（周向）摩擦系数
- **μ_Ax**：轴向（沿轴线）摩擦系数

分开的原因：
1. **微观理论**上各向同性接触时 μ_T = μ_Ax；但实测往往因表面加工方向性（车削刀纹圆周方向 vs 轴向）而差 10~20%
2. **工程规范**（DIN 7190-1）对扭矩和轴向给出不同的参考值
3. **润滑状态**可能方向相关（如轴向干燥、周向仍有装配残油）

本工具允许两者独立填入；默认材料配对表会自动填相应数值。

## 公式（本工具实现）

```
扭矩容量（最小过盈端）:
   T_cap_min = μ_T · p_min · π·d·L · (d/2) / 1000      [T: N·m, p: MPa, d,L: mm]

轴向容量（最小过盈端）:
   F_cap_min = μ_Ax · p_min · π·d·L                    [F: N]

（装配阶段的 μ_Assy 单独写在另一篇术语里，请参考"interference_assembly_mu"。）
```

## 选高还是选低更保守

对**防滑校核**：
- μ 偏高 → T_cap / F_cap 偏高 → 安全系数偏高 → 偏乐观 ⚠️
- μ 偏低 → 偏保守 ✓

**工程建议**：在不确定时，**偏低 20%** 做保守估算，相当于自带一个隐藏的 0.8 折减。DIN 7190-1 参考值中位数已经偏保守，但对**关键连接**还可再折减。

## 温度、油膜、粗糙度对 μ 的影响

1. **温度**：低温摩擦增加（低温断裂风险），高温油膜失效时摩擦波动大；本工具取常温中值。
2. **油膜厚度**：轻油润滑（边界润滑）下 μ 较干燥低约 30~40%；完全油膜（流体动力润滑）下摩擦会更低，但过盈配合通常不达流体动力润滑状态。
3. **粗糙度 Rz**：Rz 越大的表面实测 μ 更高（机械互锁），但同时 Rz 越大被压平量 s 也越大、有效过盈越小。两者效果相反。

## 常见误用

- **按经验值填 0.2、0.3**：这是刚装的新面、有锈迹、无油的极端情形；用于设计会让容量虚高。按 DIN 表中值更合理。
- **两个方向填同一值**：可行，但失去分辨服役工况差异的机会；建议按表查对应方向值。
- **涂层不在内置表里按无涂层填**：化学镀层 / 防腐涂层可能显著改变摩擦；需查具体涂层供应商数据，不直接用材料表。
- **经验值直接拿供应商宣称数据**：供应商会给"最佳条件"数据；设计应用时应折减 20~30%。

## 与装配摩擦 μ_Assy 的区别

**服役摩擦 μ_T / μ_Ax**：连接稳定工作期间的静滑移摩擦系数，基于**已建立的接触**。

**装配摩擦 μ_Assy**：压入 / 滑动装配瞬间的动摩擦系数，通常**低 20~30%**：
- 装配油膜未完全刮除
- 微峰尚未完全咬合
- 表面刚接触时存在边界润滑效应

**混用的后果**：
- 用 μ_Assy 算扭矩容量 → 容量估低，过度保守
- 用 μ_T 算压入力 → 压入力估高，设备选型过度
