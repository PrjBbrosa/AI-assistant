# 接触应力安全系数 S

**一句话**：允许值 [p0] 与实际最大接触应力 p0 之比，S < 1 表示超载，S < 1.2 工具会给出偏低警告。

**怎么理解**：

```
S = [p0] / p0            [MPa / MPa = 无量纲]
```

- **S ≥ 1**：PASS，`contact_stress_ok = True`
- **S < 1**：FAIL，实际应力超过许用值
- **S < 1.2**：工具追加 warning："接触应力安全系数偏低，建议提高材料或优化接触半径/载荷。"（`core/hertz/calculator.py:175-176`）

这里的 S **不等于** 线性校核里的"安全系数"概念。因为 p0 与 F 的关系不是线性：
- 线接触 p0 ∝ √F → 想让 p0 降低 20%，需要 F 降 36%
- 点接触 p0 ∝ F^(1/3) → 想让 p0 降低 20%，需要 F 降 49%

所以 **S = 1.2 不代表"承载力还能再涨 20%"**，而是"p0 距离许用值还有 20% 裕量"。真实承载力裕量要换算到载荷维度：

| 模式 | p0 降 10% 对应 F 减少 | p0 降 20% 对应 F 减少 | p0 降 50% 对应 F 减少 |
| ---- | --------------------- | --------------------- | --------------------- |
| 线接触 (p0 ∝ √F) | F 降 19% | F 降 36% | F 降 75% |
| 点接触 (p0 ∝ F^(1/3)) | F 降 27% | F 降 49% | F 降 87.5% |

**工程实用取值**（按工况类型建议 S 下限）：

| 工况 | 建议 S 下限 | 备注 |
| ---- | ----------- | ---- |
| 静态单次加载（如过盈压装检查） | 1.0~1.1 | 允许值本身已含接触疲劳保守余量时 |
| 稳定工况齿轮 / 轴承（工业机器） | 1.2~1.5 | 考虑载荷不确定性 |
| 振动 / 频繁启停工况 | 1.5~2.0 | 冲击尖峰不确定性大 |
| 关键 / 安全部件（航空、轨道、汽车转向） | 2.0+ | 要求多重冗余 |

若 [p0] 已经按 ISO 6336 / ISO 281 的完整 Z 因子 + SH 折减过，那只要 S ≥ 1 即满足设计；工具 S < 1.2 warning 是"双保险"，提醒用户复核许用值取法。

**本工具实现**：

```python
# core/hertz/calculator.py:150
safety_factor = allowable_p0 / p0 if p0 > 0 else math.inf
pass_contact = p0 <= allowable_p0
```

- 结果字段：`result["check"]["safety_factor"]`
- 当 p0 接近 0（载荷很小或 R' 极大）时 S 会 → ∞，代码用 `math.inf` 显式表示；UI 会直接打印（可能出现"S = inf"这种无量纲表示，视为"远大于 1"）
- 在 UI 的"建议"段（`_build_recommendations`）中 S < 1.2 会额外提示"建议增加工程裕量"

**与单向拉伸安全系数的区别**：

在单向拉伸校核中 `σ_allow = σ_y / n`，安全系数 n 直接乘在载荷或屈服强度维度，属"线性裕量"。接触应力因为 p0 与 F 非线性，S 的"直觉意义"被削弱：

- **不要**拿 S = 1.5 直接解读成"可以再加 50% 载荷"
- **要**把它理解成"p0 值距离许用值还有 (S - 1) × 100% 空间"
- 若要知道真实能加多少载荷，按下表换算：

```
线接触：F_max_new / F = S²        (S = 1.5 → 可加 125%)
点接触：F_max_new / F = S³        (S = 1.5 → 可加 237%)
```

**常见坑**：

- **用 S < 1 的绝对值当"超载比例"**：类似上述，载荷换算非线性，不能直接读
- **忽略许用值的不确定性**：[p0] 自身来自手册 / 标准 / 经验，有 ±15% 量级的不确定性；S = 1.05 的"刚刚 PASS"实际上可能已接近失效
- **把 p0 ∝ √F（线接触）当成线性关系做设计迭代**：加载荷 + 改半径同时进行时，要用 p0 的完整表达而非"按 S 线性缩放"

**出处**：工程安全系数取值通常见于 VDI 2230 / ISO 6336-2 / ISO 281 等标准的附录章节。
