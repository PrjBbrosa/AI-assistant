# DIN 5480 标准花键规格字符串

**一句话**：DIN 5480-2:2015 catalog 条目的快捷标识，格式 `W d_B×m×z` 或 `N d_B×m×z`；下拉选中后自动填充几何尺寸（m、z、d_B、d_a1、d_a2、d_f1），并把几何输入模式切到`公开/图纸尺寸`。

**怎么理解**：

DIN 5480 对渐开线花键定义了标准化的规格字符串：

```
W 15 × 1.25 × 10        [外花键 (W = Welle / shaft)]
N 15 × 1.25 × 10        [内花键 (N = Nabe / hub)]
  │      │       │
  │      │       └── 齿数 z
  │      └────────── 模数 m [mm]
  └──────────────── 参考直径 d_B [mm]
```

对应到工具内置 catalog：W 15×1.25×10 的 catalog 原值为 `d_B = 15.0、m = 1.25、z = 10、d_a1 = 14.75、d_f1 = 12.1、d_a2 = 12.5`（见 `core/spline/din5480_table.py` 的 `DIN5480_CATALOG`）。

**本工具实现**：
- 下拉只列**外花键 W 系列**（ cataldg 的主键），与之配对的内花键 N 共享相同的 d_B / m / z 与 d_a2 —— 因此 `spline.standard_designation` 选中一条就足以锁定内外花键全部几何尺寸。
- 选中后自动把 `spline.geometry_mode` 切到`公开/图纸尺寸`，并把 `module_mm / tooth_count / reference_diameter_mm / tip_diameter_shaft_mm / root_diameter_shaft_mm / tip_diameter_hub_mm` 六个字段锁为 `AutoCalcCard` 禁用态（用户不能编辑）。
- 选`自定义`时解锁所有几何字段，允许用户按实测值 / 图纸值填入；几何输入模式仍可在`近似推导`与`公开/图纸尺寸`之间切换。

**catalog 覆盖范围（当前版本）**：
- 压力角：**仅 30°** 系列（DIN 5480-2:2015 的主系列）；37.5° / 45° 系列未内置。
- d_B：15, 20, 25, 30, 35, 40, 45, 50 mm 等常见规格。
- m：0.8, 1.0, 1.25, 1.5, 1.75, 2.0, 2.5, 3.0 mm 附近。
- z：按每档模数的 catalog 齿数范围（通常 10~50）。

若手头规格不在 catalog 里，或压力角不是 30°，选`自定义`并手动录入几何。

**典型值 / 场景**：

- `W 15×1.25×10`：小直径、中模数、少齿 —— 常见于伺服电机输出轴、小型变速箱输入轴。
- `W 25×2×11`：中等直径、中模数 —— 常见于车用变速器中间轴。
- `W 40×3×12`：大直径、大模数、少齿 —— 重型传动、大扭矩输出。
- `W 50×1.25×38`：大直径、小模数、多齿 —— 精密分度、薄齿高承载面但强度对冲击敏感。

**常见误用**：

- **按"实测 d_B 接近某 catalog 条目"就选中它**：catalog 条目除了 d_B 还锁定了具体的 m × z。必须三者都匹配才可以选。d_B 相近但 m 或 z 不同时，catalog 填入的 d_a1 / d_a2 / d_f1 与实物不符。
- **选中后手动改 m 或 z**：UI 会把这些字段锁为 AutoCalcCard（只读）；若强行切到`自定义`再修改，相当于脱离了 catalog，就应该走`自定义`的完整路径。
- **误以为 W / N 系列参数不同**：DIN 5480 的设计原则是**内外花键共用同一条 catalog**（就是要啮合）——d_B、m、z 都相同，只是 d_a2（内花键齿顶圆）和 d_a1（外花键齿顶圆）对应不同的工件。本工具下拉里用 `W` 前缀只是约定，不是筛选。

**出处**：DIN 5480-2:2015（渐开线花键·尺寸系列·30° 压力角）；GB/T 3478.1-2008 为国标对照。

> Cannot verify against original DIN standard —— catalog 条目数值按公开文献整理，未与 DIN 5480-2:2015 原文逐条比对。实际选型时建议用 catalog 条目作为起点，再用图纸或实测尺寸核对。
