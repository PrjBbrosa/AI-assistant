# 被夹件等效外径 DA

**一句话**：把不规则截面被夹件近似为等效圆柱体时所用的外径，圆柱体柔度模型的关键几何输入，直接决定被夹件截面积和刚度。

**怎么理解**：

柔度公式需要被夹件截面积 A_P，而被夹件实际截面往往不是圆形——方板、矩形凸缘、带加强筋的壳体等。工程近似做法是换算出一个"等效外径 D_A"，使圆环截面积等于实际承载截面积：

```
A_P = pi/4 * (D_A^2 - d_h^2)              # 圆柱模型截面积 [mm^2]
```

D_A 取得准不准，直接影响 delta_P 的准确性：

- **D_A 偏小**：高估被夹件柔度 delta_P → Φ_N 偏高 → 外力被螺栓分担过多 → 预紧力需求被低估，疲劳校核偏危险
- **D_A 偏大**：低估柔度 → Φ_N 偏低 → 看似安全，但实际被夹件刚度没那么高，接触面压强被低估

## 取值建议

| 几何情形 | D_A 估算方法 | 备注 |
|---|---|---|
| 圆形法兰 | 法兰外径 D_flange | 最直接 |
| 正方形板（边长 a）| a（或 0.9·a 更保守）| 取短边近似 |
| 矩形板（a × b）| sqrt(4·A_plate / pi)，但 ≤ min(a, b)·1.5 | 等面积换算，但不超过约束 |
| 不对称凸缘 | min(a, b)（最保守取法）| 避免高估承载面 |
| 锥台适用边界 | 若 D_A > D_Km,o + lK，圆柱模型失效 | 工程师应手动核查，改用锥台模型 |

> 几何参数应由工程师依据图纸确认，上表估算规则来自 VDI 2230 §5.1.2.3 的表述逻辑，供初步设计参考。

## 验证 D_A 有效性

圆柱体模型的适用条件（VDI 2230 §5.1.2.2）：

```
D_A <= D_Km,o + lK          # D_Km,o = 螺栓头/螺母支承外径 [mm]
```

若 D_A > D_Km,o + lK，被夹件截面积超出螺栓头下方的应力扩散范围，圆柱模型对截面外围部分的刚度估计不准。此时应改用锥台模型（基础实体类型选"锥台"），D_A 字段在锥台模式下失去原本含义。

**工具当前不自动检测此条件**，工程师应在填写 D_A 前手动验证 D_A ≤ D_Km,o + lK。若超出该范围，建议将"基础实体类型"切换为"锥台"，此时 D_A 字段含义改变，不再用于圆柱截面积计算。

## 对 Φ_N 和疲劳的影响

以 M10 连接为例（d_h = 11 mm，lK = 20 mm，E = 210000 MPa，delta_S = 3.5e-6 mm/N）：

| D_A [mm] | A_P [mm^2] | delta_P [mm/N] | Φ_N (n=0.5) |
|---|---|---|---|
| 18（小）| 160 | 5.9e-6 | 0.32 |
| 24（标准）| 357 | 2.7e-6 | 0.21 |
| 36（大）| 923 | 1.0e-6 | 0.09 |

D_A 从 18 mm 增大到 36 mm，Φ_N 从 0.32 降到 0.09，螺栓疲劳载荷幅下降约 72%。可见 D_A 的选取对疲劳设计的影响极为显著。

## 常见误用

- **用通孔直径 d_h 代替 D_A**：D_A 是被夹件外径，d_h 是通孔内径，两者方向相反；用 d_h 填入 D_A 字段会导致面积公式变负，工具会抛错。
- **多层被夹件每层 D_A 不同时只填一个值**：本工具当前单一 D_A 适用于简化圆柱模型；若各层外径差异 >30%，建议用等效值或改走锥台模型。
- **未核查 D_A ≤ D_Km,o + lK 就直接使用圆柱模型**：工具不自动提示此边界，需工程师手动验证。若 D_A 超出范围而仍用圆柱模型，会系统性低估被夹件刚度，在疲劳载荷场景下偏危险方向失效。
