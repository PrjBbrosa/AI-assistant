# 有效旋合深度（m_eff）

**一句话**：螺栓实际旋入内螺纹的承载长度，决定螺纹脱扣校核 R8 是否启用以及能承担多大剪切力。

**怎么理解**：

螺栓穿过内螺纹（螺母孔或壳体螺纹孔）时，不是"全长都在承力"。承力区间是 "螺栓第一扣 接触第一扣内螺纹" 到 "末扣还在内螺纹"的那一段 —— **有效旋合深度 m_eff**（mm）。

经典对应：

- **用标准螺母**：螺母高度约 0.8d – 1.0d（ISO 4032/4034/DIN 934）；承力 m_eff 稍小于螺母高度（开头末尾不完全啮合），工程近似取 **0.8d**
- **直接旋入螺纹孔**（螺纹孔连接）：m_eff = 螺纹孔设计深度（不是总盲孔深度，是螺纹加工到的那部分）

## R8 校核（本工具实现）

```
A_SB = π · d3 · m_eff · C1        # 外螺纹剪切面积 [mm²]
A_SM = π · d  · m_eff · C3        # 内螺纹剪切面积 [mm²]
F_strip_B = A_SB · τ_BS           # 外螺纹剪切承载力 [N]
F_strip_M = A_SM · τ_BM           # 内螺纹剪切承载力 [N]
S_strip = min(F_strip_B, F_strip_M) / F_bolt,max     # 安全系数
R8 通过: S_strip ≥ S_strip,req (典型 1.25)
```

C1 = 0.75、C3 = 0.58（ISO 公制标准常用近似）。

**m_eff 直接出现在分子**：m_eff 越大 → 剪切面积越大 → 承载力越高 → R8 越安全。

## 什么时候留空（跳过 R8）

**m_eff 留空**：工具跳过 R8，整体结论标为 "incomplete"（不完整），不给 PASS 绿灯。

**可以留空的场景**：
- 标准螺母 + 螺栓组合（ISO 4032 / DIN 934），螺母材料强度 ≥ 螺栓 → 螺纹脱扣几乎不可能（标准保证）
- 已有上游评审通过（如第三方 FEA 或产品认证）
- 深度盲孔钢基体（m_eff ≫ d），压倒性的承载裕度

**不能留空的场景**：
- 螺栓旋入铝 / 铸铁 / 镁 / 塑料基体（内螺纹材料强度远低于螺栓）
- 盲孔深度不够（m_eff ≤ 1.0d 且非钢基体）
- 关键连接需完整校核记录

## 典型值

| 连接场景 | m_eff 典型 |
|---|---|
| M10 标准螺母 + 8.8 螺栓 + 钢法兰 | 0.8d = 8 mm |
| M10 法兰螺母 + 8.8 螺栓 | 0.6d = 6 mm（法兰螺母通常更薄） |
| M10 螺纹孔（钢） | 1.0d = 10 mm（基础设计底线） |
| M10 螺纹孔（钢，重载）| 1.5d = 15 mm |
| M10 螺纹孔（铝 6061）| 2.0d = 20 mm |
| M10 螺纹孔（铸铁 HT200）| 1.5d = 15 mm |
| M10 螺纹孔（塑料 PA66）| ≥ 3.0d 加金属螺纹衬套 |

**规律**：内螺纹材料越软，需要的 m_eff 越深。铝壳体普遍需要 1.5–2.0d。

## 何时必须与 τ_BM 一起填

m_eff 和 τ_BM 是一对耦合输入：

- **填 m_eff 但不填 τ_BM**：工具抛 `InputError: thread_strip.tau_BM 必须 > 0`
- **填 m_eff 且填 τ_BM**：R8 被启用，校核 S_strip ≥ S_strip,req
- **留空 m_eff**：R8 跳过，τ_BM 可选

也就是说：**启用 R8 就两样都填；不启用就都留空**。不要只填一个。

## 常见误用

- **m_eff 填螺母全高**：开头第 1 扣和末尾几扣承载不均匀，实际 m_eff 比全高略小。工程上按 0.8d 取"有效长度"。
- **螺纹孔深度 = 盲孔深度**：盲孔深度 = 螺纹加工深度 + 钻孔底部安全余量（约 0.5d）。m_eff 是**螺纹部分**的长度，不含钻孔未切螺纹的底部。
- **铝壳体按"m_eff = 螺母高度" 处理**：铝的 τ_BM 显著低于钢，需要加深螺纹孔，否则 R8 必 FAIL。
- **试图通过减小 S_strip,req 让 R8 过**：S_strip,req 是设计底线（1.25 是普遍值），降低它不是工程合理的做法；该加深 m_eff 或换材料。
