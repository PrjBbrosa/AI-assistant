# 配合长度（L）

**一句话**：轴向有效接触长度；线性放大接触面积 π·d·L，直接线性放大扭矩容量、轴向容量和压入力。

**怎么理解**：

L 是过盈连接最简单的"规模参数"——它不影响单位面积的接触压力，不影响柔度比例，只**线性放大接触面积**。工程上增加传递能力时，增加 L 通常是最便宜、最安全的做法。

但 L 并非越长越好：
1. **端部应力集中**：实际接触压力在两端存在衰减 / 反凸起，接触长度越长相对影响越小，但**压入力** / 热装阻力会线性增加，可能超出设备能力。
2. **同轴度要求**：L 越长，轴向若不平行会产生局部高压（"V 型偏斜"）；制造同轴度公差必须跟上。
3. **加工成本**：磨削 / 车削长配合面的成本非线性上升。

## 公式（本工具实现）

```
接触面积:  A = π · d · L                                      [mm²]

扭矩容量:  T_cap = μ_T · p · A · (d/2) / 1000                 [T: N·m]
轴向容量:  F_cap = μ_Ax · p · A                               [F: N]
压入力:    F_press = μ_Assy · p · A                           [F: N]

附加压强（张口缝）：
   p_radial  = F_r,design / (d · L)                            [p: MPa]
   p_bending = 2.25 · M_b,design · 1000 / (d · L²)             [p: MPa]
```

注意 `p_bending` 对 L 是**二次方反比**：L 翻倍，弯矩引起的附加压强降到 1/4。所以存在显著弯矩时，L 是最有力的对抗手段。

## 典型值（L/d 经验范围）

| 工况 | L/d 推荐 | 说明 |
|---|---|---|
| 纯扭矩传递，无附加载荷 | 0.5 ~ 1.0 | 足够传递，避免过长 |
| 扭矩 + 轻度轴向（皮带轮、链轮） | 0.8 ~ 1.2 | 多一点抗窜动余量 |
| 扭矩 + 弯矩（中间齿轮毂、伞齿轮毂） | 1.0 ~ 1.5 | 抵抗张口缝 |
| 仅抗窜动 / 轴向锁紧 | 0.5 ~ 1.0 | 轴向容量足够即可 |
| 重载 + 冲击 | 1.2 ~ 2.0 | 大裕量 |

**本工具对 L/d 不做硬限制**，但 L/d > 2 时压入力 / 热装阻力可能显著超出普通设备能力，应评估工艺可行性。

## 工程建议

1. **能加 L 就别加 δ**：增加 L 是线性放大容量；增加 δ 需要放宽公差或更严格加工，工艺成本高。
2. **弯矩场合重点关注 L²**：L 对 p_bending 是 L² 反比，对 F_r 只是 L 反比；有弯矩时 L 价值更高。
3. **L/d < 0.25 时 fretting 评估不适用**：本工具的重复载荷简化式要求 L/d > 0.25，否则标记为"not applicable"。

## 常见误用

- **把"轮毂宽度"当 L**：轮毂宽度可能包括两端倒角、法兰增厚段；**有效接触长度**才是 L。一般要扣掉倒角和表面不承压区域。
- **L 填 0 或负值**：不允许；本工具按 `> 0` 校验。
- **倒角 / 圆角区域算入 L**：轴端倒角区域接触压力为 0，不应算入；小倒角可忽略，大圆角（> 2 mm）应扣除。
- **L 超过轴承跨距的一半**：过盈接触长度超过一定比例会让整根轴刚度表现异常，结构设计时需要考虑。
