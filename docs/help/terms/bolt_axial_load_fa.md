# 最大轴向工作载荷（FA,max）

**一句话**：连接服役时外部作用于螺栓-被夹件组合沿螺栓轴线方向的最大拉力（或拉力循环的峰值），驱动服役应力校核、残余夹紧力校核、疲劳校核。

**怎么理解**：

FA,max 是所有轴向 VDI 2230 校核的**外部输入**：

```
F_bolt,work,max = FM,max + Φ_N · FA,max           [N]   # 螺栓承担部分
σ_ax,work = F_bolt,work,max / As                  [MPa] # 服役轴向应力
FM,work_remaining = FM,min − FZ − Fth − (1−Φ_N)·FA,max  # R3 残余夹紧力
σ_a = Φ_N·FA,max / (2·As)                         [MPa] # 疲劳交变幅值
σ_m = (FM,max + 0.5·Φ_N·FA,max) / As              [MPa] # 疲劳平均应力
```

FA,max 越大，螺栓服役应力越高、残余夹紧力越低、疲劳越严峻。

## 填什么值

**填服役中的峰值**，不是平均值：

- 静载：就是那个静态力
- 循环载：波形最高值（不是 RMS、不是平均）
- 冲击载：含动载放大系数后的峰值（参考载荷谱或 DIN 743）
- 随机载：统计 3σ 或极值分布的 p99

### 动载放大

若外载含冲击（如冲压机、冲击扳手、地震），**必须用动载放大系数 Kd** 乘静载：

```
FA,max = Kd · FA,static             [N]
```

- Kd = 1.0：纯静载
- Kd = 1.2–1.5：轻冲击（常见机械）
- Kd = 2.0–3.0：重冲击（冲床、锤击）
- Kd = 3.0+：极端冲击（按载荷谱实测）

本工具**不自动计算 Kd**；用户需要自己把 Kd 乘进来再填 FA,max。

## 轴向 vs 横向

**FA**（轴向）：沿螺栓轴线（拉 / 压）；压向少数场合连接**不校核**，只校核拉向。
**FQ**（横向）：垂直于螺栓轴线（剪切）；通过防滑校核（要求 FM 产生的摩擦力 > FQ）。

一个连接可以**同时**承受 FA 和 FQ，但 VDI 2230 把两者分开校核：

- **FA 进 R3/R4/R5/疲劳** ← 本术语
- **FQ 进 R1 防滑**（`FM,min ≥ FQ,max / (qF·μT)`）

**本工具对压力工况的处理**：FA 填负值代表压力，R4/R5 等效应力仍成立（σ_ax 变负，σ_vm 取绝对值）。但对疲劳循环，FA 若 < 0 需要把它理解为"对称循环的一端"而非"平均压载荷"，本工具不自动做此区分；请手动只填拉向峰值，压向作为循环特性另行考虑。

## 常见误用

- **填平均值而非峰值**：对循环载，VDI 2230 要求按 σ_m + σ_a 拆分；FA,max 必须是峰值。
- **忘记动载放大**：纯手感估算会低估 30–50%；对冲击敏感工况请用载荷谱或 DIN 743 取 Kd。
- **把 FA 和 FQ 搞混**：轴向拉 / 剪切填错会让校核结果"对的方向、错的量级"，R3 / R4 都虚高。先搞清楚连接几何与载荷方向。
- **不填（留空）**：服役校核链直接跳过，R3/R4/R5/疲劳结果不可信。即使"只有预紧力，无外载"也应填 FA,max = 0。

**出处**：VDI 2230-1:2015（外载定义、FA / FQ 拆分相关章节）；DIN 743（动载放大系数）
