# 载荷循环次数 ND

**一句话**：预期服役期内外部轴向载荷 FA 从 min → max 的循环次数，决定有限/无限寿命疲劳校核。

**怎么理解**：

VDI 2230 R6 疲劳校核以 ND 分段：ND < 2×10⁶ 取 S-N 曲线上的有限寿命段，允用应力幅高于无限寿命值；ND ≥ 2×10⁶ 取无限寿命 sigma_A，不随 ND 继续增加而提高。

ND 的估算关键在于**确认每个工作循环产生几次有效载荷峰值**。机器启停仅有几十次，但若稳态运行中存在振动或压力脉冲，ND 可能高出数量级。低估 ND 会落在有限寿命段、虚高允用幅值，导致疲劳判断偏冒进。

## 典型工程场景

| 应用 | ND 估算 |
|---|---|
| 起重机吊钩（每天 50 次 × 20 年）| 3.6×10⁵ |
| 汽车发动机曲轴螺栓（转速 3000 rpm × 50 万公里）| 5×10⁹ |
| 压力容器周期（每天 1 次 × 30 年）| 10⁴ |
| 风电塔筒（50 年设计寿命）| 10⁷-10⁸ |
| 周期性紧固/拆卸 | 仅几十次到几百次 |

## 本工具的疲劳极限计算方式

本工具不区分 ASV（热处理后滚压）和 SV（热处理前滚压）两个子分支，而是通过"螺纹表面处理"下拉选择"轧制"或"切削"来控制 sigma_ASV 的修正系数：

- **轧制**（rolled）：使用 VDI 2230 基准 sigma_ASV 查表（按公称直径线性插值）。
- **切削**（cut）：在轧制基准上乘以折减系数 `0.65`，反映切削螺纹的疲劳劣势。

有限寿命修正（ND < 2×10⁶ 时）按幂律因子叠加：

```
sigma_asv_eff = sigma_ASV(d, surface_treatment) * (2e6 / ND)^0.08    [MPa]
```

ND >= 2×10⁶ 时取无限寿命极限，不再随 ND 增大而继续提高。ASV 与 SV 的详细区别属于 VDI 2230 标准概念；若需完整分类，应参考标准原文并手动选取合适的疲劳极限基准值。

## 常见误用

- **把工作小时换算成 ND 忘了循环频率**：ND = 频率(次/h) × 工作小时，二者均不可省略。
- **把启停次数当 ND，漏算稳态振动**：泵、压缩机等设备虽然启停次数少，但运行中的压力脉动频率可达几十 Hz，ND 相差 10⁶ 倍。
- **风振/海浪等附加载荷漏计**：海洋结构、风电设备的环境载荷频次与幅值应单独统计叠加，不可仅计功能载荷循环。
