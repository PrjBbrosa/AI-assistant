# 被夹件数量（layers）

**一句话**：参与夹紧的独立零件数，决定柔度计算中的串联项数、界面数与沉陷预紧力损失 F_Z 的累积量。

**怎么理解**：

螺栓夹紧多个零件时，每一层被夹件在轴向上相当于一个弹簧，所有层串联叠加：

```
delta_P,total = sum_i (t_i / (E_i * A_i))               [mm/N]
```

每增加一层额外零件，带来两项影响：

1. **多一项柔度**：柔度总量增大，载荷因数 Φ_N 偏高（外力更多被螺栓承受）。
2. **多一个界面沉陷损失**：两个接触面之间的微观粗糙峰在预紧后塑性压平，导致预紧力下降。VDI 2230 Table 5.4b 给出每个界面的沉陷量 f_z（单位 μm），累计沉陷预紧力损失：

```
F_Z = (sum_j f_z,j) / (delta_S + delta_P,total)          [N]
```

注意垫圈、间隔套、密封件均作为独立层计入；多一层零件（part_count +1）= +1 个界面——工具界面数公式为 part_count + 2（DSV）或 part_count + 1（ESV），每增加一层恰好增加一个界面。

## 何时必须分层

以下情形必须分层输入，不能用单等效层替代：

- **材料差异大**（如钢件 + 铝件，E 差 3 倍）：等效弹性模量误差>20%
- **层厚差异大**（如 2 mm 薄板 + 40 mm 厚凸缘）：厚层支配刚度，需独立建模
- **有密封件或弹性垫片**：密封件 E 通常远小于钢（橡胶 ~5 MPa vs 钢 210000 MPa），对柔度影响巨大
- **校核精度要求高**（关键连接，安全系数 <1.3）：模型误差直接影响最终校核判断

## 沉陷损失参考值（界面级）

以下数据来自 VDI 2230 Table 5.4b，仅供估算参考，精确值以原标准查询：

| 表面类别 | 单界面 f_z [μm] |
|---|---|
| 粗糙（Ra > 10 μm）| 3–6 |
| 中等（Ra 3–10 μm）| 2–4 |
| 光洁（Ra < 3 μm）| 1–2 |

典型 2 层通孔连接（DSV，4 个界面：螺栓头下/层1-层2间/层2-螺母间/螺母下，即 part_count+2 = 4）：
- 中等表面 → 总沉陷 ≈ 4 × 2.5 = 10 μm
- 对 M10 短夹紧连接（lK = 20 mm，delta_S ≈ 2.2e-6 mm/N，delta_P ≈ 3.1e-6 mm/N）：
  - F_Z ≈ 10e-3 mm / (2.2e-6 + 3.1e-6) mm/N ≈ **1887 N**（约 F_M 的 5–10%，不可忽略）

典型 2 层螺纹孔连接（ESV，3 个界面：螺栓头下/层1-层2间/层2-螺纹孔端，即 part_count+1 = 3）：
- 中等表面 → 总沉陷 ≈ 3 × 2.5 = 7.5 μm（比 DSV 少一个螺母界面）

## 常见误用

- **忘记计入垫圈**：平垫圈是独立层，part_count 加 1 对应新增 1 个界面；弹簧垫圈刚度很低，会显著增大 delta_P_total，误忽略会低估 Φ_N。
- **用"等效单层"处理钢-铝混合**：直接取面积加权弹性模量误差大，多层模型才能正确反映各层柔度比例。
- **界面计数搞错**：界面数取决于连接类型（VDI 2230 §5.4.2）。通孔连接（DSV）N 层被夹件有 N+2 个界面（螺栓头下 1 + 层间 N-1 + 螺母下 1，合计 N+2）；螺纹孔连接（ESV）无螺母，共 N+1 个界面。

**出处**：VDI 2230:2015 §5.1.2.4（多层被夹件柔度计算）；VDI 2230:2015 Table 5.4（界面沉陷 f_z 参考值）
