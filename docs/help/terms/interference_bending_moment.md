# 需求弯矩（M_b,req）

**一句话**：施加在过盈连接上的**外部弯矩**（N·m）；不直接消耗摩擦容量，但会在接触面端部让压力分布**倾斜**，可能引起局部张口缝。

**怎么理解**：

弯矩的物理效应：想象过盈接触面是一块长方形（长 L × 半径 d/2）的摩擦区。弯矩让整个接触面**绕垂直于轴线的方向转动**一点点——一端压力上升、对端压力下降、甚至归零（张口）。

与径向力不同：
- **径向力** 在接触面的一侧（按周向）抬压力、另一侧降压力，但沿轴向均匀
- **弯矩** 在接触面的一端（按轴向）抬压力、另一端降压力，绕周均匀

两者都会让局部接触压力归零，但发生的位置不同。

## 公式（本工具实现）

```
设计弯矩:    M_b,design = KA · M_b,req                                [M: N·m]

附加接触压强需求（弯矩方向，本工具保守简化）:
   p_bending = 2.25 · M_b,design · 1000 / (d · L²)                    [p: MPa, M: N·m, d,L: mm]

张口缝总需求压强:
   p_gap = p_radial + p_bending

张口缝 PASS 判据:
   p_min ≥ p_gap
```

**系数 2.25 的来源**：经典 Lamé-bending 分析取 QW = 0（QW 是弯矩分布修正系数）的**上界保守简化**。真实系数随轮毂刚度在 1.5 ~ 3.0 之间变化；本工具统一取 2.25 作为安全方向。

**Cannot verify against original DIN standard** —— 精确 QW 取值与 L/d 相关，本工具未实现完整公式。对弯矩主导的工况建议走 FEA。

## 单位约定

**本工具要求 N·m**，不是 N·mm。这是与扭矩 T 单位一致的约定。若图纸用 N·mm，需除 1000。

## 典型值（工程经验）

| 应用 | M_b,req 典型 |
|---|---|
| 纯扭矩传动件 | 0 N·m（无弯矩） |
| 皮带 / 链传动的轴端毂 | 5 ~ 100 N·m（带轴跨距） |
| 齿轮啮合的弯矩（离主轴承远） | 20 ~ 500 N·m |
| 悬臂齿轮 / 凸轮 | 可能数百 ~ 数千 N·m |
| 带偏心块的主轴 | 非常大，按离心力 × 臂长算 |

**关键判断**：若过盈连接**跨坐**在主轴承上，弯矩在过盈面处接近 0；若过盈连接**悬挑**在轴承外侧，弯矩按 "F × 悬挑长度" 放大。

## 什么时候弯矩是瓶颈

本工具的 p_bending 与 L² 成反比：

```
p_bending ∝ 1 / L²
```

所以**短配合长度 + 大弯矩**是最危险组合：
- L = 20 mm, M = 50 N·m, d = 40 mm → p_bending = 2.25·50·1000 / (40·400) ≈ 7.03 MPa
- L 翻倍到 40 mm → p_bending 降到 7.03/4 ≈ 1.76 MPa

增加 L 是对弯矩最有效的对抗手段。

## 常见误用

- **单位填错**：把 N·mm 当 N·m 填（差 1000 倍）或相反。
- **忘填悬臂弯矩**：悬挑安装的连接件必定有弯矩；若遗漏，张口缝校核会偏乐观。
- **仅看扭矩通过**：扭矩通过但张口缝失败的场景很常见，必须两项都看。
- **用总弯矩 vs 连接面弯矩**：应填**该过盈连接面所在位置的弯矩**，不是全轴的最大弯矩。
- **往复动作下的弯矩幅值**：双向弯矩会让 fretting 风险大幅上升；应同时启用 Step 5 fretting 评估。
- **2.25 系数不等于标准**：这是本工具保守简化；若用户手上有更精确的 QW 值，本工具当前不支持自定义。

## 弯矩与张口缝的工程后果

- **张口缝位置**：接触面轴向的一端（弯矩受拉侧）
- **一旦张口**：摩擦容量该端失效、应力集中、fretting 萌生
- **对抗措施**（按优先级）：
  1. 加长 L（1/L² 效应，最有效）
  2. 增加最小过盈 δ_min（提高 p_min）
  3. 减小弯臂（结构重新布置）
  4. 改用键连接 / 花键（过盈不是抗弯设计的最佳选择）
