# 粗糙度压平模型（fit 标准版本选择）

**一句话**：选择粗糙度压平系数 k 的**标准版本**；DIN 7190-1:2017 用 k=0.4（新）、DIN 7190:2001 用 k=0.8（旧）、自定义 k 可手填；k 越大扣减越大、估算越保守。

**怎么理解**：

配合面装配时，微观凸峰会被**塑性压平**，带走一部分名义过盈量。压平量 s 的经验公式：

```
s = k · (Rz_shaft + Rz_hub)           [um, k: 无量纲]
```

k 是**系数**，取值依赖标准版本。历史上 DIN 7190:2001 用 0.8（偏保守），新版 DIN 7190-1:2017 修订为 0.4（重新拟合试验数据）。两者差 2 倍，对同一 Rz 组合估出来的 s 也差 2 倍。

## 本工具的三个选项

```
"DIN 7190-1:2017（k=0.4）"   新版标准，默认选项
"DIN 7190:2001（k=0.8）"     老版标准，偏保守
"自定义k"                    手动填入 roughness.smoothing_factor
```

选择前两项时 UI 自动填入 k 并锁定字段卡；选"自定义 k"时解锁，可填 0 ~ 1.0 之间任意值。

## 为什么要允许两个版本

不同行业 / 企业内部文档可能仍沿用旧标准；本工具允许用户按**自己引用的标准版本**选择：
- 新设计按新标准（k=0.4）
- 旧图纸维修 / 验证按老标准（k=0.8），保持与历史计算一致

**Cannot verify against original DIN standard** —— 两个 k 值来自公开文献记载，本工具未查证 DIN 7190-1:2017 与 DIN 7190:2001 原文的精确 k 定义。

## 公式与影响（本工具实现）

```
s = k · (Rz_shaft + Rz_hub)                  [um]
δ_eff,min = max(0, δ_min − s)                [um]
δ_eff,max = max(0, δ_max − s)                [um]

接触压力:
   p_min = (δ_eff,min / 2) / c_total · (1/1000)   [MPa]
   p_max = (δ_eff,max / 2) / c_total · (1/1000)   [MPa]
```

举例（d=40, Rz_shaft=Rz_hub=6.3 um）：

| 模型 | k | s | δ_eff（设 δ=30 um） | p 相对 |
|---|---|---|---|---|
| 不扣减 | — | 0 | 30 um | 100% |
| DIN 7190-1:2017 | 0.4 | 5.04 | 24.96 um | 83% |
| DIN 7190:2001 | 0.8 | 10.08 | 19.92 um | 66% |

k=0.8 比 k=0.4 估出的压力**低 20%**，工程结论可能从 PASS 变 FAIL。

## 工程选择建议

1. **新设计默认选 k=0.4（DIN 7190-1:2017 版本）**
2. **维修 / 与历史计算一致**：按当时使用的标准版本选
3. **关键件（后果严重）偏保守**：选 k=0.8 或手填 0.5~0.6 做中间值
4. **已有实测标定**：若企业有内部测量数据，用"自定义 k"填入实测值

## 为什么不直接用 k=1.0

k=1 意味着"微峰被完全压平"；但实际 Rz 测量时既包含锐利凸峰也包含缓坡波纹，真正被塑性压平的只是尖峰部分。k=0.4 ~ 0.8 反映了"只压平一部分"的物理真实。

## 常见误用

- **选了标准版本又手改 k**：UI 会自动覆盖回标准值；想用自定义必须先切到"自定义 k"再填。
- **忽略压平效应（k=0）**：等于高估压力 20~40%，在 Rz 大的表面上会让结论虚假乐观。
- **旧图纸计算不切老版**：用 k=0.4 算 DIN 7190:2001 时代设计的连接，可能得到比原始设计文件乐观的结论；维修时应按当时标准。
- **Rz 填了 Ra 的值**：Ra 通常是 Rz 的 1/4~1/6；填错会让 s 估低，压力虚高。
- **忽略批量生产散差**：本工具的 Rz 是单值；实际批量中 Rz 可能按正态分布散差，严酷时取 +2σ 上限。
