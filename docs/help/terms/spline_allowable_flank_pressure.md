# 许用齿面压力（p_zul）

**一句话**：花键齿面能长期承受的许用平均接触压力，单位 MPa；场景 A 的 PASS / FAIL 直接由 `p_flank vs p_zul` 决定。

**怎么理解**：

p_zul 是一个"设计许用"级别的量 —— 不是材料的屈服极限（那会立即屈服），也不是疲劳极限（更精确但需要 S-N 曲线），而是**为保证花键长期（10⁷ 次以上循环、典型寿命）不磨损 / 不剥落 / 不永久变形**的工程上界。

在工具里 p_zul 由`载荷工况`下拉一次性设定四档（100 / 150 / 60 / 80 MPa），或选`自定义`手动输入。数值单位是**MPa**（即 N/mm²）。

**p_zul 如何进入判据**：

```
flank_safety = p_zul / p_flank            [MPa / MPa = 无量纲]
flank_ok     = (flank_safety >= flank_safety_min)
```

换句话说，工具算出的齿面平均压力 `p_flank` 必须至少 `S_flank,min` 倍低于 p_zul。p_zul 越大，允许的 p_flank 也越大，结构越紧凑；p_zul 越保守，越偏安全。

**典型值（按材料 / 载荷 / 工艺分档）**：

- **调质钢（45 / 40Cr / 42CrMo）**：
  - 静载、固定连接：**100 MPa**
  - 脉动载荷、固定连接：**60 MPa**
  - 调质钢滑移连接（工具未内置）：约 30~40 MPa
- **渗碳淬火（20CrMnTi / 20CrMo）**：
  - 静载、固定连接：**150 MPa**
  - 脉动载荷、固定连接：**80 MPa**
- **氮化钢（38CrMoAl）**：静载固定约 130 MPa，脉动约 70 MPa（工具未内置，走`自定义`）。
- **不锈钢（304 / 316）**：静载约 70 MPa（强度较低，工具未内置）。

**规律**：

- 渗碳 / 氮化 > 调质 > 正火 / 退火（硬度 / 强度排序）。
- 静载 > 脉动 ≈ 0.6 × 静载（疲劳折减）。
- 固定 > 滑移 ≈ 0.5 × 固定（磨损折减）。

这些系数是公开工程手册常用值，与 DIN 6892 标准精确原文可能有 ±10% 偏差；详细设计需按实测材料硬度 + 实际载荷谱重新校核。

**常见误用**：

- **把材料的 σ_s（屈服极限）当作 p_zul**：调质 45 钢 σ_s ≈ 600 MPa，比 p_zul = 100 MPa 高了 6 倍。直接填 600 会严重低估风险。p_zul 的"长期许用"口径已经包含了：(a) 局部接触应力峰值对平均值的放大；(b) 长期磨损和塑性累积的叠加；(c) 加工表面缺陷引起的应力集中。
- **按"强度等级"类比螺栓去取**：螺栓 Rp0.2 对应 8.8 级 640 MPa；花键 p_zul 对应同材料只有 100 MPa 左右。两个概念根本不同。
- **`自定义`模式下填得过高**：若取 p_zul > 200 MPa 必须有实测 + 寿命试验依据。常规设计默认四档已经涵盖绝大多数工况。
- **忘记温度影响**：高温工况（> 150°C）材料软化，p_zul 按每 100°C 大约降 10~15% 的经验扣减，但工具不自动做这个处理。

**出处**：DIN 6892:2012、《机械设计手册》（成大先主编）、《机械工程手册》花键连接卷；工具默认四档数值见 `app/ui/pages/spline_fit_page.py:67-72`。
