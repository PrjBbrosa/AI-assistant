# 参考直径（d_B，也写作 d）

**一句话**：DIN 5480 花键规格的"基本尺寸"，在 catalog 条目里与模数、齿数一起锁定；理想几何下满足 `d = m · z`，但实际 catalog 允许小偏差（变位或圆整）。

**怎么理解**：

DIN 5480 的规格字符串 `W 15 × 1.25 × 10` 中的 `15` 就是 d_B，单位 mm。它是：

- 名义上的分度圆直径（渐开线在这个直径上切割标准齿形）。
- catalog 里的"规格定位"数值 —— 用户看图纸 / 查目录时先找 d_B，再找 m × z 匹配。
- 齿顶圆 d_a1 / d_a2 和齿根圆 d_f1 的派生基准。

数学上 `d = m · z`（理想），但在 DIN 5480-2 catalog 里**大量条目**按"轴径级别圆整"故意偏离 m·z。典型例子：`W 15×1.25×10` 的 `m · z = 12.5`，但 d_B 标为 `15`，偏差 20%；`W 20×2×8` 的 m·z=16 vs d_B=20，偏差 25%。工具会在`公开/图纸尺寸`模式下把用户填的 d 与 m·z 比对，相对偏差 > 5% 会追加 messages 警告（不阻断）。

**重要：5% 警告并非可靠的录入错误指示器**——选择工具内置 DIN 5480 下拉规格会直接把 catalog 值填入，而这些 catalog 条目本身就会触发 warning。**把 warning 当参考信号，别当 "必须修正" 信号**。真正的录入错误判据应看 m 与 z 是否匹配图纸标注。

**本工具实现**：

- 场景 A 几何派生优先看 d_B（如果四个尺寸都给了），再算 h_w、d_m。
- 近似模式下 d 直接等于 m·z（忽略 catalog 圆整）。
- d_B 与 m·z 偏差 > 5% 的警告阈值见 `core/spline/geometry.py:69-75`。

**典型值（DIN 5480 常见规格）**：

| d_B [mm] | 应用场景 |
|---|---|
| 10~15 | 小型伺服电机输出轴、手持工具传动 |
| 20~30 | 常规工业减速机、机床传动轴 |
| 35~50 | 汽车变速器轴、中型齿轮箱 |
| 60~100 | 重型传动、矿山 / 冶金设备 |
| > 100 | 大型工业设备、船用推进（catalog 通常不覆盖，走定制） |

**常见误用**：

- **把齿顶圆 d_a1 当 d_B**：d_a1 ≠ d_B。对 DIN 5480 30° 系列，`d_a1 = d_B - 0.5·m` 附近（小一圈）。误填会让后续 catalog 匹配失败。
- **把齿根圆 d_f1 当 d_B**：`d_f1 = d_B - 2·m` 附近（小两圈）。同样错位。
- **把 5% 警告当 "必须修正"**：catalog 标准条目（`W 15×1.25×10`、`W 20×2×8` 等）选中后就会触发；真正的录入错误应对照图纸核实 m 与 z，而不是依赖 warning。

**出处**：DIN 5480-1:2006（术语定义）、DIN 5480-2:2015（catalog 条目）。

> Cannot verify against original DIN standard —— d_B 的精确定义与 catalog 圆整规则未逐条比对原文。
