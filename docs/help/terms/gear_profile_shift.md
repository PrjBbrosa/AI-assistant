# 变位系数（x）

**一句话**：齿轮加工时刀具相对标准位置的径向移动量与模数的比值，用来微调齿廓位置以改善强度、干涉或中心距。

**怎么理解**：

标准齿轮刀具中心线正好过分度圆（x = 0）。若向外移一点（x > 0，正变位），齿根变厚、齿顶变尖，齿根弯曲强度提升，但齿顶容易过尖或发生顶切；若向内移（x < 0，负变位），齿根变薄，通常用于补偿中心距、避开干涉或组对小齿数齿轮。

在齿轮副中一般分配 x1（小齿轮/蜗杆）与 x2（大齿轮/蜗轮）。常见选择模式：

- **等变位**：x1 = -x2，保持中心距不变，提升小齿轮齿根
- **正变位传动**：x1 + x2 > 0，中心距增大，综合提升
- **零变位**：x1 = x2 = 0，标准齿轮对

蜗轮传动中 x1 通常取 0（蜗杆不变位），仅用 x2 调节蜗轮。**塑料蜗轮常用 x2 > 0**，因为塑料齿根疲劳更敏感，加厚齿根提升寿命。

**公式**：变位后齿根圆 `df = d − 2 × (ha* − x) × m`，齿顶圆 `da = d + 2 × (ha* + x) × m`（标准齿 ha* = 1）

**典型值**：

- x = 0：标准齿，最常用
- x = +0.3 到 +0.5：小齿数齿轮（z < 17）避免根切
- x = +0.5 到 +1.0：塑料蜗轮、齿根强度敏感的场合
- x = -0.3 到 -0.5：调整中心距凑整或匹配非标配对
- |x| > 1.0：特殊设计，需复核齿顶是否过尖

**出处**：DIN 3960:1987（齿轮变位相关章节），DIN 3975-1:2017（蜗轮变位相关章节）
