# 直径系数（q）

**一句话**：蜗杆分度圆直径 d1 与模数 m 的比值，用一个数描述蜗杆的"粗细程度"。

**怎么理解**：

同样模数下，q 越大蜗杆越粗，q 越小蜗杆越细。q 直接影响两件事：一是蜗杆刚性（q 大不易挠曲），二是导程角（q 小则 tan γ = z1/q 变大，蜗杆更陡）。

从效率角度看，q 小 → γ 大 → 传动效率高，但蜗杆刚性下降，容易在大跨距下挠曲；从自锁角度看，q 大 → γ 小，蜗轮难以反推蜗杆，适合需要防倒转的场合。因此 q 的选择是**效率、自锁、刚性**三者的平衡。

标准化的好处在于，刀具（滚刀）按 q 值备料。选一个常用的 q 可以直接用现成刀具加工蜗轮，非标 q 则要专用刀具，成本显著上升。

**公式**：`q = d1 / m`，且与导程角关联 `tan γ = z1 / q`

**典型值**：

- q = 7–9：蜗杆较细，导程角大，用于追求效率的多头蜗杆
- q = 10–12：主流通用区间，覆盖大多数工业减速机
- q = 13–16：蜗杆较粗，刚性好，适合重载、长跨距或单头自锁场合
- q > 18：专用场合，通常为大模数小型化设计

经验建议：小模数（m ≤ 2）选 q = 10–12；中等模数（m = 3–6）选 q = 9–11；大模数选 q = 8–10 控制体积。

**出处**：DIN 3975-1:2017 §4.2（直径系数定义），GB/T 10085-2018
