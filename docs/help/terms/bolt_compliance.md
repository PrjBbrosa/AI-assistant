# 柔度（δ）与刚度（c = 1/δ）

**一句话**：螺栓柔度 δs 和被夹件柔度 δp 分别描述"1N 力下螺栓伸长 / 被夹件压缩多少 mm"，两者共同决定载荷因数 Φ_N —— VDI 2230 核心链路中最关键的中间量。

**怎么理解**：

把螺栓连接想象成**两根并联的弹簧**：

- 螺栓像一根**被拉长的弹簧**（装配时预紧 → 拉长 → 蓄能）
- 被夹件像一堆**被压缩的弹簧**（被螺栓头和螺母夹住 → 压缩 → 蓄能）

外力 FA 作用时，这两根弹簧按各自的"软硬程度"分担外力：

```
Φ_N = n · δp / (δs + δp)            # 载荷因数（无量纲）
                                                 0 < Φ_N < 1
```

- **δs 大（螺栓软）、δp 小（被夹件硬）**：Φ_N 小 → 螺栓分担外力少，好
- **δs 小（螺栓硬）、δp 大（被夹件软）**：Φ_N 大 → 螺栓分担外力多，坏

工程中常追求 Φ_N 小的设计（薄被夹件 + 长螺栓 + 刚性被夹件 + 柔性螺栓），让外力尽量不作用到螺栓身上。

## 单位与物理含义

```
δ ~ mm/N             # 1N 力下变形多少 mm（弹性系数的倒数）
c = 1/δ ~ N/mm       # 产生 1 mm 变形所需的力
```

两套单位表达同一件事，互为倒数。本工具可填 δ **或** c（二选一），计算器内部统一为 δ。

## 典型量级

| 连接规模 | δs (螺栓) | δp (被夹件) | Φ_N 典型 |
|---|---|---|---|
| 薄板（lK ≈ d）+ 短 M10 螺栓 | 1e−6 mm/N | 2e−6 mm/N | 0.6 |
| 中等夹紧长度（lK ≈ 3d）+ M10 | 3e−6 mm/N | 2e−6 mm/N | 0.35 |
| 长螺栓（lK ≈ 5d）+ M10 | 5e−6 mm/N | 2e−6 mm/N | 0.25 |
| 刚性大法兰（DA/d ≈ 3）+ 螺栓 | 2e−6 | 0.5e−6 | 0.18 |

**规律**：
- 螺栓越长越软（δs 增大），Φ_N 下降，疲劳越安全
- 被夹件越厚越硬（δp 下降），Φ_N 下降，疲劳越安全
- 薄被夹件是疲劳灾难的源头（δp 大且 Φ_N 大）

## 两种获取路径（本工具）

### 路径 A — 手动输入（默认）

直接填 δs 和 δp（或 cs 和 cp）。来源：

- VDI 2230 柔度公式手算（公开教科书常用分段近似）
- 厂家给出的经验值或样本数据
- FEA 提取（两点位移除以作用力）
- 试验：施加已知 F、测量 δL

**注意：填 δ 就不填 c，填 c 就不填 δ。填错会让自动计算路径激活。**

### 路径 B — 自动计算（勾选"自动计算"）

让工具按 VDI 2230 柔度公式族估算，需填入：

- 总夹紧长度 lK
- 被夹件等效外径 DA
- 基础实体类型（圆柱 / 锥体 / 套筒）
- 螺栓弹性模量 E_bolt、被夹件弹性模量 E_clamped

工具内部调用 `core/bolt/compliance_model.py` 计算。**单层 / 多层**、**圆柱 / 锥体 / 套筒**各有特定近似 —— 详见 `_section_clamped.md`。

## 螺栓柔度 δs 的物理分段

VDI 2230 把螺栓柔度拆成串联的几段：

```
δs = δ_head + δ_shank1 + δ_shank2 + ... + δ_nut
                      ↑（各段按几何算）
δ_segment = L_segment / (E_bolt · A_segment)          [mm/N]
```

典型分段：
- 螺栓头：高度 ≈ 0.4d，按 As 计算（近似）
- 未啮合螺纹段：剩余螺纹长度，按 As 计算
- 螺纹啮合段：按 d3 的 1/3 啮合长度算（VDI 2230 经验折减）
- 螺母段（通孔连接）：螺母高度，按 d 算

本工具自动计算已实现这套分段（`core/bolt/compliance_model.py`）。

## 常见误用

- **δ 和 c 同时填**：工具会报警或按"谁不为零用谁"处理，预期行为不稳定。**二选一**。
- **勾"自动计算"又填 δ**：看起来矛盾 —— 实际 calculator 优先使用填入值，自动计算失效。保持"勾了就留空"原则。
- **δs = 0 或负数**：物理不可能，工具抛 InputError。
- **多层被夹件没分层填**：勾"自动计算"时需要填每层的厚度 / E / α，否则多层模型退化为单层。
