# 螺栓材料

**一句话**：下拉选择螺栓母材，自动填入热膨胀系数 α_bolt，是温度诱导预紧力损失 FZ,T 计算的前提；弹性模量 E 需在刚度章节单独输入。

**怎么理解**：

不同材料的屈服强度量级差 3 倍（钛合金 TC4 Rp0.2 ≈ 830 MPa，碳钢 4.6 级仅 240 MPa）；热膨胀系数差近 2 倍（钢 11.5×10⁻⁶/K，铝合金 23.0×10⁻⁶/K）。**必须先确定母材，再谈强度等级**——同是"10.9 级"，碳钢与合金调质钢的疲劳极限相差可达 30%。

在温度校核（R6）中，螺栓与被夹件材料共同决定温差载荷：

```
FZ,T = (α_parts - α_bolt) · ΔT · lK · KP · Φ_n    [N]
```

α_bolt 和 α_parts 差异越大、温差越大，FZ,T 绝对值越高；异种材料配对（铝壳体 + 钢螺栓）温差 100 K 可产生数千 N 的预紧力损失或增益。

## 选材与热膨胀系数联动

选定母材后，UI 仅自动填充 **α_bolt**（热膨胀系数）并锁定为蓝灰只读；切"自定义"后字段变白可手动输入。**弹性模量 E_bolt 不在此联动**，需在刚度章节的 E_bolt 字段单独填写（默认 210000 MPa）：

- **钢**：α_bolt 自动填入 11.5×10⁻⁶/K，字段只读
- **不锈钢**：α_bolt 自动填入 16.0×10⁻⁶/K，字段只读
- **自定义**：α_bolt 字段解锁为手动输入（蓝灰 AutoCalcCard 变为白色 SubCard）；铝合金、钛合金、合金调质钢走这条路径

**强度等级下拉与螺栓材料无联动**：等级选项固定为 4.6 / 4.8 / 5.6 / 5.8 / 8.8 / 9.8 / 10.9 / 12.9 及"自定义"，选择材料不会切换等级体系。使用不锈钢螺栓时，请在等级下拉选"自定义"并手动输入对应的 Rp0.2。

## 材料与温度诱导预紧力

异种材料配对时 FZ,T 不可忽略：

| 螺栓 | 被夹件 | α 差 [×10⁻⁶/K] | ΔT=100 K、lK=50 mm 时 FZ,T 数量级 |
|---|---|---|---|
| 钢 | 钢 | ≈ 0 | < 100 N（可忽略） |
| 钢 | 铝合金 | 11.5 | 数千 N（需计入） |
| 钢 | 不锈钢 | 4.5 | 数百至千余 N |
| 钛合金 | 铝合金 | 14.9 | 极大，设计需专项校核 |

具体值由 core 计算模块按 VDI 2230 §5.4.3 公式给出。

## 常见误用

- **把螺栓材料填成 "钢" 但实际用不锈钢**：α 差 4.5×10⁻⁶/K，100 K 温差时 FZ,T 误差可超过 800 N，导致预紧力校核结果偏乐观。
- **自定义模式下忘记修改 α_bolt**：切换到"自定义"后字段解锁但默认值仍为钢的 11.5×10⁻⁶；必须手动输入实际材料值。
- **铝螺栓当作钢螺栓校核 E_bolt**：E 差 3 倍（70000 vs 210000 MPa），螺栓柔度 δS 计算完全错误，影响载荷因数 Φ_n。

**出处**：GB/T 3098.1 / ISO 898-1（碳钢螺栓机械性能）；GB/T 3098.6 / ISO 3506（不锈钢紧固件）；GB/T 228.1（金属拉伸试验方法）；VDI 2230:2015 §5.4.3（温度诱导预紧力损失）
