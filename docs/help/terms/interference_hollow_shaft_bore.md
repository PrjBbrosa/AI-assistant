# 轴内径（d_i，空心轴）

**一句话**：轴的内孔直径；d_i = 0 表示实心轴，d_i > 0 表示空心轴；空心轴相对实心轴**更柔**，同等过盈下接触压力下降、传递能力下降。

**怎么理解**：

很多实际应用（减重、走线、油道）需要把轴做成空心。空心轴的径向刚度与壁厚相关：孔越大、壁越薄，轴对过盈量的"吸收能力"越强，实际产生的接触压力越低。

本工具用**柔度放大因子**处理空心轴：

```
bore_ratio = d_i / d              [0 ~ 1，0 = 实心]

F_hollow(bore_ratio, ν_s) = [ (1 + bore_ratio²)/(1 − bore_ratio²) − ν_s ] / (1 − ν_s)
```

bore_ratio = 0 时 F_hollow = 1（实心参考值）；bore_ratio = 0.5 时 F_hollow ≈ 1.67（轴柔度增加 67%）；bore_ratio = 0.7 时 F_hollow ≈ 2.45（柔度增加 ~145%）。

## 公式（本工具实现）

```
bore_ratio = d_i / d                                              [无量纲]

F_hollow(b, ν) = [(1 + b²)/(1 − b²) − ν] / (1 − ν)                [无量纲]
  （当 b = 0 时 F_hollow = 1，对应实心轴）

轴径向柔度:
   c_shaft = (d/2) / E_s · (1 − ν_s) · F_hollow                   [mm/MPa]

轴应力几何因子（影响 von Mises）:
   K_shaft = 1                               （实心）
   K_shaft = (d² + d_i²) / (d² − d_i²)       （空心）
```

空心轴的两个后果同时发生：
1. **柔度放大** → 同等过盈 δ 下压力 p 下降（不利，传递能力降）
2. **应力几何因子放大** → 同等压力 p 下轴 von Mises 升高（不利，屈服裕度降）

## 典型值

| d_i / d 比值 | F_hollow（ν=0.3） | 工程评价 |
|---|---|---|
| 0 | 1.00 | 实心轴，基准 |
| 0.2 | 1.12 | 小孔，影响轻微 |
| 0.4 | 1.40 | 中等空心，需注意 |
| 0.5 | 1.67 | 柔度增加 67%，建议校核 |
| 0.6 | 2.06 | 显著空心 |
| 0.7 | 2.45 | 薄壁空心，建议按管件处理 |
| 0.8 | 3.47 | 已是厚壁管，考虑更精细模型 |

**工程建议**：bore_ratio ≤ 0.5 可按本工具模型；bore_ratio > 0.7 时推荐走有限元分析，因为柔度放大已接近线弹性模型的可信边界。

## 常见误用

- **把 d_i 填成半径**：孔半径 6 mm 应输入 d_i = 12 mm；差 2 倍会让柔度因子严重偏差。
- **把 d_i 填成 d 或更大**：本工具要求 0 ≤ d_i < d；等于或大于 d 直接报错。
- **实心轴填 d_i = 0.01**：几乎没有实际物理意义，但会让 F_hollow 轻微 > 1；建议**真正实心就填 0**。
- **空心轴工况用重复载荷 / fretting 简化估算**：本工具的 repeated_load 简化式只适用于实心轴；空心轴会被标 `not applicable`，不输出简化 fretting 边界。
- **忽略孔内应力集中**：本工具不校核轴内壁的应力；当 bore_ratio > 0.5 时内壁也可能接近屈服，需要额外校核。
