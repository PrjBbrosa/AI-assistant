# 载荷工况（固定 / 脉动 × 调质 / 渗碳）

**一句话**：根据连接类型（固定 vs 滑移）、载荷类型（静载 vs 脉动）、材料热处理状态（调质 vs 渗碳），一次性查表得到许用齿面压力 p_zul；选中后 p_zul 字段自动填充并锁为只读；选`自定义`时 p_zul 解锁手动输入。

**怎么理解**：

花键齿面的许用压力 p_zul 不是一个"材料固定值"，而是**材料硬度 × 载荷类型 × 连接类型**共同决定的许用量：

- **材料硬度**（由热处理决定）：调质钢 σ_s ≈ 600~800 MPa，渗碳淬火表面硬度 55~62 HRC，两者齿面承压上限差约 1.5 倍。
- **载荷类型**：静载（单向稳态）下 p_zul 最大；脉动（反复加载，应力幅 ≈ 均值）下必须降低到约 60%。
- **连接类型**：固定连接（过盈或静态啮合）下 p_zul 最大；滑移连接（运转中可轴向滑动）会磨损加剧，p_zul 降更多——但本工具 UI 仅提供"固定连接"四档，未单列"滑移连接"条目。

**本工具实现的四档值**（见 `app/ui/pages/spline_fit_page.py:67-72`）：

| 工况 | p_zul [MPa] | 适用场景 |
|---|---|---|
| 固定连接，静载，调质钢 | 100 | 常规工业固定花键，调质 45 钢 / 40Cr |
| 固定连接，静载，渗碳淬火 | 150 | 渗碳 20CrMnTi / 20CrMo 等 |
| 固定连接，脉动载荷，调质钢 | 60 | 反复加载的固定花键，调质材料 |
| 固定连接，脉动载荷，渗碳淬火 | 80 | 反复加载 + 渗碳 |
| 自定义 | 用户填 | 非标材料 / 特殊热处理 / 滑移连接 |

**为什么 p_zul 在选中四档时被锁为只读**：避免用户同时选"固定+静载+调质钢"又手动改 p_zul，导致选项与数值不一致，报告里说不清依据。切到`自定义`就解锁。

**p_zul 的取值基础**：
- 调质钢 p_zul = 100 MPa（静载）≈ 0.14 · σ_s（若 σ_s ≈ 700 MPa）。
- 渗碳 p_zul = 150 MPa（静载）≈ 0.30 · HV_core 映射。
- 脉动降档 60%（100→60、150→80）对应于疲劳下的 Smith 或 Goodman 折减经验。

这些是**公开工程资料常用值**，不是直接引用 DIN 6892 原表；实际工程中取值还会考虑温度、润滑、表面处理细节。

**典型值 / 场景**：

- 汽车变速器固定花键（20CrMnTi 渗碳）、脉动负载 → 渗碳 + 脉动档 80 MPa。
- 工业风机（45 钢调质）、静态负载 → 调质 + 静载档 100 MPa。
- 精密伺服（长期反复启停 + 40Cr 调质）→ 调质 + 脉动档 60 MPa。

**常见误用**：

- **取静载档做脉动工况**：脉动工况下 p_zul 要打六折。取静载档会高估许用 60%，导致 PRECHECK PASS 但实际可能在疲劳下齿面剥落。
- **把 p_zul 当"材料强度"直接等同屈服**：p_zul 远低于屈服（是为了长期不磨损）。把 p_zul = 600 MPa 填进去相当于按"短时不屈服"口径，与本工具的"长期承载"定义不匹配。
- **滑移花键选"固定连接"档**：本工具未内置滑移档；实际滑移花键 p_zul 再打 50% 左右折扣（磨损加剧），应切到`自定义`并手动填入保守值。

**出处**：DIN 6892:2012 与各工程手册（《机械设计手册》《ROLOFF/MATEK》的 p_zul 推荐表）；具体四档数值见 `app/ui/pages/spline_fit_page.py:67-72`。

> Cannot verify against original DIN standard —— p_zul 表格数值按公开工程资料整理，未逐条比对 DIN 6892 原文。
