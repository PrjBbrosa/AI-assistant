# 泊松比（ν）

**一句话**：材料在单向拉伸时横向收缩应变与纵向拉伸应变的比值（取正），描述受力后"鼓不鼓"。

**怎么理解**：

把橡皮筋拉长，它会变细——变细的相对量与拉长的相对量之比就是泊松比。数值上绝大多数固体在 0.2 到 0.5 之间：越"软"的材料（橡胶、塑料）ν 越大接近 0.5（近似不可压缩）；越"硬"的金属 ν 越小接近 0.3。软木等特殊材料 ν 可接近 0。

在应力计算中 ν 出现在两个地方：一是过盈配合、厚壁筒这类二维/三维应力问题，决定径向与轴向应力的耦合；二是赫兹接触的等效模量公式 E* = E1·E2 / (E1(1-ν2²) + E2(1-ν1²))。单单替换 ν 从 0.28 到 0.35，接触应力的影响一般在 5% 以内，属于"不能漏但精度不敏感"的参数。

**典型值**：

- 碳钢、合金钢：ν = 0.28–0.30，工程默认 0.30
- 铸铁：ν ≈ 0.25
- 铝合金：ν ≈ 0.33
- 铜合金 / 青铜：ν ≈ 0.34–0.35，蜗轮青铜常用 0.35
- 钛合金：ν ≈ 0.34
- 工程塑料（PA66、POM）：ν ≈ 0.35–0.42，塑料不可压缩倾向较强
- 橡胶（不可压缩近似）：ν ≈ 0.49

选择依据：材料手册未给就按上述类别估取，金属取 0.30，塑料取 0.40 即可——在大多数工程计算中误差不构成决定性影响。

**出处**：GB/T 22315、ISO 527-1；各材料数据手册
