# 屈服强度（Re，过盈配合）

**一句话**：材料进入塑性变形的应力阈值；过盈配合中用作**应力安全系数**的分母：`S = Re / σvm`，决定最大过盈端是否触发屈服。

**怎么理解**：

过盈配合的第二类失效是"最大过盈端压坏"。σvm 是计算出的 von Mises 应力，Re 是材料屈服强度；二者比值就是安全系数。Re 越高，同等应力下安全系数越高，允许的 δ_max 越大，从而允许的公差窗口越宽，传递能力越稳定。

Re 与 Rm（抗拉强度）的区别：
- **Re** = 屈服强度（yield strength）—— 进入塑性的门槛
- **Rm** = 抗拉强度（ultimate tensile）—— 最大承受应力、接近断裂

工程上校核过盈配合用 Re，不用 Rm。Rm 大约是 Re 的 1.3~1.8 倍，用 Rm 代替会让安全系数虚高 30~80%。

## 公式（本工具实现）

```
最大过盈端应力（基于 p_max）:
   σvm_shaft,max = p_max · VM_shaft               [MPa]
   σvm_hub,max   = p_max · VM_hub                 [MPa]

安全系数:
   S_shaft = Re_shaft / σvm_shaft,max   [Re: MPa, σvm: MPa → S: 无量纲]
   S_hub   = Re_hub   / σvm_hub,max     [同上]

PASS 判据:
   S_shaft ≥ S_σ,min（来自"校核目标"章节）
   S_hub   ≥ S_σ,min
```

两侧分开填入、分开校核。薄弱侧（通常是轮毂）先失效。

## 材料选择策略

**轴选高 Re**：
- 轴常承担多功能（扭矩、弯矩、应力集中位置），Re 低会成为薄弱侧
- 建议 Re_shaft ≥ 500 MPa（除非设计选用了低强度材料的特殊场景）

**轮毂选择原则**：
- 若轮毂只承担过盈应力：Re_hub ≥ 0.5 · Re_shaft 即可
- 若轮毂本身是齿轮 / 凸轮等承载件：Re_hub 应接近 Re_shaft
- 轮毂厚壁（D/d > 2.5）时，Re_hub 可略低；薄壁时必须提高

## 热处理状态对 Re 的影响

同一牌号在不同热处理状态下 Re 差异极大：

| 45 钢热处理 | Re 典型（MPa） |
|---|---|
| 退火 | 280 ~ 350 |
| 正火 | 350 ~ 450 |
| 调质（一般） | 550 ~ 620 |
| 调质（高硬度） | 650 ~ 700 |
| 表面淬火（心部） | 500 ~ 600 |

**必须按真实热处理状态取值**；按退火值算会偏乐观，按调质值算退火件会偏危险。

## 常见误用

- **用 Rm 代替 Re**：系数虚高 30~80%，是过盈配合最常见的评审失误。
- **灰铸铁用钢的默认值**：灰铸铁没有真正屈服点，本工具按 Re 字段处理；建议加大 S_σ,min 到 1.8+ 补偿。
- **高温工况按常温 Re**：碳钢 350°C 以上 Re 快速下降；需按工作温度查表。
- **焊接件按母材 Re**：焊缝附近热影响区 Re 可能下降 30~50%；过盈面若跨焊缝需特别评估。
- **软表面硬心部（渗碳件）**：表面经过硬化、心部仍是基体材料；过盈应力沿径向深入时要考虑从外到内的 Re 变化，本工具单值模型无法表达。
