# 等效曲率半径 R'

**一句话**：两接触体主曲率半径的合成量，越小表示接触越"尖锐"，同载荷下 p0 越高。

**怎么理解**：

两弹性体在接触点附近可看作两个圆或球相切。R' 把两侧的"尖锐度"合并成一个单一几何量：

```
1 / R' = 1 / R1 + 1 / R2            [mm^-1]
```

- 两球同半径（R1 = R2 = 10 mm） → R' = 5 mm（相对尖锐）
- 一球一平面（R1 = 10, R2 = 0） → R' = 10 mm（较平缓）
- 两圆柱同半径（R1 = R2 = 50 mm） → R' = 25 mm
- 一个凸 R1 = 20，一个等半径凸 R2 = 20 → R' = 10 mm
- 一个凸 R1 = 10，一个大凹面 R2 ≈ -100（工程上"凹" 用负值处理；本工具需手动折算） → 等效 R' = 1 / (1/10 - 1/100) ≈ 11.1 mm（凹面使有效贴合更平缓 → R' 变大 → p0 变低）

**为什么 R' 出现在赫兹公式中**：

```
线接触：b = √(4·q·R' / (π·E'))  → b ∝ √R'
点接触：a = ((3·F·R') / (4·E'))^(1/3)  → a ∝ R'^(1/3)

线接触：p0 = 2·q / (π·b)  → p0 ∝ 1/√R'
点接触：p0 = 3·F / (2·π·a²)  → p0 ∝ R'^(-2/3)
```

所以：**R' 翻倍，线接触 p0 降低 29%，点接触 p0 降低 37%**。在齿轮/轴承设计里增大有效曲率半径（比如齿轮变位使得齿面曲率更平缓）是降低 p0 最直接的手段。

**本工具实现**：`core/hertz/calculator.py:33-38` 的 `_radius_inverse()` 把 R = 0 显式映射为 1/R = 0（平面）；第 108 行 `inv_r = _radius_inverse(r1) + _radius_inverse(r2)`，然后 `r_eq = 1.0 / inv_r`。若 `inv_r <= 0` 抛 `InputError`（两平面不能接触）。

结果写入 `result["derived"]["r_eq_mm"]`，同时回显两个曲率倒数 `inv_r1_per_mm` / `inv_r2_per_mm`（方便下游调试）。

**典型场景下的 R' 数值**：

| 场景 | R1 (mm) | R2 (mm) | R' (mm) | 备注 |
| ---- | ------- | ------- | ------- | ---- |
| 钢球 φ10 压平面 | 5 | 0 | 5 | 点接触典型 |
| 钢球 φ10 vs φ10 | 5 | 5 | 2.5 | 小接触斑 |
| 滚子 φ20 压滚道平直 | 10 | 0 | 10 | 线接触典型 |
| 齿轮分度圆 R = 30 vs 60 | 30 | 60 | 20 | 大齿轮啮合 |
| 蜗轮齿面凸 5 vs 蜗杆凸 40 | 5 | 40 | 4.44 | 蜗轮接触极集中 |

**单位约定**：mm。

**与"凹面"的折算**：

标准赫兹公式对凸凸接触用 `1/R' = 1/R1 + 1/R2`；对凸凹接触（内滚道 / 蜗杆蜗轮齿沟）数学形式为 `1/R' = 1/R_凸 - 1/R_凹`。本工具不直接支持负 R（`_positive(..., allow_zero=True)`），使用时：

1. 手算等效 R'：例如球 R = 6 压入凹面 R = 40 → `1/R' = 1/6 - 1/40 ≈ 0.142 → R' ≈ 7.06`
2. 把 R1 填 7.06、R2 填 0；工具派生的 R' = 7.06 与实际一致

**常见坑**：

- **忘了 R'、R1、R2 都是 mm**：切勿混用 cm 或 m
- **R1 = R2 = 0** → 两平面接触，抛 `InputError`
- **把等效 R' 直接填到 R1 位置，R2 填 0，再让工具再"派生一次"**：这正是凹面折算的标准做法，物理上等价；只需记录"本次 R1 是折算值"免得将来复核时困惑
- **忽略凹面把 R2 直接写成负值**：当前实现会在 `_positive` 校验阶段抛错
