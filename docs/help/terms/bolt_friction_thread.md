# 螺纹摩擦系数（μG）

**一句话**：螺栓牙面与内螺纹牙面之间的滑动摩擦系数，决定拧紧扭矩 MA 中有多少比例消耗在"克服螺纹斜面 + 螺纹摩擦"上，直接影响同一扭矩下得到多少预紧力。

**怎么理解**：

拧螺栓时手臂给的扭矩 MA 分三部分：

1. 克服螺纹斜面把螺栓往上"推"（约 10–20%）
2. 克服螺纹牙面摩擦（约 40–50%，**μG 控制**）
3. 克服螺栓头/螺母下方与工件的摩擦（约 40–50%，μK 控制）

也就是说，**90% 的扭矩在摩擦上燃烧**，只有 10–15% 真正转化为预紧力。μG 的小幅波动（0.08 vs 0.14）能让同一 MA 产生的 FM 差出 **±25%**。

## 扭矩公式（本工具实现）

```
MA = FM · (0.5·d2·(P/(π·d2) + 1.155·μG) + 0.5·DKm·μK)     [N·mm]
```

其中 `1.155 = 1/cos(30°)` 是 60° 牙型角带来的法向压力修正（三角函数推导）。

## 典型值

**VDI 2230 摩擦参考值常用取值**：

| 表面状态 / 润滑 | μG 典型范围 |
|---|---|
| **干燥、无涂层、微量油膜** | 0.10–0.14 |
| **装配油 / MoS2 膏** | 0.08–0.12 |
| **镀锌螺栓** | 0.12–0.20（锌镀层摩擦大且散差大） |
| **镀镉 / 达克罗** | 0.10–0.15 |
| **特氟龙 / PTFE 涂层** | 0.06–0.10（专用低摩擦） |
| **不锈钢对不锈钢（咬死风险）** | 0.18–0.35（或用防咬合膏） |

**本工具默认 μG = 0.12**，是干燥 / 微量油膜的中值；工程应对照实际工艺调整。

## 为什么 μG 和 μK 要分开填

早期简化公式把两者合并为 μ_total；但实测螺纹摩擦与支承面摩擦**可以不同**：

- 装配油只涂螺纹 → μG 降、μK 不变
- 只头下方垫专用垫圈 → μK 降、μG 不变
- 镀锌螺栓 + 钢螺母 → μG 高、μK 取决于压头面

分开填更贴近实际；本工具 μG 默认 0.12、μK 默认 0.14，略有区分。

## 选高还是选低更保守

**工程目标**是在"高摩擦偏差"场景下仍保证 FM,min。同一 MA：

- **μ 偏高**：FM 偏低，装配阶段就达不到最小预紧力 → **R3 / 密封 / 防滑风险上升**
- **μ 偏低**：FM 偏高，装配阶段易过紧 → **R4 / 屈服风险上升**

所以"选高还是选低"取决于你怕什么：

- **校核 R4 装配应力**：按 μG 偏低（FM,max 偏高）校核更保守
- **校核 R3 残余夹紧力**：按 μG 偏高（FM,min 偏低）校核更保守
- **VDI 2230 拧紧系数 αA ≥ 1.4 已经吸收了 μ 的双向散差**，所以手册值一般取中值

## 常见误用

- **干装配默认 μG = 0.08**：这是 MoS2 润滑值，干装配最低 0.10，常见 0.12–0.14。
- **不区分螺栓表面处理**：镀锌螺栓的实测 μG 可以高达 0.20 且散差大；按 0.12 设计容易 FM 不足。
- **反复拧松再拧紧导致 μ 变化**：同一颗螺栓多次使用 μG 可能下降 10–20%（螺纹被研磨光），关键连接建议使用一次性。
