# 材料最小安全系数（S_sigma,min）

**一句话**：`联合`模式下场景 B 光滑段的材料应力校核门槛；要求轴 / 轮毂的等效应力对屈服极限有 S_sigma,min 倍裕度。**仅在`联合`模式下参与计算**。

**怎么理解**：

过盈配合产生的不只是接触压力，还同时在轴内产生压应力、在轮毂内产生拉应力（按 Lamé 厚壁筒解析解）。过盈量越大，应力越高 —— 因此**应力校核看最坏过盈情况 `δ_max`**：

```
sigma_shaft  = Lamé公式(p_max, d, d_i, E_shaft, ν_shaft)   [MPa]
sigma_hub    = Lamé公式(p_max, d, D,   E_hub,   ν_hub)     [MPa]

stress_sf_shaft = Re_shaft / sigma_shaft_vm   [无量纲]
stress_sf_hub   = Re_hub   / sigma_hub_vm
PASS 条件：    两者都 >= S_sigma,min
```

`sigma_vm` 是 von Mises 等效应力（由切向、径向、轴向三个主应力合成）。Re 是材料屈服极限（45 钢 ≈ 355 MPa、40Cr ≈ 785 MPa、42CrMo ≈ 930 MPa，见 UI 材料库下拉）。

**本工具实现**：场景 B 委托给 `core/interference/calculator.py` 的 DIN 7190 Lamé 解析解。花键模块只负责在入口把 `stress_safety_min` 传进去并把结果打包回 UI。

**为什么用 `δ_max` 而不是平均值**：

过盈量有公差，装配后的实际 δ 是 `[δ_min, δ_max]` 之间某一点。**应力校核必须按最大过盈（最高应力）来做**，否则加工 / 公差累积恰好跑到 δ_max 的那一批零件会屈服。这与防滑用 δ_min 的逻辑互为镜像。

**典型值 / 场景**：

- **1.1 ~ 1.3**：材料屈服数据可靠、过盈公差精确控制；本工具默认 1.20。
- **1.3 ~ 1.5**：材料批次差异或热处理一致性稍差。
- **1.5 ~ 1.8**：关键件 / 长期运转 / 高温工况（屈服强度随温度显著下降）。
- **≥ 2.0**：按 DIN 7190 的严格侧取值；或材料疲劳 / 腐蚀环境下额外加倍。

**常见误用**：

- **把 Re 填成 Rm（抗拉极限）**：屈服 Re 通常只有 Rm 的 60~80%。若填 Rm 会让 SF 虚高一截（例如 42CrMo Rm ≈ 1080 MPa vs Re ≈ 930 MPa），实际裕度被稀释。
- **轴与轮毂用同一个 S_sigma,min**：工具把两侧都比对同一个门槛，因此轴 / 轮毂任何一侧不合格就整体 FAIL。若需要分别设定（例如轴是关键件要求更高 SF），只能在外部按更严的那一侧填。
- **误以为 p_max 越大 → 防滑裕度也越大**：防滑看 δ_min 下的 p_min，屈服看 δ_max 下的 p_max；两个安全系数是相反方向的"拉锯"。过盈公差带必须同时满足两端。

**出处**：DIN 7190-1:2017 给出了 Lamé 解析解和屈服校核；具体取值基于工程经验。
