# 轴向载荷循环区间（FA_min / FA_max）

**一句话**：一个服役载荷循环中，作用在螺栓上的最小与最大轴向拉力 [N]；两者共同决定疲劳平均应力 σ_m 和疲劳应力幅 σ_a。

**怎么理解**：

本模块面向"螺栓拧入螺纹对手件、无被夹件"场景。和一般 VDI 2230 夹紧连接不同——**没有夹紧链，没有载荷系数 Φ_N**——外部轴向力全部进入螺栓：

```
F_bolt,min = F_preload,max + FA_min        [N]   # 服役最小螺栓拉力
F_bolt,max = F_preload,max + FA_max        [N]   # 服役最大螺栓拉力
```

单位链：`F_preload_max [N] + FA [N] → F_bolt [N]`。

因为没有被夹件吸收部分外载，**FA_max 全部叠加到螺栓预紧之上**。这是本模块与标准夹紧连接校核的核心差异：同一 FA_max 下，tapped axial 螺栓承受的应力比夹紧连接更高。

## 为什么需要填两个端点

疲劳校核按 Goodman 折线评估。一个循环的最小值和最大值定义了：

```
σ_m = (F_preload,max + 0.5·(FA_min + FA_max)) / As    [F: N, As: mm² → σ: MPa]
σ_a = (FA_max − FA_min) / (2·As)                       [F: N, As: mm² → σ: MPa]
```

其中：
- **σ_m**：平均应力，决定 Goodman 折减因子 `1 − σ_m/(0.9·Rp0.2)`
- **σ_a**：应力幅，与许用幅 σ_a,allow 比较判疲劳是否通过
- **As**：应力截面积（自动由 d、p 派生）[mm²]
- **F_preload,max**：最大装配预紧力 `α_A · F_preload_min` [N]

FA_min 和 FA_max 的**差**决定应力幅（疲劳强度），**和**决定平均应力（Goodman 折减）。两者缺一不可。

## 典型填法

| 工况 | FA_min | FA_max | 说明 |
|---|---|---|---|
| 纯静载 | = FA_max | 静载值 | 应力幅 σ_a = 0；疲劳几乎不起作用 |
| 拉拉循环（脉动） | 0 | 峰值 | 常见场景：法兰偶尔加压、螺栓传动 |
| 拉拉循环（非零下限） | 小值（> 0） | 峰值 | 某些管道、阀盖始终带预压 |
| 对称载荷（含压向） | **填 0** | 拉向峰值 | 见下节："负值输入受限" |

## 负值输入受限

本模块的 `_positive(..., allow_zero=True)` 对 FA_min 和 FA_max 都要求 **≥ 0**（`core/bolt/tapped_axial_joint.py:220-221`）。填负数会抛 `InputError`。

如果实际工况含压向（外力交替把连接"推紧—拉松"）：

- 压向不会让螺栓进一步拉伸，反而使对手件更贴合；螺栓应力被"压向"上限压制在 F_preload 附近
- 对疲劳而言，**压向对螺栓应力是"减载"**，算成 FA_min = 0 是**保守**处理（把应力幅按"完全释放到预紧水平"计算）
- 若需精确分析交变压-拉工况，应参考 VDI 2230 相关附录或 DIN 743，本工具范围之外

## 取值来源

FA_max 要填**峰值**，不是平均、不是 RMS：

- 静载：直接静态力
- 循环载：波形最高值
- 冲击载：含动载放大系数 `Kd` 后的峰值（轻冲击 Kd=1.2–1.5，重冲击 2.0–3.0）
- 随机载：统计极值分布的 p99 或 3σ

本工具**不自动加 Kd**——用户需把放大系数乘进 FA_max 再输入。

## FA_min > FA_max 的校验

`core/bolt/tapped_axial_joint.py:222-224` 主动拦截：

```
if fa_min > fa_max:
    raise InputError(...)
```

填错顺序会立即报错，不会静默给错结果。

| 压力容器螺栓（小型） | 2 000 – 10 000 |

| 重载法兰（DN500+） | 50 000 – 200 000 |

## 常见误用

- **FA_min = FA_max 却不是纯静载**：这会让 σ_a = 0，疲劳校核失效。循环载荷必须把最低 / 最高点分别填准。
- **填"平均载荷"**：疲劳不能只看平均，必须显式给最小 / 最大。用平均代替峰值会低估应力幅并高估 Goodman 因子。
- **忽略动载放大**：机械冲击场景未乘 Kd 会低估 FA_max 30%–50%，疲劳判错方向。
- **把残余夹紧力 / FK_req 填进 FA_min**：FA 是外部载荷，不是夹紧保留力——本模块根本没有 FK_req 概念。
- **负值压载荷直接填负数**：工具会拒绝；压向保守处理为 FA_min = 0。
