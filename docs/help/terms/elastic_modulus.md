# 弹性模量（E）

**一句话**：材料抵抗弹性变形的能力，同样应力下 E 越大变形越小，是刚度计算的基础。

**怎么理解**：

E 是应力与应变的比值（σ = E · ε），描述材料"有多硬"。在工程上它决定的是**变形**而非**强度**——高强度钢和普通碳钢的 E 几乎相同（都是 200–210 GPa），意味着同样载荷下挠曲或压缩量差不多，但普通钢更早屈服。

在齿轮、蜗轮、滚动接触等校核里，E 进入赫兹接触公式（E* = E1·E2 / (E1(1-ν2²) + E2(1-ν1²))），影响接触应力的大小：两件钢 vs 一钢一塑料，接触应力差一个量级。温度对 E 有显著影响，塑料尤其敏感：PA66 在 23°C 约 3 GPa，到 80°C 可能降到 1.5 GPa 以下，塑料齿轮设计必须考虑工作温度。

单位习惯上机械设计用 MPa 或 N/mm²（数值 = GPa × 1000）。

**公式**：`σ = E × ε`（虎克定律）；赫兹接触等效模量 `E* = E1·E2 / (E1(1 − ν2²) + E2(1 − ν1²))`

**典型值**：

- 碳钢、合金钢：E ≈ 200–210 GPa，工程默认用 206 GPa
- 铸铁（灰口）：E ≈ 100–120 GPa，脆性材料
- 铝合金（6061、7075）：E ≈ 70–72 GPa，约为钢的 1/3
- 铜合金 / 青铜：E ≈ 100–120 GPa，蜗轮铸锡青铜取 110 GPa
- 钛合金：E ≈ 110 GPa
- 工程塑料 PA66（23°C 干态）：E ≈ 3 GPa，高温/吸水后显著下降
- POM：E ≈ 2.8 GPa
- 纤维增强塑料 PA66-GF30：E ≈ 9–10 GPa

选择依据：材料牌号查各自手册；温度变化大的塑料场合必须按工作温度取值。

**出处**：GB/T 22315-2008（金属弹性模量测试），各材料手册；ISO 527（塑料拉伸）
