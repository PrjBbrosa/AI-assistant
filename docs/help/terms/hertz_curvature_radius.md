# 曲率半径 R1 / R2

**一句话**：两个接触体在接触点处的主曲率半径；凸面填正值，平面填 0，本工具不支持负值（凹面需要折算）。

**怎么理解**：

接触体局部可以看作一个"小球帽"或"小圆柱段"——R 就是那个局部贴合圆的半径。R 越大表示"越平"；R = ∞ 就是完全平面。两个体的 R 组合起来决定了两者"在接触点局部贴得有多紧"，用**等效曲率半径 R'** 表示：

```
1 / R' = 1 / R1 + 1 / R2            [R1, R2, R': mm]
```

所以：
- 两个半径都很大（例如 R1 = R2 = 1000 mm 的平面滚轮） → R' ≈ 500 mm（接触区大，压强低）
- 一个体是平面（R2 = 0 → 内部映射 1/R2 = 0） → R' = R1（只按有限曲率的一侧算）
- R1、R2 都是 5 mm 的小球接触 → R' = 2.5 mm（压强很集中）

**本工具的"0 表示平面"约定**：`core/hertz/calculator.py:33-38` 的 `_radius_inverse()` 函数把 R = 0 显式映射为 `1/R = 0`，等价于无限大半径（平面）。所以：
- `R1 = 30, R2 = 0` → 1/R' = 1/30 + 0 → R' = 30 mm（圆柱压平面）
- `R1 = 30, R2 = 30` → 1/R' = 2/30 → R' = 15 mm（两等半径体对压）
- `R1 = 0, R2 = 0` → 1/R' = 0 → **InputError**：两平面接触赫兹理论不适用

**凹面（如轴承外圈内滚道）的处理**：代码不接受负 R 值（`_positive(..., allow_zero=True)`）。若要计算凹凸配对（凸滚动体压在凹滚道上），数学上的等效曲率应为 `1/R' = 1/R_凸 - 1/R_凹`，工程做法：

1. 先离线算好等效 R'（例如轴承外圈：R_ball = 6 mm 球压在 R_race = 40 mm 的凹面上，`1/R' = 1/6 - 1/40 = 0.142` → R' ≈ 7.06 mm）
2. 在本工具中把 R1 填为等效值 7.06，R2 填 0（假想一个平面），这样工具派生的 R' 就与实际一致
3. 在"允许接触应力"或"说明"字段里记录这是折算输入

**典型值**（工程场景 → R 取值）：

- 齿轮齿面接触：R = 齿面曲率半径（与压力角、分度圆、齿形相关），通常 5–100 mm
- 滚子轴承滚子直径 D → R_滚子 = D / 2（圆柱滚子直径 20 mm → R = 10 mm）
- 球轴承钢球直径 D → R_球 = D / 2（钢球直径 12.7 mm → R = 6.35 mm）
- 凸轮-挺杆：挺杆端面半径 R_凸 + 凸轮基圆半径 R_cam
- 平面配合件：R = 0（直接填入 0）

**单位约定**：全部用 mm。不要把直径填进来，R 是半径。

**公式对照**：

```
1/R' = 1/R1 + 1/R2           [mm^-1]
R' 用于线接触：b = √(4·q·R' / (π·E'))   [b: mm]
R' 用于点接触：a = ((3·F·R') / (4·E'))^(1/3)  [a: mm]
```

**本工具实现**：`core/hertz/calculator.py:106-111` 读入 R1/R2，经 `_radius_inverse` 处理后求和；若 `inv_r <= 0` 抛 `InputError("等效曲率为 0...")`。

**常见坑**：

- **填直径代替半径**：滚子直径 φ20 时应填 10 mm，不是 20
- **凹面填负值**：工具拒绝负值，必须自行折算为等效凸 R
- **两平面接触**：1/R' = 0，工具拒绝输入；赫兹理论不适用

**出处**：Hertz 接触理论（Johnson 《Contact Mechanics》整理）。
