# 需求传递扭矩（T_req）

**一句话**：过盈连接需要传递的**名义**扭矩（N·m）；会被工况系数 KA 放大成设计扭矩 T_design 后，再与扭矩容量 T_cap 比较。

**怎么理解**：

过盈连接的主要功能是传递扭矩。T_req 是设计意图层面的"要求传递的扭矩"，通常对应：
- 电机额定输出扭矩传到此连接的那部分
- 传动系前级齿轮副传出的扭矩
- 外力矩（如制动器、离合器）

**注意**：T_req 应是**服役阶段的峰值稳态**扭矩，不是时均值。瞬态冲击由工况系数 KA 在下一步放大。

## 公式（本工具实现）

```
设计扭矩:     T_design = KA · T_req                    [T: N·m]

扭矩容量（最小过盈端）:
   T_cap_min = μ_T · p_min · π·d·L · (d/2) / 1000      [T: N·m, p: MPa, d,L: mm]

扭矩安全系数:
   S_torque = T_cap_min / T_design                      [无量纲]

PASS 判据:
   T_cap_min ≥ S_slip,min · T_design
   <=> S_torque ≥ S_slip,min
```

T_req 填 0 表示无扭矩传递需求（纯过盈固定件）；此时扭矩校核按"无限安全"处理。

## 单位约定

**本工具统一用 N·m**，不是 N·mm 也不是 kN·m。常见换算：
- 1 kW @ 1000 rpm ≈ 9.55 N·m
- 1 kW @ 3000 rpm ≈ 3.18 N·m
- 1 马力（HP）@ 1500 rpm ≈ 4.75 N·m

若图纸或功率表只有 kW / rpm，先用 `T = 9550 · P_kW / n_rpm` 换算。

## 典型值

| 连接类型 | T_req 典型 | 说明 |
|---|---|---|
| 小型轴承内圈固定 | 0 ~ 5 N·m | 主要抗滑移，非扭矩 |
| 皮带轮 / 链轮毂（小电机） | 10 ~ 100 N·m | 常配合 0.5 kW ~ 5 kW 电机 |
| 小齿轮毂（一般工业） | 50 ~ 500 N·m | 减速机小齿轮 |
| 大齿轮毂（重工业） | 500 ~ 10000 N·m | 轧机、起重机 |
| 主轴 / 大锥毂 | 10000 N·m+ | 特殊装备 |

## 填写注意

**场景 1：稳态电机**
- 填额定扭矩 T_rated 即可，冲击由 KA 吸收

**场景 2：启停冲击频繁**
- 若已知启动扭矩 T_start（通常 2~3 倍额定），可选：
  - T_req = T_rated + KA 吸收冲击（推荐）
  - T_req = T_start，KA = 1（直接填峰值，KA 不再放大）
- 不要两边都放大（KA × T_start），会过度保守

**场景 3：双向扭矩（正反转）**
- 填绝对值最大的方向扭矩
- 启用 Step 5 fretting 评估，考虑循环效应

**场景 4：高速离心主轴**
- 离心力未纳入本工具模型，需要单独评估；T_req 按传统静扭矩填入

## 常见误用

- **单位混淆 N·m vs N·mm**：差 1000 倍；本工具内部也会用 N·mm，但输入字段固定 N·m。
- **把时均扭矩当 T_req**：真实工况里峰值扭矩可能是均值的 2~3 倍；过盈连接需要峰值通过，填峰值或用 KA 吸收。
- **只看电机标签**：电机铭牌是输出端，连接位置可能有多级传动，实际扭矩随速比放大。
- **短路 / 急停瞬态没考虑**：电机短路或紧急制动可能产生 3~5 倍额定扭矩；若此工况是设计约束，必须把它作为 T_req 或在 KA 里吸收。
