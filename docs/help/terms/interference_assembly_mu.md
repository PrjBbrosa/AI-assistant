# 装配摩擦系数（μ_Assy / μ_press_in / μ_press_out）

**一句话**：过盈连接**装配瞬间**的动摩擦系数；用于估算压入力 / 压出力，与服役摩擦 μ_T, μ_Ax 不同；典型值比服役摩擦低 20~30%。

**怎么理解**：

装配过程是动态摩擦：轴在向轮毂内**滑动**，接触面处于**动摩擦**状态。与服役阶段（已建立稳态接触、静摩擦临界滑移）不同：

1. **油膜残余**：装配油没被完全刮除，仍有流体边界润滑
2. **微峰未完全咬合**：装配瞬间刚开始互锁，还未达到服役的粘结平衡
3. **热效应**：压入摩擦发热可能局部降低黏度（轻微润滑加剧）

所以装配摩擦通常比服役摩擦低。

## 本工具的三个相关字段

```
friction.mu_assembly  (μ_Assy)     manual_only / shrink_fit 模式下的通用装配摩擦
assembly.mu_press_in  (μ_in)       force_fit 模式下压入方向
assembly.mu_press_out (μ_out)      force_fit 模式下压出方向
```

**字段联动**：
- `manual_only`：仅用 μ_Assy 估算通用压入力
- `shrink_fit`：几乎无摩擦（热装间隙下动滑），μ_Assy 仍被用作报告展示值
- `force_fit`：μ_in 用于压入力估算，μ_out 用于压出力；μ_Assy 在结果展示时被 μ_in 覆盖

## 公式（本工具实现）

```
manual_only / shrink_fit 模式:
   F_press = μ_Assy · p · π · d · L                    [F: N, p: MPa, d,L: mm]

force_fit 模式:
   F_press_in  = μ_in  · p · π · d · L                 [F: N]
   F_press_out = μ_out · p · π · d · L                 [F: N]
```

三个工况（p_min, p_mean, p_max）下都会计算压入力曲线，UI 显示最小 / 平均 / 最大三档。

## 典型值

| 场景 | μ_Assy 典型 | μ_in 典型 | μ_out 典型 |
|---|---|---|---|
| **钢-钢干摩擦（常温干装）** | 0.10 ~ 0.14 | 0.08 ~ 0.12 | 0.06 ~ 0.10 |
| **钢-钢 + 装配油 / MoS2 膏** | 0.06 ~ 0.10 | 0.05 ~ 0.08 | 0.04 ~ 0.06 |
| **钢-铝干摩擦** | 0.08 ~ 0.12 | 0.06 ~ 0.10 | 0.05 ~ 0.08 |
| **钢-铸铁干摩擦** | 0.08 ~ 0.12 | 0.06 ~ 0.10 | 0.05 ~ 0.08 |
| **热装（shrink_fit）瞬间** | ≈ 0.05 或更低 | — | — |

**本工具默认值**：
- μ_Assy = 0.12
- μ_in = 0.08
- μ_out = 0.06

## 为什么 μ_out < μ_in

首次压入后接触面发生轻微塑性变形：
- 微峰被"展平"，表面变光
- 残余装配油形成油膜
- 机械互锁程度下降

所以**再次滑动（压出）时摩擦比首次压入低约 20~30%**。这对设备选型有现实意义：
- **压入设备**：按 μ_in 算最大压入力选型
- **维修拆卸**：按 μ_out 估算抽出力，可能比压入时低

## 压入力超 250 kN 的工程建议

本工具在 F_press_max > 250 kN 时会发出警告，建议改为：
- **热装**（shrink_fit）：轮毂加热 100~200°C，装配间隙下几乎无摩擦
- **液压装配**（hydraulic-dilation）：在配合面加入高压油短暂撑开孔；本工具未内置
- **分段装配**（stepped interference）：分多段不同过盈量，降低单段阻力；本工具按均匀过盈处理

## 常见误用

- **μ_Assy = μ_T**：服役摩擦直接用于压入力估算，会把压入力估高 20~30%，设备选型过度。
- **μ_in = μ_out**：实测压出通常更容易，用同一值不会错但会保守。
- **热装工况仍填 μ_in = 0.1**：热装瞬间理论上几乎无摩擦（装配间隙），μ_Assy 值仅用作报告展示；校核重点应转向**热装温度**是否超轮毂允许上限。
- **默认值照搬不看工况**：默认值是通用中值；油污环境 / 特殊涂层 / 非标工艺必须按实际数据填入。
- **忘了分 in / out**：force_fit 模式下本工具允许两者独立；若手边只有一个数据，建议 μ_in 按该值填、μ_out 按 μ_in × 0.75 填。
