# 表面粗糙度（Rz）

**一句话**：表面轮廓**峰谷高度**的代表值（um）；过盈配合用 Rz（非 Ra）做粗糙度压平计算，因为 Rz 直接关联微观凸峰的塑性压平量。

**怎么理解**：

粗糙度有多种描述参数，工程最常用的是 Ra 和 Rz：
- **Ra**：算术平均偏差（中心线平均粗糙度）
- **Rz**：轮廓最大高度（峰谷最大差）

两者关系（对钢磨削面大致）：`Rz ≈ 4~6 · Ra`。

为什么过盈配合用 Rz 不用 Ra：
- **Rz 反映最高的"锐峰"**，那些峰是最容易被塑性压平的部分
- Ra 是平均值，包括缓坡波纹，不直接对应压平损失
- DIN 7190-1 的压平公式 `s = k·(Rz_s + Rz_h)` 用 Rz 做输入

## 公式（本工具实现）

```
粗糙度压平量:
   s = k · (Rz_shaft + Rz_hub)                  [um, k: 无量纲（见 smoothing_factor 术语）]

有效过盈量:
   δ_eff = max(0, δ − s)                        [um]
```

## Rz 典型范围（按加工工艺）

| 加工工艺 | Rz 典型（um） | Ra 等价（大致 Ra = Rz/5） |
|---|---|---|
| **精密磨削**（镜面感） | 0.4 ~ 1.6 | Ra 0.05 ~ 0.4 |
| **精磨**（高精度机件） | 1.6 ~ 3.2 | Ra 0.4 ~ 0.8 |
| **粗磨 / 精车 + 粗磨**（一般工业） | 3.2 ~ 6.3 | Ra 0.8 ~ 1.6 |
| **车削 / 铣削**（未磨削） | 6.3 ~ 16 | Ra 1.6 ~ 3.2 |
| **粗车** | 16 ~ 32 | Ra 3.2 ~ 6.3 |
| **铸造表面** | 25 ~ 100+ | Ra 6.3 ~ 25 |

**过盈配合常用范围**：Rz 1.6 ~ 6.3 um（即 Ra 0.4 ~ 1.6）。

**本工具默认 Rz_shaft = Rz_hub = 6.3 um**（精车 + 粗磨，工业普遍）。

## Ra → Rz 估算

若图纸只标 Ra，需估算 Rz：
```
粗加工表面 (Ra 0.8+):      Rz ≈ 4 · Ra
磨削表面 (Ra 0.2~0.8):     Rz ≈ 5 · Ra
镜面 (Ra < 0.2):           Rz ≈ 6~8 · Ra
```

例：Ra=0.8 → Rz ≈ 4 um 到 5 um（推荐填 4 um 偏保守）。

**标准换算参考**：GB/T 1031 / ISO 4287 / DIN 4768（旧）/ ISO 21920（新）。不同定义下数值可能相差 10~20%；建议按本企业图纸惯例统一。

## 工程影响

**Rz 越大的表面**：
- 压平量 s 越大 → 有效过盈越小 → 接触压力越低 → 传递能力越低
- 但也**促进机械互锁**，实测摩擦系数可能上升 5~15%（本工具不体现这个效应）

**Rz 越小的表面（镜面级）**：
- 压平量 s 接近 0 → 几乎全部 δ 都是有效过盈 → 压力、容量最大
- 但**加工成本显著上升**；只有高精度主轴、轴承圈这类场合才值得做到 Rz < 1.6

## 工程选择建议

1. **默认 Rz = 6.3 um**：一般过盈配合足够，不必过度追求镜面
2. **大过盈连接可放宽**：δ 足够大时粗糙度压平影响相对较小
3. **小过盈 / 精密件必须 Rz ≤ 3.2 um**：否则压平量接近 δ 的一半，非常不稳定
4. **批量生产统一检测口径**：老图纸 DIN 4768 vs 新图纸 ISO 21920 的 Rz 定义差 10~20%；建议企业内部统一

## 常见误用

- **填 Ra 值到 Rz 字段**：差 4~6 倍，会让 s 估低，压力虚高、结论虚假乐观。
- **图纸标 Ra 0.8**：不是 Rz 0.8；需换算到 Rz ≈ 4 um 填入。
- **忽略批量散差**：真实生产批的 Rz 分布有方差；保守填 +2σ 上限。
- **同时忽略 Rz 效应**（填 0）：等于不扣粗糙度压平，明显乐观。
- **旧定义 vs 新定义**：DIN 4768（1970s）与 ISO 21920（2021）的 Rz 略有差异；数值级别相近但不是完全等同。

## 两侧 Rz 差异的处理

本工具把 `Rz_shaft + Rz_hub` 两个值**相加**代入公式：

```
s = k · (Rz_shaft + Rz_hub)
```

这是标准做法（压平发生在两侧的微峰）。实际场合下：
- 若一侧是软材料（铝），其凸峰容易被硬侧（钢）完全压平，压平效应可能偏向软侧
- 若两侧硬度接近，压平近似均分

本工具按**和**处理，属 DIN 标准做法，不细分硬度。
