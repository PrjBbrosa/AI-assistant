# 需求径向力（F_r,req）

**一句话**：施加在过盈连接上的**外部径向力**（N）；不直接消耗摩擦容量，但会在接触面局部**抬高需求接触压力**，可能引起张口缝。

**怎么理解**：

径向力不是靠摩擦抵抗的（轴向力才是）；它作用的是"把接触面局部压得更紧 / 局部压松"。对过盈配合来说危险在于：
- 径向力方向上的接触面被**额外压紧**（没问题，只是应力更高）
- 径向力**反向**的接触面（180°对面）**压力减小**，可能**归零或分离**

一旦局部分离，摩擦力瞬间失去，扭矩 / 轴向容量局部失效；即使"整体扭矩容量还够"，实际是从分离侧开始滑移。

常见径向力来源：
- 齿轮啮合的径向力（任何齿轮都有）
- 皮带 / 链的张力合成径向分量
- 不平衡旋转质量的离心力
- 外部静载（如悬挂重物压在一段传动轴上）

F_r,req = 0 表示无径向力（纯轴对轴套）。

## 公式（本工具实现）

```
设计径向力:    F_r,design = KA · F_r,req                  [F: N]

附加接触压强需求（径向方向）:
   p_radial = F_r,design / (d · L)                        [p: MPa, F: N, d,L: mm]

张口缝总需求压强（径向 + 弯矩合成）:
   p_gap = p_radial + p_bending

张口缝 PASS 判据:
   p_min ≥ p_gap
```

`p_radial = F_r / (d·L)` 是**均匀平均**的保守简化：实际径向力会让压力分布沿周向呈 cos(θ) 波动，最危险位置应力可能更高。本工具按平均值处理，属工程估算。**Cannot verify against original DIN standard** —— 精确分布需走 FEA。

## 典型值

| 应用 | F_r,req 典型 |
|---|---|
| 皮带轮（张紧侧合力） | 500 ~ 3000 N |
| 链轮毂 | 1000 ~ 5000 N |
| 直齿小齿轮（径向分力） | 200 ~ 2000 N |
| 伞齿 / 锥齿（径向分力） | 500 ~ 5000 N |
| 蜗杆径向推力 | 1000 ~ 10000 N |
| 悬臂齿轮 / 离心飞轮 | 视工况，可能极大 |

## 张口缝的工程意义

**什么情况下张口缝会成为瓶颈**：
- 配合长度短（L/d < 0.5）：径向力集中在小面积上
- 过盈量偏小（p_min 本身不大）
- 径向力大（高速皮带 / 大齿轮）
- 弯矩大（悬臂安装）

**张口缝一旦发生的后果**：
- 接触面分离 → 该区域摩擦失效 → 局部滑移
- 分离端面应力集中 → 疲劳裂纹萌生
- 加速 fretting corrosion（界面微滑移 + 氧化）

## 常见误用

- **忘记填**：径向力最容易被忽略；很多"纯扭矩连接"其实有齿轮或皮带附加径向分力。
- **只填扭矩支线的分量**：齿轮/皮带的径向分力与扭矩分力方向不同，两者都要单独填。
- **KA 重复计入**：KA 在工具内部对所有载荷统一放大；不要把 KA × F_r 填进 F_r,req 字段。
- **静/动不分**：高速旋转的不平衡离心力会随转速平方增大；按最高工作转速下的 F_r 填。
- **忽略悬臂 → 弯矩耦合**：悬臂安装的齿轮同时带径向力 + 弯矩；本工具的 p_gap 是两者之和，必须同时填。

## 与弯矩的组合

径向力作用在"某个点"时，若该点与过盈连接中心有距离，会同时产生弯矩 `M_b = F_r · a`。本工具对径向力和弯矩的影响**线性叠加**：

```
p_gap = p_radial + p_bending
      = F_r / (d·L) + 2.25·M_b·1000 / (d·L²)
```

悬臂工况 `a = L/2` 时：
- p_radial = F_r / (d·L)
- p_bending = 2.25·(F_r·L/2)·1000 / (d·L²) = 1.125·F_r·1000/(d·L)
- p_gap ≈ 2.125·F_r·1000/(d·L)

相当于"径向力被放大 2 倍"，这是悬臂连接的典型保守结果。
