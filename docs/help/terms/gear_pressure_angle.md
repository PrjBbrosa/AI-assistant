# 压力角（α）

**一句话**：齿面啮合点上法向力方向与节圆切线的夹角，是齿轮几何的核心设计参数。

**怎么理解**：

压力角决定了齿廓的胖瘦与接触面承力方式。α 越大，齿根越宽厚，弯曲强度越高，且接触线更陡有利于抗点蚀；但同时径向分力（把轴往外推的力）增大，对轴承不利，也更容易咬住。α 越小则相反：齿细效率略高、径向分力小，但强度余量少。

20° 是目前工业界压倒性主流的选择，刀具、齿条、各类齿轮成品几乎全按 20° 生产，除非特殊场合否则不建议偏离。14.5° 是美式旧标准，如今仅在维修老设备时出现；25°（或 22.5°）主要用于飞机、汽车变速箱等高功率密度场合。

对蜗杆蜗轮，压力角通常指**法向压力角 αn**（沿螺旋齿法线方向测量），与直齿轮的含义略不同但工程用法一致。DIN 3975 定义了 4 种蜗杆齿形（用车刀、铣刀还是砂轮加工决定齿面形状），各自压力角测量方向略有出入：

- **ZA（阿基米德型）**：轴向剖面齿廓是直线，轴向测量 α = 20°；车刀加工得到
- **ZN（法向直齿型）**：法向剖面齿廓是直线，法向测量 αn = 20°；铣刀加工
- **ZI（渐开线型）**：端面齿廓是渐开线，法向 αn = 20°；磨齿砂轮加工，精度最高
- **ZK（圆弧锥形）**：砂轮锥面加工，齿廓略带圆弧

标称值都是 20°，差异主要在齿面微观形状与加工工艺上，普通设计场景不必深究。本模块按 ZI 齿形假设，法向压力角 αn 直接输入即可。

**公式**：径向分力 `Fr = Ft · tan α` [Ft、Fr: N，α: rad]（切向力 Ft 已知的情况下计算把轴向外推的径向力）

**典型值**：

- 14.5°：旧式美国标准，仅老设备维修沿用
- 20°：现代国际通用标准，覆盖 90% 以上齿轮与蜗杆应用
- 22.5° / 25°：高负载齿轮（变速箱、航空）、小齿数齿轮避免根切
- 蜗杆蜗轮：标称 αn = 20°，ZI 齿形法向压力角就是 20°

除非客户图纸另有规定或配对齿已定，**默认选 20°**。

**出处**：GB/T 1356-2001（通用机械基本齿轮齿条齿形）、DIN 867:1986（直齿轮基准齿廓）、DIN 3975-1:2017（蜗杆 ZA/ZN/ZI/ZK 齿形定义）、ISO 53。Cannot verify against original DIN standard —— 本术语的 ZA/ZN/ZI/ZK 描述基于公开教科书整理。
