# 退刀槽宽度（relief_groove_width_mm）

**一句话**：花键齿段与相邻光滑过盈段之间的让刀凹槽宽度，用于加工退刀；工具计算光滑段时会把**名义配合长度**减去退刀槽宽度得到真实的有效配合长度。

**怎么理解**：

花键零件加工时，齿与光滑段之间不能平直过渡，否则加工花键的滚刀 / 拉刀无法"干净地收刀"。典型的做法是在交界处做一个宽度 2~5 mm 的环形凹槽，让刀具有让位空间：

```
  花键齿段           退刀槽（宽度 w）         光滑过盈段
 ├─────────────┤ ├──────────────┤   ├──────────────────┤
  齿高 h, 长 L_spline  这段齿完全消失     名义长度 L_nominal
                                        有效 L_fit = L_nominal - w
```

这一段**既没有齿也不做过盈配合**（凹槽底径比光滑段小），因此：
- 它不在花键齿面承压校核的有效长度 L 里（场景 A 的 L 只计实际啮合齿长）。
- 它不在光滑段过盈校核的有效长度 L_fit 里（要从名义长度中扣除）。

工具在 `core/spline/calculator.py:_calculate_scenario_b` 里显式做这个扣减：
```python
l_fit = l_nominal - relief_groove
if l_fit <= 0:
    raise InputError("退刀槽宽度 >= 配合长度，有效配合长度 <= 0")
```

因此**用户需要填的 `smooth_fit.fit_length_mm` 是名义长度（含退刀槽）**，而不是已扣减的有效长度；工具自动完成扣减。

**典型值**：

- **2 ~ 3 mm**：小模数花键（m ≤ 1.25），加工时刀具让位要求小。工具默认 3.0 mm。
- **3 ~ 5 mm**：中等模数（m = 1.5 ~ 2.5），主流工业范围。
- **5 ~ 8 mm**：大模数（m ≥ 3）或加工刀具直径大；重型传动常见。
- **> 10 mm**：特殊场合（例如需要安放挡圈 / 弹性挡圈）；此时凹槽不只是退刀，还是功能槽。

**本工具实现的补充规则**：
- `relief_groove_width_mm < 0` → `InputError: 退刀槽宽度不能为负数`。
- `relief_groove_width_mm >= fit_length_mm` → `InputError: 退刀槽宽度 >= 配合长度，有效配合长度 <= 0`。
- 退刀槽宽度仅进入**场景 B**（光滑段），对场景 A（花键齿面）没有影响。

**常见误用**：

- **把光滑段名义长度 fit_length_mm 就填成"已扣减后的有效长度"**：双重扣减。工具再减一次会让 L_fit 严重偏小、p_mean 虚高、防滑虚低。正确做法：按图纸标注的"光滑段名义长度"填，工具自行扣减。
- **忘记退刀槽直接填 0**：相当于把整段视作有效承载；对短轴（L_nominal < 20 mm）会让扭矩容量高估 10~20%。
- **误把齿段与光滑段的长度混进一个字段**：`spline.engagement_length_mm` 是**齿段**有效啮合长度，`smooth_fit.fit_length_mm` 是**光滑段**名义配合长度。两段位于轴向相邻位置但独立。
- **不同的退刀槽宽度填给花键段**：花键段不做退刀槽扣减（齿段自身的 L 已经是用户按"有效齿啮合"直接给的）。退刀槽字段只影响光滑段。

**出处**：本字段属于加工工艺约定，无直接 DIN 标准约束；退刀槽宽度取值参考《机械设计手册》花键加工章节。

> Cannot verify against original DIN standard —— 退刀槽宽度属于加工工艺参数，DIN 5480 / DIN 6892 / DIN 7190 原标准未给具体数值。
