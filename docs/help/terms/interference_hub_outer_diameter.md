# 轮毂外径（D）

**一句话**：轮毂外圆直径；决定厚壁轮毂的几何因子 (D²+d²)/(D²−d²)；D 越大，轮毂越刚，同等过盈下接触压力越高，但轮毂应力水平**不一定**更低。

**怎么理解**：

轮毂用作过盈配合的"外包圈"；它必须抵抗从接触面传来的径向压力。用**厚壁圆筒** Lamé 解建模：

```
几何因子 K_hub = (D² + d²) / (D² − d²)
```

K_hub 描述"轮毂对径向压力的几何放大"。D/d 越接近 1（薄壁），K_hub 越趋近无穷大；D/d 越大（厚壁），K_hub 越趋近 1。

| D / d | K_hub | 轮毂类型 |
|---|---|---|
| 1.2 | ≈ 5.5 | 极薄壁，Lamé 解近失效 |
| 1.5 | ≈ 2.6 | 薄壁轮毂 |
| 2.0 | ≈ 1.67 | 中等壁厚（工程默认起点） |
| 2.5 | ≈ 1.38 | 厚壁 |
| 3.0 | ≈ 1.25 | 厚壁（接近极限） |
| ∞ | 1.00 | 无限大实体（理想刚性） |

**工程建议**：D/d ≥ 1.5 才满足厚壁圆筒假设；D/d < 1.3 时本工具模型偏乐观（低估轮毂应力），应走 FEA。

## 公式（本工具实现）

```
几何因子：    K_hub = (D² + d²) / (D² − d²)

轮毂径向柔度（半径方向变形 / MPa 压力）：
   c_hub = (d/2) / E_h · (K_hub + ν_h)                 [mm/MPa]

轮毂 von Mises 系数：
   VM_hub = sqrt(1 + K_hub + K_hub²)                   [无量纲]

同等接触压力 p 下：
   σvm_hub = p · VM_hub                                [MPa]
   周向应力（内壁最大）: σθ = p · K_hub                [MPa]
```

## 轮毂外径对校核的双向影响

**D 增大**：
- K_hub 下降 → 轮毂更刚 → 同等过盈 δ 下接触压力 p 提升（有利于扭矩容量）
- VM_hub 下降 → 轮毂 von Mises 下降（有利于轮毂应力校核）
- 轴应力（von Mises 系数 VM_shaft 不变）→ 轴侧应力反而可能上升（p 变高）

所以不是"D 越大越好"：D 过大时，压力放大可能让**轴侧应力**先出问题。工程上常见的甜点区是 D/d = 2.0 ~ 2.5。

**D 减小**：
- K_hub 急剧上升 → 轮毂柔度上升，同等过盈下压力下降（不利传递能力）
- VM_hub 上升 → 薄壁轮毂更容易屈服

## 典型值

| 应用 | D/d 典型 |
|---|---|
| 齿轮 / 皮带轮毂 | 1.6 ~ 2.5 |
| 轴承内圈（等效 D≈外径） | 1.4 ~ 1.8 |
| 锥套 / 胀套 | 1.8 ~ 2.2 |
| 大齿轮体（带辐板） | 2.5 ~ 3.5 |

**本工具未内置"阶梯轮毂"模型**：若轮毂有多台阶，建议按最薄壁处（最小 D）取值做保守估算，或走 FEA。

## 常见误用

- **填半径 = 40 当 D = 40**：D 是直径；填半径会让几何因子严重偏差。
- **D ≤ d**：本工具直接报错；D 必须大于 d。
- **薄壁（D/d < 1.3）**：Lamé 解假设"轮毂壁无轴向约束、径向压力均匀"；薄壁 + 短配合长度时这两条都不满足，本工具会显著低估轮毂应力。
- **带辐板 / 幅条的轮毂**：本工具按实心厚壁圆筒近似；辐板型轮毂实际刚度远低于纯圆筒，应选等效"包裹圆筒"的最小 D 做保守估算。
- **轮毂外有法兰 / 键槽**：法兰会局部加厚、键槽会局部削弱；本工具不区分，需手工判断最薄弱位置的等效 D。
