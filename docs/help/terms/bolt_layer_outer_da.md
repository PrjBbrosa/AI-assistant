# 第 n 层外径（layer_DA_n）

**一句话**：第 n 层被夹件的外径；多层模式下每层须独立填写，工具不会自动从全局 D_A 继承。

**怎么理解**：

VDI 2230 用等效圆柱/锥台模型计算被夹件柔度，柔度公式需要每层的截面积：

```
A_i = pi * (DA_i^2 - d_h^2) / 4        [mm^2，第 i 层有效截面]
delta_i = t_i / (E_i * A_i)            [mm/N，第 i 层柔度]
```

常见多层法兰外径并非相同：例如底座直径 80 mm、中间法兰法兰外径 120 mm、顶部盖板直径 95 mm，三层 DA 不同，若统一使用全局 DA 会导致某些层的截面积严重偏差，进而误算柔度和 Phi_N。

## 逐层不同的适用场景

- **台阶法兰**：连接体因安装需要做成台阶，不同截面层厚不等；
- **止口配合带定位销套**：套筒外径独立于法兰本体；
- **异形夹持（异形截面）**：中间有衬套，内外径与其余层均不同；
- **混合材料法兰**：铝合金盖 + 钢本体，外径可能相差 10–30 mm。

## 填写要求

多层模式下，每层的 DA_n 字段须独立填写——工具不从全局 `clamped.D_A` 自动继承。DA_n ≤ 0 会触发 `InputError` 报错；正值无上下界校验，填错（如远小于 d_h 的正值）会在柔度计算中传播为错误结果。工程师需自行确保：

- 每层 DA_n 均已正确填写，不可留空；
- DA_n 应大于等于螺栓头支承面外径 D_w，以保证等效圆柱体模型有意义；
- 外径均匀的法兰可将各层填写相同值，便于核对。

## 常见误用

- **把螺纹孔边距当作 DA**：DA 是被夹件本体外径，与孔心距、边距无关；孔径 d_h 已通过螺纹公称直径自动计算。
- **多层时遗漏逐层 DA 输入**：多层模式下必须逐层填写 DA_n；留空会触发"不能为空"的 InputError，填 0 或负值会触发 `_positive(D_A, ...)` 的正值校验 InputError；两种错误都会中止计算。各层外径差异 > 20% 时柔度误差可超 40%，应特别注意逐层准确填写。

**出处**：VDI 2230:2015 §5.1.2.5
