# 装配利用系数（ν）

**一句话**：装配阶段允许螺栓等效应力达到屈服强度 Rp0.2 的百分比（典型 0.9），决定 R4 校核的允许值 σ_allow,asm = ν · Rp0.2；ν 越小越保守，但允许预紧力越低。

**怎么理解**：

R4 装配等效应力校核：

```
σ_vm,asm = √(σ_ax,asm² + 3·τ_asm²)    [MPa]     # 装配等效应力
σ_allow,asm = ν · Rp0.2                [MPa]     # 允许值
R4 通过: σ_vm,asm ≤ σ_allow,asm
```

ν 是工程师预留的**屈服裕量**：

- **ν = 0.9**（本工具默认）：装配允许应力达到屈服的 90%，留 10% 裕量。工业主流。
- **ν = 0.95**：更激进，用到 95% 屈服。需要极高的工艺控制（转角法、液压拉伸）。
- **ν = 0.8**：更保守，只允许 80% 屈服。用于疲劳敏感或反复拆装的连接。

## 为什么不直接让 σ_vm,asm ≤ Rp0.2（即 ν = 1.0）

- 实际屈服曲线过渡平滑，Rp0.2 本身是约定值（0.2% 塑性变形），不是"瞬时屈服点"
- 材料批次偏差 ±5% 常见
- 拧紧过程中实际应力会超过扳手读数推算值（摩擦波动、震动）
- 留裕量避免微量塑性变形引起预紧力松弛

所以工程实践默认 ν = 0.9。

## ν 与拧紧方式的关系

**拧紧精度越高，允许 ν 越大**：

| 拧紧方式 | 推荐 ν | 理由 |
|---|---|---|
| 扭矩法 | 0.85–0.9 | 摩擦散差大，留更多屈服裕量 |
| 转角法 | 0.9–0.95 | 避开摩擦，可直接控制到屈服段 |
| 液压拉伸法 | 0.9–0.95 | 高精度 |
| 热装法 | 0.9 | 温度控制不确定性 |

**转角法的特殊意义**：转角法本身就是"拧到屈服开始塑性变形 + 再转一定角度"，所以 ν 实际可以取到 **1.0 以上**（允许少量塑性），但本工具 UI 上限锁在 0.95。若实际工艺明确按屈服控制法（转角法变种），需要按 VDI 2230 附录另计。

## R4 公式展开（本工具实现）

```
σ_ax,asm = FM,max / As                           [MPa]  # 装配轴向应力
M_thread = FM,max · 0.5·d2·(P/(π·d2) + 1.155·μG) [N·mm] # 螺纹摩擦扭矩
τ_asm   = 16·M_thread / (π·d3³)                  [MPa]  # 装配扭转应力
σ_vm,asm = √(σ_ax,asm² + 3·τ_asm²)               [MPa]  # 等效应力
σ_allow,asm = ν · Rp0.2                          [MPa]  # 允许值
```

## 与服役屈服安全系数 S_F 的区别

ν 和 S_F 是**不同阶段**的不同参数：

- **ν**（装配阶段）：ν · Rp0.2 是允许装配应力
- **S_F**（服役阶段）：Rp0.2 / S_F 是允许服役应力

两者数值上没有直接关系。常见工程取值：

- 装配 ν = 0.9 + 服役 S_F = 1.1 → 装配允许 0.9·Rp0.2，服役允许 Rp0.2/1.1 ≈ 0.91·Rp0.2
- 装配 ν = 0.9 + 服役 S_F = 1.3 → 装配允许 0.9·Rp0.2，服役允许 Rp0.2/1.3 ≈ 0.77·Rp0.2

**规律**：服役阶段允许值 ≤ 装配阶段允许值（因为服役时还要加上外力贡献的 σ_ax_work）。

## 常见误用

- **ν 填 1.0**：装配允许直接到屈服。数值与公式上可行，但任何工艺波动会导致瞬时塑性变形。**除非有屈服控制工艺和数据支撑，不要填 1.0**。
- **ν 填 0.7**：过于保守，允许 FM,max 很低，R4 易过但 R3 困难（FM,min 也被连带降低）。除非疲劳特别严格否则不建议。
- **ν 当成"安全系数"的倒数理解**：0.9 = 1/1.11，数学上是安全系数的倒数。但文档里 VDI 2230 明确叫 "Anziehfaktor / utilization" —— 它是"用多少"，不是"余多少"。
