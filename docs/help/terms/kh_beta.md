# 齿宽载荷分布系数（KHβ）

**一句话**：载荷沿齿宽方向（beta 方向）分布不均造成的局部接触应力放大倍数，反映轴系刚性与装配精度。

**怎么理解**：

理想情况下载荷沿整个齿宽均匀分布，但实际上轴在扭矩下会发生弹性挠曲（特别是悬臂布置或长跨距），齿轮副会出现"一头压得紧、另一头几乎脱开"的偏载。加上加工时的齿向偏差、装配时的平行度误差、箱体变形，齿宽一端的实际接触应力可能比平均值高 20% 到 50%，这就是 KHβ 要表达的。

KHβ 受三类因素主导：**轴系刚度**（跨距、直径、悬臂或对称支承）、**齿面精度**（齿向偏差 Fβ）、**齿向修形**（螺旋修形、鼓形修形可显著降低 KHβ）。即便齿轮本身精度很高，若安装到挠曲大的长轴上，KHβ 仍然会很高——所以 KHβ 本质上是**系统特性**，不是单独零件特性。

DIN 3990 / ISO 6336 的 C 法按齿宽 b 与小轮直径 d1 的比值（b/d1）查图得到基础 KHβ0，再做支承方式修正；严格做法则按 A/B 法直接代入挠曲变形计算。对蜗轮，该系数在 DIN 3996 中有专门形式，一般不单独输入。

**典型值**（齿轮 DIN 3990-1 §8）：

- 对称支承、b/d1 ≤ 0.5、精密齿轮：KHβ = 1.05–1.10
- 对称支承、b/d1 = 0.5–1.0、常规精度：KHβ = 1.10–1.25
- 非对称支承 / 较长跨距 b/d1 > 1：KHβ = 1.20–1.40
- 悬臂布置（齿轮伸出支承外）：KHβ = 1.30–1.70
- 无齿向修形的大齿宽斜齿轮：KHβ 经常超过 1.50

选择依据：工业通用齿轮（对称支承、常规精度）缺省取 1.15–1.25；机器人等刚性结构取 1.10；输出端悬臂蜗轮轴、搅拌器输出轴等取 1.30–1.50 并配合鼓形修形。

**出处**：DIN 3990-1:1987 §8，ISO 6336-1:2019 §8
