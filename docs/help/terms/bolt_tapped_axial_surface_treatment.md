# 螺纹表面处理（rolled / cut）

**一句话**：螺栓外螺纹的加工方式，**轧制（rolled）**还是**切削（cut）**，直接决定疲劳极限 σ_ASV 的取值——切削螺纹比轧制螺纹的疲劳寿命显著更短（约 0.65 倍）。

**怎么理解**：

螺纹加工有两种主流方式：

- **滚轧成型（rolled）**：通过滚压模让螺纹金属纤维塑性变形排列，牙根留下残余压应力；疲劳裂纹难在压应力牙根处形核。**绝大多数工业螺栓**（8.8、10.9、12.9 标准紧固件）都是滚轧螺纹。
- **切削加工（cut）**：车削 / 攻丝 / 板牙等方式切断金属纤维，牙根是残余拉应力 + 加工痕迹。易成为疲劳裂纹起源。**常见于专用件、螺纹杆件、后加工螺纹**。

本工具的 σ_ASV 基表是**轧制螺纹**的值（rolled）；选切削螺纹（cut）时乘以折减系数：

```
_CUT_THREAD_FACTOR = 0.65      # core/bolt/tapped_axial_joint.py:32
σ_ASV_effective = σ_ASV_table[d] · cut_factor · cycle_factor       [MPa]
```

`cut_factor = 1.0`（rolled）或 `0.65`（cut）。0.65 意味着**切削螺纹疲劳许用幅只有轧制的 65%**——同样的 σ_a，切削螺纹的疲劳安全系数只剩 2/3。

## 怎么判断是 rolled 还是 cut

- **标准紧固件**（ISO 4014 / GB/T 5782 六角头螺栓、8.8/10.9/12.9）：几乎全部是滚轧螺纹。**默认选 rolled**。
- **专用长螺纹杆、自制螺栓**：常见切削螺纹。
- **大规格螺栓**（M > 30）：工艺可能是切削，查产品说明或采购规格。
- **后加工螺纹**（如在光杆上现场攻丝）：必为 cut。
- **螺纹衬套 / 预制孔螺纹**：通常切削或铣削。

如果不确定：**填 cut（更保守）**——高估 σ_a 风险远高于低估。工程经验上除非采购文件明确写"rolled threads"，否则专用件按切削考虑。

## σ_ASV 基表（部分）

| 6 | 50 | 32.5 |
| 8 | 47 | 30.6 |
| 10 | 44 | 28.6 |
| 12 | 41 | 26.7 |
| 16 | 38 | 24.7 |
| 20 | 36 | 23.4 |
| 24 | 34 | 22.1 |
| 30 | 32 | 20.8 |

表格外直径按工具线性插值；**小于 M6 取 M6 值**、**大于 M36 取 M36 值**（保守钳位，`core/bolt/tapped_axial_joint.py:139-144`）。

随直径增大 σ_ASV 下降的物理原因：大螺栓的螺纹牙根几何尺寸更大、加工缺陷概率更高、**尺寸效应**使疲劳极限下降。

## 表面处理 vs 寿命修正的相对影响

疲劳许用幅按以下级联计算：

```
σ_a,allow = σ_ASV_table[d] · surface_factor · cycle_factor · goodman_factor    [MPa]
```

数量级感知：

- surface_factor：0.65（cut）或 1.0（rolled）→ 最大影响 ~35%
- cycle_factor：0.9 – 1.8 → 最大影响 ~80%
- goodman_factor：0.0 – 1.0 → 可能归零（σ_m ≥ 0.9·Rp0.2 时）

一个常见误区：认为"切削 vs 轧制影响小"——实际上切削就是**直接砍掉 35%** 疲劳许用，比 Miner 寿命修正影响还显著。

## 边界检查

`core/bolt/tapped_axial_joint.py:134-137` 要求字符串严格匹配 `rolled` 或 `cut`；其他值抛 `InputError`。**不支持用户自定义折减系数** —— 如果实际加工工艺（如磨削、滚轧后磨削 / 喷丸强化）有更好的疲劳性能，需另做评估；本工具保守按 rolled / cut 二选一。

## 常见误用

- **后加工螺纹按 rolled 选**：例如购买光杆，自己用板牙攻丝，螺纹是**切削**的，必须选 cut；选 rolled 会高估疲劳许用 ~54%（1 / 0.65）。
- **表面强化（喷丸、滚压）选 cut**：若有**后续喷丸强化**工艺，疲劳极限可能比 rolled 还高——本工具不支持此修正，只能在文档/评审中说明，校核值按 rolled 处理并做工程判断。
- **忽略 σ_ASV 随直径下降**：大螺栓（M30+）σ_ASV 显著低，许用幅小；误以为"大螺栓更耐疲劳"是**错的**——大螺栓承载高但单位面积疲劳极限更低。
- **把"surface_treatment" 当作覆层/镀层意思**：本字段**只**指螺纹成形工艺（rolled/cut），**不是指**电镀、达克罗、涂料等防腐层。
