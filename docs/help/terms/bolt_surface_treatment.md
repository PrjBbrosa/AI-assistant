# 螺纹表面处理（surface_treatment）

**一句话**：螺纹的加工方式——轧制或切削，决定疲劳极限 σ_ASV 的高低；轧制螺纹比切削螺纹具有更高的疲劳强度。

**怎么理解**：

螺纹加工方式从根本上影响螺纹牙根的残余应力状态和表面完整性：

- **轧制螺纹**（搓丝 / 滚压）：牙根引入压缩残余应力，材料晶粒沿螺旋方向流线分布，表面加工硬化。这些因素共同提高了疲劳强度。
- **切削螺纹**（车螺纹 / 丝锥攻丝）：牙根无压缩残余应力，切削刀痕可成为疲劳裂纹起始点，疲劳极限低于轧制工艺。

本工具按 VDI 2230 Table A1 存储轧制螺纹的 σ_ASV 参考值，切削螺纹则乘以折减系数 0.65：

```
# core/bolt/calculator.py
_CUT_THREAD_FACTOR = 0.65   # 切削螺纹约为轧制的 65%
sigma_ASV = sigma_ASV_rolled * 0.65   # 切削工艺
```

**实际影响举例**（M10 螺栓，VDI 2230 Table A1）：

| 加工方式 | σ_ASV (MPa) |
|---|---|
| 轧制 | 44 |
| 切削 | ~29（= 44 × 0.65）|

疲劳裕度 S_A = σ_ASV / σ_a，切削螺纹的裕度比轧制螺纹低约 35%。

## 下拉选项

| 选项 | 对应 surface_treatment | σ_ASV 计算 |
|---|---|---|
| 轧制 | `"rolled"` | 按 VDI 2230 Table A1 直接查表 |
| 切削 | `"cut"` | 查表值 × 0.65 |

默认为"轧制"——标准紧固件（螺栓、螺柱）通常为轧制螺纹；攻丝孔、现场切削螺纹应选"切削"。

## 与其他字段的关系

- 本字段仅影响疲劳极限 σ_ASV（疲劳校核 R6），**与螺纹摩擦系数 μG 无关**。
- 摩擦系数由 `tightening.mu_thread`（字段 `bolt_friction_thread`）单独控制。
- 本字段的 `mapping = None`，不直接传入计算器 payload；计算器从 `options.surface_treatment` 读取。

## 何时选切削

- 现场攻丝（铸铁箱体直接攻 M 螺纹）
- 细长零件上直接车螺纹（不能滚压的直径/长度比）
- 改制、修复件
- 图纸或工艺卡明确标注"切削螺纹"

标准外购螺栓（ISO 4014/4017 等）均为轧制螺纹，选"轧制"。

## 常见误用

- **工厂用车床批量加工螺杆，选了"轧制"**：导致 σ_ASV 虚高约 54%，疲劳校核结果偏乐观。
- **把表面处理（镀锌/发黑等）与螺纹加工方式混淆**：涂层/润滑影响摩擦系数 μG，不影响 σ_ASV；两者是不同字段，不要互换填写。

**出处**：VDI 2230:2015 Table A1（σ_ASV 轧制螺纹疲劳极限）；VDI 2230:2015 §5.5.2.2（切削螺纹折减系数）；DIN 267-31（滚压螺纹）
