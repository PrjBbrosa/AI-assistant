# 基础实体类型（圆柱体 / 锥台）

**一句话**：被夹件刚度 K_P 的几何模型选择，直接影响柔度 δ_P 的计算，进而影响载荷因数 Φ_N 和所需预紧力 F_M。

**怎么理解**：

VDI 2230 §5.1.2 提供两种被夹件柔度的解析模型：

**圆柱体模型**：将被夹件理想化为均匀截面圆柱，截面积由等效外径 D_A 和通孔直径 d_h 确定。适合被夹件截面沿夹紧方向变化不大的紧凑件。

**锥台模型**：螺栓头（或螺母）下方，力传递路径向外扩散，形成截面积随深度增大的锥台（frustum of cone）。适合薄板或 D_A 相对螺栓头支承直径 D_w 较大时的情形。本工具锥台模型使用 Lori-Engel 近似锥角，按 D_A、lK、D_w 三个几何量自动估算锥角，无需用户手动输入锥角。

```
圆柱体：A_clamped = pi/4 · (D_A^2 - d_h^2)         [均匀截面，mm^2]

锥台（Lori-Engel 近似）：
  锥角 phi = atan(0.362 + 0.032*ln(D_A/(2*lK)) + 0.153*ln(lK/D_w))
  delta_P  = 2*ln(...) / (E * pi * d_h * tan(phi))  [mm/N，见公式简表]
```

两种模型对 Φ_N 的影响：锥台模型因截面积随深度增大，等效刚度比圆柱更高（δ_P 更小），Φ_N 更小，即外力被螺栓分担得更少——若误选圆柱模型，将高估 Φ_N，进而导致 FM_min 偏低（偏不保守）；同时螺栓所承担的附加应力偏高，疲劳侧偏危险。

## 选型准则

简化判据（来自 VDI 2230 §5.1.2 的工程惯例）：

| 条件 | 推荐模型 |
|---|---|
| D_A ≤ D_Km,o + lK | 圆柱体（截面不"溢出"夹紧范围）|
| D_A > D_Km,o + lK | 锥台（截面明显超出夹紧范围）|
| 多层异材料（E 差异 > 2×）| 建议锥台 |
| 薄板（lK < 3d）| 建议锥台（锥形扩散效应更显著）|
| 厚凸缘（lK > 5d），截面均匀 | 圆柱体通常足够 |

本工具当前支持圆柱体和锥台两种模型；套筒模型（用于空心被夹件）为选项之一，柔度公式与圆柱相似但内径不同。

## 柔度公式简表

**圆柱体**（VDI 2230 Eq. 5.1c）：

```
delta_P = lK / (E_P * pi/4 * (D_A^2 - d_h^2))           [mm/N]
```

**锥台**——本工具采用 Lori-Engel 近似锥角公式（`compliance_model.py`，非固定 30° 锥角）：

```
phi = atan(0.362 + 0.032*ln(D_A/(2*lK)) + 0.153*ln(lK/D_w))   [rad，近似锥角]

delta_P = 2*ln((D_w + d_h)*(D_A - d_h) / ((D_w - d_h)*(D_A + d_h)))
          / (E_P * pi * d_h * tan(phi))                  [mm/N]
```

其中 D_w 为螺栓头/螺母支承面外径。锥角 phi 由 D_A/lK、lK/D_w 两个比值决定，通常在 15°–50° 之间随几何条件自动适应。VDI 2230 §5.1.2 Eq. 5.1d 给出包含固定锥角的完整积分解析式，精度更高但需已知锥角；本工具使用 Lori-Engel 近似可省去锥角输入，适合快速预校核。若需更精确结果，建议使用手动输入柔度模式并参考 VDI 2230 标准原文。

## 典型数值对比（M10，D_A = 24 mm，d_h = 11 mm，lK = 20 mm，钢 E = 210000 MPa）

| 模型 | delta_P [mm/N] | 对应 c_P [N/mm] |
|---|---|---|
| 圆柱体 | ~3.1e-6 | ~3.2e5 |
| 锥台（alpha=30°）| ~2.4e-6 | ~4.2e5 |
| 锥台（alpha=45°）| ~2.0e-6 | ~5.0e5 |

锥台比圆柱刚约 30–60%，对 Φ_N 的影响不可忽视。

## 常见误用

- **统一选"圆柱体"不论几何**：薄板多层连接应用锥台，用圆柱会低估被夹件刚度（高估 Φ_N），导致 FM_min 偏低、螺栓附加应力偏高，对疲劳侧偏危险，不是"偏保守可接受"的方向。
- **锥角 alpha 随意取 45° 而非 30°**：VDI 2230 推荐 30° 作为保守默认；45° 仅在经过试验验证的紧凑钢件中使用。
- **套筒模型漏掉内孔**：套筒被夹件有内孔，面积公式应用 D_out 和 D_in，与实心圆柱不同；本工具套筒模型已区分。
