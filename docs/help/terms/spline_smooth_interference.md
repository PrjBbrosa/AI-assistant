# 光滑段过盈量（δ_min / δ_max）

**一句话**：联合模式下光滑圆柱段与轮毂孔之间的过盈量**直径值**，以 μm 为单位；δ_min 决定防滑下限，δ_max 决定屈服上限，两端必须同时满足。

**怎么理解**：

过盈量 δ 定义为**轴直径与孔直径之差**（直径值，不是半径值）：

```
δ = d_shaft - d_hole     [μm]
```

轴直径略大于孔直径 → 压装后产生径向压缩，从而生成接触压力 p。加工公差下 δ 不会是定值，而是一个区间 `[δ_min, δ_max]`，对应到：

- 最紧配合：d_shaft 在上公差 + d_hole 在下公差 → δ 达到 **δ_max**（最大过盈）。
- 最松配合：d_shaft 在下公差 + d_hole 在上公差 → δ 降到 **δ_min**（最小过盈）。

**防滑校核用 δ_min**（看下限）、**屈服校核用 δ_max**（看上限）—— 这是工具场景 B 的核心判据设计，两侧都必须 PASS。

**在工具里的位置**：

- `smooth_fit.delta_min_um` / `smooth_fit.delta_max_um`：直径值，单位 **μm**（微米，不是 mm）。
- 传入 `calculate_interference_fit` 的 Lamé 厚壁筒解析解后，得到 `pressure_mpa.p_min` / `p_max`（对应接触压力下限 / 上限）。
- 三档数据（min / mean / max）沿轴向和圆周各做简化均布假设。

**δ 如何转换成接触压力**（DIN 7190 Lamé 解析解，简化表达）：

```
p = δ / (d · [ (d² + d_i²)/(d² - d_i²)/E_shaft + (D² + d²)/(D² - d²)/E_hub + ν_hub/E_hub - ν_shaft/E_shaft ])
                                                                          [mm, MPa]
```

- δ 按 μm 填入，公式内部换算到 mm（除以 1000）。
- 轴越厚（d_i 小 → 空心薄壁让 p 降低）或轮毂越厚（D 大 → 轮毂硬 → p 升高），同样 δ 下的 p 越大。
- 两种材料 E / ν 几乎相同时，p ∝ δ 近似线性。

**典型值（钢轴压入钢轮毂，d = 40 mm 附近）**：

- **过渡配合 H7/k6**：δ = 5 ~ 20 μm；接触压力较低（约 20~40 MPa），防滑能力有限。
- **小过盈 H7/n6**：δ = 10 ~ 30 μm；接触压力约 30~60 MPa，防滑足够多数轻载工况。
- **中过盈 H7/p6 或 H7/s6**：δ = 20 ~ 60 μm；接触压力 60~120 MPa。本工具 UI 默认 δ_min=20 / δ_max=45 μm 对应此档。
- **大过盈 H7/u6 或 H7/x6**：δ = 50 ~ 120 μm；接触压力 > 120 MPa；需校核屈服（必要时热装）。
- **δ > 200 μm**：极端大过盈，通常需热装 / 液压压装；实心轴要特别校验屈服。

**常见误用**：

- **填成半径值**：`smooth_fit.delta_min_um` 是**直径值**。若按半径填，实际过盈被缩小一半，算出的 p 低估一半。
- **δ_min > δ_max**：工具不抑制这种输入，但数学上 p_min > p_max 违反物理 —— 后续结果会混乱。填之前先确认范围顺序。
- **单位写成 mm**：`spline.smooth_fit.delta_*_um` 的下标 `_um` 是 μm（微米）；填 0.02 会被当成 0.02 μm（几乎零过盈）；应填 20。
- **忽视表面粗糙度折减**：实际接触时表面微凸被压平，有效过盈会小于名义 δ（约 0.6 · Rz_shaft + 0.6 · Rz_hub，DIN 7190 推荐折减）。工具在 `calculate_interference_fit` 里自动做这个折减，用户不用重复扣减 —— 只填**名义**过盈量（装配前测量的 d_shaft - d_hole）。

**出处**：DIN 7190-1:2017（过盈配合计算）；ISO 286-1/-2（公差配合代号）；GB/T 1800.1-2009（国标对照）。
