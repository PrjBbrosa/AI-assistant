# 许用齿根应力（σF,lim）

**一句话**：齿根在指定循环次数下不发生弯曲疲劳断裂的最大应力，是齿根弯曲强度校核的判据。

**怎么理解**：

齿轮每转过一次啮合区，齿根纤维就经历一次拉—压循环，和金属疲劳试棒一样可能在足够多次循环后出现裂纹，最终齿断裂失效。σF,lim 描述的就是"**齿根在对应疲劳循环次数（通常 3×10^6 脉动循环）下不断裂的最大弯曲应力**"，是从标准齿形试件弯曲疲劳试验拟合的材料特性。

σF,lim 是"原始材料弯曲疲劳极限"，校核时要按工况折减：

```
σFP = σF,lim · YN · YX · Yδ · YRrelT / SF
```

各符号含义（都是 >0 的无量纲系数）：
- **YN —— 寿命修正**：与齿面 ZN 同理，寿命短于基准 3×10⁶ 次脉动循环时 YN > 1
- **YX —— 尺寸修正**：大模数齿轮里缺陷更大，许用值下调（YX < 1）
- **Yδ —— 相对缺口敏感性修正**：齿根圆角越尖、材料越脆，Yδ 越小；退火低碳钢不敏感，高硬度渗碳钢敏感
- **YRrelT —— 表面状态修正**：喷丸强化、精磨齿根 YRrelT 可到 1.1+；铸造齿根只有 0.9 左右
- **SF —— 齿根安全系数**：工业常取 1.4–1.6，比齿面 SH 高，因为齿根断裂是突发破坏

实际齿根应力 σF 必须小于 σFP。**双向载荷**（齿轮反向工作或惯性反转）会把脉动疲劳转为对称疲劳，σF,lim 要再乘一个 0.7 左右的折减系数（参考 DIN 3990-3 双向载荷相关章节；Cannot verify against original DIN standard）。

**工程经验**：未定详细系数时，先用 σF,lim / SF 做粗估（YN、YX、Yδ、YRrelT 都取 1.0）；方案通过后再细化。塑料蜗轮要额外考虑温度折减。

齿根失效后果通常比齿面点蚀严重——点蚀是逐步的可检测磨损，断齿则是突然破坏，因此齿根安全系数 SF 通常比 SH 取得更高（SF ≥ 1.4–1.6，SH ≥ 1.0–1.2）。

**典型值**（脉动载荷、无限寿命）：

- 渗碳淬火钢齿轮（20CrMnTi 等）：σF,lim ≈ 400–500 MPa
- 调质合金钢齿轮（40Cr、42CrMo）：σF,lim ≈ 270–360 MPa
- 正火碳钢齿轮：σF,lim ≈ 180–230 MPa
- 锡青铜蜗轮（CuSn12）：σF,lim ≈ 80–110 MPa
- 铝青铜蜗轮（CuAl10Ni5）：σF,lim ≈ 115–160 MPa
- 灰铸铁齿轮：σF,lim ≈ 90–130 MPa
- 工程塑料蜗轮（PA66、POM，23°C）：σF,lim ≈ 30–70 MPa，高温下还要下调
- 纤维增强塑料（PA66-GF30）：σF,lim ≈ 70–110 MPa

选择依据：齿轮查 DIN 3990-5 / ISO 6336-5，蜗轮查 DIN 3996 附录。**双向载荷按脉动值 × 0.7 使用**；高温塑料必须按工作温度下的数据，不能用常温值。

**出处**：DIN 3990-3:1987（齿根强度计算流程）、DIN 3990-5:2016（材料疲劳数据）、DIN 3996:2019（蜗轮）、ISO 6336-3/5。Cannot verify against original DIN standard —— 本术语的系数分解与典型值基于公开教科书整理，精确数值应以标准原文为准。
