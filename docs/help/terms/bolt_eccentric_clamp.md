# 夹紧偏心 e_clamp

**一句话**：螺栓轴线到被夹件"刚度中心"的距离；非对称夹持时引起被夹件弯矩，影响力比 Φ。

**怎么理解**：

e_clamp = 0 对应对称夹紧——螺栓轴线正好穿过被夹件的刚度中心（重心近似），
两侧受压均匀，被夹件只承受纯压缩，无弯曲附加。

实际工程中螺栓偏心非常常见：

- **边缘法兰**：螺栓孔靠近法兰边，零件受夹时产生弯矩
- **角部连接**：夹件形状不对称，刚度中心偏移
- **悬臂底板**：螺栓在一侧，被夹件重心在另一侧

当 e_clamp ≠ 0 时，被夹件受压的同时产生附加弯曲变形，等效柔度增大，
进而改变力分配比（力比 Φ），影响螺栓工作载荷估算。

## 如何量

在装配截面图上：

1. 确定被夹件的**刚度中心**（截面积均匀时近似为重心，截面变化大时需计算）
2. 量螺栓轴线到刚度中心的**垂直距离**（沿被夹件承压方向的投影）
3. 该距离即为 e_clamp（单位 mm）

沿被夹件对称截面量测，不含螺栓头或螺母的偏移——只量被夹件本身的几何偏心。

## 影响公式（VDI 2230 参考）

VDI 2230 §5.1.3 给出偏心夹持的柔度修正公式：

```
被夹件柔度修正：δP* = δP + e_clamp² / (E · I)
其中：
  δP   = 对称夹持柔度（VDI 2230 标准公式）
  e_clamp = 偏心距（mm）
  E    = 被夹件弹性模量（MPa）
  I    = 被夹件在螺栓方向的截面惯性矩（mm⁴）
```

修正后柔度 δP* > δP，力比 Φ 相应增大，螺栓分担更多外载。

> **本工具当前版本未实现此修正**：`e_clamp` 字段目前仅作设计记录用途，不参与柔度或力比的自动计算。若连接存在明显夹紧偏心，需工程师按上式手动修正等效柔度，或直接在"柔度手动输入"模式中填入修正后的 δP 值。

## 典型量级

| 连接形式 | e_clamp 典型值 |
|---|---|
| 中心对称螺栓（圆法兰中心孔） | 0 mm |
| 标准法兰边孔（孔偏边缘） | 0.3–0.8 倍节圆半径 |
| 矩形底板角部螺栓 | 可达 1–2 倍螺栓直径 |
| 异形悬挑件 | > 2 倍螺栓直径 |

对称设计（螺栓在截面中心）可将 e_clamp 归零，是减小附加弯矩的最简单手段。

## 常见误用

1. **测到零件边缘而非刚度中心**：e_clamp 是到**刚度中心**的距离，
   不是到零件几何边缘的距离。薄板件重心即为几何中心，厚件或异形件需计算截面形心。

2. **对称件填非零值**：若螺栓穿过被夹件的对称轴，e_clamp = 0，
   填非零值会引入虚假的柔度修正，使校核偏保守或失准。

3. **e_clamp 与 e_load 混淆**：e_clamp 描述**夹紧几何**（螺栓到被夹件刚度中心），
   e_load 描述**载荷几何**（外力作用线到螺栓轴线）。两者方向不同、
   物理含义不同，必须分别量测和填写。
