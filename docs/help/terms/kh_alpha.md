# 横向载荷分配系数（KHα）

**一句话**：一对齿与多对齿交替啮合时，载荷沿齿廓方向（alpha 方向）分配不均造成的实际最大接触应力放大倍数。

**怎么理解**：

齿轮运转时重合度通常大于 1，即某些时刻有两对齿同时承载，另一些时刻只有一对齿承载。理想情况下两对齿平分载荷（各担一半）；但由于齿距偏差、齿廓偏差、弹性变形分担不均，实际上某一对齿会承担更大的份额。KHα 就是把"理论均分"修正为"实际最大承担"的放大系数。

直齿轮由于齿面瞬间啮合，KHα 波动小，通常取 1.0–1.2；斜齿轮由于齿线倾斜进入啮合，本身分配更平滑，但精度差时 KHα 反而可能偏大。齿面接触应力校核用 KHα（H 表示 Hertz/接触），齿根弯曲校核用 KFα（F 表示 foot/齿根），两者数值一般取同一档。

若采用 DIN 3990 C 法（简化），一般把 KHα ≈ KFα ≈ 1.0–1.2 作为工程经验值直接代入，不必精算；只在精度争议或失效分析时走 A/B 法。

**典型值**（DIN 3990-1 §7 / ISO 6336-1 §7）：

- 直齿轮 Q ≤ 6（高精度硬齿面）：KHα = 1.00
- 直齿轮 Q = 7–8（中等精度）：KHα = 1.00–1.10
- 直齿轮 Q ≥ 9（低精度）：KHα = 1.10–1.20
- 斜齿轮中等精度：KHα = 1.00–1.15
- 斜齿轮精度差 / 齿向修形不足：KHα 可达 1.20–1.40

选择依据：工业通用齿轮缺省 1.10；蜗轮单对齿啮合本质上 KHα = 1（DIN 3996 蜗轮负载能力校核通常不引入该系数）；精密减速机（机器人、航天）走 1.0，但需图纸明确精度。

**出处**：DIN 3990-1:1987 §7，ISO 6336-1:2019 §7
