# 最大过盈量（δ_max）

**一句话**：加工公差组合下**可能出现的最大过盈量**（直径方向，单位 um）；决定最大接触压力 p_max，从而决定是否触发轴 / 轮毂屈服。

**怎么理解**：

δ_max 对应**过盈最大的加工组合**——轴按上极限偏差（es 偏大）、孔按下极限偏差（EI 偏小）时得到的过盈量。这组数据在**应力方面最关键**：只要 δ_max 给出的压力 p_max 不让两侧屈服，整批产品都不会在装配时压坏。

## 公式（本工具实现）

```
基本关系（偏差换算模式）:
   δ_max = 轴上偏差 es − 孔下偏差 EI                [um, 直径方向]

粗糙度压平后的有效过盈:
   s = k · (Rz_shaft + Rz_hub)                      [um]
   δ_eff,max = max(0, δ_max − s)                    [um]

最大接触压力:
   p_max = (δ_eff,max / 2) / c_total · (1/1000)     [p: MPa, δ_eff: um, c_total: mm/MPa]

最大工况下的应力:
   σvm_shaft,max = p_max · VM_shaft                 [MPa]
   σvm_hub,max   = p_max · VM_hub                   [MPa]

安全系数:
   S_shaft = Re_shaft / σvm_shaft,max
   S_hub   = Re_hub   / σvm_hub,max
```

所有**应力类校核**（轴、轮毂屈服裕度）都基于 δ_max 工况。

## 输入约束（本工具实现）

```
δ_max > 0
δ_max ≥ δ_min                         上限必须 >= 下限
δ_eff,max > 0                         粗糙度压平后的最大过盈必须仍 > 0
```

## 最大过盈端会触发什么警报

本工具按以下优先级发出提示：

1. **轴侧应力安全系数不足**（S_shaft < S_σ,min）→ 降低 δ_max 或提高 Re_shaft
2. **轮毂应力安全系数不足**（S_hub < S_σ,min）→ 通常比轴侧先失效（轮毂材料 Re 一般更低）；增大 D 或换更高强度轮毂
3. **压入力过大**（F_press > 250 kN）→ 建议改为热装或液压装配

## 典型值（按直径参考）

| 配合直径 d | 轻度过盈（H7/p6） | 中度过盈（H7/s6） | 重度过盈（H7/u6） |
|---|---|---|---|
| 10 ~ 18 mm | δ_max ≈ 20 um | δ_max ≈ 27 um | δ_max ≈ 34 um |
| 18 ~ 30 mm | δ_max ≈ 29 um | δ_max ≈ 39 um | δ_max ≈ 49 um |
| 30 ~ 50 mm | δ_max ≈ 42 um | δ_max ≈ 59 um | δ_max ≈ 73 um |

（数据来源 ISO 286 优选配合；本工具内置表覆盖 6~50 mm。）

## 工程选择建议

1. **δ_max 尽量取在应力安全系数 S_σ ≥ S_σ,min 的边界以下**：过大会让最坏装配件压坏；过小会牺牲传递能力稳定性。
2. **轮毂通常是薄弱侧**：轮毂材料 Re 一般 < 轴材料 Re；设计 δ_max 时先算 S_hub，再算 S_shaft。
3. **带冲击或热变化工况**：δ_max 应更保守（留疲劳余量、热松弛余量）。

## 最大过盈端的其他后果

除屈服外，δ_max 还会增大：
- **压入力 F_press**：若压入力 > 250 kN，本工具提示考虑热装 / 液压。
- **热装所需轮毂温度**：shrink_fit 时 T_hub 按 δ_max 反算，若超轮毂允许温度上限会警告。
- **轴内壁应力**（空心轴）：本工具当前不校核内壁应力，但 δ_max 过大时内壁也可能接近屈服。

## 常见误用

- **以"图纸上标的最大过盈"为准，不看加工能力**：图纸上允许的上限不等于实际能做到的上限。加工能力决定散差窗口的宽度。
- **δ_max 填成 δ_min 两倍**：工程常见 δ_max / δ_min ≈ 1.5~2.0；过小的窗口不切实际，过大的窗口说明公差等级过宽。
- **仅看扭矩能力通过就接受 δ_max**：应力校核可能反而失败；必须两侧兼顾。
- **δ_max 超出优选配合表上限**：本工具内置表有限，超出请手工填入或切到"偏差换算"模式。
