# 压力角（α）

**一句话**：齿面啮合点上法向力方向与节圆切线的夹角，是齿轮几何的核心设计参数。

**怎么理解**：

压力角决定了齿廓的胖瘦与接触面承力方式。α 越大，齿根越宽厚，弯曲强度越高，且接触线更陡有利于抗点蚀；但同时径向分力（把轴往外推的力）增大，对轴承不利，也更容易咬住。α 越小则相反：齿细效率略高、径向分力小，但强度余量少。

20° 是目前工业界压倒性主流的选择，刀具、齿条、各类齿轮成品几乎全按 20° 生产，除非特殊场合否则不建议偏离。14.5° 是美式旧标准，如今仅在维修老设备时出现；25°（或 22.5°）主要用于飞机、汽车变速箱等高功率密度场合。

对蜗杆蜗轮，压力角通常指**法向压力角 αn**（沿螺旋齿法线方向测量），与直齿轮的含义略不同但工程用法一致。ZA/ZN/ZI/ZK 不同齿形蜗杆对应的 α 定义略有出入，但标称值都是 20°。

**公式**：径向分力 `Fr = Ft × tan α`（切向力 Ft 既定情况下）

**典型值**：

- 14.5°：旧式美国标准，仅老设备维修沿用
- 20°：现代国际通用标准，覆盖 90% 以上齿轮与蜗杆应用
- 22.5° / 25°：高负载齿轮（变速箱、航空）、小齿数齿轮避免根切
- 蜗杆蜗轮：标称 αn = 20°，ZI 齿形法向压力角就是 20°

除非客户图纸另有规定或配对齿已定，**默认选 20°**。

**出处**：GB/T 1356-2001、DIN 867:1986、ISO 53
