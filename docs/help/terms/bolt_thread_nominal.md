# 螺纹公称直径（d）

**一句话**：外螺纹沿螺纹牙顶量出的最大外径，是螺栓"M10"这种标称的那个数，标识一颗螺栓的"基本尺码"。

**怎么理解**：

d 是螺纹规格的**名片**。M10 = 公称直径 10 mm、粗牙螺距 1.5 mm；M10×1.25 = 同外径但细牙。选 d 就像选鞋码：大了撑不下、小了承载不够。

d 不直接参与大部分强度公式（那些用 As、d3、d2），但它决定：

- **应力截面积 As**：`As = π/4 · (d − 0.9382·p)²`，As 才是拉伸强度公式分母里用的
- **支承面积（螺栓头下方）**：与内径 DKm,i 配合决定 R7 支承面压强校核
- **螺纹脱扣内侧面积**：`A_SM = π · d · m_eff · C3`（R8 内螺纹剪切）
- **柔度自动计算**：螺栓刚度段按 d 折算

## 选 d 的决策思路

不是"越粗越安全"。过粗会：
- 需要更大的孔、更重、更贵
- 需要更大的扳手空间
- 拧紧扭矩 MA 随 d² 增大，工人拧不动或需要更大扳手力臂

工程经验（单颗 8.8 级螺栓承载力上限参考）：

- **M6**：轴向允许 ~3–5 kN
- **M8**：~6–10 kN
- **M10**：~10–18 kN
- **M12**：~16–28 kN
- **M16**：~30–50 kN
- **M20**：~50–80 kN
- **M24**：~80–120 kN
- **M30 及以上**：重型机械、压力容器

## 标准系列

本工具内置 GB/T 196 / ISO 724 粗牙与部分细牙：M3 / M4 / M5 / M6 / M8 / M10 / M12 / M14 / M16 / M18 / M20 / M22 / M24 / M27 / M30 / M33 / M36 / M39 / M42 / M48。

选下拉选项时 d/p/d2/d3/As 会自动按标准表填入；选"自定义"时需要自己输入 d 与 p，d2/d3/As 会按 ISO 898-1 公式派生。

## 典型值 / 常见选择

- 薄板 / 钣金 / 电器外壳：M3 – M6
- 通用工业设备 / 传动箱 / 法兰：M8 – M16（最主流）
- 重型设备 / 大型电机底座：M20 – M30
- 压力容器 / 核电 / 风电塔筒：M36+

## 常见误用

- **选粗牙还是细牙？** 同一规格 M12 有粗牙 p=1.75 和细牙 p=1.25 / 1.5 三种。**粗牙是默认选择**，装拆方便、容差大；细牙仅用于需要精细调整预紧（如轴承预紧螺母）或薄壁螺纹孔（增加啮合扣数）。
- **"M10" 与 "公称外径 10 mm" 是否等价？** 是，严格按 ISO 724 定义。实测牙顶直径会因加工公差略小（6g 公差约 9.78–9.97 mm），但 Rp0.2 / As 等计算仍按标称 d = 10 mm。

**出处**：GB/T 196-2003、ISO 724:1993（公制螺纹尺寸）、ISO 898-1:2013（螺栓尺寸与性能）
