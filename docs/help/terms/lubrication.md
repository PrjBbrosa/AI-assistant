# 润滑方式

**一句话**：齿轮、蜗轮、轴承等运动副在齿面之间建立油膜、带走热量的供油形式，直接决定磨损寿命与承载能力。

**怎么理解**：

润滑方式要解决两个问题：**在啮合处形成油膜**和**把摩擦生成的热带走**。低速场合热量少，润滑脂或浸油即可；高速场合表面温升快、油膜破坏风险高，必须用循环供油强制冷却。

蜗轮传动特别依赖润滑，因为蜗杆—蜗轮之间是**滑动接触**（不像齿轮的滚动+少量滑动），摩擦生热显著，若润滑不良会迅速胶合或磨损报废。选油品时除了黏度，还要关注 EP（极压）添加剂，工业蜗轮油通常选用极压合成油（如 PAG、PAO 基础油）。

一般按滑动速度 v（或蜗杆圆周速度）划档选用方式：

- **油浴润滑（splash）**：蜗杆浸入油池，转动时将油带到啮合区。结构最简单，用于低中速闭式传动
- **飞溅润滑**：靠齿轮/蜗杆旋转甩油到箱体再回流，适合中等速度
- **强制喷油（circulation）**：油泵把油通过喷嘴打到啮合区，同时经油冷器降温，用于高速大功率
- **脂润滑**：油脂填入啮合腔，低速、小功率、免维护场合

**典型值**（按蜗杆—蜗轮滑动速度 vs 或圆周速度参考）：

- vs < 1 m/s：脂润滑即可，常见于手动机构、电动阀门
- vs = 1–10 m/s：油浴润滑主流区间，涵盖绝大多数工业蜗轮减速机
- vs = 10–25 m/s：需飞溅+油池+冷却筋，或强制喷油
- vs > 25 m/s：必须强制喷油循环加外置油冷器

选油黏度经验：低速重载选高黏度（ISO VG 460–680），高速轻载选中等黏度（VG 220–320）；工业蜗轮优先选 PAG 合成蜗轮油。
