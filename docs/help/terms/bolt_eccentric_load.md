# 载荷偏心 e_load

**一句话**：外部轴向载荷 FA 作用线到螺栓轴线的距离；决定螺栓附加弯矩 MB = FA · e_load。

**怎么理解**：

e_load = 0 意味着外力对中——载荷沿螺栓轴线作用，螺栓只受纯拉伸，无弯曲附加。

e_load ≠ 0 时，载荷相对螺栓偏移，产生附加弯矩进入螺栓应力组合：

- 小偏心（e_load < 0.5d）：弯曲附加可忽略，通常含在安全系数里
- 中等偏心（0.5d–2d）：需显式校核螺栓弯曲应力
- 大偏心（> 2d）：弯矩主导，单螺栓抗弯能力有限，宜考虑多螺栓或改变连接方式

注意：e_load 关注的是**外力施加点**相对螺栓轴线的偏移，
与 e_clamp（夹紧几何偏心）是两个独立的几何量。

## 如何量

在受力图上：

1. 确定外部载荷的**合力作用线**（均布载荷取几何重心线，集中力取作用点）
2. 量该作用线到**螺栓轴线**的垂直距离
3. 该距离即为 e_load（单位 mm）

均布载荷的有效作用点是载荷区域的几何中心；若载荷非均匀分布，
应用力矩等效原理求合力作用点。

## 影响公式（VDI 2230 参考）

VDI 2230 §5.1.3 给出载荷偏心引起的附加弯矩及弯曲应力：

```
附加弯矩：MB = FA · e_load               [N·mm]

螺栓弯曲应力：σb = MB / WS              [MPa]
其中 WS = π · d3³ / 32                  [mm³]
  d3 = 螺栓小径（应力截面直径）

组合应力（拉伸 + 弯曲）：
σ_total = σ_axial + σb ≤ Rp0.2 / S_F
```

弯曲应力 σb 直接叠加到轴向拉伸应力上，增大断裂风险。

> **本工具当前版本未实现此校核**：`e_load` 字段目前仅作设计记录用途，不参与弯矩或组合应力的自动计算。若连接存在明显载荷偏心，需工程师单独按上式计算 MB 和 σb，并手动叠加到工具输出的轴向应力结果上进行综合判断。

## 典型工程场景

| 场景 | e_load 典型值 |
|---|---|
| 中心受压短柱（螺栓在形心） | 0 mm |
| 标准法兰密封（均布内压） | ≈ 0（内压均匀作用） |
| 偏载法兰（载荷偏向一侧） | ~ 法兰偏心臂长 |
| 悬臂梁根部固定（螺栓在支座） | ~ 支座到加载点全距 |
| 角撑连接（单侧受力） | ~ 受力臂 |

悬臂场景是 e_load 最大的典型案例——螺栓在根部，载荷在悬臂端，
e_load 等于悬臂臂长，产生很大弯矩。

## 与 e_clamp 独立性

| 参数 | 几何意义 | 影响对象 |
|---|---|---|
| e_clamp | 螺栓轴线到被夹件刚度中心的距离 | 被夹件等效柔度 δP、力比 Φ |
| e_load | 外力作用线到螺栓轴线的距离 | 螺栓附加弯矩 MB、弯曲应力 σb |

两者几何方向不同，物理机制不同，必须分别填写。
e_clamp = 0 不代表 e_load = 0，反之亦然——它们是独立的设计参数。

## 常见误用

1. **e_load 与 e_clamp 互换**：两者名字相近但意义完全不同。
   e_clamp 是夹持几何偏心（影响柔度），e_load 是载荷位置偏心（影响弯矩），
   绝不能混用或相互替代。

2. **均布载荷不求重心直接取边缘距离**：均布载荷的合力作用点是分布区域中心，
   不是边缘。用边缘距离会高估 e_load 导致过度设计。

3. **对中设计忘记检查实际载荷路径**：名义上对中的连接，若安装偏差或
   制造公差导致实际载荷偏移，e_load 不为零。重要连接应核查最大可能偏心量。
