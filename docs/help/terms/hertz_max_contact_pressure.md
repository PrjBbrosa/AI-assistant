# 最大接触应力 p0

**一句话**：接触区中心处的峰值法向压强，是赫兹校核的核心输出；p0 一旦超过材料接触疲劳极限就会点蚀剥落。

**怎么理解**：

接触区内压力分布不是均匀的。在线接触上压力沿宽度方向呈**半椭圆**（垂直于接触线方向）分布，中心高两边低；在点接触里压力呈**半球形**（沿接触圆径向）分布，中心最高边缘为零。接触区中心的峰值压强即 p0，也就是工程师最关心的"最大接触应力"。

关键特征：
- p0 是**压应力**（单位 MPa 或 N/mm²）；工程上数值很大（碳钢齿轮齿面 p0 常在 1000~2000 MPa 量级，轴承球可达 2000~3500 MPa）
- p0 远大于接触区的平均压强 p_mean：线接触 p0 = (4/π) × p_mean ≈ 1.27 × p_mean；点接触 p0 = 1.5 × p_mean
- p0 **不是** σ_Tresca / σ_Mises；接触区内的最大剪应力实际位于次表面（深度 ≈ 0.47·b 或 0.48·a），该次表面剪应力才是点蚀寿命的实际起源点。本工具仅给 p0，不算次表面剪应力。

**公式**（两种模式 p0 的直接表达式）：

```
线接触：p0 = 2·q / (π·b)          [q: N/mm, b: mm → p0: MPa]
        其中 q = F / L            [F: N, L: mm → q: N/mm]
              b = √(4·q·R' / (π·E'))

点接触：p0 = 3·F / (2·π·a²)       [F: N, a: mm → p0: MPa]
        其中 a = ((3·F·R') / (4·E'))^(1/3)
```

代入化简后的等价表达式（便于敏感度分析）：

```
线接触：p0 = √(q·E' / (π·R'))      [q: N/mm, E': MPa, R': mm → p0: MPa]
点接触：p0 = (1 / π) × ((6·F·E'²) / R'²)^(1/3)
```

从这两个形式可以看出：
- **线接触**：p0 与 √q、√E' 成正比，与 √R' 成反比 → 载荷翻倍 p0 增 41%、E' 翻倍 p0 增 41%、R' 翻倍 p0 降 29%
- **点接触**：p0 与 F^(1/3)、E'^(2/3) 成正比，与 R'^(2/3) 成反比 → 载荷翻倍 p0 增 26%、E' 翻倍 p0 增 59%、R' 翻倍 p0 降 37%

**本工具实现**：`core/hertz/calculator.py:138-148` 计算 p0 并写入 `result["contact"]["p0_mpa"]`。后续 `safety_factor = allowable_p0 / p0`，`checks.contact_stress_ok = (p0 <= allowable_p0)`。

**典型值**（工程经验 p0 允许范围）：

| 接触副类型 | 常见 p0 范围 (MPa) | 备注 |
| ---------- | ------------------ | ---- |
| 碳钢齿轮齿面（正火） | 800~1300 | 接触疲劳极限约 900 MPa |
| 合金钢齿轮齿面（调质） | 1100~1600 | 接触疲劳极限约 1200 MPa |
| 表面淬火齿轮（渗碳） | 1500~2500 | 接触疲劳极限约 1500–1650 MPa |
| 滚动轴承钢（GCr15） | 2000~4000 | 短期峰值允许；长期寿命需低于 3000 |
| 塑料齿轮（PA66 vs 钢） | 50~100 | 塑料接触疲劳极限很低 |

**常见坑**：

- **用 p0 和材料抗拉强度 σ_b 比较**：p0 是接触压应力（三向应力），σ_b 是单向拉伸极限；两者不能直接比。应与接触疲劳许用值 [p0] 比较。
- **忽略 p0 循环次数修正**：许用值通常对应 10^7 循环；长寿命应用需乘寿命系数 < 1（从许用值角度看更严格）。
- **把 p_mean 当 p0 填到下游**：p0 / p_mean 比例已解释；切勿混用。
- **用"传感器测到的平均压强"作 p0**：压敏纸 / 压敏膜 / 测力传感器给的是整体压强或等效平均；p0 需要通过赫兹公式从几何与载荷反算，或用微探针 / 光弹 / 数字图像相关法直接测。

**出处**：Hertz 接触理论（Johnson 《Contact Mechanics》）；各工程手册接触应力章节。
