# 载荷导入系数（n）

**一句话**：外载作用点在"螺栓-被夹件"这个弹性组合体上的相对位置修正系数（0.2–1.0），决定外力对螺栓多少"有效传递"，直接乘入载荷因数 Φ_N。

**怎么理解**：

想象一颗螺栓把两块厚法兰夹紧。现在有一股外力 FA 要"撬开"这两块法兰：

- **情况 1**：外力直接作用在螺栓头下方的法兰外缘 → 几乎全部 FA 拉向螺栓 → **n ≈ 1.0**
- **情况 2**：外力作用在两块法兰内部（靠近螺栓中段）→ 被夹件自身能"消化"一部分外力，剩下的才到螺栓 → **n ≈ 0.5**
- **情况 3**：外力通过薄壳分布导入（容器罐壁） → 被夹件体系分担更多 → **n 可低至 0.2**

n 的物理效应：

```
Φ_N = n · δp / (δs + δp)         # 载荷因数
            ↑ n 的位置
n 越小 → Φ_N 越小 → 螺栓分担外力越少 → 服役应力低、疲劳安全
```

## 取值指南（VDI 2230 §5.5.1）

| 外力作用几何 | n 取值 |
|---|---|
| 外力作用于螺栓头 / 螺母正下方（紧贴） | 1.0 |
| 外力作用于被夹件表面（离螺栓头不远） | 0.7–0.9 |
| 外力作用于被夹件中部（lK 中线附近） | 0.5 |
| 外力通过薄壳 / 法兰外缘传导 | 0.3–0.5 |
| 外力通过壳体广泛分布导入 | 0.2 |

**工程经验**：**不确定时取 n = 1.0**（本工具默认）。保守路线 —— 认为螺栓承担全部外力，服役应力与疲劳取最严校核。

## n 与 Φ_N 的区别

**n 是输入，Φ_N 是输出**。常被新手混淆：

- 你填 n（在 UI 里）→ 工具算 Φ_N = n · δp / (δs + δp)
- Φ_N 不是手动填，是计算结果显示

## n 选小的代价

看起来 n 小是好事（螺栓轻松），但：

```
R3 残余夹紧力 = FM,min − FZ − Fth − (1−Φ_N)·FA,max
                                      ↑ (1−Φ_N) 增大
Φ_N 小 → (1−Φ_N) 大 → R3 要求的 FM,min 更大
```

也就是 **n 小让螺栓服役应力下降，但要求装配预紧力更高**。装配阶段压力增大 → R4/R7 更紧张。

**规律**：n 是一种"再分配"，不改变总外载，只改变螺栓 vs 被夹件的分配比。选对 n 的收益不是"凭空减载"，而是让几何更贴近真实。

## 工程实践

- **法兰密封连接**：通常 n = 0.7–0.9（密封力在螺栓头附近）
- **机械底座连接**：通常 n = 0.5（外力从机械体内传来）
- **厚壁压力容器**：通常 n = 1.0（内压直接顶螺栓头）
- **薄壁容器 / 壳体**：通常 n = 0.3–0.5（外力分布导入）

具体工程无现成公式，需要：

1. 画连接的"力流图"，判断外力入口位置
2. 量化入口到螺栓头的"距离比例" e / lK
3. 查 VDI 2230 表 5.5/3 或附录图表

本工具**不内置 n 查表**，需要用户根据经验填入。

## 常见误用

- **把 n 设为 0**：意味着"螺栓完全不受外力"，数学上 Φ_N = 0，外力全给被夹件。物理上不可能，除非外力真的作用在被夹件内部最理想位置。
- **n 和 Φ_N 搞反**：填的是 n（几何量），Φ_N 是计算结果。
- **复杂连接单独填 n**：一个连接可能有多个载荷路径（静载 + 冲击 + 热），不同路径 n 可能不同；工具目前只支持单一 n。若需分别处理，手动取保守的最大 n。
- **在薄壳场合取 n = 1**：保守，但可能过度设计。壳体工程建议按 VDI 2230 图表查更精确的 n（0.3–0.5）。
