# 支承面材料（bearing_material）

**一句话**：螺栓头/螺母下方被接触件的材料，选定后自动填入许用支承面压强 p_G,allow；本工具预设钢 700 MPa、铝合金 300 MPa，其余材料用"自定义"手动输入。

**怎么理解**：

支承面校核（R7）检验的不是螺栓强度，而是**被夹件支承区域的抗压强度**：拧紧螺栓时，螺栓头或螺母将压力集中传递到接触环形面，若局部压应力超过材料抗压极限，接触区会发生永久塑性压陷（"塌陷"），导致预紧力下降。

```
p_G = FM,max / A_bearing <= p_G,allow    [MPa]
```

p_G,allow 取决于**被接触件**的材料硬度和强度，而非螺栓材料。

## 下拉选项与预设 p_G,allow

| 材料 | p_G,allow [MPa]（自动填入值）|
|---|---|
| 钢 | 700 |
| 铝合金 | 300 |
| 自定义 | 手动输入 |

这两个预设值来自本工具 `BEARING_MATERIAL_PRESETS` 字典，属于工程简化值，适用于通用结构钢和普通铝合金型材的初筛估算。

## 扩展参考（需选"自定义"）

以下材料不在下拉预设中，需选"自定义"手动输入：

| 材料 | 常用 p_G,allow 参考值 [MPa] | 来源说明 |
|---|---|---|
| 铸铁（灰口 HT200）| ~550 | 按 Rm/硬度推算，应查原始标准 |
| 高强钢（Rm ≥ 800 MPa）| ~1000–1200 | VDI 2230 Table A9，视等级而定 |
| 淬火钢（HRC > 50）| ~1500–1800 | 表面强化后，须图纸确认 |
| 6061/6082 铝合金（T6 态）| ~350–400 | 热处理提升后稍高于普通铝预设 |

注意：上表为参考区间，精确值须查 VDI 2230:2015 Table A9 或材料手册，按实际热处理状态填入。

## 选定后联动 p_G_allow 字段

选择支承面材料后：

- `p_G_allow` 字段自动填入对应推荐值，但字段**仍可编辑**；用户可直接覆盖，无需切换到"自定义"
- 选"自定义"时字段内容清空，光标自动跳入，方便直接输入——适用于表面硬化、镀层或图纸指定值等特殊情况
- 更换材料后字段立即刷新

## 支承面面积计算

p_G 中的 A_bearing 由螺栓头下支承环面决定：

```
A_bearing = pi/4 * (DKm,o^2 - DKm,i^2)    [mm^2]
```

DKm,o 为支承外径（通常取螺栓头对边宽约 1.5d），DKm,i 为螺栓通孔直径（约 1.1d）。面积参数在支承面章节的 `bearing_d_inner` / `bearing_d_outer` 字段填写。

## 常见误用

- **用螺栓 Rp0.2 估算 p_G,allow**：p_G,allow 是被接触件参数，用螺栓屈服强度代替会高估允许压强（铝件配高强螺栓时，错误倍数可达 2–4×）。
- **铝合金热处理状态不确认就用默认 300 MPa**：T6 处理的铝合金允许压强可高于预设值，若追求精度应选"自定义"并填入更准确的值。
- **换用软垫圈后未更新 bearing_material**：加铝垫圈后，有效支承件变为垫圈材料，p_G,allow 应随之降低。
- **高温工况忽略 p_G 衰减**：铝合金在 150 °C 以上屈服强度显著下降，应按高温修正值填入，否则热态可能发生支承面塌陷。

**出处**：VDI 2230:2015 §5.5.1（支承面压强校核）及 Table A9（p_G,allow 参考值）；GB/T 3098.1（螺栓机械性能）
