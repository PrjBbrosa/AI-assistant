# 齿顶圆与齿根圆（d_a1 / d_a2 / d_f1）

**一句话**：花键齿在径向上的三个边界圆——外花键齿顶圆 d_a1、内花键齿顶圆 d_a2、外花键齿根圆 d_f1——决定有效齿高 h_w、平均直径 d_m，是场景 A 齿面压力公式的直接几何输入。

**怎么理解**：

外花键（轴）剖面上，一颗齿在径向从 d_f1（齿根）延伸到 d_a1（齿顶）。内花键（轮毂）齿顶圆 d_a2 比 d_a1 小一点（否则轴花键的齿顶会顶到轮毂花键的齿顶，两者咬死）。在 DIN 5480 的典型几何里：

```
d (参考) > d_a1 (外花键齿顶) > d_a2 (内花键齿顶) > d_f1 (外花键齿根)
```

这个顺序约束在 `core/spline/geometry.py:60-64` 里显式校验，不满足就抛 `GeometryError`。

**三个量怎么组合成 h_w 与 d_m**：

```
h_w = (d_a1 - d_a2) / 2        [有效齿高，单侧]
d_m = (d_a1 + d_a2) / 2        [接触区平均直径]
```

- **h_w**：外花键齿顶与内花键齿顶的**半径差**。这是齿真正"顶着传扭矩"的那一段径向高度；齿根段（d_f1 到 d_a2 之间）其实没有配对面接触，不承载。因此 h_w 而不是 "齿全高" 才进入齿面压力公式。
- **d_m**：接触区的平均半径 × 2，作为 `T / d_m` 算切向力的基准。

**d_f1 有什么用**：工具把它作为**几何顺序校验的一部分**（d_f1 < d_a2 保证内花键齿顶够得着外花键齿根）。d_f1 本身不直接进入齿面压力计算，但会影响：
- 轴剖面的抗扭刚度（齿根越深，轴芯越细，抗扭刚度越低）。
- 轴剖面的应力集中（齿根圆角是应力集中区）——但本工具未校核齿根弯曲强度。

**典型值（DIN 5480 30° 系列 catalog 实例）**：

| 规格 | d_B [mm] | d_a1 [mm] | d_a2 [mm] | d_f1 [mm] | h_w [mm] | h_w/m |
|---|---|---|---|---|---|---|
| W 15×0.8×17 | 15.0 | 14.36 | 12.84 | 12.56 | 0.76 | 0.95 |
| W 15×1.25×10 | 15.0 | 14.75 | 12.5 | 12.1 | 1.125 | 0.90 |
| W 20×2×8 | 20.0 | 18.6 | 16.5 | 15.6 | 1.05 | 0.525 |
| W 25×2×11 | 25.0 | 24.5 | 21.0 | 20.0 | 1.75 | 0.875 |
| W 25×2.5×8 (0.5m 下限实例) | 25.0 | 23.75 | 21.25 | 20.375 | 1.25 | 0.5 |

从表中可见，**h_w/m 的正常范围**大致在 0.5~1.0 之间，主流条目在 0.85~1.0；真正贴近 0.5 下限的是 `W 25×2.5×8` 这类条目（见 `core/spline/din5480_table.py:98-100`）。本工具近似模式取的 0.5 是 **catalog 保守下限**（`core/spline/geometry.py:110-111` 的 `h_w = (d_a1 - d_a2)/2` 派生，此处 `d_a1 = d - 0.5m`、`d_a2 = d - 1.5m` 刚好给出 h_w = 0.5m）。

**本工具实现 **：
- `公开/图纸尺寸`模式下四个值全给，工具只做顺序校验和 m·z ↔ d 一致性比对。
- `近似推导`模式下只给 m × z，工具按 `d_a1 = d - 0.5m`、`d_a2 = d - 1.5m`、`d_f1 = d - 2.0m` 派生，**h_w 固定 0.5·m**。

**常见误用**：

- **把齿全高 (d_a1 - d_f1)/2 当 h_w**：这是"齿从根到顶的总高度"，包括非接触的齿根部分。h_w 只算 d_a1 与 d_a2 之间那一段。
- **d_a1 与 d_a2 填反**：d_a1 是**外花键**齿顶（应该更大），d_a2 是**内花键**齿顶（应该更小）。填反会导致 h_w 为负，工具抛 `GeometryError`。
- **d_a1 填成 d_B**：新手偶尔以为"齿顶就在参考圆上"。实际 d_a1 < d_B，因为 DIN 5480 的齿顶做了 0.5·m 左右的齿顶降低（避免齿顶过尖）。

**出处**：DIN 5480-1:2006（几何定义）、DIN 5480-2:2015（catalog 数值）。
