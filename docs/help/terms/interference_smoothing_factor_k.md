# 压平系数 k

**一句话**：过盈配合中将表面粗糙度换算为**过盈量损失** `s = k · (Rz_s + Rz_h)` 的无量纲系数；k 越大，扣减的有效过盈越多。

**怎么理解**：

配合面装配时，两侧的微观凸峰（高度为 Rz）互相接触并发生塑性压平。**压平的净效应**等于"某一比例 k 的凸峰被压掉"，这一比例写作无量纲 k。

物理上 k 的取值依赖：
- 材料硬度比：软硬差异大时凸峰更容易被塑性压平，k 偏高
- 加工工艺：车削刀纹 vs 磨削纹理 → 微观几何差异
- 装配速度：动态装配比静态压入的瞬时接触压力更高，压平更完全
- 润滑状态：有装配油膜时摩擦低，压平更"顺滑"

DIN 7190 标准基于试验给出了两个版本的默认值：
- **DIN 7190-1:2017**：k = 0.4（基于新试验数据）
- **DIN 7190:2001**：k = 0.8（老标准）

## 公式（本工具实现）

```
粗糙度压平量:
   s = k · (Rz_shaft + Rz_hub)                    [um, k: 无量纲]

有效过盈量:
   δ_eff = max(0, δ − s)                          [um]

接触压力:
   p = (δ_eff / 2) / c_total · (1/1000)           [MPa, δ_eff: um, c_total: mm/MPa]
```

```
校验规则（本工具）:
   k > 0                 不允许 0 或负
   k ≤ 1.0               大于 1 时报错（建议值）
   δ_eff,max > 0         粗糙度压平后的最大过盈必须仍 > 0
                         （否则报错：过盈量不足以抵消表面压平）
```

## 典型值

| k 值 | 来源 / 工况 | 工程含义 |
|---|---|---|
| **0.4** | DIN 7190-1:2017（新版本默认） | 现行标准，本工具默认 |
| **0.6** | 中间值 / 企业规范 | 比新版保守但比老版乐观 |
| **0.8** | DIN 7190:2001（老版本默认） | 历史图纸维修时使用 |
| **0.5** | 工程保守估算 | 不依赖特定版本的中值 |
| 0 | 忽略压平效应（不推荐） | 明显高估压力 |
| > 1.0 | 异常取值（不允许） | 物理上 k 不应超过 1 |

**本工具内置选择器**通过 `roughness.profile` 切换：新版 / 老版 / 自定义。选新版或老版时 k 自动填入并锁定；选自定义时解锁。

## 为什么 k 不该超过 1

k = 1 意味着"表面凸峰全部被压平"；但实际 Rz 既包含锐峰也包含缓坡波纹，不可能全部塑性压平。k > 1 不符合物理意义，本工具按 `> 1 报错` 校验。

## 工程选择建议

1. **新设计默认 k = 0.4**（DIN 7190-1:2017）
2. **关键件偏保守**：k = 0.5 ~ 0.6，留一些对试验散差的余量
3. **维修 / 与老设计对齐**：按图纸当时使用的标准版本选
4. **实测数据**：若本企业有 μ/s 的实测标定，可用"自定义 k"填入

## 常见误用

- **k = 1.0**：物理上不成立，工程上对应"表面镜面 + 极高装配压力"的理论极限，与实际差距大。
- **k = 0**：等于忽略粗糙度压平，让压力估算虚高 20~40%。
- **与 Rz / Ra 混淆**：k 是系数（无量纲），Rz 是粗糙度数值（um）；填错字段会导致数量级偏差。
- **用 Ra 当 Rz 填但保留 k=0.4**：等于把 s 估低 4~6 倍，压力虚高更严重。
- **忽略材料硬度比**：纯钢-钢和钢-软铝在同一 Rz 下实际压平程度不同；本工具单值 k 做统一近似。
