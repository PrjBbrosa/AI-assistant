# 优选配合代号（H7/p6, H7/s6, H7/u6 等）

**一句话**：ISO 286 体系中常用的"孔基制"过盈配合代号；字母表示公差位置（s/u 为过盈、p 为过盈偏小），数字表示公差等级（6 为标准精度）；本工具内置有限子集用于快速反推过盈窗口。

**怎么理解**：

ISO 286 把轴孔公差分为两部分：
- **字母**（H / h / s / p / u / ...）→ 公差带相对基本尺寸的**位置**
- **数字**（5 / 6 / 7 / 8 / ...）→ 公差带的**宽度**（IT 级，数字越大公差越宽）

约定：大写字母是孔、小写字母是轴。"孔基制"约定孔用 H（下偏差为 0），轴带各种字母形成不同配合。

## 本工具内置的三种配合（本工具实现）

| 代号 | 过盈强度 | 工程语境 |
|---|---|---|
| **H7/p6** | 轻度过盈 | 可拆装、低扭矩传递、轴承内圈固定（有时） |
| **H7/s6** | 中度过盈 | 齿轮毂、皮带轮、一般传动的工程默认 |
| **H7/u6** | 重度过盈 | 高扭矩传递、热装工艺、大扭矩毂 |

**只覆盖直径 6~50 mm**；超出范围需切到"直接输入过盈量"或"偏差换算"。

## 配合代号的典型过盈窗口（本工具内置表子集）

按 ISO 286 子集表，d=30~50 mm 段示例：

| 代号 | 轴偏差 (es / ei) | 孔偏差 (ES / EI) | δ_min / δ_max |
|---|---|---|---|
| H7/p6 | +35 / +22 um | +25 / 0 um | 22 − 25 = **−3** 不合格 ❗ |
| H7/s6 | +48 / +35 um | +25 / 0 um | 35 − 25 = **10** / 48 − 0 = **48** |
| H7/u6 | +61 / +48 um | +25 / 0 um | 48 − 25 = **23** / 61 − 0 = **61** |

（数据摘自工具内置 `_PREFERRED_FIT_TABLE`；实际 H7/p6 在 30-50 段已接近过渡，大直径下常不是纯过盈配合；本工具用表值直接换算，不额外告警。）

## 公式（本工具实现）

```
δ_min = shaft_lower − hole_upper             [um, 直径方向]
δ_max = shaft_upper − hole_lower             [um, 直径方向]

查表逻辑:
   按 (fit_name, nominal_diameter_mm) 查表
   若 d 不在内置直径段，返回 (0, 0) 并附带 warning
   若 fit_name 未知，抛 InputError
```

## 为什么只内置三种代号

工程上常用的过盈配合代号还有 p7、r6、t6、n6 等，但本工具当前只覆盖最核心的三种。理由：
1. **覆盖 80% 工程场景**：普通过盈多用 H7/s6；轻负载 H7/p6；重载 H7/u6
2. **保持 UI 简洁**：下拉选项过多会增加新手误选
3. **兼容 ISO 286**：三种代号的表值在 ISO 286-2 公开摘录中查到

其他代号可通过"偏差换算"模式手工输入偏差数值实现。

## 工程选择建议

1. **常规传动先试 H7/s6**：工程默认中度过盈，能覆盖大部分场景。
2. **冲击 / 大扭矩上 H7/u6**：但要特别校核 δ_max 下的应力（容易超屈服）。
3. **轻扭矩 / 可拆装用 H7/p6**：注意小直径段可能是过渡配合，不稳定。
4. **精度要求高的主轴用更窄 IT 级**：如 H6/p5、H6/s5；本工具不覆盖，走偏差换算。

## 常见误用

- **误以为所有直径段都是过盈配合**：H7/p6 在大直径段（30~50 mm）可能出现 δ_min 接近零甚至过渡；必须看真实表值。
- **忽略直径超范围告警**：直径 100 mm 选 H7/s6 时，本工具会返回 (0, 0) 并保留该值，导致校核全失败；此时应切模式手填真实值。
- **混淆 "孔基制"（H 优先）与 "轴基制"（h 优先）**：本工具默认孔基制；JIS / 某些行业标准用轴基制，代号写法不同。
- **把 IT 级误写成小数**：IT6 不是 0.6、0.06；直接选代号 h6、p6 即可。

## 出处

- ISO 286-1 / ISO 286-2 公差与配合（优选配合表）
- GB/T 1800（等同 ISO 286）
- 本工具内置 `core/interference/fit_selection.py`（ISO 286 curated preferred-fit subset，直径 6~50 mm）
