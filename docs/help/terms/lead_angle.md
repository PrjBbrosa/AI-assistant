# 导程角（γ）

**一句话**：蜗杆每转一圈螺旋上升的角度，决定蜗杆"陡峭程度"与传动效率。

**怎么理解**：

γ 越大，蜗杆螺旋越陡，推动蜗轮越省力，传动效率越高；但同时自锁能力下降（蜗轮反向也能推蜗杆）。γ 越小（< 5°），蜗杆接近水平螺纹，蜗轮几乎推不动蜗杆回转，即"自锁"，适合需要防倒转的场合，如电动窗帘、吊装提升机、大减速比低速传动。

多头蜗杆（z1 = 2–4）一般走大导程角路线换效率；单头蜗杆（z1 = 1）则为自锁与大传动比设计。导程角不是独立输入，通常由 z1 和 q 直接算出：`tan γ = z1 / q`。

注意：自锁不是绝对保险。振动、冲击、长期润滑下都可能破坏自锁条件，安全制动场合仍应加机械锁止或制动器。

**公式**：`tan γ = z1 / q`

其中 z1 为蜗杆头数，q 为直径系数（= d1 / m）。

**典型值**：

- 单头蜗杆（z1 = 1）：γ = 3°–6°，自锁主导，用于需要防倒转与大减速比
- 两头蜗杆（z1 = 2）：γ = 8°–12°，自锁与效率折中
- 多头蜗杆（z1 = 3–4）：γ = 15°–25°，效率主导，不自锁，用于电机—减速机输出级
