# 连接形式（DSV / ESV）

**一句话**：DSV（Durchsteckverbindung）= 通孔 + 螺母夹紧；ESV（Einschraubverbindung）= 螺栓直接拧入基体螺纹孔；两者柔度模型不同，ESV 通常需填写 m_eff 以激活 R8 螺纹脱扣校核（DSV 可跳过）。

**怎么理解**：

DSV 中，螺栓穿过被夹件通孔，另一端用螺母锁紧。螺栓头下和螺母下各有一个支承面，扭紧时两端都产生支承扭矩 M_K。

ESV 中，螺栓直接拧入基体（铸件、厚板、机体），基体本身充当螺母角色。只有螺栓头端有支承面，螺纹旋合长度由基体材料和有效啮合深度 m_eff 决定。

两种形式对校核流程的影响如下：

```
DSV（通孔连接）:
  支承扭矩 M_K  = 0.5 · FM · mu_K · DKm          # 仅螺栓头端
  螺纹扭矩 M_G  = FM · d2/2 · tan(phi + rho)
  总拧紧扭矩 M_A = M_G + M_K                       # 一端支承

ESV（螺纹孔连接）:
  支承扭矩 M_K = 0.5 · FM · mu_K · DKm            # 同上，仍仅螺栓头
  总拧紧扭矩 M_A = M_G + M_K                       # 同 DSV
  额外要求：基体螺纹脱扣 R8 校核 → m_eff 不得过短
```

> 注意：两种形式支承扭矩公式形式相同，核心差异在于 ESV 必须做 R8，而 DSV 的螺母脱扣一般由螺母标准保证（不单独校核）。

## 如何选型

选型核心判据：

| 判据 | 推荐形式 |
|---|---|
| 基体厚度 ≥ 1.0d（钢）或 ≥ 1.5d（铝/铸铁） | ESV 可用 |
| 需要频繁拆装（>1000 次）| DSV（避免 ESV 螺纹孔磨损）|
| 空间限制，一侧无法放螺母 | ESV 必选 |
| 两侧均可操作，载荷较重 | DSV 更可靠（螺母承载面更大）|
| 基体为铝，动载振动 | DSV + 防松螺母（ESV 铝攻牙易松动）|

基体最小厚度建议（VDI 2230 §R8 推论）：

- **钢基体**：m_eff ≥ 1.0d（即基体厚 ≥ d）
- **铸铁基体**：m_eff ≥ 1.25d
- **铝基体**：m_eff ≥ 1.5d（推荐 2.0d，若动载）

不满足上述最小啮合深度时，R8 螺纹脱扣可能不通过，应改用加长螺纹孔、螺纹镶套（E-螺套）或改为 DSV。

## 对校核的影响

joint_type 会在校核流程以下几个分支中产生不同行为：

1. **支承扭矩 M_K**：两种形式的计算公式形式相同（都只有螺栓头端支承），不因 joint_type 分叉。历史上有版本把 DSV 算两端，属于误解。

2. **R8 螺纹脱扣校核**：
   - 无论 DSV/ESV，R8 **仅在用户填写 `m_eff`（有效啮合长度）时激活**；`m_eff` 留空则 R8 标记为"未校核"。
   - ESV 通常需要填写 m_eff 并执行 R8（基体螺纹孔深度有限，脱扣风险真实存在）。
   - DSV 的螺母强度由螺母标准保证，通常无需额外做 R8，可留空 m_eff 跳过该项。

3. **有效螺纹啮合长度 m_eff 的定义**：
   - DSV：m_eff 为螺母的有效高度，标准螺母已规定
   - ESV：m_eff 为基体螺纹孔深度减去 1–2 牙（首尾不完整牙折减）

4. **柔度模型中的螺纹旋合段**：
   - ESV 的有效螺栓长度取 `lK + 0.4·d + 0.33·d`；DSV 取 `lK + 0.4·d + 0.4·d`（均由工具内部自动处理，见 `bolt_clamp_length_lk`）。

## 常见误用

- **ESV 忘做 R8**：认为"拧进去了就不会脱扣"——实际薄壁铝件 ESV 的螺纹脱扣是最常见的失效模式之一。
- **DSV 两端各算一次支承扭矩**：高估 M_A 约 2 倍，导致拧紧扭矩过大，甚至超出螺栓屈服。VDI 2230 DSV 中 M_K 只算螺栓头端（螺母端由操作工固定，不产生额外摩擦扭矩传入螺栓）。
- **ESV 螺纹孔深不够就填 m_eff = 全孔深**：孔底 1–2 牙不完整，应折减；精确数值按 VDI 2230 §R8 中的 D_eff 公式计算。

**出处**：VDI 2230:2015 §3.2（连接形式定义）；VDI 2230:2015 §R8（螺纹脱扣校核）；GB/T 16823.1（螺纹连接的基本要求）
