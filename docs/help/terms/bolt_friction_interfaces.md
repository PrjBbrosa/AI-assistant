# 摩擦面数 qF

**一句话**：横向载荷 FQ 传递路径上由螺栓压紧的被夹件间摩擦界面数量。

**怎么理解**：

防滑校核的核心逻辑是：工具计算维持连接不滑移所需的最小残余夹紧力：

```
F_slip_required = FQ_max / (μT · qF)      [N]
```

qF 是其中的关键除子——每增加一个摩擦界面，所需夹紧力就线性减少一份，即防滑能力线性增加。
**但计数有严格限制**：必须是螺栓压紧力直接作用的 part–part 界面，
螺栓头下的支承面和螺母下的支承面**不算**摩擦面。

## 怎么数

统计被螺栓夹紧的"零件-零件"接触面数量：

| 场景 | qF |
|---|---|
| 2 件直接对夹（A 和 B） | 1（A-B 界面） |
| 3 件叠（A-B-C） | 2（A-B + B-C） |
| 4 件叠（A-B-C-D） | 3（A-B + B-C + C-D） |
| 垫圈夹在中间（算独立层） | 每加一层 qF+1 |

通用规律：**qF = 被夹件层数 − 1**（按力流路径数）。

## 与 μT 的关系

- **qF** 是几何量：连接的层叠数决定，与材料无关
- **μT** 是物理量：被夹件材料与表面状态决定，与层叠数无关

两者**独立输入**。当各层界面的 μT 不同时，保守做法是取所有界面中最小的 μT：

```
μT_effective = min(μT_界面1, μT_界面2, ..., μT_界面N)
```

不能对各界面 μT 取平均——一旦最差界面先滑动，整体防滑即失效。

## 防滑校核公式

本工具按 VDI 2230 §5.5.4 计算防滑所需最小残余夹紧力：

```
F_slip_required = FQ_max / (μT · qF)      [N]
```

其中：
- `FQ_max`：最大横向载荷（设计值）
- `μT`：接触界面摩擦系数（取最差界面）
- `qF`：摩擦界面数（本字段）

工具将 `F_slip_required` 与 `FK,R`（残余夹紧力）对比，FK,R ≥ F_slip_required 则防滑通过。工具不内置 S_T 安全系数乘子；如需留有安全裕量，工程师可在 FQ_max 输入中自行乘以安全系数后再填入。

## 常见误用

1. **把螺栓头下支承面算成摩擦面**：螺栓头支承面承受轴向力，不传递横向力 FQ，
   不计入 qF。同理螺母下支承面也不算。

2. **3 件叠填 qF = 1**：3 件叠有 2 个 part–part 界面（A-B 和 B-C），应填 qF = 2；
   填 1 会低估防滑容量 50%，导致过度设计或错误判断。

3. **不同界面 μT 差异大时取平均**：正确做法是取最小值（最差界面）。
   若一个界面是涂漆面（μT ≈ 0.07）、另一个是干燥钢面（μT ≈ 0.15），
   应按 μT = 0.07 校核，而非取 0.11 的平均。

**出处**：VDI 2230:2015 §5.5.4 Eq. 5.45
