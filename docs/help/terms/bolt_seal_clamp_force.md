# 密封所需残余夹紧力（FK,req）

**一句话**：为了保证连接在工作状态下密封、防滑、定位等功能，螺栓至少需要维持的最低剩余压紧力；R3 残余夹紧力校核的判据。

**怎么理解**：

螺栓装配后，被夹件之间相互压紧。外力 FA 作用时，一部分预紧力会被"抵消"，剩下的**残余夹紧力** FK,work 维持接触面压力。只有当 FK,work ≥ FK,req 时：

- 密封垫片保持接触压强 → 不漏
- 接触面保持摩擦力 → 不滑移
- 被夹件之间无缝隙 → 定位稳定

公式（本工具实现）：

```
FK,work = FM,min − FZ − Fth − (1−ΦN)·FA,max      [N]
R3 通过: FK,work ≥ FK,req
```

## FK,req 来自哪里

**不是你随便估的数字**，而是根据连接功能具体算或实测：

### 1. 密封功能

```
FK,req,seal = p_seal × A_seal         [N]   (p_seal: MPa, A_seal: mm²)
```

- p_seal（密封接触压强）= 密封垫片要求的最小压强（厂家样本）
- A_seal（密封垫片面积）= 垫片轴向投影面积

典型值：
- **橡胶 O 形圈**：p_seal ≈ 2–4 MPa（弹性变形靠形状）
- **软石棉 / 纤维垫**：p_seal ≈ 10–20 MPa
- **金属缠绕垫**：p_seal ≈ 25–40 MPa
- **金属实心垫（Ring Type Joint）**：p_seal ≈ 60–100 MPa

例：DN100 法兰用纤维垫（Dout 125, Din 100），A_seal = π/4·(125²−100²) ≈ 4420 mm²，p_seal = 15 MPa → FK,req,seal ≈ 66 kN（这是**整个法兰**所有螺栓要合力提供的）。

### 2. 防滑功能

```
FK,req,slip = FQ,max / (qF · μT)       [N]
```

- qF = 摩擦界面数（通常 1 或 2）
- μT = 防滑摩擦系数（钢-钢干 0.15–0.20，钢-钢油 0.10–0.12）
- 若连接同时需要密封 + 防滑，取 **max** 不加和

### 3. 定位 / 刚度功能

按连接刚度要求估算。一般比密封 / 防滑的要求小，工程上常取：

```
FK,req,position ≈ 0.25 · FM,min        # 25% 装配预紧力作为"维持接触"底线
```

本工具**不自动算 FK,req**，需要用户填入。

## 填 0 的含义

**FK,req = 0**：不要求任何残余夹紧力。物理含义：允许连接"刚好不分开"。

工程实际：FK,req = 0 很少见，因为：
- 大多数连接有某种密封或防滑要求
- 即使没有，残余夹紧力 = 0 意味着接触面可能有微动 → 长期疲劳失效

**工程建议**：无特殊密封 / 防滑需求时，仍取 FK,req ≥ 0.25 · FM,min 作为兜底（防微动、防松）。

## 典型值

| 应用场景 | FK,req 典型（每颗螺栓） |
|---|---|
| 家电 / 电器外壳（无密封） | 100–500 N |
| 发动机油底壳（低压油密封） | 1–3 kN |
| 燃气管道法兰（低压） | 5–15 kN |
| 液压管路法兰（高压） | 20–80 kN |
| 压力容器（高压，大直径） | 50–200 kN |
| 风电塔筒法兰（防滑 + 密封） | 100–500 kN |

## 设计模式 vs 校核模式

- **设计模式**（本工具默认）：你填 FK,req，工具反推 FM,min。R3 自动满足（设计保证）。
- **校核模式**：你填 FM,min（已装配），工具算 FK,work。再把 FK,work 与 FK,req 比较得出 R3 PASS/FAIL。

## 常见误用

- **FK,req = 0 但实际需要密封**：R3 会 PASS 但密封漏，工具无法识别这种语义错误。
- **按"整个法兰所需"填每颗螺栓的 FK,req**：DN100 法兰 66 kN 总密封力，若用 8 颗 M12 → 每颗 FK,req = 66/8 ≈ 8 kN，而不是 66 kN。
- **密封 + 防滑取和**：取 max 不取和。同一螺栓同时提供摩擦和密封，同样的压紧力服务两个目的。
- **忘了乘以动载 / 安全系数**：密封垫片典型按 2× 设计余量；防滑按 qF·μT 折算时 μT 取偏低保守值。
