# 脱扣安全系数要求 S_BS,req

**一句话**：R8 螺纹脱扣校核的目标安全系数；实际 S_BS = F_B / F_max 必须 ≥ S_BS,req。

**怎么理解**：

螺纹脱扣是"弱螺纹剥离"或"强螺纹拉断"的边界校核。内螺纹（螺纹孔）和外螺纹（螺栓）的剪切强度 tau_BM / tau_BS 与有效啮合长度 m_eff 组合决定极限承载力 F_B；S_BS,req 是工程师在设计时留的裕度。

与常规强度安全系数不同，S_BS,req 的选择要考虑材料塑性、螺纹加工质量和后果严重性。软基体（铝合金、铸铁）螺纹孔脱扣风险远高于钢质螺纹孔，需取偏保守值或延长 m_eff。

## 典型取值

| 应用等级 | S_BS,req 参考 |
|---|---|
| 常规机械连接 | 1.25 |
| 关键连接（压力容器、结构件）| 1.5 |
| 高可靠场景（航空、核工业）| 2.0 |
| 临时/试验工装 | 1.1 |

## 与 m_eff 的联动

若计算所得 S_BS < S_BS,req，软件会建议以下改进方向：

- **延长 m_eff**：增加螺纹旋合长度（常见措施：使用螺纹嵌套 Helicoil，或加深螺纹孔）。
- **换高级螺栓**：提高 tau_BS（螺栓抗剪强度），适用于螺栓拉断先于孔脱扣的情形。
- **换更硬基体**：提高 tau_BM（内螺纹材料抗剪强度），适用于软金属螺孔脱扣的情形。

## 校核公式（VDI 2230 §5.5.5）

本工具按如下公式计算（`core/bolt/calculator.py` R8 段）：

```
外螺纹（螺栓侧）剪切面积：A_SB = pi * d3 * m_eff * C1       [mm^2]
  C1 = 0.75（ISO 公制螺纹啮合修正系数，可覆盖）
  d3 = 螺栓小径（应力截面当量直径）

内螺纹（螺母/壳体侧）剪切面积：A_SM = pi * d * m_eff * C3   [mm^2]
  C3 = 0.58（内螺纹参与承载的有效比例，可覆盖）
  d  = 螺纹公称直径

外螺纹承载力：F_strip_bolt = A_SB * tau_BS
内螺纹承载力：F_strip_nut  = A_SM * tau_BM
  tau_BS = 螺栓材料剪切强度（默认 Rp0.2 * 0.6，可覆盖）
  tau_BM = 内螺纹材料剪切强度（必须填写，无默认值）

F_B = min(F_strip_bolt, F_strip_nut)    # 取先脱扣的一侧
S_BS = F_B / F_bolt_max                 # F_bolt_max 为工况最大螺栓拉力
需 S_BS >= S_BS,req（默认 1.25）
```

## 常见误用

- **默认 1.0 等同无裕度**：S_BS = 1.0 意味着恰好在极限载荷下脱扣，任何实际散差（材料批次、加工误差）均可能导致失效；通常最低取 1.25。
- **ESV 忽略螺纹孔脱扣，只看螺栓拉断**：单纯校核 AS × Rm 不等于 R8；孔脱扣（tau_BM 控制）在软基体中往往比螺栓拉断先发生。
- **临时工装用常规值过设计**：试验工装寿命短、人员监控严格，可适当放宽至 1.1；但切勿将试验件标准沿用至量产件。

**出处**：VDI 2230:2015 §5.5.5; DIN EN ISO 898-2
