# 配合直径（d）

**一句话**：过盈配合中轴与轮毂**公称接触直径**；决定接触面积 π·d·L、接触半径 d/2，并影响所有容量与应力换算。

**怎么理解**：

d 是过盈连接的"基准尺寸"——所有几何因子、接触面积、扭矩力臂都从它派生。配合直径变化 1 倍，接触面积也变 1 倍（线性），但**扭矩力臂还会再变 1 倍**，所以扭矩容量随 d 的增大近似按 d² 增长（固定 L 时）。

在三种尺寸里容易搞混的差别：
- **公称直径 d**（本字段）：图纸标注的"轴 Ø40"这类名义尺寸
- 轴实际直径 d_shaft（= d + es~ei 范围内）
- 孔实际直径 d_hub_bore（= d + ES~EI 范围内）

本工具不关心装配后的瞬时值，只要过盈量窗口 (δ_min, δ_max) 给出，d 就按**名义尺寸**参与几何与柔度计算。

## 公式（本工具实现）

```
接触半径:   r = d / 2                                    [mm]
接触面积:   A = π · d · L                                [mm²]
轮毂厚壁因子: K_hub = (D² + d²) / (D² − d²)              [无量纲]

柔度（径向变形 / MPa 接触压力）:
  c_shaft = (d/2) / E_s · (1 − ν_s) · F_hollow(d_i/d, ν_s)   [mm/MPa]
  c_hub   = (d/2) / E_h · (K_hub + ν_h)                       [mm/MPa]

扭矩容量:
  T_cap = μ_T · p · A · r / 1000
        = μ_T · p · π · d · L · (d/2) / 1000              [T: N·m, p: MPa, d,L: mm]
```

注意 T_cap 里出现了两次 d（一次在 A 里、一次在 r 里），所以**扭矩容量随 d 按二次方增长**。

## 典型值（过盈配合常见适用范围）

| 连接场景 | d 典型范围 | 说明 |
|---|---|---|
| 小传动件（齿轮、皮带轮、凸轮） | 10 ~ 50 mm | 过盈配合经典范围 |
| 中等传动（中小型电机轴、泵轴） | 40 ~ 150 mm | 过盈 + 键槽常见 |
| 大机械（轧机辊、大齿轮毂） | 150 ~ 600 mm | 常用热装 |
| 超大直径（舰船主轴、水轮机转子） | 600 mm+ | 特殊工艺，超出本工具适用范围 |

**本工具优选配合表**只内置 6 ~ 50 mm；超出范围请切到"直接输入过盈量"或"偏差换算"模式。

## 常见误用

- **填半径 = 20 当 d = 20**：半径 20 对应的是 d = 40；容易把图纸上的半径误当直径。
- **把装配后的实际尺寸填进去**：应填**公称**直径。过盈量 δ 已经独立地捕捉了实际尺寸偏差。
- **用英制直径**：本工具以 mm 为单位，1 inch = 25.4 mm；英制图纸需先换算。
- **d 小于 6 mm**：过盈配合在小直径上很难加工出稳定的过盈量；实际工艺建议 d ≥ 6 mm。
