# 光滑段材料屈服强度（Re_s / Re_h）

**一句话**：联合模式下场景 B 屈服校核用的轴 / 轮毂材料屈服极限（Rp0.2 或 σ_s），单位 MPa；与 von Mises 等效应力相比得到屈服安全系数。下拉选常用材料时自动填充。

**怎么理解**：

过盈配合压装后轴承受径向压缩（切向压应力）和轴向压缩，轮毂承受切向拉应力（箍一圈的圆环）和径向压应力。合成的 von Mises 等效应力 σ_vm 必须低于材料屈服极限 Re：

```
stress_sf = Re / sigma_vm           [MPa / MPa = 无量纲]
PASS 条件：stress_sf >= S_sigma,min
```

轴与轮毂两边用各自材料的 Re 分别校核；工具里两者都使用同一个 S_sigma,min 门槛。

**Re 是什么**：

- **Rp0.2（塑性变形 0.2% 时的应力）**：常用于没有明显屈服台阶的调质钢 / 合金钢。工程上视为"屈服强度"。
- **σ_s**（明显屈服台阶的下屈服强度）：低碳钢 / 退火钢。
- 热处理状态会显著改变 Re：45 钢正火 Re ≈ 355 MPa、调质 Re ≈ 600 MPa、淬火 + 低回火 Re ≈ 800 MPa。

**工具特别约定**：`_on_material_changed` 里有一段 mode-aware 逻辑——`仅花键`模式下即便选`自定义`，E / ν / Re 仍然保持 `AutoCalcCard` 锁定（因为整个 smooth_* 段对仅花键模式不可用）；切到`联合`时才解锁。这保证 "mode 是权威" 的不变量（见 `spline_fit_page.py:644-670`）。

**典型值（按 Re 从低到高）**：

- **低碳钢（Q235、10 钢）**：Re ≈ 180~235 MPa。很少做花键轴，但轮毂可能用。
- **中碳钢正火（45 钢正火）**：Re ≈ 355 MPa。常规轮毂。
- **中碳钢调质（45 钢调质）**：Re ≈ 600 MPa。常规花键轴。
- **低合金调质（40Cr / 42CrMo 调质）**：Re ≈ 785~930 MPa。关键花键轴 / 重载连接。
- **高合金渗碳淬火（20CrMnTi / 20CrMo）**：表面 Re 接近抗拉极限 Rm（1100~1400 MPa），但心部 Re 仍约 600 MPa。过盈校核应按**心部 Re**取，因为过盈压力深入轴芯。

**常见误用**：

- **用 Rm（抗拉极限）代替 Re**：Rm 比 Re 高 25~50%。用 Rm 会让 stress_sf 虚高一截，实际裕度不足。
- **渗碳件取"表面 Re"**：渗碳层只有 0.5~2 mm，过盈应力分布到整个轴芯；应取心部 Re。
- **不同批次、不同热处理共用一个 Re**：热处理不稳定的情况下 Re 波动 ±15%。取名义值时留足 S_sigma,min 余量（建议 ≥ 1.3）。
- **把 Re 填成 E（弹性模量）**：E ≈ 210 000 MPa（钢），Re 在 200~1000 MPa 之间。填错会让屈服 SF 虚高 100+ 倍，结果异常。

**出处**：GB/T 699-2015（优质碳素钢）、GB/T 3077-2015（合金结构钢）、ISO 898-1（类比螺栓钢性能）。

> Cannot verify against original standard —— 材料库的 Re 下限按公开标准整理，实际批次值以材料证明 / 入厂复验为准。
