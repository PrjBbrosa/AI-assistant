# 第 n 层被夹件厚度（layer_t_n）

**一句话**：多层夹紧时第 n 层的轴向厚度，各层 t_i 求和得总夹紧长度 lK，直接影响整体柔度链。

**怎么理解**：

被夹件可由多层不同材料叠合而成（例如底座 + 法兰 + 垫圈）。每层独立参与柔度串联：

```
delta_i = t_i / (E_i * A_i)        [mm/N，第 i 层压缩柔度]
delta_p = sum(delta_i)             [总被夹件柔度]
```

- 厚度越大，该层柔度越大（越"软"），承载相同外力时压缩变形越多。
- 若各层材料弹性模量 E 差距显著（如钢 E=210000 MPa、铝 E=70000 MPa），不能用平均值代替，必须逐层独立计算；否则会低估铝层柔度约 3×，进而高估 lK 等效刚度。

## 量法

- 从装配图轴向尺寸标注直接读出；
- **不含**螺栓头高度、螺母高度、螺纹啮合段（这些属于螺栓柔度 delta_s 分段）；
- 垫圈、间隔套、衬套按独立层处理（填独立厚度 + 材料），而非并入相邻层；
- 若法兰端面有倒角/斜面，取净轴向接触厚度（垂直于螺栓轴向的投影值）。

## 与 total_thickness 的关系

工具提供两种模式：
- **单层模式**（`part_count=1`）：填写 `total_thickness` 单值，走等效单层圆柱体模型，layer_t_n 不显示；
- **多层模式**（`part_count>=2`）：显示逐层 t_i 输入，工具内部将各层求和作为 total_thickness，无单独的 total_thickness 输入字段。

两种模式互斥，不存在"同时填写 total_thickness 与各层 t_i 并自动比对"的逻辑。工具不会对 sum(t_i) 与 total_thickness 做差值警告。建议工程师在多层模式下人工核对 sum(t_i) 是否等于实际夹紧长度，避免漏填某一层。

## 常见误用

- **把螺栓穿透段长度当作 lK**：螺栓总长 L_bolt 包含螺栓头高、螺纹外露段；lK 只是被夹件夹紧段净长。
- **漏填垫圈层**：垫圈（弹簧垫、平垫）若刚度低，会显著提升 delta_p，导致 Phi_N 被低估，计算偏危险但实际连接更松。
- **ESV 连接把螺纹咬合段（m_eff）也算入厚度**：螺纹咬合段属于螺栓柔度分段，已单独处理，勿重复计入 layer_t。
