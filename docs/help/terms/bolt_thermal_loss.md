# 热预紧力损失（Fth）

**一句话**：螺栓与被夹件在服役温度下热膨胀不一致，导致预紧力相对装配时增加或减少的那一部分；本模块按 VDI 2230 §5.4.1 温差公式自动估算。

**怎么理解**：

装配在室温（通常 20°C）完成。设备运行后温度升至 T_work。如果螺栓与被夹件**热膨胀系数不同**，或**两者温度不同步**，弹性几何就"装配时自然长度" vs "服役时自然长度"错配。这会让预紧力变化。

极端例子：
- **钢螺栓 + 铝被夹件**，全部升到 80°C：
  - 铝被夹件"想膨胀" 更多（α_Al ≈ 23×10⁻⁶ /K）
  - 钢螺栓"只膨胀一点"（α_steel ≈ 11.5×10⁻⁶ /K）
  - 被夹件推开螺栓头 → 螺栓被"拉得更长" → 预紧力**增加**
  - 反过来降到 −20°C，铝收缩得多 → 螺栓被压缩 → 预紧力**下降**

## 公式骨架（本工具实现）

基于 VDI 2230 §5.4.1 温差公式：

```
ΔL_thermal = [α_bolt · (T_bolt − T_ref) − α_parts · (T_parts − T_ref)] · l_K   [mm]
                                                                              # l_K: 总夹紧长 [mm]
Fth = ΔL_thermal / (δs + δp)                                                   [N]
                                                                              # δs,δp: mm/N
```

符号含义：
- **α_bolt**（螺栓热膨胀系数，1/K）：钢 11.5×10⁻⁶，不锈钢 16×10⁻⁶，自定义可手填
- **α_parts**（被夹件热膨胀系数，1/K）：按被夹件材料表（钢 11.5e−6 / 铝 23e−6 / 铸铁 10.5e−6 / 不锈钢 16e−6）
- **T_bolt, T_parts**（温度，°C）：工作温度；若螺栓与被夹件温度差异大（如排气法兰），分别填入
- **T_ref**：装配参考温度（本工具默认 20°C）

**注意符号**：ΔL > 0 表示热装后被夹件更长，预紧力**增加**；ΔL < 0 表示预紧力**下降**。Fth 在本工具中记为**损失**（降低预紧力），所以：

```
Fth = max(0, −ΔL_thermal / (δs + δp))    # 仅计负向（损失）部分
```

升温且膨胀系数一致时 Fth ≈ 0。

## 本工具自动估算

在工况章节填入螺栓/被夹件材料 + 温度后，工具会：

1. 查表填入 α_bolt、α_parts
2. 按上述公式计算 ΔL
3. 回填到装配章节的 Fth 字段

**若你在装配章节手动填 Fth > 0**：手动值优先，自动估算被覆盖。

## 典型场景

| 场景 | α_bolt | α_parts | ΔT | Fth 方向 |
|---|---|---|---|---|
| 全钢连接 / 等温 | 11.5e−6 | 11.5e−6 | 0 | 0（无损失） |
| 全钢连接 + 降温 80°C | 11.5e−6 | 11.5e−6 | −80 | 0（材料相同，消差） |
| 钢螺栓 + 铝被夹件 + 升温 100°C | 11.5e−6 | 23.0e−6 | +100 | **正**（预紧力增加，Fth = 0） |
| 钢螺栓 + 铝被夹件 + 降温 40°C | 11.5e−6 | 23.0e−6 | −40 | **负**（被夹件更收缩，预紧力下降） |
| 不锈钢螺栓 + 铸铁被夹件 + 升温 200°C | 16e−6 | 10.5e−6 | +200 | **负**（螺栓膨胀更多 → 预紧力下降） |

## 警告阈值

本模块 R10 热损失校核判据：

```
thermal_loss_ratio = Fth / FM,min
R10 通过: thermal_loss_ratio ≤ 0.25     # 热损失不超过 25% 装配预紧力
```

超过 25% 说明材料热膨胀组合不合理，工程上应：
- 换用热膨胀系数相近的材料组合
- 预紧时预留"热余量"（FM,min 加大）
- 使用带碟簧的螺栓（吸收温差变形）

## 常见误用

- **铝螺栓 + 钢被夹件**：α_bolt 大于 α_parts，升温时螺栓膨胀多 → 预紧力损失大，不建议的材料组合。
- **把 Fth 手动填 0 忽略**：高温 / 大温差场景下这会让 R3 虚高（"以为预紧力够"）。即使不做自动估算，也应人工按温差估一个保守值。
- **不同温度但填同温**：排气法兰典型"螺栓 150°C + 被夹件 200°C"，不是"一起 200°C"。分别填入两个温度。
