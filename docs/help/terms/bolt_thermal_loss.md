# 热预紧力损失（Fth）

**一句话**：螺栓与被夹件在服役温度下热膨胀不一致，导致预紧力相对装配时流失的那一部分；本模块按 VDI 2230 温差公式自动估算（取绝对值，仅计损失）。

**怎么理解**：

装配在室温（通常 20°C）完成。设备运行后温度升至 T_work。如果螺栓与被夹件**热膨胀系数不同**，或**两者温度不同步**，装配时的弹性几何与服役时"自由长度"错配，预紧力会随之改变。物理上这种改变**可正可负**（既可能增加、也可能损失预紧力），取决于 α 差与温差的组合方向。

## 本模块如何处理（重要：单向损失近似）

本模块**不对预紧力做有向建模**。计算 `core/bolt/calculator.py:337-391` 始终取 `abs(Δα · ΔT) · (c_s·c_p)/(c_s+c_p) · l_K` 的绝对值，把结果都当作 **损失**（记为正的 Fth 扣到 FM,min 上）。原因：

- VDI 2230 R3 残余夹紧力校核必须按"最坏情况"设计。哪怕实际工况下温差会**提升**预紧力，设计师也不应该依赖这一提升来满足 FM,min
- 反之，若温差真的让预紧力上升，R4 装配应力会自动承受那一部分 —— 但设计端只需保证 R3 的最低线

这意味着工具**会低估**"正向"温差场景的安全裕度（温差实际有利时被当作有害），但**不会低估**损失场景的风险。保守方向。

## 公式骨架（本工具实现）

```
Δα = α_bolt − α_parts                              [1/K]
ΔT = T_bolt − T_parts                              [K]
c_s = 1/δs, c_p = 1/δp                             [N/mm]
Fth = |Δα · ΔT| · (c_s · c_p)/(c_s + c_p) · l_K    [N]
                                                   # l_K: 总夹紧长 mm
```

符号含义：
- **α_bolt**（螺栓热膨胀系数，1/K）：钢 11.5×10⁻⁶，不锈钢 16×10⁻⁶，自定义可手填
- **α_parts**（被夹件热膨胀系数，1/K）：按被夹件材料表（钢 11.5e−6 / 铝 23e−6 / 铸铁 10.5e−6 / 不锈钢 16e−6）
- **T_bolt, T_parts**（温度，°C）：工作温度；若螺栓与被夹件温度差异大（如排气法兰），分别填入
- **Δα, ΔT**：差值，**取绝对值**再乘刚度 / 长度

## 本工具自动估算

在工况章节填入螺栓/被夹件材料 + 温度后，工具会：

1. 查表填入 α_bolt、α_parts
2. 按上述公式计算 |Δα·ΔT| · 刚度项 · lK
3. 回填到装配章节的 Fth 字段（始终 ≥ 0）

**若你在装配章节手动填 Fth > 0**：手动值优先，自动估算被覆盖。

## 典型场景（本模块输出均为 Fth ≥ 0）

| 场景 | α_bolt | α_parts | ΔT = T_bolt − T_parts | 本模块 Fth | 物理实际方向 |
|---|---|---|---|---|---|
| 全钢连接 / 等温 | 11.5e−6 | 11.5e−6 | 0 | 0 | 无变化 |
| 全钢连接 + 全升 / 全降温 | 11.5e−6 | 11.5e−6 | 0 | 0 | 无变化（Δα = 0） |
| 钢螺栓 + 铝被夹件，螺栓比被夹件 **热 80K** | 11.5e−6 | 23.0e−6 | +80 | 正（大） | 螺栓伸长多 → 预紧力 **下降**，模型与物理一致 |
| 钢螺栓 + 铝被夹件，全部 **同升** 80K | 11.5e−6 | 23.0e−6 | 0 | **0** | 物理上预紧力会**增加**（铝胀更多），但本模型按 ΔT 差计算 → **低估裕度**（保守） |
| 钢螺栓 + 铸铁 + 升温，螺栓比铸铁 **冷 50K** | 11.5e−6 | 10.5e−6 | −50 | 正 | 本模型按 \|Δα·ΔT\| 计损，一律扣 |

注意：**本模型只捕捉两者温度不同步带来的损失**。若 T_bolt = T_parts 但 α 不同（同升同降），本模型返回 Fth = 0，忽略物理上的预紧力变化。若你的工况是"螺栓和被夹件同升但材料不同"且预紧力变化显著，必须手动估算 Fth 或走 FEA。

## 警告阈值

本模块 R10 热损失校核判据：

```
thermal_loss_ratio = Fth / FM,min
R10 通过: thermal_loss_ratio ≤ 0.25     # 热损失不超过 25% 装配预紧力
```

超过 25% 说明材料热膨胀组合不合理，工程上应：
- 换用热膨胀系数相近的材料组合
- 预紧时预留"热余量"（FM,min 加大）
- 使用带碟簧的螺栓（吸收温差变形）

## 常见误用

- **铝螺栓 + 钢被夹件**：α_bolt 大于 α_parts，升温时螺栓膨胀多 → 预紧力损失大，不建议的材料组合。
- **把 Fth 手动填 0 忽略**：高温 / 大温差场景下这会让 R3 虚高（"以为预紧力够"）。即使不做自动估算，也应人工按温差估一个保守值。
- **不同温度但填同温**：排气法兰典型"螺栓 150°C + 被夹件 200°C"，不是"一起 200°C"。分别填入两个温度。
