# 允许最大接触应力 [p0]

**一句话**：用户按材料接触疲劳极限、寿命、润滑状态等**自行折算**得出的允许值，工具用它做 PASS/FAIL 判据。

**怎么理解**：

本工具的赫兹校核只有一条硬判据：

```
p0 ≤ [p0]  →  PASS
p0 >  [p0] →  FAIL
```

其中 p0 由赫兹公式计算，[p0] 由用户输入。这里必须强调：**本工具不自动从材料牌号查 [p0]，也不做寿命折减 / 润滑修正**。用户必须自己把下列因素算进去后填入最终许用值：

- 材料接触疲劳极限 σH,lim（来自材料手册 / 齿轮规范）
- 寿命系数 ZN（循环次数 ≠ 10^7 时的折算）
- 润滑系数 ZL（油膜厚度修正）
- 粗糙度系数 ZR（表面处理影响）
- 尺寸系数 ZX（齿尺寸 / 滚子尺寸影响）
- 安全系数下限 SH（工况可靠性要求）

**符号说明**（ISO 6336-2 体系的 Z 因子家族）：
- σH,lim — 材料接触疲劳极限（MPa），基础值
- ZN — 寿命修正因子（通常 0.85 ~ 1.1）
- ZL — 润滑油膜修正因子（通常 0.9 ~ 1.0）
- ZR — 表面粗糙度修正因子（通常 0.9 ~ 1.0）
- ZX — 尺寸修正因子（通常 0.9 ~ 1.0）
- SH — 许用安全系数（通常 1.0 ~ 1.3）

完整表达（如需 ISO 6336 齿面接触）：
```
[p0] = σH,lim × ZN × ZL × ZR × ZX / SH    [MPa]
```

实际用本工具时若只做快速估算，可直接取表格值：

| 材料 / 工况 | 建议 [p0] (MPa) | 说明 |
| ----------- | --------------- | ---- |
| 碳钢正火（45 钢、20 钢） | 400~600 | 未热处理普通钢件 |
| 调质合金钢（42CrMo、40Cr） | 700~1100 | 常规齿轮 / 轴 |
| 表面淬火 / 渗碳硬化钢（齿面） | 1200~1650 | 渗碳 HRC58-62 的齿面 |
| 滚动轴承 GCr15 淬火硬化 | 2500~3000 | 轴承短期峰值允许 |
| 灰铸铁 HT250（齿面） | 400~600 | 铸铁齿轮 |
| 青铜 / 黄铜（蜗轮齿面） | 200~400 | 青铜蜗轮接触疲劳低 |
| 工程塑料 PA66（干态 23°C） | 40~80 | 塑料接触疲劳极低 |

**默认值**：工具 UI 默认 [p0] = 1500 MPa，对应调质合金钢齿轮齿面的中间值，**并非"合理缺省"**，一定按实际材料手册重填。

**本工具实现**：

- `core/hertz/calculator.py:120-123` 用 `_positive` 校验必须 > 0（未填入则走默认 1500）
- `calculator.py:150` 计算安全系数 `safety_factor = allowable_p0 / p0`
- `calculator.py:151` 布尔判据 `pass_contact = p0 <= allowable_p0`
- `calculator.py:175-176` 安全系数 < 1.2 时追加 warning："接触应力安全系数偏低..."

**常见坑**：

- **误把材料屈服强度 Rp0.2 当接触许用值填**：碳钢 Rp0.2 ≈ 350 MPa，接触许用值可达 1000+ MPa；两者量级不同不能互换。
- **长寿命场合不做 ZN 折减**：设计 10^8 循环以上时要把手册给的 σH,lim 乘 ZN（< 1），否则允许值偏乐观。
- **润滑恶劣时不做 ZL 折减**：脂润滑 / 稀油混用 / 边界润滑场合 ZL 可能取到 0.85，而不是 1.0。
- **一颗螺栓 / 一个接触点重复使用同一 [p0]**：本工具每次运行一个接触副；多处接触应分别按各自的工作载荷、材料、寿命分别校核。
