# 服役屈服安全系数（S_F）

**一句话**：服役阶段等效应力 σ_vm,work 与屈服强度 Rp0.2 之间希望保留的裕量倍数，典型 1.1–1.3，值越大越保守。

**怎么理解**：

R5 服役等效应力校核：

```
σ_vm,work = √(σ_ax,work² + 3·(k_τ·τ_asm)²)       [MPa]
σ_allow,work = Rp0.2 / S_F                        [MPa]
R5 通过: σ_vm,work ≤ σ_allow,work
```

S_F 是设计师预留的裕量：

- S_F = 1.0 → 允许服役应力达到屈服（零裕量，不推荐）
- S_F = 1.1（本工具默认）→ 留 10% 裕量
- S_F = 1.2–1.3 → 中等保守
- S_F ≥ 1.5 → 高度保守（疲劳敏感、不可检修、极端工况）

## 与装配利用系数 ν 的关系

ν 和 S_F 是**不同阶段的独立参数**：

- **ν**（装配）：装配允许应力 = ν · Rp0.2（ν 典型 0.9）
- **S_F**（服役）：服役允许应力 = Rp0.2 / S_F（S_F 典型 1.1）

两者数值的相对大小：

- 装配允许 = 0.9 · 640 = **576 MPa**（M10 8.8）
- 服役允许 = 640 / 1.1 ≈ **582 MPa**

装配与服役允许值相近，这是合理的 —— 服役时应力 = 装配应力 + ΦN·FA 贡献，工程目标是"两者都不超过屈服的 91%"。

## 取值指南

| 连接性质 | S_F 推荐 |
|---|---|
| 普通工业连接 | **1.1**（本工具默认） |
| 重要设备连接 | 1.2 |
| 密封关键（压力容器、管道）| 1.3 |
| 疲劳敏感 / 高循环 | 1.3–1.5 |
| 不可检修（卫星、植入医疗） | 1.5–2.0 |
| 法规强制（核电、航空） | 按法规（常 1.5–2.5） |

## S_F 高的代价

R5 校核越严 → 允许服役应力越低 → 需要：

1. **更粗的螺栓**（同外力下 As 更大，σ_ax 更低）
2. **更高的强度等级**（Rp0.2 更大，分子更大）
3. **或减小 Φ_N**（让螺栓少分担外力）

不能无底线提高 S_F，否则设计成本飙升。工程实践：先按 S_F = 1.1 基线设计，若关键程度要求提升再调到 1.2 / 1.3。

## 与其他安全系数的区别

本模块里有多个"安全系数"，各自角色不同：

- **S_F**：**服役屈服安全**（Rp0.2 与 σ_vm,work）
- **S_strip**：**螺纹脱扣安全**（螺纹剪切承载力与螺栓拉力比），典型 ≥ 1.25
- **Goodman 折减系数**：**疲劳**内部的平均应力修正，不是用户输入
- **αA**：**装配散差**（FM,max / FM,min），不是裕量而是工艺波动

不要混淆它们。S_F 只影响 R5。

## 常见误用

- **把 S_F 当"安全系数 k"直接乘到外载**：不是这样用。S_F 是除数，作用于 Rp0.2 分母。
- **S_F = 1.0 且期望 R5 通过**：等于要求服役应力精确等于屈服值。数学上可能通过，物理上已经临界 —— 任何工艺波动都会失败。
- **S_F 很高但 FA 还是过大**：再大 S_F 都不能把 σ_ax,work = FA/As 关进允许值。真正的解法是减小 FA 或加大 As（更粗螺栓 / 多螺栓并联）。
