# 支承面摩擦系数（μK）

**一句话**：螺栓头（或螺母）压向被夹件表面时，二者之间的滑动摩擦系数，决定拧紧扭矩 MA 中有多少比例消耗在"克服支承面摩擦"上（约 40–50%）。

**怎么理解**：

拧螺栓时，螺栓头下方与工件接触面之间会相对转动——"螺栓头转了一圈，工件没动"。这个滑动界面的摩擦吃掉约 **40–50% 的扭矩**。摩擦面积大（支承直径大），吃的扭矩就多。

μK 进入扭矩公式的最后一项：

```
MA = FM · (0.5·d2·(P/(π·d2) + 1.155·μG) + 0.5·DKm·μK)     [N·mm]
                                              ↑ 支承面贡献
DKm = (DKm,i + DKm,o) / 2    # 有效支承面直径 [mm]
```

μK 的波动（0.10 vs 0.20）同样可让同一 MA 的 FM 差出 ±20%。

## 典型值（VDI 2230 摩擦参考值节选）

| 界面状态 | μK 典型范围 |
|---|---|
| **钢-钢，干净无油** | 0.12–0.18 |
| **钢-钢，装配油** | 0.08–0.14 |
| **钢-铸铁** | 0.10–0.16 |
| **钢-铝** | 0.10–0.14（铝易粘着，警惕咬合） |
| **镀锌螺栓-钢** | 0.14–0.22（散差大） |
| **达克罗螺栓-钢** | 0.10–0.15 |
| **带平垫圈（钢垫圈）** | 0.10–0.16 |
| **带法兰螺栓（大支承面）** | 0.10–0.14 |

**本工具默认 μK = 0.14**，略高于 μG = 0.12 以反映常见钢-钢干燥状态。

## μK 和 μG 为什么可以不一样

见 `bolt_friction_thread` 术语的说明 —— 两个摩擦界面在实际工艺里可以独立调整（例如只在螺纹涂装配油）。同时本模块将 μK 单独列出，是为了：

- 给用户一个机会按"支承面到底是什么配对"精确取值
- 支承面若使用特殊垫圈（锯齿、达克罗垫）时可精确建模

## 支承面材料 → μK 默认值的联动

选"支承面材料 = 钢"时本模块默认 μK = 0.14；选"铝合金"时稍低（0.12 左右）。但因为数值差异小且**用户通常应按实际界面状态填**，本模块选材料主要驱动的是**许用支承面压强 p_G**，不自动改 μK。

## 支承面压强与 μK 同时影响 R7 校核

R7 校核支承面不被压溃：

```
p_bear = FM,max / A_bearing        [N / mm² = MPa]
A_bearing = π/4 · (DKm,o² − DKm,i²)   [mm²]
R7 通过: p_bear ≤ p_G
```

μK 不直接进入 R7，但它影响 FM,max（同 MA 下 FM 大小）。摩擦 μK 高 → FM 低 → 支承面压强低 → R7 轻松；但同一摩擦下 R3/R4 更紧张。

## 常见误用

- **μK 填 0（无摩擦）**：会让扭矩公式里支承面项消失，计算出极高的 FM。物理上不可能。
- **不填但留空**：工具会用默认 0.14。建议保留或按实际工艺修改，不要留空期望工具"跳过 μK"。
- **用螺纹防咬合剂涂到支承面**：MoS2 / 二硫化钼膏会把 μK 降到 0.06–0.08，FM 严重偏高，容易过紧屈服。
