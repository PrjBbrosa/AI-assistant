# 附加防松扭矩 MA,prev

**一句话**：防松元件（锁紧螺母、螺纹锁固胶）产生的与预紧力无关的基础阻力扭矩。

**怎么理解**：

普通螺母转动时的阻力完全来自预紧力产生的摩擦：预紧力为 0 时，阻力也为 0。防松元件则不同——尼龙圈、金属形变锁圈、螺纹锁固胶等在预紧力为 0 时仍有显著阻力扭矩。装配时要克服这部分阻力，才能继续拧紧螺栓。

装配扭矩校核必须把 MA,prev 加回去，否则"剩余用于产生预紧力的扭矩"会被高估，实际预紧力低于计算值。在小螺栓（M10 以下）或高强防松胶场合，MA,prev 占总装配扭矩的比例可达 20-30%，不可忽略。

## 典型防松元件与 MA,prev 范围

| 防松类型 | MA,prev 典型值 |
|---|---|
| 无（普通螺母）| 0 |
| Nyloc（尼龙圈自锁）| M10: 2-6 N·m；M16: 5-15 N·m |
| 全金属自锁（Stover 等）| M10: 3-8 N·m；M16: 8-20 N·m |
| 螺纹锁固胶 Loctite 243 | M10: 1-4 N·m（低强度）|
| 螺纹锁固胶 Loctite 263 | M10: 6-15 N·m（高强度）|
| 齿形锁紧垫圈 | 视型号 1-5 N·m |

（精确取供应商规格书）

## 影响公式

```
装配总扭矩：MA = MG + MK + MA,prev
其中 MA,prev 占比可达 10-30%（小尺寸螺栓更显著）
```

MA,prev 仅出现在装配扭矩输出中，**不影响 FM_min**。FM_min 由密封要求、防滑要求等夹紧需求决定，与装配扭矩路径无关。

忽略 MA,prev 会导致装配扭矩 MA 被低估——工具按照 FM_min 反推出的 MA 偏小，实际拧到该扭矩时，部分扭矩被 MA,prev 消耗，真实预紧力低于目标值 FM_min。

## 常见误用

- **普通螺母填非零 MA,prev**：普通六角螺母无防松结构，MA,prev = 0；误填正值会低估预紧力，过于保守。
- **Nyloc 第一次拧入与反复拧松后 MA,prev 不同**：尼龙圈磨损后 MA,prev 显著降低，DIN 267-28 规定首次拧入值；重复使用应降额或更换。
- **锁固胶固化前后差异巨大**：Loctite 的 MA,prev 在固化后才生效（通常 24 h）；固化前装配扭矩仅靠螺纹摩擦，不可混淆固化前后的预紧状态。

**出处**：VDI 2230:2015 §5.4.3; DIN 267-28; DIN 25201
