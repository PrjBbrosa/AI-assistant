# 应力截面积（As）

**一句话**：螺栓抗拉计算所用的等效圆形截面积，落在中径 d2 与小径 d3 之间，代表螺栓在轴向拉力下**真正承载的横截面**。

**怎么理解**：

螺栓看起来"外径 d"，但螺纹牙根一圈比牙顶细得多。拉伸时真正工作的是牙根附近的横截面。把这一段看作一个等效圆柱，它的直径 ds = (d2 + d3) / 2，截面积就是 As。ISO 898-1 给出更精确的定义：

```
As = π/4 · (d − 0.9382·p)²     [d,p: mm → As: mm²]
```

这个公式等价于用直径 `ds = d − 0.9382·p`（介于 d2 与 d3 之间）画出来的圆面积。

## 为什么总是用 As 而不是 π/4·d² 或 π/4·d3²

- **π/4·d²**（公称外径面积）：高估，实际螺纹牙根处薄得多
- **π/4·d3²**（小径面积）：低估，真实断面并不全在小径
- **As**（应力面积）：工程实践折衷，与屈服 / 抗拉试验数据对齐

所有 VDI 2230 应力公式都用 As：

```
σ_ax,asm  = FM,max / As                  # 装配轴向应力 [MPa]
σ_ax,work = (FM,max + ΦN · FA,max) / As  # 服役轴向应力 [MPa]
```

**细牙 As 大约比粗牙大 5–10%**，这是选细牙做高承载螺栓的一个理由。

## 何时手动覆盖

- 标准规格（M6–M48）：**由下拉自动填入**，不要手动改，改了会偏离 ISO 表。
- 自定义规格：工具按 `π/4 · (d − 0.9382·p)²` 派生；若你手里有加工图纸规定的非标 As（例如特殊阶梯轴螺纹），可以手动覆盖。
- **若手动输入值与派生值相对偏差 > 1%**，工具会抛 `InputError` 提示不自洽（轴向受力螺纹连接模块已实现这个校验，VDI 2230 模块按 GB 表取值，偏差容忍更宽松）。

## 常见误用

- **把公称外径 d 的面积当 As**：高估 20–40%，会得到虚高的允许载荷。
- **M8 粗牙与细牙 As 混用**：细牙 M8×1.0 的 As = 39.2 ≠ 粗牙的 36.6。选择规格时注意下拉显示的螺距。
- **自定义螺纹不填 p**：工具按 `p=1.5` 默认派生，可能与实际加工牙型不符。

**出处**：GB/T 196-2003、ISO 898-1:2013 附录（As 派生公式），GB/T 3098.1-2010 表 3（强度等级允许载荷与 As 乘积）

> Cannot verify against original ISO standard —— As 派生系数 0.9382 对应 60° 标准牙型在 ISO 898-1 附录的推导；本术语列表对应 GB/T 196 粗牙（部分细牙）标准值。精确数值请以原始标准为准。
