# 许用支承面压强（p_G）

**一句话**：螺栓头或螺母下方压向被夹件表面允许的最大接触压强，作为 VDI 2230 R7 支承面压强校核的判据（防止表面被压溃）。

**怎么理解**：

预紧力 FM,max 集中作用在螺栓头（或螺母）与被夹件之间一圈小面积 `A_bearing = π/4·(DKm,o² − DKm,i²)` 上。局部压强 `p_bear = FM,max / A_bearing` 可能非常高（几百 MPa）。若 p_bear 超过被夹件表层的屈服承载能力，表面会被压出凹坑 —— 带动预紧力逐步松弛，最终连接失效。

R7 校核：

```
p_bear = FM,max / A_bearing       [N/mm² = MPa]
R7 通过: p_bear ≤ p_G
```

p_G **取配对两侧的软侧**。钢螺母压在铝法兰上，p_G 按铝取，不按钢取。

## 怎么让 R7 通过（p_bear 偏高时）

1. **加钢垫圈**（或带法兰螺栓）：增大 A_bearing，把压力分散到更大面积
2. **提高被夹件表层硬度**：表面淬火、渗碳、氮化
3. **换更大规格螺栓**：同 FM 下 DKm,o 更大，A_bearing 更大
4. **降低 FM,max**（慎用）：可能让 R3/R4 失败，不推荐

## 常见误用

- **直接取被夹件 Rp0.2 作为 p_G**：错。p_G 是**局部允许接触压强**（含三向应力约束），工程上比 Rp0.2 高 1.5–2 倍。VDI 2230 表 B.5 给出标准取值；本模块内置值已经按 B.5 折减。
- **钢-铝装配 p_G 按钢取**：必须按铝侧取 ~300 MPa；按钢侧 700 MPa 会让铝法兰被压溃而工具判 PASS。
- **带波形垫圈 / 弹垫不改 p_G**：波形垫圈变形吸能，实际接触面积**远小于**外形尺寸；p_bear 可能比平垫圈高 2–3 倍。弹性垫圈不用于校核 p_G，高强连接慎用。
- **忘填 DKm,i 或 DKm,o**：R7 公式分母 A_bearing 变为 0 或负数会抛异常。

## R7 与 R4 的关系

R4（装配等效应力）和 R7（支承面压强）检查的是**两种不同的破坏模式**：

- **R4**：螺栓自身屈服（螺栓身被拉断或永久变形）
- **R7**：被夹件表层压溃（坑洼导致预紧力松弛）

两者都要通过才算连接合格。R7 往往在"小螺栓 + 软材质被夹件"时成为 FM 的真正上限。
