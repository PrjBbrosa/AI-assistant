# 模数（m）

**一句话**：齿轮与蜗轮传动的基础尺寸单位，决定齿的大小、强度上限以及整个齿轮副的尺寸规模。

**怎么理解**：

模数 m 是齿距 p 除以 π 得到的标准化量（p = π·m），单位 mm。模数越大，齿越高越粗，承载能力越强，但齿轮体积、重量、加工成本也随之上升。选模数本质上是"挑一个既能抗住工作扭矩、又不致过度浪费材料"的折中。

实际选型按扭矩反推：先估算齿面接触应力或齿根弯曲应力，查许用值，算出最小模数，再在标准模数系列里就近向上取。两个配对齿轮的模数必须完全相同，蜗杆与蜗轮也是如此，否则无法正确啮合。

分度圆直径与模数、齿数的关系非常直观：d = z · m。所以知道齿数和模数就能锁定齿轮的基本轮廓。

**公式**：`d = z × m`，`p = π × m`

**典型值**：

标准模数系列（GB/T 1357、DIN 780 第一系列优先选用）：

- 小模数（0.5 / 0.8 / 1 / 1.25 / 1.5 mm）：仪表、小功率减速、电动工具
- 中等模数（2 / 2.5 / 3 / 4 / 5 mm）：常规工业减速机、机床传动
- 大模数（6 / 8 / 10 / 12 / 16 mm）：冶金、矿山、大型齿轮箱

选择依据：小扭矩小空间选小模数；大冲击、长寿命选大模数；第二系列（1.75、2.25、2.75 等）一般避免使用。

**出处**：GB/T 1357-2008、DIN 780-1、ISO 54
