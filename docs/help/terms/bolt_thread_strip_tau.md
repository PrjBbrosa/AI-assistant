# 螺纹剪切强度（τ_BM / τ_BS）

**一句话**：R8 螺纹脱扣校核里，**内螺纹材料** τ_BM（螺母 / 壳体）与**外螺纹材料** τ_BS（螺栓）各自的剪切强度 MPa；典型为 Rp0.2 的 0.6 倍。

**怎么理解**：

螺纹脱扣（滑牙）本质是**螺纹牙根被剪断**。剪切强度就是材料承受"沿平面滑动剪切"的能力。工程近似：

```
τ_shear ≈ 0.6 · Rp0.2          [MPa]       # 公式
```

R8 校核里两个剪切强度是**分开填**的：

- **τ_BS**（外螺纹 / 螺栓侧）：就是螺栓材料的剪切强度；本工具默认按 `0.6 · Rp0.2` 自动填入，手动可覆盖
- **τ_BM**（内螺纹 / 螺母或壳体侧）：内螺纹所在基体材料的剪切强度；**必填**，无默认值

分开填的原因是 τ_BM 往往是**瓶颈**：钢螺栓（τ_BS ≈ 400 MPa）装入铝壳体（τ_BM ≈ 180 MPa），内螺纹先滑牙。

## 典型值

### τ_BS（螺栓侧，τ_BS ≈ 0.6 · Rp0.2）

| 强度等级 | Rp0.2 [MPa] | τ_BS 默认 [MPa] |
|---|---|---|
| 4.6 | 240 | 144 |
| 5.8 | 400 | 240 |
| **8.8** | **640** | **384** |
| 9.8 | 720 | 432 |
| 10.9 | 900 | 540 |
| 12.9 | 1080 | 648 |

### τ_BM（内螺纹基体侧，按基体材料查）

| 基体材料 | τ_BM 典型 [MPa] | 备注 |

| 10 级螺母（Rp0.2 ≈ 900） | 540 | 重载螺母 |
| **结构钢 Q235** | **140** | 低碳钢 |

## 填值工作流

最常见的填法：

1. 强度等级选定后，τ_BS **留空**（工具按 0.6·Rp0.2 自动填）
2. 基体材料决定 τ_BM：查上表或材料手册
3. 勾选是否启用 R8（填了 m_eff + τ_BM 就自动启用）

**注意**：选择"标准螺母"（ISO 4032 配对螺栓强度等级的螺母）通常 τ_BM ≈ τ_BS，R8 很容易通过。τ_BM 成为瓶颈主要出现在"螺栓旋入非螺母基体"场景。

## 螺母强度等级的配对规则

ISO 898-2 给出"螺栓强度等级 → 配对螺母强度等级"：

| 螺栓 | 配对螺母（ISO 898-2） |
|---|---|
| 4.8 / 5.8 | 5 级螺母 |
| 8.8 | 8 级螺母 |
| 10.9 | 10 级螺母 |
| 12.9 | 12 级螺母 |

**核心原则**：螺母强度 ≥ 螺栓强度。反之（小号螺母配大号螺栓）会导致螺母先滑牙失效。本工具**不自动检查这种配对**，需要设计人员自己保证。

## 常见误用

- **τ_BM 填 Rp0.2**：高估 67%（0.6·Rp0.2 是 0.6 倍不是 1 倍）。
- **τ_BM 留空**：R8 校核启用但 τ_BM 未定义 → 工具抛 `InputError: thread_strip.tau_BM 必须 > 0`。
- **τ_BS 手动覆盖为一个过大值**：绕过 R8 校核的人为做法；不推荐。若有理由覆盖（如螺栓材料比标称强），用供应商测试报告数据，不要拍脑袋。
- **铝壳体取 τ_BM = 400 MPa**：常见错误。铝合金的 τ 远低于钢；按铝实际材料查表（6061 ≈ 160、7075 ≈ 300）。

**出处**：ISO 898-1 / -2（螺栓与螺母机械性能）、VDI 2230-1:2015 §5.5.5

> Cannot verify against original ISO / VDI standard —— τ ≈ 0.6 · Rp0.2 是工程常用近似；精确数值（延性、加工硬化影响）请查具体材料规格书；螺母强度等级对应的精确剪切值请以 ISO 898-2 表 2 为准。
