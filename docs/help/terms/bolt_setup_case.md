# 载荷工况（setup_case）

**一句话**：下拉记录螺栓连接承受的载荷类型（轴向/横向/组合/自由），用于设计文档描述；当前为 UI 记录字段，不直接路由计算路径。

**怎么理解**：

VDI 2230 将螺栓连接中的外部载荷分为：

- **轴向载荷 FA**：平行于螺栓轴线，直接增减螺栓拉伸力，影响残余夹紧、应力幅和疲劳校核。
- **横向载荷 FQ**：垂直于螺栓轴线，通过摩擦或剪切传递，影响防滑校核（R9）。
- **轴向 + 横向组合**：最常见的实际工况，两类载荷同时存在。
- **自由输入**：载荷方向或形式不适合上述分类时，手动标注说明。

```
本字段 mapping = None — 不传入计算器 payload，
FA / FQ 的实际值分别由 loads.FA_max 和 loads.FQ_max 字段填写。
```

## 下拉选项说明

| 选项 | 含义 | 后续填写重点 |
|---|---|---|
| 轴向载荷 | 仅有轴向拉/压载荷 | 填 FA_max；FQ_max 可填 0 |
| 横向载荷 | 仅有横向剪切 | 填 FQ_max；需填摩擦面数 qF |
| 轴向+横向 | 两类载荷共存（默认选项）| FA_max + FQ_max 都需填写 |
| 自由输入 | 特殊或混合工况 | 按实际情况填载荷，说明见设计备注 |

## 与其他字段的关系

- 实际参与计算的载荷量是 `loads.FA_max`、`loads.FQ_max`，而非本字段。
- 疲劳校核路径由 `options.check_level`（basic / thermal / fatigue）和 `operating.load_cycles` 控制，与本字段无关。
- **本字段不改变安全系数**——服役屈服安全系数由 `checks.yield_safety_operating` 字段单独设定；当前版本没有独立的 S_A 设定字段，支承面压强允许值通过 `bearing.p_G_allow` 输入。

## 工程意义

选择正确的工况类型有助于：

1. **核对填写完整性**：轴向工况下发现 FA_max 未填，提示遗漏。
2. **设计文档存档**：导出计算书时显示工况描述，便于后续评审识别。
3. **未来扩展**：后续版本可能据此自动路由横向校核路径（如剪切销校核），当前版本尚未实现。

## 常见误用

- **认为选"静态"或"疲劳"工况会改变安全系数**：本字段不控制计算路径，请用 `check_level` 激活疲劳校核，用 `checks.yield_safety_operating` 设定服役屈服安全系数。
- **横向载荷为主却选"轴向载荷"**：不影响计算结果（横向载荷通过 FQ_max 传入），但导出文档描述不符，给评审造成混淆。

**出处**：VDI 2230:2015 §5（载荷种类分类）
