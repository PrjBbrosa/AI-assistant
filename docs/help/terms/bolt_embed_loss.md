# 嵌入损失（FZ）

**一句话**：装配后接触面粗糙峰被压平，导致预紧力在几小时到几天内缓慢流失的那一部分，VDI 2230 嵌入损失参考表按粗糙度等级给出典型值。

**怎么理解**：

螺栓刚拧紧时 FM 是装配读数。但接触面（螺栓头下方、被夹件层间、螺母上方）其实不是光滑的平面 —— 粗糙峰尺度在几微米到十几微米。这些峰在高压下逐步压平，导致弹性系统"被压扁一点点"。系统压扁 δ 时刻，FM 同时下降：

```
FM_after_embed = FM_initial − FZ         [N]
```

FZ 是这部分**总预紧力损失**，以力为单位（N）；物理来源是**嵌入量** fZ（μm）乘以系统柔度的倒数：

```
fZ_total = 各界面嵌入量求和            [μm]
FZ = fZ_total · 1e−3 / (δs + δp)      [N]     # 1e−3 把 μm 换成 mm
```

## 本工具的简化估算模型（重要：与 VDI 原表不同）

**本模块并没有按 VDI 2230 原表做"按界面类型（螺纹 / 头下方 / 板板）分类"的逐界面估算**。实际代码只用了一个"按粗糙度等级取单一常数 × 界面数"的简化模型（见 `core/bolt/calculator.py:178-214`）：

```python
# 本模块使用的简化单界面嵌入量（μm，不按界面类型区分）
_EMBED_FZ_PER_INTERFACE = {
    "rough":  3.0,   # 粗糙 (Ra ≈ 6.3 μm)
    "medium": 2.5,   # 中等 (Ra ≈ 3.2 μm)
    "fine":   1.0,   # 精细 (Ra ≈ 1.6 μm)
}
# 界面数：通孔连接 = part_count + 2；螺纹孔连接 = part_count + 1
n_interfaces = part_count + (2 if joint_type == "through" else 1)
fZ_total_mm = fz_per_if * n_interfaces * 1e-3     # μm → mm
FZ = fZ_total_mm / (δs + δp)                       # N
```

举例（M10 通孔连接、中等粗糙度、单层被夹件 part_count=1）：

```
n_interfaces = 1 + 2 = 3
fZ_total = 2.5 μm × 3 = 7.5 μm
FZ = 7.5e−3 mm / (2.2e−6 + 3.1e−6) mm/N ≈ 1415 N
```

**VDI 2230 原表更精细**：对螺纹、螺栓头下方、板板接触面分别给不同的 μm 值（常见范围 1–4 μm）。本模块选用**每类一个常数**是为了工程初筛；若你需要精确的界面分类估算，请按 VDI 2230 原表逐面手算后在 UI 里手填 FZ（大于 0 的值会覆盖自动估算）。

## 本工具自动估算触发条件

**若填 FZ = 0 且已选"接触面粗糙度"**，工具按上述简化模型自动估算 FZ 并回填（`_estimate_embed_loss` 在 `core/bolt/calculator.py:186`）。

**若 FZ 填一个大于 0 的值**：工具不做自动估算，直接用你填的值（手动覆盖）。

## 粗糙度选择建议

| 粗糙度等级 | 常见加工 |
|---|---|
| 粗糙 (Ra≈6.3 μm) | 粗铣、粗车、粗糙铸件毛面 |
| 中等 (Ra≈3.2 μm) | 普通铣 / 车，磨削前状态 |
| 精细 (Ra≈1.6 μm) | 精铣、精磨、精刮 |

**工厂实际**：法兰密封面通常精细（1.6 μm 以下）；通用机械接触面多是中等（3.2 μm）。

## 嵌入损失的时间特性

FZ 不是瞬时发生，而是：

- **前 10 分钟**：损失 ~40% 总 FZ（快速相，粗糙峰塑性变形）
- **前 24 小时**：损失 ~80% 总 FZ（慢速相，微蠕变）
- **前 1 周**：基本稳定

因此"拧完立刻测扭矩" 与 "一天后测扭矩" 读数会不同。工程实践：关键连接装配后静置 24 小时再复拧一次（或一次拧紧时按 FM,min + FZ 额外增量）。

## 常见误用

- **FZ 填 0 且没选粗糙度**：工具按 FZ = 0 处理，实际会损失 5–15% 预紧力 → R3 校核虚高。
- **FZ 与"拧紧松弛率"混为一谈**：嵌入损失只是 FM 损失的一部分，还有热损失 Fth、振动松弛、涂层压实等。本工具只自动处理 FZ + Fth。
- **自动估算的 FZ 过大难过 R3**：这是工具提醒你接触面粗糙度真的影响预紧力 —— 应考虑改用精细加工、或加大 FM,min 设计余量，而不是手动覆盖 FZ 为小数值。
