# 嵌入损失（FZ）

**一句话**：装配后接触面粗糙峰被压平，导致预紧力在几小时到几天内缓慢流失的那一部分，VDI 2230 表 5.4/1 按粗糙度等级给出典型值。

**怎么理解**：

螺栓刚拧紧时 FM 是装配读数。但接触面（螺栓头下方、被夹件层间、螺母上方）其实不是光滑的平面 —— 粗糙峰尺度在几微米到十几微米。这些峰在高压下逐步压平，导致弹性系统"被压扁一点点"。系统压扁 δ 时刻，FM 同时下降：

```
FM_after_embed = FM_initial − FZ         [N]
```

FZ 是这部分**总预紧力损失**，以力为单位（N）；但物理来源是**嵌入量** fZ（μm）乘以螺栓柔度的倒数。VDI 2230 的原始公式：

```
fZ = Σ fZ,i                              [μm]   # 各界面嵌入量求和
FZ = fZ / (δs + δp) · 1000               [N]    # μm → mm 换 N
```

## VDI 2230 表 5.4/1 单界面嵌入量

表按"承压面"分类（螺纹、螺栓头下方、板板接触面），每类再按"粗糙度等级"给单界面典型值：

| 粗糙度 | 螺纹 fZ,th | 头下方 fZ,h | 板板 fZ,p |
|---|---|---|---|
| 粗糙 (Ra≈6.3 μm) | 4 μm | 4 μm | 4 μm |
| 中等 (Ra≈3.2 μm) | 3 μm | 2.5 μm | 2.5 μm |
| 精细 (Ra≈1.6 μm) | 2 μm | 1.5 μm | 1.5 μm |

**总嵌入量 = 各界面之和**。举例（M10 通孔连接、中等粗糙度、单层被夹件）：

```
fZ,total = fZ,th (螺纹) + fZ,h (螺栓头下方) + fZ,p (板板) + fZ,h (螺母下方)
        = 3 + 2.5 + 2.5 + 2.5 = 10.5 μm

FZ = fZ,total / (δs + δp) · 1000
   = 10.5 / (2.2e-6 + 3.1e-6) · 1000
   = 10.5 / 5.3e-6 · 1000  ≈ 1981 N
```

## 本工具自动估算

**若填 FZ = 0 且已选"接触面粗糙度"**，工具按 VDI 2230 表 5.4/1 自动估算 FZ 并回填（`_estimate_embed_loss` 在 `core/bolt/calculator.py:186`）。

**若 FZ 填一个大于 0 的值**：工具不做自动估算，直接用你填的值。

## 粗糙度选择建议

| 粗糙度等级 | 常见加工 |
|---|---|
| 粗糙 (Ra≈6.3 μm) | 粗铣、粗车、粗糙铸件毛面 |
| 中等 (Ra≈3.2 μm) | 普通铣 / 车，磨削前状态 |
| 精细 (Ra≈1.6 μm) | 精铣、精磨、精刮 |

**工厂实际**：法兰密封面通常精细（1.6 μm 以下）；通用机械接触面多是中等（3.2 μm）。

## 嵌入损失的时间特性

FZ 不是瞬时发生，而是：

- **前 10 分钟**：损失 ~40% 总 FZ（快速相，粗糙峰塑性变形）
- **前 24 小时**：损失 ~80% 总 FZ（慢速相，微蠕变）
- **前 1 周**：基本稳定

因此"拧完立刻测扭矩" 与 "一天后测扭矩" 读数会不同。工程实践：关键连接装配后静置 24 小时再复拧一次（或一次拧紧时按 FM,min + FZ 额外增量）。

## 常见误用

- **FZ 填 0 且没选粗糙度**：工具按 FZ = 0 处理，实际会损失 5–15% 预紧力 → R3 校核虚高。
- **FZ 与"拧紧松弛率"混为一谈**：嵌入损失只是 FM 损失的一部分，还有热损失 Fth、振动松弛、涂层压实等。本工具只自动处理 FZ + Fth。
- **自动估算的 FZ 过大难过 R3**：这是工具提醒你接触面粗糙度真的影响预紧力 —— 应考虑改用精细加工、或加大 FM,min 设计余量，而不是手动覆盖 FZ 为小数值。
