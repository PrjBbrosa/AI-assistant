# 防滑摩擦系数 μT

**一句话**：被夹件之间（part–part）的静摩擦系数，决定横向载荷能否靠压紧而不是剪切螺栓传递。

**怎么理解**：

VDI 2230 中涉及三个不同的摩擦系数，极易混淆：

- **μG**（螺纹摩擦系数）：螺栓螺纹与螺母/孔的接触摩擦，影响拧紧扭矩
- **μK**（支承面摩擦系数）：螺栓头/螺母与被夹件的接触摩擦，同样影响拧紧扭矩
- **μT**（防滑摩擦系数）：**被夹件互相接触的界面**，决定横向载荷能不能靠摩擦传递

μT 是**被夹件之间**的摩擦，与螺栓头/螺纹完全无关。
防滑校核能不能成立，取决于 μT 是否足够大，使摩擦力 > 横向载荷 FQ。

工程中 μT 最常被"想当然"设成 0.15，但不同表面状态差异可达 10 倍。

## 典型 μT 值

| 界面状态 | μT 区间 |
|---|---|
| 干燥钢/钢 机加工面 | 0.10–0.20 |
| 涂漆（有机漆层） | 0.05–0.10 |
| 镀锌件对镀锌件 | 0.15–0.30 |
| 喷砂/滚花钢面 | 0.30–0.50 |
| 铝/铝（阳极氧化） | 0.15–0.30 |
| 钢/铸铁 | 0.20–0.30 |
| 润滑油膜 | < 0.05 |

上表为工程参考区间，**取设计值时用下限**（保守）。
关键连接建议做样件剪切试验验证实际 μT。

## 工程取值策略

1. **取下限**：不同来源数据区间较宽，选最小值作为设计输入
2. **多层时取最差界面**：qF > 1 时，μT 取所有界面中最小值（见摩擦面数 qF 条目）
3. **样件试验**：关键或高精度连接，按实际装配表面做剪切试验（夹紧力已知→
   横向加载至滑动→记录 FQ_slip → μT = FQ_slip / FK）
4. **考虑长期效应**：温升导致油脂流失、腐蚀/氧化层变化、振动松弛——
   服役条件与装配状态的差异可使 μT 漂移 ±30%

## 与 μG / μK 的区别

| 参数 | 接触位置 | 影响对象 |
|---|---|---|
| μG | 螺栓螺纹 ↔ 螺母/孔螺纹 | 拧紧扭矩 MA、预紧力 FM |
| μK | 螺栓头/螺母面 ↔ 被夹件表面 | 拧紧扭矩 MA、支承面压强 |
| μT | 被夹件 ↔ 被夹件 | 防滑容量 FQ_allow |

三者来源不同，绝不能互换。μG 和 μK 由螺栓表面处理决定；μT 由被连接件的
配合面状态决定——可以是完全不同的材料/工艺。

## 常见误用

1. **把 μG 当 μT**：μG 是螺纹摩擦（涂油后常为 0.10–0.14），
   μT 是被夹件界面摩擦，两者控制的现象不同，不可互换。
   用 μG 做防滑校核会得出偏保守或偏危险的错误结论。

2. **混合界面取平均**：3 层叠有两个界面，μT 分别为 0.10 和 0.25，
   取平均 0.175 做校核是错误的——最差界面先滑动，正确做法是取 0.10。

3. **忽略长期服役 μT 漂移**：装配后初始 μT 可能满足要求，
   但使用环境（振动、腐蚀、油脂渗入）会使界面状态改变，μT 下降。
   关键连接应在最恶劣服役状态下验证，而非仅在装配态。
