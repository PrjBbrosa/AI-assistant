# 光滑段摩擦系数（μ_torque / μ_axial / μ_assembly）

**一句话**：联合模式下光滑过盈段在三种工况下的摩擦系数——传扭矩时的 μ_T、抵抗轴向力时的 μ_ax、装配压入时的 μ_M；三者物理上相关但**数值可以不同**，都是经验取值。

**怎么理解**：

过盈配合的防滑能力来自径向接触压力 p × 摩擦系数 μ × 接触面积：

```
T_slip  = p · A · μ_T  · (d/2)   [p: MPa, A: mm², μ: 无量纲, d: mm → T_slip: N·mm]
F_slip  = p · A · μ_ax           [p: MPa, A: mm², μ: 无量纲 → F_slip: N]
F_press = p · A · μ_M            [p: MPa, A: mm², μ: 无量纲 → F_press: N]
```

三种摩擦系数数值上可以不同，主要原因：

1. **切向与轴向的表面运动方向不同**：有些表面（如精加工的长条轴向纹理）在轴向方向 μ_ax 偏低、切向 μ_T 偏高。通常实际数据两者差别在 ±10% 内。
2. **静摩擦 vs 装配摩擦**：μ_M 对应**装配过程中**（压入时有相对运动）的**滑动摩擦**，通常略低于静摩擦。工具 UI 默认 μ_M = 0.12 < μ_T = μ_ax = 0.14。
3. **润滑状态**：装配常用油润滑降低 μ_M（以免装配力过高损伤表面），运行时油被挤出、进入干 / 边界摩擦，μ_T 与 μ_ax 回升。

**本工具中的用法**：

- `smooth_friction.mu_torque` → 场景 B 的防滑校核、扭矩容量计算。
- `smooth_friction.mu_axial` → 场景 B 的轴向防滑校核。
- `smooth_friction.mu_assembly` → 场景 B 的 `press_force_curve`（压入力 vs 过盈量曲线）。

三者各自独立填入 payload，对应到 `core/interference/calculator.py` 的不同输出字段。

**典型值**：

| 接触副 | μ 范围（干燥、微油） | 备注 |
|---|---|---|
| 钢-钢，精车 + 光洁磨 | 0.10 ~ 0.14 | **工具默认 0.14**（保守侧） |
| 钢-钢，粗车 | 0.12 ~ 0.18 | 粗糙度高 → 接触更"牙齿状" |
| 钢-铸铁 | 0.10 ~ 0.15 | 常规工业取值 |
| 钢-铜合金 / 青铜 | 0.08 ~ 0.12 | 青铜软、表面易被塑性变形填平 → μ 略低 |
| 钢-铝 | 0.08 ~ 0.10 | 铝表面氧化膜降低 μ |
| 压装油润滑（装配时）| 0.06 ~ 0.10 | 仅装配阶段；运行时油被挤出 μ 回升 |
| 胶合（用 Loctite 等锁固胶）| 0.20 ~ 0.40 | 本工具未内置；需`自定义`手动填 |

**DIN 7190 的取值习惯**：对"不确定的干配合"推荐 μ = 0.10（保守下限）；对"清洁干燥的钢-钢配合"推荐 μ = 0.14~0.15。本工具默认 0.14 对应第二档。

**常见误用**：

- **查教科书"钢对钢静摩擦 μ ≈ 0.3"并填入**：教科书常引用的是**干净、干燥、无氧化**的理想表面值。实际工业加工表面带残留油、氧化层、微小颗粒，μ 只有 0.10~0.15。填 0.3 会让防滑 SF 虚高 100%。
- **μ_T 与 μ_ax 取极端不同的值**：除非有实验数据支持，否则两者差距不应超过 ±20%。默认 μ_T = μ_ax = 0.14 是安全选择。
- **μ_M 取得比 μ_T 还高**：装配时常用油润滑便于压入，μ_M 应 ≤ μ_T。若填 μ_M > μ_T 会让 press_force 曲线虚高。工具默认 μ_M = 0.12 是合理的。
- **认为"过盈大 → μ 也大"**：摩擦系数由表面状态决定，与接触压力几乎无关。不要把摩擦系数与过盈量做线性关联。
