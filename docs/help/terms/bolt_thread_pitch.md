# 螺距（p）

**一句话**：相邻两牙螺纹牙尖之间沿轴向量的距离，决定螺栓"每转一圈前进多远"、螺纹的粗细、以及派生的 As / d2 / d3。

**怎么理解**：

p 不是你随便选的数。每个公称直径 d 对应一组标准螺距：**粗牙**和若干**细牙**。例如：

- **M10**：粗牙 p = 1.5 mm；细牙可选 1.25 或 1.0 mm
- **M12**：粗牙 p = 1.75 mm；细牙 1.5 / 1.25 mm
- **M20**：粗牙 p = 2.5 mm；细牙 1.5 mm

本工具下拉选项会根据所选 d 过滤合法 p。选"自定义"时可填任意非标值，但派生几何（d2、d3、As）将按 ISO 898-1 公式重算。

## 粗牙 vs 细牙怎么选

| 场景 | 选择 |
|---|---|
| 通用装配、拆装频繁、容差大、材料脆弱 | **粗牙**（默认） |
| 需要精细调整预紧（轴承预紧螺母、调节螺杆） | 细牙 |
| 薄壁螺纹孔、需要更多啮合扣 | 细牙（同深度下扣数更多） |
| 振动环境需防松 | 细牙（自锁性略好）或加锁紧垫圈 |
| 重载螺栓但空间紧凑 | 细牙（同 d 下 As 略大） |

## 公式（本工具实现）

派生几何：

```
As = π/4 · (d − 0.9382·p)²     [d,p: mm → As: mm²]
d2 = d − 0.64952·p             [mm]   # 中径
d3 = d − 1.22687·p             [mm]   # 小径
```

扭矩 / 摩擦：

```
MA = FM · (0.5·d2·(P/(π·d2) + 1.155·μG) + 0.5·DKm·μK)    [N·mm; P=p 在本工具内部变量]
```

螺距 p 进入扭矩公式通过 `P/(π·d2)` 这一项，称为**导程角贡献**。粗牙 p 大 → 导程角大 → 同一扭矩下产生的预紧力**较小**（因为更多扭矩消耗在克服螺纹斜面上）。

## 典型值

| d | 粗牙 p | 常见细牙 p |
|---|---|---|
| M6 | 1.0 | — |
| M8 | 1.25 | 1.0 |
| M10 | 1.5 | 1.25, 1.0 |
| M12 | 1.75 | 1.5, 1.25 |
| M16 | 2.0 | 1.5 |
| M20 | 2.5 | 1.5 |
| M24 | 3.0 | 2.0 |

## 常见误用

- **自定义 p 小于规格粗牙**：不是常规可选螺距（如 M10 × 0.75），需要非标加工或专用工艺，成本高。除非你确切知道有非标刀具供应，**不要自选非表内值**。
- **把 p 和"螺纹长度"搞混**：p 只是相邻牙距，不是总螺纹长度。螺纹加工长度 b 需要单独确定。

**出处**：GB/T 196-2003、ISO 724:1993（螺距系列）、ISO 898-1:2013（As / d2 / d3 派生公式）

> Cannot verify against original ISO standard —— As / d2 / d3 派生系数（0.9382 / 0.64952 / 1.22687）来自公开教科书，对应于 ISO 898-1 附录对 60° 标准牙型的推导。精确数值请以标准原文为准。
