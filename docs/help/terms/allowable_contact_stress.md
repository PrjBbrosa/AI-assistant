# 许用齿面接触应力（σH,lim）

**一句话**：齿面在无限寿命下能承受的最大赫兹接触应力（对应特定循环次数的疲劳极限），是齿面点蚀校核的判据。

**怎么理解**：

齿轮或蜗轮工作时，齿面反复承受接触应力循环，到一定次数后表层出现点蚀（pitting）——金属细微颗粒从齿面脱落，形成麻点。σH,lim 就是"**在对应疲劳循环次数（通常 10^9）下，齿面不发生点蚀的最大接触应力**"。它是从大量疲劳试验数据拟合出的材料特性，与材料、表面处理、润滑、表面粗糙度直接挂钩。

在校核时要比较实际接触应力 σH 与许用值 σHP。σH,lim 是"原始材料疲劳极限"，但实际能用多少要按工况折减。DIN 3990 / DIN 3996 把折减拆成几个系数相乘再除以安全系数：

```
σHP = σH,lim · ZN · ZL · ZR · ZX / SH
```

各符号含义（都是 >0 的无量纲系数）：
- **ZN —— 寿命修正**：你设计的齿轮只跑 10⁷ 次循环还是 10⁹ 次？寿命短于"无限寿命基准"时，许用值可以放大（ZN > 1）
- **ZL —— 润滑油膜修正**：干润滑、脂润滑、油池润滑形成的油膜厚度不同，带走的热量和抑制点蚀能力不同
- **ZR —— 齿面粗糙度修正**：齿面越光滑越不易启裂（ZR ≈ 1.0 磨齿，ZR < 1 铸造齿）
- **ZX —— 尺寸（大齿轮）修正**：大齿轮里缺陷尺寸也大，许用值要下调
- **SH —— 齿面安全系数**：工程目标，工业传动常取 1.0–1.2，重要装备取 ≥ 1.3

**工程经验**：没有详细资料时，先直接用 σH,lim 除以目标安全系数做估算（ZN、ZL、ZR、ZX 都先取 1.0），得到"粗估 σHP"。方案选定后再按附录查精确系数细化。若实际使用寿命远低于 10⁹ 循环，ZN 可适当放大许用值（有限寿命设计）。

蜗轮传动中钢蜗杆—青铜蜗轮是主流组合，σH,lim 取青铜（较弱方）的值。塑料蜗轮的 σH,lim 低一个量级，且强烈依赖温度与时间，DIN 3996 专门给出塑料蜗轮的 σH,lim 曲线。

**典型值**（无限寿命、常规润滑条件）：

- 钢蜗杆 + 锡青铜蜗轮（CuSn12）：σH,lim ≈ 350–450 MPa
- 钢蜗杆 + 铝青铜蜗轮（CuAl10Ni5）：σH,lim ≈ 450–550 MPa，抗冲击更好
- 钢蜗杆 + 黄铜蜗轮（CuZn）：σH,lim ≈ 250–320 MPa，低价低承载
- 钢蜗杆 + 灰铸铁蜗轮：σH,lim ≈ 200–280 MPa，仅用于低速轻载
- 渗碳淬火钢齿轮 σH,lim ≈ 1300–1650 MPa（齿轮用，蜗杆副不代表）
- 调质合金钢齿轮：σH,lim ≈ 500–700 MPa
- 塑料蜗轮（PA66、POM、23°C）：σH,lim ≈ 30–80 MPa，80°C 以上还要进一步下调

选择依据：蜗轮按材料牌号查 DIN 3996 附录 B；齿轮查 DIN 3990-5 / ISO 6336-5。若材料手册只给 σHlim 范围，按中值取；寿命需求显著低于 10^9（如 10^7 次启停）时按寿命系数 ZN 放大。

**出处**：DIN 3990-5:2016（齿轮材料疲劳数据相关章节）、DIN 3996:2019（蜗轮材料允许应力相关章节）、ISO 6336-5:2016。
