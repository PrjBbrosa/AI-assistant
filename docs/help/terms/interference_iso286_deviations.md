# 轴孔偏差（es / ei / ES / EI）

**一句话**：ISO 286 描述公差位置的四个符号；大写给孔、小写给轴、e/E 表示上偏差、i/I 表示下偏差；所有单位为 um（微米）。

**怎么理解**：

ISO 286 把"公差"拆成**位置**（偏差的上下限）与**宽度**（公差带 IT 等级）。在过盈配合上下文里，四个符号分别是：

| 符号 | 全称 | 含义 |
|---|---|---|
| **es** | écart supérieur（轴） | 轴实际尺寸相对名义的**上限**（轴最大比名义大多少） |
| **ei** | écart inférieur（轴） | 轴实际尺寸相对名义的**下限**（轴最小比名义大多少） |
| **ES** | écart supérieur（孔） | 孔实际尺寸相对名义的**上限**（孔最大比名义大多少） |
| **EI** | écart inférieur（孔） | 孔实际尺寸相对名义的**下限**（孔最小比名义大多少） |

**符号约定**：过盈配合里，轴通常比名义**大**（es、ei 为**正**），孔通常比名义**小**（ES、EI 为**负**或接近 0）。

## 公式（本工具实现）

```
过盈量窗口由四个偏差直接换算:
   δ_min = ei − ES                              [um, 直径方向]
   δ_max = es − EI                              [um]

校验规则（本工具）:
   es ≥ ei                轴上偏差 ≥ 下偏差
   ES ≥ EI                孔上偏差 ≥ 下偏差
   δ_max > 0              必须形成过盈
   δ_min ≥ 0              不是过渡 / 间隙配合
```

**直径方向**：所有偏差都是**直径**方向，不是半径。图纸上标 "Ø40 es=+35, ei=+22" 就是直径方向。

## 典型值（d=30~50 mm 段示例）

**孔侧（H7 基孔制）**：
- ES = +25 um
- EI = 0 um

**轴侧（不同配合）**：
| 配合 | es | ei | 过盈窗口 (δ_min, δ_max) |
|---|---|---|---|
| p6（轻过盈） | +35 | +22 | (22−25, 35−0) = (−3, 35) → 实际过渡 |
| s6（中过盈） | +48 | +35 | (35−25, 48−0) = (10, 48) |
| u6（重过盈） | +61 | +48 | (48−25, 61−0) = (23, 61) |

（数据摘自 ISO 286 子集。）

## 什么时候用偏差换算模式

- **自制件 / 非标件**：图纸上标注了具体偏差数值，没有标准配合代号
- **超出内置代号 / 直径范围**：本工具只内置 H7/p6、H7/s6、H7/u6，d=6~50 mm；其他情况需要手填
- **混合公差（轴 p5 + 孔 H7）**：任意组合都可以换算

## 常见误用

- **把孔偏差填成正值**：基孔制里 EI 通常是 0、ES 是 +25 这类小正值；若图纸上标 ES = −10 这类负值说明是特殊公差，按原值填入，**不要改符号**。
- **符号填反**：Shaft 和 Hub 的字段容易搞混；UI 上按"轴上偏差 es / 轴下偏差 ei / 孔上偏差 ES / 孔下偏差 EI"四个字段，位置固定。
- **δ_min 算出负值**：说明当前公差组合是**过渡或间隙**配合，不属于过盈；本工具直接报错。想继续必须调偏差。
- **把 IT 级数字当偏差值**：IT6 是公差**宽度**（不是偏差值本身）；换算需要查 ISO 286 表格，不能直接填 "IT6 = 6"。
- **偏差上下限顺序反**：es < ei 会让本工具报错；必须保证上限 ≥ 下限。

## 输入单位约定

本工具统一使用 **um（微米，= 0.001 mm）**。若图纸标注 mm（例如 es = +0.035 mm），需乘 1000 变成 +35 um 输入。

## 为什么过盈配合不能有间隙分量

"过渡配合"意味着同批产品里有的是过盈、有的是间隙。间隙件不能通过摩擦传递载荷，会立即滑脱。所以**过盈配合的 δ_min 必须 ≥ 0**；δ_min < 0 的组合不在本模块的适用范围。
