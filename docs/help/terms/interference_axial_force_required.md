# 需求轴向力（F_req）

**一句话**：过盈连接需要抵抗的**名义轴向力**（N）；会被工况系数 KA 放大成设计轴向力 F_design，用于校核抗窜动能力。

**怎么理解**：

过盈连接除了传递扭矩，很多时候还要抵抗轴向窜动：
- 皮带 / 链的张力分量
- 热膨胀引起的伸缩推力
- 振动 / 冲击引起的往返力
- 齿轮啮合的轴向分力（斜齿、锥齿、蜗杆）
- 外力推拉（活塞推力、组装工具拉出力）

F_req 是设计意图层面需要**过盈摩擦抵抗**的轴向力。填 0 表示无轴向力需求（纯扭矩或纯定位）。

## 公式（本工具实现）

```
设计轴向力:    F_design = KA · F_req                     [F: N]

轴向容量（最小过盈端）:
   F_cap_min = μ_Ax · p_min · π · d · L                  [F: N, p: MPa, d,L: mm]

轴向安全系数:
   S_axial = F_cap_min / F_design                         [无量纲]

PASS 判据:
   F_cap_min ≥ S_slip,min · F_design
   <=> S_axial ≥ S_slip,min

联合作用（与扭矩同时存在时）:
   u_combined = sqrt( (T_design/T_cap_min)² + (F_design/F_cap_min)² )
   S_combined = 1 / u_combined
   PASS: S_combined ≥ S_slip,min
```

## 单位约定

**本工具统一用 N**，不是 kN 或 daN（十牛）。

常见换算：
- 1 kgf ≈ 9.81 N
- 1 kN = 1000 N
- 1 daN = 10 N（某些欧洲图纸使用）

## 典型值

| 应用场景 | F_req 典型范围 |
|---|---|
| 皮带轮（张紧侧） | 500 ~ 3000 N |
| 链轮毂（主动链传动） | 1000 ~ 5000 N |
| 斜齿 / 锥齿轮轴向分力（中小型） | 500 ~ 5000 N |
| 蜗杆轴向推力 | 2000 ~ 20000 N |
| 组装工具推力 | 100 ~ 500 N |
| 活塞 / 液压缸推拉 | 按液压面积 × 压力计算 |
| 重载减速机输出端 | 10000 N+ |

## 工程选择建议

1. **没有轴向力源就填 0**：纯扭矩传动件常见；填 0 后本项校核按"无限安全"处理，不影响其他结果。
2. **斜齿 / 锥齿要计算轴向分力**：按齿轮几何分解，不是"猜一个数"。
3. **热膨胀推力常被遗漏**：长跨距轴在温差工况下产生轴向力；若过盈连接是**固定端**（另一端有间隙或浮动支撑），需评估。
4. **冲击 / 碰撞力建议专项评估**：偶发大冲击用 F_req 体现可能偏保守；可走单独的"冲击载荷"校核，不完全依赖过盈摩擦。

## 与扭矩的联合作用

扭矩 + 轴向同时存在时，单项能力都通过**不等于**联合作用通过。本工具按 RSS 合成：

```
使用度 = sqrt((T/T_cap)² + (F/F_cap)²)
联合 PASS: 使用度 ≤ 1/S_slip,min
```

例子：
- T/T_cap = 0.6，F/F_cap = 0.6 → 使用度 = √(0.36 + 0.36) = 0.85
- 若 S_slip,min = 1.2 → 允许上限 = 1/1.2 = 0.833
- 联合作用 **FAIL**，尽管单项都在 0.6 以下

单项都应控制在 0.7 以下，联合才有余量。

## 常见误用

- **不填就当 0**：本工具默认 0，若实际有轴向力不填，校核会漏检。
- **把扭矩换算成"等效轴向力"**：扭矩和轴向力是不同方向的摩擦消耗；必须分开填，由本工具按联合作用校核。
- **kN / kgf 当 N 填**：差 10~1000 倍；本工具要求 N。
- **忽略温差推力**：100°C 温差 × 钢的 α=11·10⁻⁶/°C × 1 m 跨距 × E=210000 MPa × 截面积 = 几千 ~ 几万 N；对短跨距不显著，但长轴必评估。
- **轴向力仅在一个方向**：过盈连接的摩擦对两个方向都有效；若轴向力是单向（例如只抵抗重力），仍用绝对值填入。
