# 接触类型（线接触 / 点接触）

**一句话**：决定使用哪一套赫兹公式——两圆柱轴线平行走线接触（矩形斑），两球相切走点接触（圆斑）。

**怎么理解**：

赫兹接触按接触区形状分两大类。同样的载荷 F 下，接触区越集中，最大接触应力 p0 越高：

- **线接触**：两个圆柱轴线平行，或圆柱压在平面上——接触区呈现细长矩形，面积 `A = 2·b·L`。接触斑半宽 b 与曲率半径、材料、**单位长度载荷 q = F / L** 有关；载荷沿长度方向摊薄，所以接触长度 L 越长 p0 越低。
- **点接触**：两球相切，或球压在平面上——接触区呈现圆形，面积 `A = π·a²`。接触斑半径 a 与曲率半径、材料、**总载荷 F** 有关；点接触将载荷集中在一个点上，所以 p0 通常比同载荷线接触高很多。

选错模式会让 p0 差一个数量级：比如把长度 20 mm 的滚子错按点接触算，公式里会忽略长度 L，用 F = 总载荷而非 q = F/L，结果 p0 严重偏高且物理意义错乱。

**公式对照**（两种模式 R'、E' 的定义相同，但接触斑公式不同）：

```
线接触：b = √(4·q·R' / (π·E'))       [q: N/mm, R': mm, E': MPa → b: mm]
        p0 = 2·q / (π·b)               [q: N/mm, b: mm → p0: MPa]
        p_mean = q / (2·b)
        p0 / p_mean = 4/π ≈ 1.27

点接触：a = ((3·F·R') / (4·E'))^(1/3) [F: N, R': mm, E': MPa → a: mm]
        p0 = 3·F / (2·π·a²)            [F: N, a: mm → p0: MPa]
        p_mean = F / (π·a²)
        p0 / p_mean = 3/2 = 1.5
```

符号说明：q 为单位长度载荷，F 为总法向载荷，R' 为等效曲率半径，E' 为等效弹性模量，b 为线接触半宽，a 为点接触半径，p0 为最大接触应力，p_mean 为平均接触应力。

**选择依据**（工程场景 → 模式）：

- 圆柱齿轮齿面啮合（两齿面在接触线上分布） → **线接触**
- 滚子轴承（圆柱滚子 / 滚针） → **线接触**
- 凸轮-挺杆（挺杆为圆柱端面） → **线接触**
- 球轴承（钢球 vs 内/外滚道） → **点接触**（严格说为椭圆接触，本工具以球接触近似）
- 球销 / 球铰接（球面配合） → **点接触**
- 滑块 - 导轨的角点接触 → **点接触**（若支撑体为球状；否则为混合型）

**模式切换时 UI 行为**：
- 从线接触切到点接触后，"接触长度 L" 输入框变灰（只读 / AutoCalcCard 样式），不参与计算但值保留
- 从点接触切回线接触后，"接触长度 L" 解锁；若值为默认 20 请按实际滚子长度修改

**本工具实现范围**：`core/hertz/calculator.py:138-148` 根据 `geometry.contact_mode` 字段分两个分支计算；未实现椭圆接触（滚子-滚道交叉曲率的一般情况）、未实现切向摩擦接触。
