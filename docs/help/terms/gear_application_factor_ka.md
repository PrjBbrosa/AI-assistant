# 使用系数（KA）

**一句话**：把工况的冲击与不平稳性折算到齿轮名义载荷上的放大系数，校核强度时用 KA 乘上理论扭矩。

**怎么理解**：

KA 要回答的问题是："这台机器正常工况下的载荷波动能有多大？"电机驱动风机这种连续平稳工况，KA 接近 1；冲床、破碎机这种每秒几十次瞬时冲击的工况，KA 可以高达 2.5 甚至 3。忽略 KA 直接按额定扭矩算强度，实际运行寿命会大打折扣。

KA 由**驱动机特性**与**工作机特性**共同决定。电机平稳、液压马达冲击中等、内燃机冲击大；负载端轻载传送、重载搅拌、冲压各不相同。DIN 3990 / ISO 6336 给出驱动端×工作端的矩阵表格，直接查表即可。

KA 只是"工况冲击的统计估计"，真正测得的过载（如启动瞬时扭矩）若超过 KA·T 仍需额外校核。对于存在已知峰值（如伺服启停）的场合，应按实际峰值扭矩代入，不能只靠 KA。

**典型值**（DIN 3990-1 Table 1 / ISO 6336-6 简化值，按工作机特性）：

- **均匀平稳**（连续发电、带式输送机、离心泵）：KA = 1.00
- **轻微冲击**（机床进给、造纸机辊、小风扇）：KA = 1.00–1.25
- **中等冲击**（金属切削主轴、搅拌机、冶金辊道）：KA = 1.25–1.75
- **重度冲击**（冲床、破碎机、轧钢机主传动）：KA = 1.75–2.50
- **极重冲击**（挖掘机主链、往复泵主轴）：KA ≥ 2.50

选择依据：电机驱动且下游无大惯量负载从 1.00 起步；若有频繁启停、反向、或惯量突变，至少 1.25；若无法判断，按 DIN 3990 查表，不要"凭感觉"给 1.0。

**出处**：DIN 3990-1:1987 Table 1，ISO 6336-6:2019 Annex A
