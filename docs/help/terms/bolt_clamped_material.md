# 被夹件整体材料（clamped_material）

**一句话**：单层被夹件的材料选择，选定后**自动填入热膨胀系数 α_parts**，用于热工况预紧力损失估算；弹性模量 E_clamped 需在刚度章节单独填写。

**怎么理解**：

VDI 2230 将连接体分为螺栓侧与被夹件侧，两侧的柔度之比决定载荷分配系数：

```
phi = delta_P / (delta_S + delta_P)        # 基本 Φ
phi_n = n * phi                            # 含载荷导入因数 n 的载荷因数 Φ_n
```

其中 `n`（`stiffness.load_introduction_factor_n`）描述外力作用点在夹紧体中的位置（0 = 被夹件外端，1 = 分离面处）。`delta_P` 由 `stiffness.E_clamped`（刚度章节字段）和夹紧几何决定，**不由本字段直接控制**。

**本字段实际作用**：仅驱动 `operating.alpha_parts`（热膨胀系数）的**自动填入**。选材后 α_parts 字段（蓝灰 AutoCalcCard）自动填入并锁定为只读；切"自定义"后解锁为白色 SubCard，可手动输入。

## 下拉选项与热膨胀系数

| 材料 | α_parts [1/K]（自动填入值） |
|---|---|
| 钢 | 11.5×10⁻⁶ |
| 不锈钢 | 16.0×10⁻⁶ |
| 铝合金 | 23.0×10⁻⁶ |
| 铸铁 | 10.5×10⁻⁶ |
| 自定义 | 手动输入 |

弹性模量 E_clamped（影响柔度和 Φ_n）需单独在**刚度**章节的 `stiffness.E_clamped` 字段填写，或开启自动柔度建模。

## 材料对热工况的影响

铝被夹件与钢螺栓的热膨胀系数差异显著，温度变化会产生预紧力波动：

```
F_th ≈ |alpha_parts - alpha_bolt| * |Delta_T| * c_S * c_P / (c_S + c_P) * l_K    [N]
```

- α_parts（铝）≈ 23e-6，α_bolt（钢）≈ 11.5e-6：升温时铝膨胀更多，热载荷叠加
- 降温时预紧力减小，可能趋近最小夹紧力下限

## 单层 vs 多层模式

- 本字段仅在**单层模式**（`part_count = 1`）下可见，驱动全局 α_parts。
- 多层模式（`part_count = 2` 或"自定义"）下本字段隐藏，每层用 `layer_material` 独立指定热膨胀系数。

## 常见误用

- **以为选材后 E_clamped 也会自动填入**：本字段只自动填 α_parts，E_clamped 需在刚度章节手动填写，或用自动柔度建模替代。
- **多层被夹件时只填整体材料**：`part_count > 1` 时本字段不参与逐层柔度计算，须切到多层模式并逐层填写 `layer_material`。
- **自定义模式下只改 α，忘填 E_clamped**：两个字段功能不同，只改其中一个会导致热工况和刚度计算不一致。

**出处**：VDI 2230:2015 §5.2（被夹件柔度模型）；VDI 2230:2015 §5.4（热预紧力损失估算）
