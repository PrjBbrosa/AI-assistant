# 第 n 层材料（layer_material_n）

**一句话**：第 n 层材料选择；自动填入该层热膨胀系数 alpha_n（用于温度诱导损失估算）；弹性模量 E_n 需工程师单独填写，不会自动联动。

**怎么理解**：

多层被夹件中每层材料独立，各层进入柔度链的 E_n 与进入热损失计算的 alpha_n 均需逐层区分。选择材料下拉后，alpha_n 会自动填入对应预设值；E_n 需工程师自行填写（参考各材料的弹性模量典型值）：

```
delta_i = t_i / (E_i * A_i)                   # 柔度（影响 Phi_N）
FZ_T = sum((alpha_bolt - alpha_i) * t_i * ΔT) * c_T   # 温度诱导损失 [N]
```

钢/铝混合法兰是最典型案例：铝层 E=70000 MPa 仅为钢的 1/3，对柔度贡献约为等厚钢层的 3 倍；铝 alpha=23.5×10⁻⁶/K 约为钢的 2 倍，温差 ΔT=100 K 下该层热损失是纯钢连接的 ~2×。

## 下拉选项与 alpha 预设值

| 材料     | alpha 预设 [10⁻⁶/K] | E_n 处理   |
|----------|----------------------|------------|
| 钢       | 11.5                 | 用户手动填 |
| 铝合金   | 23.0                 | 用户手动填 |
| 铸铁     | 10.5                 | 用户手动填 |
| 不锈钢   | 16.0                 | 用户手动填 |
| 自定义   | 用户填               | 用户手动填 |

选择预设材料后，**仅** alpha_n 字段自动填入并锁定（AutoCalcCard 蓝灰色标识）；E_n 字段不会自动联动，需工程师根据实际材料规格书手动填写。切换为"自定义"时 alpha_n 字段解锁，可手动输入。

## 热工况影响

同温度差 ΔT 下，钢/铝复合连接比纯钢连接的温度诱导预紧力损失约高 2×。高温应用下忽略逐层材料区别会导致：

- 低估 FZ_T，进而低估最小夹紧力 F_Kmin 的余量需求；
- 判断"PASS"的安全系数可能在工作温度下已不足。

**实用建议**：工作温度差 ΔT > 50 K 且混合材料时，必须逐层填材料；ΔT < 20 K 且全钢结构时可接受统一材料简化。

## 常见误用

- **所有层默认留"钢"不修改**：铝合金盖板未改材料，导致 E 高估约 3×，柔度被低估，Phi_N 偏小（看似更安全，实则错误）。
- **自定义后忘记修改 alpha**：切到"自定义"后 alpha 字段解锁，但数值仍是上次预设值；**务必核对并手动修改**，否则与实际材料不符。E_n 字段始终需要手动填写，无论选哪种材料。
- **铸铁填 100000 或 130000 随意选择**：铸铁 E 因牌号差异大（灰铸铁约 100000–120000 MPa，球铁约 150000–170000 MPa），建议选"自定义"并填材料规格书中的实测值。
