# 螺栓强度等级

**一句话**：螺栓材料抗拉性能的标准化等级标识（如 8.8、10.9、12.9），由抗拉强度 Rm 和屈服强度 Rp0.2 共同决定，是选材与后续所有应力校核的起点。

**怎么理解**：

螺栓强度等级是**两位数的紧凑编码**，形式为 `x.y`：

- **第一位 x**：抗拉强度 Rm 的百位（Rm ≈ x × 100 MPa）
- **第二位 y**：屈服比（Rp0.2 / Rm × 10），即屈服强度占抗拉的比例 × 10

举例：
- **8.8 级**：Rm ≈ 800 MPa，Rp0.2 / Rm = 0.8 → Rp0.2 ≈ 640 MPa
- **10.9 级**：Rm ≈ 1000 MPa，Rp0.2 / Rm = 0.9 → Rp0.2 ≈ 900 MPa
- **12.9 级**：Rm ≈ 1200 MPa，Rp0.2 / Rm = 0.9 → Rp0.2 ≈ 1080 MPa

本工具内置强度等级 → Rp0.2 的查表（GB/T 3098.1 / ISO 898-1）：

| 等级 | Rp0.2 [MPa] | 典型材料 |
|---|---|---|
| 4.6 | 240 | 低碳钢（Q235、A3） |
| 4.8 | 320 | 低碳钢 |
| 5.6 | 300 | 低碳钢 |
| 5.8 | 400 | 低碳钢 |
| **8.8** | **640** | **中碳钢调质（35#、45#），最常用** |
| 9.8 | 720 | 中碳钢调质 |
| 10.9 | 900 | 中碳合金调质（35CrMo、40Cr） |
| 12.9 | 1080 | 合金调质（42CrMo、34CrMo4） |

## 怎么选

1. **默认从 8.8 开始**：工业界最通用等级，价格、可用性、强度平衡点；绝大多数通用机械螺栓都是 8.8。
2. **10.9 / 12.9 用于重载紧凑空间**：承载相同 FA 但螺栓规格需要小一号；代价：价格高、脆性较大、对氢脆敏感、不建议镀锌（需走达克罗或机械镀）。
3. **4.6 / 5.8 用于轻载或怕脆断场合**：韧性好，但 Rp0.2 低意味着允许预紧力低，大多数工况 FM 不够。
4. **不锈钢螺栓走 A 级分类**（A2-70 / A4-80）：与碳钢 x.y 编码体系不同；本工具未内置，若选不锈钢强度等级请用"自定义"手动输入 Rp0.2。

## 12.9 与氢脆风险

10.9 以上等级对**氢脆**敏感：电镀锌、酸洗、潮湿环境长时间停放都可能导致延迟断裂。工程实践：

- **10.9 及以上严禁湿法电镀**（浸锌等）
- 改用机械镀锌 / 达克罗 / 锌铝涂层
- 关键连接做拉伸 + 保持 48 小时后复测

## 公式入口

```
Rp0.2 (从强度等级自动填入) → R4 装配:  σ_allow,asm  = ν · Rp0.2
                          → R5 服役:  σ_allow,work = Rp0.2 / S_F
                          → R8 脱扣:  τ_BS 默认 = Rp0.2 · 0.6
```

## 常见误用

- **选 12.9 但不做氢脆防控**：新装后 1–2 周突然断裂是典型症状，外观无缺陷。
- **把"8.8"当成 8.8 mm 外径**：这是强度等级标识，不是尺寸。
- **用碳钢 8.8 的 Rp0.2 校核不锈钢螺栓**：A2-70 的 Rp0.2 典型 450 MPa，远低于碳钢 8.8 的 640 MPa。

**出处**：GB/T 3098.1-2010（碳钢 / 合金钢螺栓机械性能）、ISO 898-1:2013、ISO 3506（不锈钢紧固件）

> Cannot verify against original ISO standard —— Rp0.2 表值与强度等级编码来自 GB/T 3098.1 / ISO 898-1 公开条款整理，与原始标准的精确测试条件（拉伸速率、温度）可能略有出入。不锈钢等级（A2 / A4）编码体系不属于本表。
