# 接触长度 L（线接触）

**一句话**：线接触中两圆柱沿接触线有效接触的长度，决定"单位长度载荷 q = F / L"；点接触模式下该参数不参与计算。

**怎么理解**：

两圆柱轴线平行相互压触时，接触区是一条宽度为 `2b`、长度为 `L` 的细长矩形。总法向载荷 F 沿接触线摊薄为**单位长度载荷 q**：

```
q = F / L                       [F: N, L: mm → q: N/mm]
```

接触半宽 b 与 p0 都由 q 决定：

```
b = √(4·q·R' / (π·E'))          [q: N/mm, R': mm, E': MPa → b: mm]
p0 = 2·q / (π·b)                [q: N/mm, b: mm → p0: MPa]
```

所以**在线接触中 L 越长，p0 越低**：L 翻倍则 q 减半 → b 变为原来的 √(1/2) ≈ 0.707 倍 → p0 变为原来的 1/√2 ≈ 0.707 倍。换句话说，**load 不变、把滚子加长一倍，p0 降低约 29.3%**。

**本工具实现**：

- 代码 `core/hertz/calculator.py:126-133`：
  ```python
  if mode == "line":
      length_mm = _positive(float(geometry.get("length_mm", 10.0)), "geometry.length_mm")
  else:
      length_mm = 0.0  # 点接触占位，不校验、不写入结果
  ```
- 线接触时 L 必须 > 0，否则抛 `InputError`
- 点接触时 L 完全忽略；UI 把该字段变灰只读

**Warning 机制**：`calculator.py:173-174` 在线接触且 L < 5 mm 时追加 warning：

```
"接触长度较短，建议核查边缘效应和三维修正。"
```

原因：有限长圆柱端部有"端部应力集中"（edge effect），严格解法要求用 Lundberg / Palmgren 线接触端部修正或有限元。本工具仍按 `q = F/L` 均匀摊薄处理；**短滚子（L < 5 mm）时端部局部 p0 可能比赫兹均匀解算出的高 10–30%**，warning 提醒用户自己复核。

**典型值**（工程场景 → L 取值）：

- 圆柱滚子轴承：L = 滚子有效长度（通常 10~30 mm 范围）
- 圆柱齿轮啮合：L = 齿宽 b（实际接触线长度随啮合位置变化，工程取齿宽作保守上界）
- 滚针轴承：L 较长（20~50 mm），p0 相对较低
- 凸轮-滚子挺杆：L = 滚子宽度（通常 5~15 mm）

**单位约定**：mm。

**典型场景下 L 与 p0 的敏感度**（R1=30 mm, R2=∞ 平面（即 R'=30 mm）, E1=E2=210 GPa, ν=0.30, F=12000 N, E'≈115385 MPa）：

| L (mm) | q = F/L (N/mm) | b (mm)   | p0 (MPa) |
| ------ | -------------- | -------- | -------- |
| 10     | 1200           | ≈ 0.6303 | ≈ 1212   |
| 20     | 600            | ≈ 0.4457 | ≈ 857    |
| 40     | 300            | ≈ 0.3151 | ≈ 606    |

**常见坑**：

- **点接触模式忘记该字段已禁用**：UI 已置灰只读，但若绕过 UI 直接改 JSON 里的 `length_mm`，点接触分支完全忽略；此为预期行为。
- **把齿宽填成接触线长度**：齿轮啮合时实际接触线会随啮合位置变，齿顶 / 齿根处实际接触线更短。**取齿宽作 L 是偏乐观 / 不保守估计**（L 偏大 → `q = F/L` 偏小 → p0 偏低 → 安全裕度虚高）。如果真实接触线显著短于齿宽，真实 p0 会高于工具输出。严格需要 ISO 6336 接触线长度因子修正。
- **填 0 试图表示"不考虑 L"**：线接触模式下 L = 0 会抛 `InputError`，必须填 > 0 的实际值。

**出处**：Hertz 接触理论（Johnson 《Contact Mechanics》）；端部效应修正见 Lundberg G. (1939) 线接触端部分析。
