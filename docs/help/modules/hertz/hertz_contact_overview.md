# 赫兹接触应力方法总览（Hertz Contact Theory）

## 一图总览

```
   线接触（圆柱-圆柱 / 圆柱-平面）
   ═══════════════════════════════
           F
           ↓
        ┌─────┐  R1
        │     │
   ─────┴──o──┴─────  接触半宽 b
              R2
      矩形接触区：2b × L
      p0 = 2·q / (π·b),  q = F / L


   点接触（球-球 / 球-平面）
   ═══════════════════════════════
           F
           ↓
          ( )  R1
         ──o──   接触半径 a
           R2
      圆形接触区：π·a²
      p0 = 3·F / (2·π·a²)
```

## 解决什么问题

两个弹性体（齿轮齿面、滚动轴承滚子/滚道、凸轮-挺杆、滑块-导轨）在外载作用下相互压触时，接触区内的压力分布呈椭球/抛物线形，**最大值 p0 出现在接触区中心**。p0 过高会导致：

- 表面塑性变形（压痕）
- 次表面最大剪应力引起的**点蚀 / 剥落**（疲劳失效）
- 胶合 / 擦伤（配合滑动速度过高时）

工程上通过 Hertz 弹性接触理论的解析解直接计算 p0，再和材料接触疲劳许用值 [p0] 对比。本工具仅做 **最大接触应力校核**——输入几何、材料、载荷，输出 p0 和 S = [p0] / p0。

## 核心流程（4 步）

1. **选接触模式**：线接触（两圆柱轴线平行 / 圆柱-平面）或点接触（两球相切 / 球-平面）。
2. **填几何**：两接触体的曲率半径 R1、R2（平面填 0）；线接触还需接触长度 L。派生 `1/R' = 1/R1 + 1/R2`。
3. **填材料**：各体的 E、ν；派生 `1/E' = (1-ν1²)/E1 + (1-ν2²)/E2`。
4. **填载荷**：法向载荷 F；按选定模式代入赫兹公式得 p0、b 或 a。

```
线接触：b = √(4·q·R' / (π·E')),  p0 = 2·q / (π·b)
        [q = F/L in N/mm, R' in mm, E' in MPa → b in mm, p0 in MPa]

点接触：a = ((3·F·R') / (4·E'))^(1/3),  p0 = 3·F / (2·π·a²)
        [F in N, R' in mm, E' in MPa → a in mm, p0 in MPa]
```

## 赫兹理论的基本假设

赫兹解析解成立需要满足以下前提（实际工程多为近似满足）：

1. **接触区远小于物体尺寸**（"半无限空间"假设）：接触斑尺寸 ≤ 最小曲率半径的约 1/10
2. **材料各向同性、线弹性**：未进入塑性；未考虑纤维方向、晶粒取向
3. **接触面光滑且无摩擦**：切向摩擦力为零；仅法向力作用
4. **小变形**：弹性位移远小于接触体尺寸
5. **单点/单线接触**：两体只有一处相互压触

实际工程中：
- 齿轮齿面啮合时切向滚滑存在，严格需要 Carter 解，但工程用赫兹 + 摩擦因子近似即可
- 滚动轴承已趋近半无限空间假设
- 轻薄壁结构（薄圆环）会明显偏离半无限假设，赫兹公式偏保守

## 本模块实现的范围 / 不实现的范围

**已实现**（见 `core/hertz/calculator.py`）：
- 线接触：圆柱-圆柱 / 圆柱-平面；接触半宽 b、最大接触应力 p0、平均接触应力 p_mean、接触面积 2bL
- 点接触：各向同性球接触（R1 = R1x = R1y，R2 = R2x = R2y）；接触半径 a、p0、p_mean、接触面积 πa²
- 允许接触应力对比 `p0 ≤ [p0]`，计算安全系数 S = [p0] / p0
- 压力-载荷曲线：按采样点数 / 载荷上限倍率生成 F → p0 曲线

**未实现**：
- **椭圆接触（点接触的一般情况）**：当两体主曲率在两个正交方向不同（滚子-滚道交叉曲率，角接触轴承）时，接触区为椭圆，需要完整第二类椭圆积分求解。本工具的"点接触"仅覆盖各向同性球接触。
- **线接触边缘效应**：有限长圆柱端部有应力集中（"端部效应"），本工具按 q = F/L 均匀分布处理，短滚子（L < 5 mm）时给出 warning 但不做数值修正。
- **弹塑性接触**：p0 接近或超过材料流动极限时实际压力重分布为 truncated Hertz + 塑性区，本工具仍按纯弹性解计算；塑性不等效自动告警，需用户自行判断。
- **切向摩擦接触（Carter / Mindlin）**：带摩擦 / 带切向载荷的接触剪应力分布
- **动载 / 冲击接触**：按准静态处理，冲击载荷需预先折算为等效法向峰值载荷
- **次表面最大剪应力与点蚀寿命**：仅给 p0，不计算最大 Tresca 剪应力位置、循环次数下的疲劳寿命
- **凹面接触的直接支持**：`_positive(..., allow_zero=True)` 拒绝负 R；凹凸配对需用户在填 R 前自行折算

## 常见误用

- **用赫兹做薄壁接触**：薄板 / 薄壁管壳的弯曲变形量级可能远大于局部压缩量，此时赫兹假设失效，实际接触压力分布会被弯曲平均化。
- **接触面有显著粗糙度**：宏观赫兹压力 × 粗糙度放大系数 ≈ 实际局部峰值，可达 3~5 倍。本工具按光滑表面计算，粗糙工况需另外乘放大系数。
- **旋转速度高引发的离心效应**：高速滚动轴承、飞轮接触中离心力会改变 R'，本工具不做修正。
- **忽略模式切换**：从线接触切到点接触后忘记重填 R1、R2；工具按新模式的公式计算，但若 R1/R2 还停留在"圆柱半径" 意义下，结果会全盘错乱。
- **用 p0 和单向拉伸的 Rp0.2 比较**：接触应力是三向压应力，允许值远高于单向拉伸屈服值；不能套用 σ_allow = Rp0.2 / 1.5 类的规则。
