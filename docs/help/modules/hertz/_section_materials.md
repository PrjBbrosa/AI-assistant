# 材料参数章节

本章节用于**填写两接触体的弹性模量 E 和泊松比 ν**。两组材料参数共同决定等效弹性模量 E'，影响接触斑大小与 p0 的数值。

## 为什么要填这些

赫兹公式里描述"两个弹性体相互抵抗变形的能力"的参数是等效模量 E'：

```
1/E' = (1 - ν1²) / E1 + (1 - ν2²) / E2        [E: MPa → E': MPa]
```

代码实现 `core/hertz/calculator.py:118`：
```python
e_eq = 1.0 / (((1.0 - nu1*nu1) / e1) + ((1.0 - nu2*nu2) / e2))
```

这个 E' 同时出现在线接触公式 `b = √(4qR'/(πE'))` 和点接触公式 `a = ((3FR')/(4E'))^(1/3)` 里——E' 越大代表整体越"硬"，接触斑越小、p0 越高。

同样的载荷、同样的几何：
- 钢-钢（E1 = E2 = 210 GPa，ν = 0.30）→ E' ≈ 115 GPa
- 钢-铝（E1 = 210 GPa, E2 = 70 GPa）→ E' ≈ 58 GPa（差一倍）
- 钢-塑料（E2 ≈ 3 GPa）→ E' ≈ 3.3 GPa（几乎由塑料决定）

## 输入 / 产出

**输入**：
- **接触体 1 材料** 下拉（42CrMo / GCr15 / 45钢 / 铸铁 HT250 / 铝合金 6061-T6 / 自定义）
  - 选中任一牌号后，E1 / ν1 自动填入并以 `AutoCalcCard` 样式锁定只读
  - 选"自定义"可手动输入任意 E1 / ν1
- **弹性模量 E1 [MPa]**：必填，> 0（默认 210000）
- **泊松比 ν1 [-]**：必填，0 < ν1 < 0.5（默认 0.29）
- **接触体 2 材料** / **E2 [MPa]** / **ν2 [-]**：同上，默认 45钢 / 210000 / 0.30

**产出**：
- `derived.e_eq_mpa = E'` — 等效弹性模量 [MPa]

## 方法差异

本工具使用 **Young 模量 + Poisson 比 → 等效模量** 的简化赫兹公式（线性各向同性弹性）。未实现：
- **各向异性材料**（纤维增强复合材料、取向晶体）
- **非线性弹塑性**（接触区进入塑性后的实际压强重分布）
- **温度影响**：塑料 E 受温度剧烈影响（PA66 23°C 约 3 GPa，80°C 可能降到 1.5 GPa），本工具按用户填入的数值不做修正；设计塑料齿轮接触时应按工作温度查材料 E。

## 常见坑

- **把 GPa 当 MPa 填**：钢 E = 210 GPa，要填 `210000` 不是 `210`。UI 单位标注为 MPa；自定义时看清单位。
- **泊松比填了 0.5**：`_nu` 校验要求 `0 < ν < 0.5`，填 0.5（不可压缩极限）会抛 `InputError`；橡胶等场合实际取 0.49 已经很接近理论极限。
- **塑料材料按常温值填，实际高温工况**：PA66 / POM 等工程塑料 E 随温度变化大，设计温升工况时需按实际工作温度查材料手册后手填。
