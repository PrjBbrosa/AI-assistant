# 校核目标章节

本章节用于**设定最大接触应力的许用值 [p0]，以及压力-载荷曲线的采样参数**。这是赫兹接触校核唯一的硬性判据：计算得到的最大接触应力 p0 与 [p0] 比较得出安全系数与 PASS/FAIL 结论。

## 为什么要填这些

赫兹公式本身只给出在某一组"几何 + 材料 + 载荷"下的接触斑尺寸和最大接触应力 p0（单位 MPa），它并不告诉你这颗接触副"合不合格"——合格与否由许用接触应力 [p0] 决定，而 [p0] 来自材料接触疲劳极限、热处理工艺、润滑状态、使用寿命要求等。

- **允许最大接触应力 [p0]**：本工具按输入的许用值直接对比 `p0 ≤ [p0]` 得出 `contact_stress_ok`，并计算 `safety_factor = [p0] / p0`。本工具**不自动查材料接触疲劳表**，需用户按材料手册 / 齿轮规范 / 滚动轴承寿命曲线自行查取后填入。
- **压力-载荷曲线采样点数**：决定结果图上 F→p0 曲线的分辨率；11~201 之间有效，超出会被钳位并给出警告。
- **曲线载荷上限倍率**：曲线终点载荷 = 设计载荷 × 倍率，1.05~2.0 有效；用于观察"如果载荷增大到 1.3 倍、1.5 倍时 p0 变化多少"。

## 输入 / 产出

**输入**：
- 允许最大接触应力 [p0] [MPa]（必填，> 0，默认 1500）
- 采样点数 [点]（默认 41）
- 载荷上限倍率 [-]（默认 1.30）

**产出**：
- `check.allowable_p0_mpa` — 用户输入的许用值（回显）
- `check.safety_factor = [p0] / p0` — 接触应力安全系数
- `checks.contact_stress_ok` — 布尔 PASS/FAIL
- `curve.force_n` / `curve.p0_mpa` — 压力-载荷曲线采样

## 方法差异

本工具只有一条判据 `p0 ≤ [p0]`；没有分"齿面接触疲劳 / 滚动接触疲劳 / 静强度接触"多档许用值。这意味着若工程场合要求区分（例如齿轮 σH,lim × ZN × ZR × ...），需要用户在填 [p0] 之前**自己算好最终许用值**，工具不做二次折减。

## 常见坑

- **把材料屈服强度 Rp0.2 当接触许用值填入**：屈服强度反映单向拉伸极限；接触应力是三向压应力，通常允许值远高于 Rp0.2（钢淬火 [p0] 可达 1500–3000 MPa）。按接触疲劳极限或 Hertz 许用表取值。
- **忽略循环次数影响**：材料手册给的接触疲劳极限往往基于 10^7 次循环；长寿命场合需要乘寿命系数 ZN（< 1），短寿命允许系数 > 1。本工具不自动处理，需预先折算。
- **采样点数调得很大期望更精确**：曲线仅用于可视化，p0 的精度由赫兹公式本身决定，不受采样密度影响。
