# 载荷输入章节

本章节用于**填写作用在接触区的法向载荷 F**。赫兹接触按纯法向载荷计算，切向摩擦力、滚动粘滑、冲击附加载荷均需用户预先折算到 F 后再输入。

## 为什么要填这个

赫兹解析解只对"垂直于切平面的法向压力"成立——它假设接触面无摩擦、两体仅有法向应力。因此所有实际工况里的横向力、滚动剪切、冲击尖峰等，都必须折到"峰值法向载荷"再输入：

- **静态加载**：F = 实际法向压力（N）
- **冲击工况**：F = F_static × Ka（载荷系数），Ka 按工况取 1.2~2.5
- **滚动接触**：F = 最大轮齿法向力（沿接触线分布的最大值，不是平均值）
- **振动工况**：F = 静载 + 动载峰值

## 输入 / 产出

**输入**：
- **法向载荷 F [N]**：作用在接触区法向方向的载荷，必填且 > 0（默认 12000）

**产出**：
- `contact.normal_force_n = F` — 回显
- `contact.p0_mpa` — 最大接触应力
- `contact.p_mean_mpa` — 平均接触应力
  - 线接触：`p_mean = q / (2·b)` [q: N/mm, b: mm → p_mean: MPa]
  - 点接触：`p_mean = F / (π·a²)` [F: N, a: mm → p_mean: MPa]
  - p0 / p_mean = 4/π ≈ 1.27（线接触）；p0 / p_mean = 3/2 = 1.5（点接触）

## 方法差异

- **单一法向载荷**：本工具不支持"同时输入法向 + 切向摩擦力"计算带摩擦的接触应力场；若需要评估摩擦导致的表面剪应力分布，需要切线 Carter/Mindlin 解或有限元，超出本工具范围。
- **静态计算，不做循环寿命推导**：输入的 F 视为单一工况的峰值；如需做接触疲劳寿命 N-p 曲线，需把 F 对应的 p0 与 S-N 曲线外接。
- **点接触忽略 L**：如章节 2 所述，点接触模式下 `geometry.length_mm` 被禁用，载荷 F 直接用于 `a = ((3·F·R')/(4·E'))^(1/3)` 和 `p0 = 3F/(2πa²)`。

## 曲线采样（可视化）

基于章节 1 的"采样点数 + 载荷上限倍率"，工具在 `[0, F · scale]` 区间均匀采样并计算对应 p0，用于观察 p0 如何随 F 变化：

- 线接触：`p0 ∝ √F`（因为 `q ∝ F`，`b ∝ √q`，`p0 = 2q/(πb) ∝ √q`）
- 点接触：`p0 ∝ F^(1/3)`（因为 `a ∝ F^(1/3)`，`p0 = 3F/(2πa²) ∝ F^(1/3)`）

这两个比例关系是赫兹公式的重要工程特性：**载荷加倍，线接触 p0 增加约 41%，点接触 p0 只增加约 26%**——说明点接触对载荷变化更"钝感"。

## 输入 / 产出单位对照

| 字段 | UI 单位 | 内部使用 | 备注 |
| ---- | ------- | -------- | ---- |
| F（法向载荷） | N | N | 正值，> 0 |
| p0（派生） | MPa | MPa | 最大接触应力 |
| p_mean（派生） | MPa | MPa | 平均接触应力 |
| q（线接触派生） | N/mm | N/mm | q = F/L |

## 常见坑

- **把扭矩当载荷填**：齿轮接触时实际的法向齿面力 `Fn = T / (r · cos α)` 需要先折算（T 为扭矩，r 为分度圆半径，α 为压力角），不是直接填扭矩值。
- **忘记乘冲击系数**：电机频繁启停、冲击工况按"额定载荷"填会严重低估 p0；建议在载荷上乘一个工况系数 Ka（类似齿轮的 KA）后再填入。
- **填入的是总载荷而非单点接触载荷**：轴承多个滚动体分担时，应按"最大受力滚动体"的法向力填，而不是整个轴承径向力。
- **曲线载荷上限倍率设得过低（< 1.05）或过高（> 2.0）**：代码会自动钳位到 [1.05, 2.0] 区间并返回 warning；若希望观察更大范围需分多次计算取值。
