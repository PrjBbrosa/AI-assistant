# 校核目标章节

本章节用于**定义校核门槛与工况系数**：防滑安全系数下限 S_slip,min、应力安全系数下限 S_σ,min、工况系数 KA 与曲线采样点数。这些设置决定每一项"是否通过"的判据门槛，也决定把名义载荷放大到多少设计载荷去校核。

## 为什么要填这些

过盈配合的通过/不通过结论，本质上是**"容量 / 需求"与"实际应力 / 允许应力"两类比值是否达到最小安全系数**。章节里的三个参数分别控制三件事：

- **S_slip,min**（防滑最小安全系数）→ 直接进扭矩、轴向力、联合作用、张口缝校核；例如扭矩通过条件为 `T_cap_min / T_design ≥ S_slip,min`。
- **S_σ,min**（材料最小安全系数）→ 进轴侧与轮毂侧 von Mises 应力校核：`Re / σvm ≥ S_σ,min`。
- **KA**（工况系数）→ 把名义扭矩、名义轴向力、名义径向力、名义弯矩**全部**乘 KA 得到设计载荷；再与容量比较；KA 是**需求侧放大**，不是能力侧折减。

曲线采样点数只影响压入力曲线图的平滑度，不改变任何校核结论。

## 输入 / 产出

**输入**：
- S_slip,min（建议 1.1~1.5，重要部件 1.3~1.8）
- S_σ,min（建议 1.1~1.8，按材料保证值与工况严酷度取）
- KA（DIN 3990 风格 1.0~2.25，冲击越大越高）
- 曲线采样点数（11~201，默认 41，仅影响显示）

**产出给下一步**：
- 设计载荷（`T_design = KA · T_req`、`F_design = KA · F_req` 等）
- 各校核项 PASS / FAIL 的阈值

## 核心校核逻辑（本工具实现）

```
设计载荷       T_design = KA · T_req                   [N·m]
               F_design = KA · F_req                   [N]

扭矩容量       T_cap = μ_T · p_min · π·d·L · (d/2) / 1000
                     [μ_T: 无量纲, p_min: MPa, d: mm, L: mm → T_cap: N·m]
                     （除以 1000 把 N·mm 转 N·m）

PASS 判据：
   扭矩能力         T_cap ≥ S_slip,min · T_design
   轴向能力         F_cap ≥ S_slip,min · F_design
   联合作用         sqrt( (T_design/T_cap)² + (F_design/F_cap)² ) ≤ 1 / S_slip,min
   张口缝           p_min ≥ p_gap
   最大过盈端覆盖   δ_max ≥ δ_required
   轴应力           Re_shaft / σvm_shaft,max ≥ S_σ,min
   轮毂应力         Re_hub   / σvm_hub,max   ≥ S_σ,min
```

**注意**：所有滑脱类校核按 **p_min**（最小过盈端）；所有应力校核按 **p_max**（最大过盈端）。章节里的两个安全系数门槛分别适用于这两条支线。

## 常见坑

- **KA 和 S_slip 双重放大误差**：KA 已经把名义放大到设计载荷；S_slip 是设计载荷之上的安全裕度。不要再在 T_req 里"预先乘一个 1.5"充当安全系数，否则校核偏保守。
- **S_slip,min < 1**：任何 < 1 的取值相当于让设计容量低于设计载荷，直接失去工程意义；本工具按 `>0` 检查，但建议不低于 1.1。
- **S_σ,min 用到屈服以上的材料**：本工具使用 Re（屈服强度）作为分母；对于要求"只许用弹性阶段"的重要件，建议把 S_σ,min 取到 1.5 以上。
- **KA 不等于冲击系数 Ks**：KA 是按"工况类型"查表得到的统一系数；DIN 3990 的 Ks/Kv/KH 等是齿轮专属，不要混淆。
