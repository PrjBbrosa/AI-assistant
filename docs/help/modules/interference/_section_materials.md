# 材料参数章节

本章节用于**填写轴与轮毂两侧的弹性常数和屈服强度**：弹性模量 E、泊松比 ν、屈服强度 Re。这些参数共同决定过盈量 → 接触压力的换算（E / ν 控制柔度），以及最大过盈端是否触发屈服（Re 控制应力安全系数）。

## 为什么要填这些

- **弹性模量 E** 进柔度：`c_shaft ∝ r/E_s`、`c_hub ∝ r/E_h`。E 越大越刚，同等过盈下压力越高。
- **泊松比 ν** 出现在柔度表达式的几何因子里；钢材间差异极小（0.28~0.30），但钢-铝或钢-铸铁配对时必须按真实材料取值。
- **屈服强度 Re** 只进应力校核：`S_shaft = Re_s / σvm_shaft,max`、`S_hub = Re_h / σvm_hub,max`，要求 S ≥ S_σ,min。

## 材料下拉与自动填充（本工具实现）

选中内置材料时，系统会把 E 和 ν 锁定为只读并填入默认值；屈服强度 Re 仍然需要手动输入（同一牌号不同热处理状态下 Re 差异较大）。

## 输入 / 产出

**输入**：
- 轴材料 + 轴 E、ν、Re
- 轮毂材料 + 轮毂 E、ν、Re

**产出给下一步**：
- 轴 / 轮毂径向柔度 c_shaft, c_hub → 合成 c_total
- 最大过盈端的应力安全系数 S_shaft, S_hub

## 材料配对对结果的影响

**两侧 E 差异越大，越需要注意**：
- 钢-铝（E 比约 3:1）：同等过盈下，铝轮毂变形吸收了更多过盈 → 实际接触压力显著低于两侧都钢的估算；软侧材料也更容易进入屈服。
- 钢-铸铁（E 比约 1.75:1）：偏差不大，但铸铁 Re 低、脆性大，要加大 S_σ,min。
- 同牌号钢：两侧近似等 E，最接近经典课本算例。

## 屈服强度 Re 的几个选择陷阱

1. **用抗拉强度 Rm 代替 Re**：本工具 Re 专指"屈服"或"0.2% 条件屈服 σ0.2"；用 Rm 会让安全系数虚高 30~50%。
2. **调质件按退火值取 Re**：40Cr 退火 Re ≈ 350 MPa、调质 Re ≈ 650 MPa，差近一倍；必须按真实热处理状态。
3. **灰铸铁没有真正屈服点**：工程上按 σ_0.2 或抗拉 × 0.6 近似；本工具把 Re 字段当作"允许应力分母"，对脆性材料建议再把 S_σ,min 上调到 1.8~2.2。

## 常见坑

- **把 E 填成 GPa 数值**（例如 210）：本工具要求 MPa，钢应填 210000。差 1000 倍会让压力估算差 1000 倍。
- **ν 填 0.5**：不允许（体积不可压缩），本工具对 0 < ν < 0.5 做开区间校验，刚好等于 0.5 会报错。
- **轮毂材料当"薄弱侧"不自觉**：压铸铝 AlSi9Cu3 轮毂配钢轴，Re_hub 约 180 MPa，先降 S_σ,min 比换材料更快见效；但注意是否已经破坏设计余量。
