# 几何与过盈章节

本章节用于**填写过盈配合的几何尺寸和过盈量窗口**：配合直径 d、轴内径 d_i、轮毂外径 D、配合长度 L，以及过盈量的来源（直接输入 / 优选配合 / 偏差换算）。这些尺寸是弹性柔度、接触压力和所有后续校核的几何基础。

## 为什么要填这些

- **配合直径 d** 决定接触半径 r=d/2 和接触面积 A=π·d·L，直接影响扭矩容量和压力→应力换算。
- **轴内径 d_i** 决定轴是实心（d_i=0）还是空心；空心轴在同样过盈下接触压力会**降低**（轴侧更柔），传递能力也随之下降。
- **轮毂外径 D** 决定厚壁轮毂的几何因子 `(D²+d²)/(D²−d²)`；D 越大，轮毂越刚，同等过盈下接触压力越高，但轮毂应力也会改变。
- **配合长度 L** 线性放大接触面积：L 翻倍 → 扭矩 / 轴向容量翻倍。
- **过盈量窗口 (δ_min, δ_max)** 是弹性柔度的直接输入；δ_min 决定最小可保证传递能力，δ_max 决定应力峰值。

## 三种过盈输入模式（本工具实现）

```
fit.mode = "直接输入过盈量"
   → 直接使用 fit.delta_min_um / fit.delta_max_um 输入

fit.mode = "优选配合"
   → 从内置 ISO 286 子集（H7/p6、H7/s6、H7/u6、6~50 mm）查表
   → 自动换算 delta_min / delta_max 填入并锁定

fit.mode = "偏差换算"
   → 输入轴上下偏差 (es, ei) 与孔上下偏差 (ES, EI)，自动换算：
        delta_min = ei − ES                         [um]
        delta_max = es − EI                         [um]
   → 若 delta_max ≤ 0 或 delta_min < 0，直接报错（当前组合不是过盈配合）
```

**重要说明**：优选配合表只内置三种代号、直径 6~50 mm；超出范围请切到"直接输入过盈量"或"偏差换算"模式。

## 输入 / 产出

**输入**：
- 配合直径 d（mm，大于 0）
- 轴内径 d_i（mm，0 表示实心；必须 < d）
- 轮毂外径 D（mm，必须 > d）
- 配合长度 L（mm，大于 0）
- 过盈窗口 δ_min / δ_max（um，直径方向）或等效的偏差 / 配合代号

**产出给下一步**：
- 接触面积 `A = π · d · L`（mm²）
- 厚壁轮毂几何因子 `(D² + d²) / (D² − d²)`
- 轴柔度修正因子（实心 = 1，空心 > 1）

## 过盈量 δ 的物理定义

δ 是**装配前轴与孔的直径差**（轴大、孔小）。输入为 um（微米），即 0.001 mm。

```
δ = d_shaft,before_assembly − d_hub_bore,before_assembly [um, 直径方向]
```

容易搞错的三个点：
1. **直径差不是半径差**：若图纸给"径向过盈 0.025 mm"，换成直径差应输入 50 um，不是 25 um。
2. **δ_min / δ_max 是公差窗口**：加工允许的最窄和最宽过盈，**不是同一连接的同一次装配**，而是**一批产品中可能出现的上下限**。
3. **粗糙度压平会先扣一部分**：有效过盈 `δ_eff = max(0, δ − s)`，其中 `s = k·(Rz_s + Rz_h)`（见粗糙度章节）。

## 常见坑

- **把轴内径填成半径**：d_i 是直径；若实际内孔半径 6 mm，应输入 12 mm。
- **让轮毂外径 ≤ d**：校核会直接报错；最小推荐 D ≥ 1.5·d，否则轮毂太薄无法作为厚壁圆筒处理。
- **δ_max < δ_min**：加工偏差符号填反常见；校核会直接报错。
- **用 δ_mean 单值**：本工具不接受单个过盈量；必须给出窗口（允许 δ_min = δ_max，但会失去公差散差信息）。
- **"直接输入过盈量"模式下填了配合代号**：此模式忽略配合代号；要联动必须切到"优选配合"。
