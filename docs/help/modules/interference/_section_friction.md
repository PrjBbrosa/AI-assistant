# 摩擦与粗糙度章节

本章节用于**分开输入服役阶段扭矩方向摩擦 μ_T、轴向方向摩擦 μ_Ax、装配阶段通用摩擦 μ_Assy，以及粗糙度压平模型**。这三个摩擦系数直接决定扭矩 / 轴向容量和装配压入力；粗糙度压平量则决定"实际参与压力换算的有效过盈量"。

## 为什么要填这些

摩擦系数的工程意义：

- **μ_T（扭矩方向）** → 扭矩容量 `T_cap = μ_T · p · A · r`
- **μ_Ax（轴向方向）** → 轴向容量 `F_cap = μ_Ax · p · A`
- **μ_Assy（装配通用）** → 压入力 `F_press = μ_Assy · p · A`；在 `force_fit` 模式下由装配章节的 μ_press_in 覆盖

分开填的原因：
1. 扭矩 / 轴向的服役摩擦理论值在**恒定接触压力**下大致相同，但实测会因表面织构方向性（刀纹圆周 vs 纵向）而差 10~20%。
2. 装配摩擦是**动摩擦 / 滑动摩擦**，通常比静摩擦低 20~30%；直接复用 μ_T 会让压入力估高、装配设备选型过度。

粗糙度压平的工程意义：

配合面上微观凸峰在装配瞬间被**塑性压平**，带走一部分名义过盈量。**有效过盈量**决定接触压力，不是铭牌值：

```
s = k · (Rz_shaft + Rz_hub)                  [um, k: 无量纲]
δ_eff = max(0, δ − s)                        [um]
p = (δ_eff / 2) / c_total · (1/1000)         [p: MPa]
```

不扣 s 会把实际压力估高 20~40%，导致工具"通过"但实物不稳。

## 输入 / 产出

**输入**：
- 表面状态（干摩擦 / 轻油润滑 / 自定义）
- μ_T、μ_Ax、μ_Assy（服役扭矩 / 轴向 / 装配通用）
- 粗糙度压平模型（选择 k 来源）
- k（压平系数，0.4 或 0.8 或自定义）
- Rz_shaft、Rz_hub（um）

**产出给下一步**：
- 有效过盈量 `δ_eff = max(0, δ − s)` 三档（min / mean / max）
- 扭矩 / 轴向 / 压入力三个容量

## 常见坑

- **μ 全填一样**：掩盖 μ_Assy 应当低于服役摩擦的常识；压入力估算会偏高。装配设备选型若按此选定，可能过度裕量。
- **把 Ra 当 Rz 填**：Ra 是算术平均、Rz 是峰谷最大差；一般 Rz ≈ 4~6 · Ra（标准钢磨削面）。误把 Ra=0.8 填进 Rz 字段会让压平量估低 4 倍，压力虚高。
- **高精磨削（Ra 0.1）仍按默认 Rz 6.3**：实际 s 会很小，不扣减几乎不影响；但若表面是车削 Rz 25，不调此值会严重偏差。
- **润滑状态跟材料配对脱钩**：本工具按 (材料配对, 表面状态) 查表；请保证下拉选择反映真实加工与装配条件。
- **老图纸上标 Rz 来自 DIN 4768 老定义**：与 ISO 21920（新定义）数值略有差异；批量生产建议统一口径。
