# 圆柱面过盈配合（DIN 7190 风格）

## 一图总览

```
  过盈量 delta (um)                      接触压力 p (MPa)
  ────────────────────    弹性柔度      ───────────────────────
  delta_min / delta_max   ──────────►   p_min / p_max
       │                                       │
       │                                       ├──► 扭矩能力  T_cap = μ_T · p · A · r
       │                                       ├──► 轴向能力  F_cap = μ_Ax · p · A
       │                                       ├──► 应力校核  σvm ≤ Re / S_σ
       │                                       └──► 压入力    F_press = μ_Assy · p · A
       │
       └──► 粗糙度压平 s = k·(Rz_s + Rz_h) 先扣除，再做上面的换算
```

本模块按 DIN 7190 圆柱面过盈配合的思路，从最小 / 最大过盈量出发，算出对应的**接触压力窗口**，然后分四条支线校核：滑脱（扭矩 / 轴向 / 联合作用）、张口缝（径向力 + 弯矩）、应力（轴 / 轮毂屈服裕度）、装配力（压入 / 压出 / 热装温度）。

## 解决什么问题

在轴与轮毂之间靠**弹性预紧产生的摩擦力**来传递扭矩或轴向力时，需要回答四个问题：

1. **能不能传得动**：最小过盈工况下扭矩 / 轴向能力是否高于设计载荷（带 KA）？
2. **会不会张口**：径向力或弯矩是否让局部接触压力归零？
3. **会不会压坏**：最大过盈工况下轴或轮毂是否超过屈服？
4. **能不能装进去**：装配压入力或所需加热温度是否在设备 / 材料允许范围内？

## 核心流程（5 步）

### Step 1 — 建立几何与材料模型

输入配合直径 d、轴内径 d_i（0 表示实心轴）、轮毂外径 D、配合长度 L，以及两侧材料的弹性模量 E、泊松比 ν、屈服强度 Re。本工具按厚壁圆筒 + 轴柔度修正建立线弹性模型：

```
c_shaft = r/E_s · (1 − ν_s) · F_hollow(d_i/d, ν_s)      [mm/MPa]
c_hub   = r/E_h · ((D² + d²)/(D² − d²) + ν_h)           [mm/MPa]
c_total = c_shaft + c_hub
```

`F_hollow(bore_ratio, ν)` 是空心轴相对实心轴柔度的修正因子（bore_ratio=0 时为 1）。`c_total` 单位 **mm/MPa**：每 MPa 接触压力引起的径向半径变化。

### Step 2 — 换算过盈窗口 → 接触压力窗口

过盈量 δ 的工程定义是**直径方向**的额外尺寸差。粗糙度会在装配瞬间把微峰压平，**有效过盈** `δ_eff = max(0, δ − s)`。

```
p = (δ_eff / 2) / c_total · (1/1000)                    [δ_eff: um, c_total: mm/MPa → p: MPa]
```

`/2` 是因为 c_total 用的是半径柔度；`/1000` 把 um 换成 mm。由此得到 p_min / p_mean / p_max 三个工况。

### Step 3 — 能力校核 + 联合作用 + 张口缝

```
T_cap = μ_T · p_min · π·d·L · (d/2) / 1000              [T_cap: N·m, p: MPa, d,L: mm]
F_cap = μ_Ax · p_min · π·d·L                            [F_cap: N]
```

联合作用校核按 RSS：`sqrt((T/T_cap)² + (F/F_cap)²) ≤ 1 / S_slip`。

张口缝校核：径向力 + 弯矩会在局部抬升所需接触压力。本工具用 `p_gap = F_r/(d·L) + 2.25·M_b·1000/(d·L²)`，并要求 `p_min ≥ p_gap`；这是取 QW=0 的保守简化，不是 DIN 标准原公式（Cannot verify against original DIN standard）。

### Step 4 — 应力校核（最大过盈端）

轮毂内壁按 Lamé 解周向应力 `σθ = p · (D²+d²)/(D²−d²)`；轴和轮毂的 von Mises 应力按标准厚壁圆筒二维应力合成。安全系数 `S = Re / σvm`，要求 `S ≥ S_σ,min`。

### Step 5 — 装配工艺校核

| 装配模式 | 关注量 | 输出 |
|---|---|---|
| `manual_only` | 仅压入力估算 | F_press = μ_Assy · p · π·d·L |
| `shrink_fit`（热装） | 需求加热温度 | T_hub = T_amb + (δ_max + clearance) / (α_h · d) |
| `force_fit`（压装） | 压入 / 压出力 | F_in = μ_in · p · π·d·L（μ_in, μ_out 分开） |

## 本模块实现的范围

**已实现**：
- 实心轴 + 空心轴（通过柔度修正）
- 厚壁对称轮毂（圆筒形）
- 分离的**服役摩擦** μ_T, μ_Ax 和**装配摩擦** μ_Assy（或 μ_in / μ_out）
- 粗糙度压平修正（k = 0.4 或 0.8，可自定义）
- 过盈来源三选一：直接输入 / 优选配合 / 偏差换算
- 联合作用（RSS）、张口缝（QW=0 保守）、应力（Lamé + von Mises）
- Fretting 风险定性评估（基于规则，不改变主 verdict）

**未实现 / 排除**：
- **阶梯轮毂** / 非对称轮毂（必须按圆柱等厚处理）
- **塑性修正**（纯弹性模型，接近屈服时偏保守）
- **离心力**（高速工况需要单独评估）
- **装配温差对服役过盈的影响**（服役温度与装配温度相同时才一致）
- **非线性接触压力分布**（端部应力集中、油膜分离等按均匀接触简化）
- **优选配合表**只内置了 H7/p6、H7/s6、H7/u6 三个代号，且仅覆盖 6~50 mm 直径段

## 常见误用

1. **把"过盈量"当作半径差**：δ 永远是**直径**方向；输入半径差会让压力翻倍，结果严重偏保守。
2. **忽略粗糙度压平**：加工 Rz=10 um 的两件组合，装配后 s ≈ 4~8 um，直接被吃掉；不扣减会让压力估高 20~40%。
3. **用 KA=1**：任何带冲击或循环的工况都不是名义载荷；KA 吃不进去会让防滑校核对真实工况虚高。
4. **只看最小过盈端**：应力和装配温度必须按**最大过盈**校核，否则会漏掉压坏或热装爆温风险。
5. **把 μ_Assy 填成 μ_T**：服役摩擦（稳态滑移极限）与装配摩擦（动态润滑）不同；混用会让压入力或扭矩能力估错。
