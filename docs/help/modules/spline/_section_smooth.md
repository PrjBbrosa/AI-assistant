# 光滑段过盈章节

本章节**仅在`联合`模式下启用**。用于录入花键轴光滑过盈段与轮毂孔之间的 DIN 7190 圆柱过盈配合参数：配合直径 d、轴内径 d_i、轮毂外径 D、配合长度 L、退刀槽宽度、最小 / 最大过盈量 δ、轴 / 轮毂材料属性、摩擦系数与表面粗糙度。`仅花键`模式下整章被锁定为 `AutoCalcCard` 禁用态，不参与计算。

## 为什么要填这些

花键齿面只校核了"齿传扭矩时会不会把齿压溃"。但实际结构中花键轴往往同时有一段光滑圆柱压入轮毂孔，起 **定位、防滑、分担扭矩 / 轴向力** 作用。这一段的承载能力必须单独走 DIN 7190 Lamé 解析解：

- **防滑**：过盈产生的接触压力 × 摩擦系数 × 接触面积 ≥ 实际切向 / 轴向载荷。
- **屈服**：过盈产生的轴 / 轮毂应力 ≤ 许用应力（按最大过盈量 δ_max 和弹性模量 E 推出）。

如果这一段强度不够，即便花键齿面预校核 PASS，连接在实际工况下仍可能打滑或轮毂胀裂。

本章节字段直接委托给 `core/interference/calculator.py:calculate_interference_fit` 处理；花键模块做的事是：
1. 用退刀槽宽度把名义配合长度扣减为**有效**配合长度（`L_fit = L_nominal - relief_groove`）。
2. 把已预乘 K_A 的 T_design 和 F_ax_design 传入，同时强制 `application_factor_ka = 1.0`，避免重复放大（见 `calculator.py:174` 注释）。
3. 把返回的 `pressure_mpa` / `capacity` / `safety` / `press_force_curve` 整块塞回场景 B 结果。

## 输入 / 产出

**输入**（全部为 `smooth_fit` / `smooth_materials` / `smooth_friction` / `smooth_roughness` section）：
- **配合直径 d**、**轴内径 d_i**（实心填 0）、**轮毂外径 D**
- **名义配合长度 L**、**退刀槽宽度**（计算时扣除）
- **δ_min / δ_max**（μm，直径值）：过盈量允许范围
- **轴 / 轮毂材料**：下拉 `45钢 / 40Cr / 42CrMo / 自定义`，选非自定义时自动填充 E / ν / Re 并锁为 `AutoCalcCard`
- **轴 / 轮毂 E、ν、Re**：弹性模量 [MPa]、泊松比、屈服强度 [MPa]
- **μ_torque、μ_axial、μ_assembly**：扭矩、轴向、装配三种摩擦系数
- **Rz_shaft、Rz_hub**：轴 / 轮毂表面粗糙度 [μm]（用于过盈折减）

**产出**：
- `pressure_mpa.p_min / p_mean / p_max`：三档过盈下的接触压力
- `capacity.torque_min / mean / max_nm`：三档过盈下的打滑扭矩容量
- `safety.torque_sf`、`safety.combined_sf`：防滑 / 综合安全系数
- `stress_mpa`：轴 / 轮毂等效应力
- `press_force_curve`：UI 显示的压入力-过盈量曲线

## 常见坑

- **δ 填成半径值**：`smooth_fit.delta_min_um` 与 `delta_max_um` 约定为**直径值**（与 DIN 7190 / ISO 286 公差标注一致）。若按半径给，会把实际过盈缩小一半，算出的接触压力低估一半。
- **退刀槽宽度漏填或填错**：实际花键-光滑段过渡处通常有一段不承载的让刀凹槽，图纸上会单独标注。填 0 相当于把整段视作有效承载长度，会高估扭矩容量；填得过大则 L_fit 变负（工具会抛 `退刀槽宽度 >= 配合长度` 的 `InputError`）。
- **轴内径 d_i 填成花键齿根圆 d_f1**：光滑段和花键段几何不同。光滑段的 d_i 指**轴的内孔直径**（空心轴），与花键齿根无关。实心轴填 0。
- **μ 选偏高**：干净干摩擦 μ 可达 0.20~0.25，但 DIN 7190 的防滑校核默认用保守值 0.10~0.14（考虑加工残留油、氧化、装配过程摩擦不均）。取 0.20 会让防滑 SF 虚高约 40%，正式设计取 0.12 附近。
- **`仅花键`模式下的默认值看起来仍然在**：字段值保留但整章锁为 `AutoCalcCard`；切回`联合`时保留上次编辑，不用重填。
