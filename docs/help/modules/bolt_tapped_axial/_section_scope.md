# 适用范围与建模假设

本章节**只是说明该模块的使用边界**，不包含任何需要输入的字段。填任何参数前，请先确认你的工况符合这里列出的假设——不符合就用错了模块。

## 这个模块适用于什么

**核心场景**：螺栓拧入螺纹对手件、**中间无被夹件**、承受**纯轴向拉载荷**。

典型例子：

- 螺栓拧入发动机缸体、变速器壳体的螺纹孔
- 螺栓锁入焊接螺母、压铆螺母的盲孔
- 球阀 / 阀盖螺栓拧入阀体本体
- 螺栓直接旋入铝合金 / 铸铁厚壁壳体的螺纹孔

这些场景有一个共同结构：

```
螺栓头 → 被"旋合"进对手件的螺纹孔 → 外部拉力想把螺栓拔出来
没有"被夹件"这一层结构
```

## 为什么不能当成普通 VDI 2230 夹紧连接

普通 VDI 2230（`bolt_page.py` 模块）是**夹紧连接模型**：

```
螺栓 → 被夹件（多层板或法兰） → 对手件
工作时：被夹件吸收部分外载（Φ_N·FA），剩余部分（1−Φ_N）·FA 进螺栓
```

夹紧连接有**载荷系数 Φ_N**、**残余夹紧力 FK_remaining**、**被夹件柔度 δ_P** 等概念。

本模块**没有被夹件**，所以：

- **没有 Φ_N**（外载不分流，**全部进螺栓**）
- **没有 FK_remaining 概念**（没有夹紧面接触压力需要维持）
- **没有被夹件刚度 / 柔度**

**错误用法**：把螺栓拧入法兰 + 垫片 + 法兰这种"带被夹件"工况硬塞进本模块——会少算被夹件吸收的那部分外载，**高估螺栓应力**，也无法校核密封所需残余夹紧力。**这种工况应使用 `bolt_page.py`（VDI 2230 夹紧连接模块）**。

## 载荷方向假设

**只接受纯轴向拉力**：

- **FA ≥ 0**（不接受负载；`core/bolt/tapped_axial_joint.py:220-221` 强制）
- **无横向力 FQ**（剪切 / 防滑校核不在本模块）
- **无弯矩**（偏心载荷、力矩臂导致的附加应力不建模）
- **无扭矩载荷**（只考虑装配扭矩 MA，不考虑服役额外扭矩）

如果实际工况带横向力 / 弯矩 / 扭矩，本模块的 von Mises 校核和疲劳校核都会**低估螺栓应力**——必须另做补充分析，或改用能考虑这些载荷的模块。

## 本模块做什么校核

四项分项独立判断：

| 校核项 | 关注什么 |
|---|---|
| **装配 von Mises** | 拧紧到最大预紧力时螺栓不屈服 |
| **服役最大 von Mises** | 叠加 FA_max 后螺栓不屈服 |
| **交变轴向疲劳** | σ_a ≤ σ_ASV 修正后的许用幅 |
| **螺纹脱扣**（可选） | 螺纹剪切承载 ≥ 服役最大拉力 |

**结论类型**（`overall_status`）：

- `pass`：所有启用项通过
- `fail`：任一启用项不通过
- `incomplete`：有项**未校核**（常见为螺纹脱扣留空 m_eff）

**`incomplete` 不会被误判为 `pass`**——与 VDI 2230 模块"默默跳过"的行为**不同**。`core/bolt/tapped_axial_joint.py:396-414` 会在有 `not_checked` 项时强制 overall = `incomplete`，同时追加 warning 提醒用户补齐输入。

## 参考标准

- VDI 2230-1:2015（单螺栓轴向拉载荷校核的公式框架）
- ISO 898-1:2013（螺栓材料、强度等级、应力截面积）
- DIN 13-1 / ISO 724（ISO 公制螺纹尺寸）

> Cannot verify against original VDI / ISO / DIN standard —— 本模块为上述标准的工程简化子集，**未查证原文**。不适用于横向载荷、弯矩、多螺栓并联或"螺栓 + 被夹件 + 对手件"夹紧链工况；这类场景应使用 `bolt_page.py`（VDI 2230 夹紧连接模块）或另做专项评估。
