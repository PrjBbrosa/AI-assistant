# 螺纹与材料参数

本章节确定螺栓的**螺纹几何尺寸**和**材料强度**：这两组参数是所有后续校核的基础——装配应力、服役应力、疲劳许用、螺纹脱扣承载都要回到这里。

## 为什么要填这些

- **d（公称直径）+ p（螺距）** → 决定螺纹的 d2（中径）、d3（小径）、As（应力截面积），应力截面积直接决定轴向应力 σ = F/As
- **Rp0.2（屈服强度）** → 装配 / 服役 von Mises 许用应力的基准，也是疲劳 Goodman 折减的 0.9·Rp0.2 平台
- **E_bolt（弹性模量）** → 在本模块**保留为输入但不进入应力公式**（见下）；出于一致性与未来扩展（热应变、变形校核）保留
- **grade（强度等级）** → 追溯用途；本工具不用它反查 Rp0.2（Rp0.2 直接由用户输入），仅用于报告标记

## 输入 / 产出

**输入**：

| 字段 | 单位 | 说明 |
|---|---|---|
| d | mm | 螺栓公称直径（M6, M8, M10, …） |
| p | mm | 螺距（M10 粗牙 1.5 / 细牙 1.25） |
| d2, d3, As | mm / mm / mm² | 可选手工输入；若填，会和 d/p 自动派生值比对，相对偏差 > 1% 抛 `InputError`。一般留空让工具自动派生 |
| Rp0.2 | MPa | 螺栓材料屈服强度（8.8: 640 / 10.9: 900 / 12.9: 1080） |
| E_bolt | MPa | 螺栓弹性模量；钢默认 210000 |
| grade | 字符串 | 强度等级（8.8 / 10.9 / 12.9） |

**自动派生**（`core/bolt/tapped_axial_joint.py:77-88`）：

```
As = π/4 · (d − 0.9382·p)² [d: mm, p: mm → As: mm²]
d2 = d − 0.64952·p         [d, p: mm → d2: mm]
d3 = d − 1.22687·p         [d, p: mm → d3: mm]
```

工具在 UI 上把 As/d2/d3 三个字段锁为 `AutoCalcCard` 只读；改 d 或 p 会立即重算。

## 本模块做什么（vs 不做什么）

**本模块做**：

- 按 ISO 公制螺纹公式自动派生截面（不支持英制螺纹、螺纹插管等）
- 用 Rp0.2 作为 von Mises 许用的基准
- `E_bolt` 保留为输入字段但**不参与应力 / 疲劳 / 脱扣公式**——本模块不计算热应变、不计算螺栓伸长量；E 只是占位以便未来扩展和报告追溯

**本模块不做**：

- **不从 grade 查 Rp0.2**：grade 字符串只做报告追溯，Rp0.2 必须由用户直接输入。**填错 Rp0.2 会直接影响所有校核**——请按 ISO 898-1 对应强度等级填正确值。
- **不支持英制 / 非 ISO 螺纹**：公式按 ISO 724 / DIN 13-1 公制螺纹；UNF/UNC 需另做转换。
- **不考虑材料温度效应**：Rp0.2 按室温填；高温工况需用户自行乘温度折减系数后输入。
- **不验证 d 与 p 是否为标准组合**：M10 × 1.5、M10 × 1.25 都接受；用户填错组合（例如 M10 × 2.0）会给出非标准截面值但不会报错。

## 输入一致性守护

`_derive_thread_geometry`（`core/bolt/tapped_axial_joint.py:91-130`）在**每次计算**都会：

1. 用 d 和 p 派生 As、d2、d3 标准值
2. 如用户手动填了这些字段，比对相对偏差 ≤ 1%
3. 超过 1% 抛 `InputError`，拒绝继续

这个守护是为了防止"改了 d/p 但忘了同步截面"导致的错误结果。UI 侧默认自动同步，手工填错时抛错而不静默使用旧值。
