# 蜗轮负载能力校核 Method B（DIN 3996:2019）

## 一图总览

```
  输入扭矩 T2
      │
      ▼
  法向力分解到齿面 Fn = f(T2, d2, γ, α)
      │
      ├─► 齿面赫兹应力 σH ──► 许用 σH_allow ──► 安全系数 SH = σH_allow / σH
      │
      └─► 齿根弯曲应力 σF ──► 许用 σF_allow ──► 安全系数 SF = σF_allow / σF
                                                         │
                                              SH、SF ≥ 目标值 → 通过
```

工况系数 KA、动载系数 Kv、齿宽方向分布 KHβ、横向分配 KHα 在应力计算前作为乘性修正进入 Fn。

## 解决什么问题

给定几何（DIN 3975 派生的 d2、γ、接触线长）、材料（钢-青铜、钢-塑料等副的许用应力）和工况（T2、转速 n、工作时长、冲击等级），判断齿面（点蚀）与齿根（折齿）在长期运转下是否安全。输出两个独立的安全系数 SH、SF，由工程师对照目标值拍板是否放行。

## 核心流程（5 步）

1. 查表取工况系数 KA（平稳驱动/轻冲击/重冲击查 DIN 3996 附录）。
2. 计算动载系数 Kv，主要依赖蜗轮节线速度与齿面精度等级。
3. 计算齿宽载荷分布系数 KHβ（齿宽方向不均匀）与横向载荷分配系数 KHα（齿廓方向分配）。
4. 合并上述修正，计算齿面赫兹应力 σH 与齿根弯曲应力 σF（均为单点解析值）。
5. 与材料许用 σH_allow、σF_allow 对比，得 `SH = σH_allow / σH`、`SF = σF_allow / σF`，对照目标安全系数判通过。

## 本模块实现的范围 / 不实现的范围

**已实现**：Method B 单点解析法；KA / Kv / KHα / KHβ 支持经验值查表或用户手工输入；钢-青铜、钢-塑料常见材料副的许用应力库。

**未实现**：
- Method A（沿接触线逐段数值积分，需要完整疲劳曲线与油膜厚度实测）
- Method C（最简粗算法，用于方案初选）
- 累积磨损与寿命积分（当前仅对单一稳态工况给安全系数，不按负载谱积累寿命）
- 齿面瞬时接触温度与润滑油膜厚度校核（需配合 ISO/TR 14521 或 AGMA 补充）

## 常见误用

- **KHα 与 KHβ 混淆**。KHα（alpha，齿廓方向横向分配）与 KHβ（beta，齿宽方向分布）是两个独立的系数，作用方向互相垂直。新手常把齿宽不均填到 KHα，或把齿精度影响填到 KHβ，导致应力整体偏高或偏低。
- **许用应力不分单双向工况**。双向（正反转）运转时，齿面/齿根都要按对称循环折减；塑料蜗轮尤其显著，σF_allow 相比单向可能要再降 30-50%。如果直接套用单向许用值，会得到虚高的安全系数。
- **把"SH = 1"当通过**。工程实践要求 SH ≥ 1.0-1.2、SF ≥ 1.3-1.5（具体目标视可靠性要求、维护难度而定）。等于 1 只代表应力刚好等于许用，没有任何裕度，不能视为合格。
