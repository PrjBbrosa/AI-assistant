# 负载能力章节

本章节用于**完成齿面接触疲劳和齿根弯曲疲劳的强度校核**：在几何、材料、工况都定好之后，把应力算出来，和许用值做比较，得出安全系数。这一章只有在"基本设置"里勾选了负载能力校核后才可见。

## 为什么要填这些

蜗轮传动最常见的失效模式是齿面点蚀/胶合（σH 超限）和齿根断裂（σF 超限）。校核要回答两个问题：实际应力是多少？许用应力是多少？实际应力由几何、材料、载荷配合一串修正系数算出，每个系数反映一种偏离理想状态的实际情况——动载、载荷集中、齿宽方向不均匀。许用应力来自材料疲劳曲线，再按寿命、可靠度、润滑状态折减。目标安全系数是你希望应力比许用值低多少倍——这是设计余量。

## 输入 / 产出

**输入**：
- 许用齿面接触应力 σHP（由材料 + 寿命 + 可靠度查表或计算，单位 MPa）
- 许用齿根弯曲应力 σFP
- 动载系数 Kv（反映啮合过程中的动态冲击，与滑动速度、制造精度相关）
- 横向载荷分布系数 KHα（反映多对齿接触时载荷在周向的不均匀分布）
- 齿宽载荷分布系数 KHβ（反映沿齿宽方向的载荷集中程度，与装配误差、轴刚度相关）
- 目标安全系数 SH_min、SF_min（常规工业传动齿面取 1.0~1.3，齿根取 1.4~1.6）

**产出**（本模块实际实现，非完整 DIN 3996 Method B 公式链）：
- 实际齿面应力 σH —— **线接触 Hertz 近似**（而非 ZE/ZH/Zβ 系数链）：
  `σH = q' / (2·b_H)`，其中 `q' = F_n·K_total / L_contact`、`b_H = √(4·q'·ρ / (π·E*))`
- 实际齿根应力 σF —— **等效悬臂梁近似**（而非 YF/YS/Yβ 系数链）：
  `σF = M_bending·6 / (b·s_Ft²)`
- 齿面安全系数 SH = σHP / σH，齿根安全系数 SF = σFP / σF
- 与目标安全系数比较的 PASS / FAIL 判定

> 本模块的 σH、σF 算式是"Method B 风格"的工程估算，用 Hertz 线接触 + 悬臂梁两套粗近似代替了完整 Method B 的 ZH/ZE/Zβ/YF/YS 系数链。Cannot verify against original DIN standard —— 原始 DIN 3996:2019 条款尚未逐项比对，结果仅供方案估算与初筛，不作最终强度合规依据。

## 方法差异

三个载荷修正系数不是独立参数，而是一个相乘链条：K_total = KA · Kv · KHα · KHβ。

- **KA** 在"工况"章节填，反映外部载荷特性。
- **Kv** 取决于滑动速度和齿轮精度等级；简化法可直接取经验值 1.0~1.3，详细法按 DIN 3990 曲线计算。
- **KHα** 通常对单头蜗杆 ≈ 1.0（只有一对齿受力），多头蜗杆才显著偏离 1。
- **KHβ** 最敏感，装配误差大或悬臂布置时可达 1.5~2.0，对称支承、精密装配可到 1.1~1.2。

新手最容易低估 KHβ；如果没有实测数据，保守取 1.3 以上，不要默认 1.0。

## 参考标准

- DIN 3990 / ISO 6336 载荷修正系数（KA / Kv / KHα / KHβ 通用定义）
