# 负载能力章节

本章节用于**完成齿面接触疲劳和齿根弯曲疲劳的强度校核**：在几何、材料、工况都定好之后，把应力算出来，和许用值做比较，得出安全系数。这一章只有在"基本设置"里勾选了负载能力校核后才可见。

## 为什么要填这些

蜗轮传动最常见的两种失效模式：

- **齿面点蚀 / 胶合**：齿面反复被压出凹坑（点蚀）或在高滑动速度下被撕裂（胶合）。判据：实际**齿面接触应力 σH**（Hertz 应力，单位 MPa）小于**许用齿面应力 σHP**。
- **齿根弯曲断裂**：齿根像悬臂梁一样反复受弯，最终疲劳断裂。判据：实际**齿根弯曲应力 σF**（MPa）小于**许用齿根应力 σFP**。

校核要回答两个问题：**实际应力是多少**？**许用应力是多少**？

- 实际应力 σH、σF —— 由几何（分度圆、齿宽）、材料（弹性模量、泊松比）、载荷（蜗轮扭矩 T₂）配合一串**载荷修正系数**算出。每个修正系数反映一种偏离理想状态的实际情况：
  - **KA（使用系数）**：铭牌扭矩 vs 实际峰值扭矩的放大倍数，反映原动机与工作机冲击
  - **Kv（动载系数）**：啮合过程中齿轮速度不均匀带来的动载放大
  - **KHα（横向载荷分布系数）**：多对齿同时啮合时载荷在齿廓方向（alpha 方向）的不均匀分配
  - **KHβ（齿宽载荷分布系数）**：沿齿宽方向（beta 方向）的载荷集中程度，装配不对中或悬臂支承时显著
- 许用应力 σHP、σFP —— 来自材料疲劳曲线，再按寿命、可靠度、润滑状态折减（具体系数含义见 `terms/allowable_contact_stress`、`terms/allowable_root_stress`）。
- 目标安全系数 SH_min、SF_min —— 你希望实际应力比许用值低多少倍（工业齿面 1.0~1.2，齿根 1.4~1.6）。这是设计余量。

## 输入 / 产出

**输入**：
- 许用齿面接触应力 σHP（材料 + 寿命 + 可靠度查表或计算，单位 MPa）
- 许用齿根弯曲应力 σFP（同上）
- 动载系数 Kv —— 啮合过程中的动态冲击，与滑动速度、制造精度相关；简化取 1.0–1.3
- 横向载荷分布系数 KHα —— 单头蜗杆 ≈ 1.0，多头蜗杆可能 > 1
- 齿宽载荷分布系数 KHβ —— 对称支承、精密装配 1.1–1.2；悬臂或不对中 1.5–2.0
- 目标安全系数 SH_min、SF_min（常规工业传动齿面 1.0~1.3，齿根 1.4~1.6）

**产出**（本模块实际实现，非完整 DIN 3996 Method B 公式链）：
- 实际齿面应力 σH —— **线接触 Hertz 近似**（而非 ZE/ZH/Zβ 系数链）：
  `σH = q' / (2·b_H)`，其中 `q' = F_n·K_total / L_contact`、`b_H = √(4·q'·ρ / (π·E*))`
- 实际齿根应力 σF —— **等效悬臂梁近似**（而非 YF/YS/Yβ 系数链）：
  `σF = M_bending·6 / (b·s_Ft²)`
- 齿面安全系数 SH = σHP / σH，齿根安全系数 SF = σFP / σF
- 与目标安全系数比较的 PASS / FAIL 判定

> 本模块的 σH、σF 算式是"Method B 风格"的工程估算，用 Hertz 线接触 + 悬臂梁两套粗近似代替了完整 Method B 的 ZH/ZE/Zβ/YF/YS 系数链。Cannot verify against original DIN standard —— 原始 DIN 3996:2019 条款尚未逐项比对，结果仅供方案估算与初筛，不作最终强度合规依据。

## 方法差异

三个载荷修正系数不是独立参数，而是一个相乘链条：K_total = KA · Kv · KHα · KHβ。

- **KA** 在"工况"章节填，反映外部载荷特性。
- **Kv** 取决于滑动速度和齿轮精度等级；简化法可直接取经验值 1.0~1.3，详细法按 DIN 3990 曲线计算。
- **KHα** 通常对单头蜗杆 ≈ 1.0（只有一对齿受力），多头蜗杆才显著偏离 1。
- **KHβ** 最敏感，装配误差大或悬臂布置时可达 1.5~2.0，对称支承、精密装配可到 1.1~1.2。

新手最容易低估 KHβ；如果没有实测数据，保守取 1.3 以上，不要默认 1.0。
