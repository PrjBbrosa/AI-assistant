# 几何参数章节

本章节用于**确定蜗杆副的基本尺寸与啮合几何**：头数、齿数、模数、直径系数、变位系数、齿宽、中心距。后续所有应力、速度、效率计算都建立在这些尺寸给出的分度圆、齿顶圆、齿根圆之上。

## 为什么要填这些

蜗轮传动的几何由一组相互约束的量决定：模数 m 与直径系数 q 决定蜗杆分度圆直径 d1 = m·q；头数 z1 与齿数 z2 决定传动比 i = z2/z1；导程角 γ 由 tan γ = z1/q 自洽得出，不是独立输入，而是用来反向校核 q 的合理性。变位系数 x1/x2 用来调整齿根厚度、避免根切、微调中心距；齿宽 b1/b2 影响接触线总长进而影响 σH。

这一步任何一个值输错，后续所有派生量都会错；中心距 a 和导程角 γ 之间的自洽性是几何章节的关键输出。

## 输入 / 产出

**输入**：
- 蜗杆头数 z1（常见 1、2、4 头）
- 蜗轮齿数 z2
- 模数 m（GB/DIN 标准系列选取，单位 mm）
- 直径系数 q（影响蜗杆刚度和导程角）
- 变位系数 x1、x2（塑料蜗轮常用中正变位 x2 > 0 补偿弯曲变形）
- 齿宽 b1（蜗杆工作长度）、b2（蜗轮齿宽）
- 中心距 a（部分场景由上述量反算）

**产出给下一步**：
- 分度圆 d1、d2；齿顶圆 da1、da2；齿根圆 df1、df2
- 导程角 γ 的自洽校验结果
- 中心距 a 与 (d1+d2)/2 + (x1+x2)·m 的一致性校验
- 供 Load Capacity 使用的接触线长度和有效啮合宽度
