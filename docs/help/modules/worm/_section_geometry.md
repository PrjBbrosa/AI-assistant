# 几何参数章节

本章节用于**确定蜗杆副的基本尺寸与啮合几何**：头数、齿数、模数、直径系数、变位系数、齿宽、中心距。后续所有应力、速度、效率计算都建立在这些尺寸给出的分度圆、齿顶圆、齿根圆之上。

## 为什么要填这些

蜗轮传动的几何由一组**相互约束**的量决定。不是挨个独立填入，填完要相互验证自洽性。先把每个符号讲清：

- **模数 m** [mm]：齿轮"齿的大小"标称尺度，GB/DIN 标准系列选取（常见 1, 1.5, 2, 2.5, 3, 4, 5, 6...）
- **蜗杆头数 z1**：一根蜗杆上有几条螺旋齿线（类似多头螺纹；常见 1、2、4 头，单头效率低但可能自锁）
- **蜗轮齿数 z2**：蜗轮总齿数（决定传动比 i = z2 / z1）
- **直径系数 q**：DIN 3975 定义 `q = d1 / m`（蜗杆分度圆直径相对模数的比值）。q 越大蜗杆越粗、刚度越高，但导程角越小
- **导程角 γ** [rad 或 deg]：蜗杆螺旋齿线相对轴线的倾斜角，由 `tan γ = z1 / q` 自洽得出
- **变位系数 x1、x2**：齿廓相对标准位置的径向移动量与模数的比值；蜗杆端通常 x1 = 0，蜗轮端塑料蜗轮常用正变位 x2 > 0 补偿弯曲变形
- **齿宽 b1、b2** [mm]：蜗杆工作齿宽与蜗轮齿宽；影响接触线总长进而影响齿面应力

关键几何关系（所有公式单位都是一致的 mm）：

```
d1 = m · q            # 蜗杆分度圆直径
d2 = m · z2           # 蜗轮分度圆直径
tan γ = z1 / q        # 导程角自洽
a = (d1 + d2) / 2     # 无变位时的中心距；有变位时再加 (x1+x2)·m
```

这一步任何一个值输错，后续所有派生量（齿顶圆、齿根圆、接触线长）都会错。**中心距 a 与 (d1+d2)/2 + (x1+x2)·m 的自洽性**是几何章节的关键检查点，UI 会在不一致时给出警告。

## 输入 / 产出

**输入**：
- 蜗杆头数 z1（常见 1、2、4 头）
- 蜗轮齿数 z2
- 模数 m（GB/DIN 标准系列选取，单位 mm）
- 直径系数 q（影响蜗杆刚度和导程角）
- 变位系数 x1、x2（塑料蜗轮常用中正变位 x2 > 0 补偿弯曲变形）
- 齿宽 b1（蜗杆工作长度）、b2（蜗轮齿宽）
- 中心距 a（部分场景由上述量反算）

**产出给下一步**：
- 分度圆 d1、d2；齿顶圆 da1、da2；齿根圆 df1、df2
- 导程角 γ 的自洽校验结果
- 中心距 a 与 (d1+d2)/2 + (x1+x2)·m 的一致性校验
- 供 Load Capacity 使用的接触线长度和有效啮合宽度
