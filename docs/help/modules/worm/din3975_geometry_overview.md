# 蜗轮副几何计算（DIN 3975-1:2017）

## 一图总览

```
        蜗杆（头数 z1，螺旋体）
     ───────────────────────
      \\    \\    \\    \\      ← 轴向模数 m
       \\    \\    \\    \\
                              ↕ 中心距 a = (d1+d2)/2 + m·(x1+x2)
            ┌──────────┐
            │          │     蜗轮（齿数 z2，盘形齿轮）
            │    ○     │     分度圆 d2 = m·z2
            │          │
            └──────────┘
```

蜗杆以分度圆 `d1 = m·q` 螺旋推进，每转一圈沿轴向前进 `z1·m`；蜗轮是一只被蜗杆螺旋推着转的齿轮。中心距 a、模数 m、直径系数 q、变位系数 x1/x2 之间彼此约束，不能独立给。

## 解决什么问题

给定 **z1、z2、m、q、x1、x2** 这组输入，反推出蜗杆与蜗轮的全部几何尺寸——分度圆、齿顶圆、齿根圆、导程角、中心距、有效啮合宽度——并校验这些量之间是否自洽。几何一旦走通，后续 DIN 3996 负载能力校核才有可靠的 d2、γ、接触线长作为输入。

## 核心流程（5 步）

1. 输入蜗杆头数 z1、蜗轮齿数 z2、模数 m、直径系数 q；得到传动比 i = z2 / z1。
2. 派生导程角 `tan γ = z1 / q`，蜗杆分度圆 `d1 = m·q`。
3. 派生蜗轮分度圆 `d2 = m·z2`；按变位系数 x1、x2 修正齿顶圆 da1/da2、齿根圆 df1/df2。
4. 派生中心距 `a = (d1 + d2)/2 + m·(x1 + x2)`；与用户输入的中心距对比。
5. 校核导程角、中心距、啮合宽度三者的自洽性，给出是否满足 DIN 3975 几何约束的结论。

## 本模块实现的范围 / 不实现的范围

**已实现**：DIN 3975-1:2017 完整的基础几何模型——分度圆、齿顶圆、齿根圆、导程角、中心距、齿宽；变位蜗杆/蜗轮（x1、x2 任意组合）；中心距自洽校核。

**未实现**：DIN 3975-2 公差体系与接触斑点形状评估；非对称极端变位（|x1 - x2| 过大时的特殊干涉复核）；动态变形与热胀修正。

## 常见误用

- **q 与 d1 不能同时输入**。两者由 `d1 = m·q` 强绑定，新手经常把 d1 当独立量填，结果与 m、q 互相矛盾。正确做法是只给 q（或只给 d1 反算 q），让模块派生另一个。
- **x1 = x2 = 0 不是强制默认**。它只是"不变位"这一初始选择。塑料蜗轮为补偿弯曲变形、钢-青铜副为调整中心距，常取 x2 > 0。
- **导程角不是压力角**。导程角 γ 是蜗杆螺旋线沿分度圆一圈的上升角（与轴线的余角），压力角 α 是齿侧廓线方向。两者出现位置不同，不能互换。
