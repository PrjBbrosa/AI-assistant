# 工况与润滑章节

本章节用于**描述传动在现场实际承受的载荷和转速**：蜗杆侧的输入扭矩、输入转速、工况系数、扭矩波动。几何章节定的是"尺寸能不能装上"，这一章定的是"装上之后受多大力、转多快"。

## 为什么要填这些

输入扭矩 T1 和转速 n1 是两个最基本的载荷参数：T1 决定蜗杆齿面上的切向力 Ft1 = 2·T1/d1，进而决定接触应力和齿根应力；n1 决定滑动速度 vs = π·d1·n1 / (60·cos γ)，滑动速度影响动载系数 Kv、油膜厚度、摩擦系数和温升。两者组合出输入功率 P1 = T1·n1/9550（kW），用来估算热平衡。

使用系数 KA 把"铭牌扭矩"放大到"实际峰值扭矩"，反映原动机和工作机的载荷特性：电机驱动平稳工况取 1.0，往复泵、破碎机、冲击负载要取到 1.5~2.0 甚至更高。扭矩波动百分比用于区分稳态和交变工况，影响疲劳寿命折算。

## 输入 / 产出

**输入**：
- 输入扭矩 T1（蜗杆轴上的名义扭矩，单位 N·m）
- 输入转速 n1（蜗杆转速，单位 r/min）
- 使用系数 KA（DIN 3990 / 6336 中载荷特性分类表查取）
- 扭矩波动百分比（峰值相对名义值的浮动范围）

**产出给下一步**：
- 蜗杆齿面切向力 Ft1、径向力 Fr1、轴向力 Fa1（按导程角、压力角分解）
- 蜗轮齿面切向力 Ft2 = Ft1·tan(γ + ρ')，反向传动分析
- 滑动速度 vs，供润滑状态判别和 Kv 取值
- 输入功率 P1 和热功率对比
- 参与疲劳校核的等效扭矩 T_eq = KA·T1

## 参考标准

- DIN 3990-1 / ISO 6336-1 使用系数 KA 载荷特性分类
- 扭矩波动无专门标准，按现场实测或电机特性曲线估计
