# 工况与润滑章节

本章节用于**描述传动在现场实际承受的载荷和转速**：蜗杆侧的输入扭矩、输入转速、工况系数、扭矩波动。几何章节定的是"尺寸能不能装上"，这一章定的是"装上之后受多大力、转多快"。

## 为什么要填这些

输入扭矩 T1（蜗杆轴，N·m）和输入转速 n1（蜗杆轴，r/min）是两个最基本的载荷参数：

- **T1 → 决定齿面力系**：先通过传动效率换算出蜗轮扭矩 T2，再由 T2 计算蜗轮切向力，公式（本模块实现）：
  `Ft2 = 2000·T2 / d2`（T2 单位 N·m，d2 单位 mm，得 Ft2 单位 N）。
  Ft2 决定齿面接触应力 σH 与齿根弯曲应力 σF。
- **n1 → 决定滑动速度**：
  `vs = π·d1·n1 / (60000·cos γ)`（d1 单位 mm，n1 单位 r/min，得 vs 单位 m/s；分母里 60000 来自 "60 s × 1000 mm/m"）。
  滑动速度影响动载系数 Kv、油膜厚度、摩擦系数和温升。
- **P1 = T1·n1 / 9550**（T1 单位 N·m，n1 单位 r/min，得 P1 单位 kW）— 用来估算热平衡。

使用系数 KA 把"铭牌扭矩"放大到"实际峰值扭矩"，反映原动机和工作机的载荷特性：电机驱动平稳工况取 1.0，往复泵、破碎机、冲击负载要取到 1.5~2.0 甚至更高。扭矩波动百分比用于区分稳态和交变工况，影响疲劳寿命折算。

## 输入 / 产出

**输入**：
- 输入扭矩 T1（蜗杆轴上的名义扭矩，单位 N·m）
- 输入转速 n1（蜗杆转速，单位 r/min）
- 使用系数 KA（DIN 3990 / 6336 中载荷特性分类表查取）
- 扭矩波动百分比（峰值相对名义值的浮动范围）

**产出给下一步**（力系分解遵循 Niemann §24 / DIN 3975，坐标约定：蜗杆切向 = 蜗轮轴向）：
- 蜗轮切向力 `Ft2 = 2000·T2 / d2`（N）
- 蜗杆切向力 / 蜗轮轴向力 `Ft1 = Fa2 = Ft2·tan(γ + φ')`（N；φ' 为当量摩擦角，φ' = atan(μ / cos αn)）
- 径向力 `Fr = Ft2·tan(αn) / cos γ`（N）
- 法向力 `Fn = Ft2 / (cos αn · cos γ)`（N，用于齿面 Hertz 应力）
- 滑动速度 vs（m/s），供润滑状态判别和 Kv 取值
- 输入功率 P1、输出功率 P2 和热功率对比
- 参与疲劳校核的设计力 = Ft2 × KA · Kv · KHα · KHβ

## 参考标准

- DIN 3990-1 / ISO 6336-1 使用系数 KA 载荷特性分类
- DIN 3975 蜗杆力系几何关系；Niemann "Maschinenelemente" Bd. III §24 力系分解示意
- 扭矩波动无专门标准，按现场实测或电机特性曲线估计
