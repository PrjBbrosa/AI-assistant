# 材料与配对章节

本章节用于**选定蜗杆和蜗轮的材料组合、旋向、润滑方式**。材料决定弹性模量 E 和泊松比 ν，进而决定赫兹接触应力的计算；材料配对还决定许用应力和磨损寿命的查表基线。

## 为什么要填这些

蜗轮传动几乎都用"硬蜗杆 + 软蜗轮"的异材组合，让蜗轮先于蜗杆磨损，便于维护更换。常见组合：淬硬钢蜗杆（20CrMnTi 渗碳、45 钢调质）配锡青铜（ZCuSn10Pb1）、铝青铜或铸铁蜗轮；中低载场合还会用到塑料蜗轮（POM、PA66）。塑料蜗轮弯曲刚度低，常用中正变位 x2 > 0 补偿齿的弯曲变形、改善啮合位置。

润滑方式直接影响接触区油膜厚度和摩擦系数：油池润滑是常规做法；喷油润滑用于高速或大功率；脂润滑仅用于低速、间歇、开式传动。旋向（左旋/右旋）不改变强度，但影响轴向力方向，装配时必须与轴承布置一致。

## 输入 / 产出

**输入**：
- 蜗杆材料（常见渗碳淬硬钢、调质钢）
- 蜗轮材料（锡青铜、铝青铜、铸铁或工程塑料）
- 旋向（左旋 / 右旋）
- 润滑方式（油池、喷油、脂润滑）
- 两侧弹性模量 E1 / E2（通常由材料选择自动填充）
- 两侧泊松比 ν1 / ν2

**产出给下一步**：
- 赫兹综合弹性模量 E* = 1 / [(1-ν1²)/E1 + (1-ν2²)/E2]，供 σH 计算
- 蜗轮材料决定的许用齿面应力 σHP、许用齿根应力 σFP 基线（DIN 3996 附录查表）
- 润滑方式对应的基础摩擦系数 μ0，供效率和自锁判据使用
- 轴向力方向（随旋向）进入受力分析

## 参考标准

- DIN 3996:2019（蜗轮材料许用应力参考表相关附录；Cannot verify against original DIN standard）
- 润滑方式无专门标准，经验做法见模块 README
