# 高级参数章节

本章节用于**对默认值不满足的场景做精细覆盖**：手动指定摩擦系数、法向压力角、工作温度、相对湿度。这些参数在绝大多数常规设计里直接使用默认值即可；只有环境恶劣、材料非标、或者需要和试验数据对齐时才需要改动。注意当前 UI 中这是几何章节内的一个组卡，不是独立章节。

## 为什么要填这些

摩擦系数 μ 默认由"蜗轮材料 + 润滑方式 + 滑动速度"自动查表得出；但实测油样、特殊添加剂、磨合后的稳定状态都可能和查表值差 20% 以上，这时就要手动覆盖。法向压力角 αn 标准值 20°，只有用旧图纸（14.5°、15°）或特殊高载设计（25°）才改。工作温度影响润滑油黏度进而影响摩擦系数和油膜厚度；相对湿度主要影响塑料蜗轮（PA66 吸水后尺寸和强度都会变化）。

## 输入 / 产出

**输入**：
- 摩擦系数覆盖值 μ（留空使用自动查表；常规钢-青铜油润滑 0.03~0.08，脂润滑 0.08~0.12）
- 法向压力角 αn（默认 20°，特殊工况可选 14.5° / 15° / 25°）
- 工作温度（环境 + 传动自身温升，用于润滑油黏度修正）
- 相对湿度（仅对塑料蜗轮有显著影响）

**产出给下一步**：
- 覆盖默认摩擦系数后重新计算啮合效率 η = tan γ / tan(γ + ρ')，其中 ρ' = arctan(μ / cos αn)
- 自锁判据：γ < ρ' 时静态自锁
- 润滑油实际运行黏度 → 油膜厚度 → 动载系数 Kv 修正
- 塑料蜗轮材料参数的湿度折减系数

## 参考标准

- DIN 3996:2019（蜗轮摩擦系数参考表相关附录；Cannot verify against original DIN standard）
- 法向压力角由齿轮刀具标准规定，一般 ISO 53 / GB 1356
