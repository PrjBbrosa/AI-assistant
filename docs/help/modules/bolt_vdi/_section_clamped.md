# 被夹紧件和实体章节

本章节用于**描述螺栓所夹紧的那叠零件的几何、材料和等效"弹性身体"**：被夹件层数与每层厚度/外径/材料、基础实体模型（圆柱体 / 锥体 / 套筒）、以及对应的柔度或刚度。被夹件的"软硬程度" δp 与螺栓的"软硬程度" δs 共同决定载荷因数 ΦN —— VDI 2230 核心链路中最关键的中间量。

## 为什么要填这些

螺栓连接的本质是**两个弹性体并联**：拧紧时螺栓像拉长的弹簧，被夹件像压缩的弹簧。当外力 FA 作用时，这两根"弹簧"分摊外载的比例由载荷因数决定：

```
ΦN = n · δp / (δs + δp)         [δs, δp 单位: mm/N 或等价的 1/(N/mm)]
```

其中：
- **δs**（螺栓柔度，mm/N）= 1N 拉力下螺栓伸长多少 mm。螺栓越长越软（δs 越大）。
- **δp**（被夹件柔度，mm/N）= 1N 压力下被夹件压缩多少 mm。被夹件越粗或越厚越硬（δp 越小）。
- **n**（载荷导入系数）= 外力作用点在螺栓-被夹件这条"弹性串"上的位置修正（典型 0.2–1.0）。
- **ΦN**（载荷因数）= 外力 FA 有多大比例分给螺栓而不是松开被夹件。本工具结果会给出该值；ΦN 越小意味着被夹件吸走更多外力，螺栓越轻松。

本章的任务是把"被夹件到底是什么形状、多软多硬"说清楚，有两种输入路径：

1. **手动输入柔度**（默认）：直接填 δs 和 δp，来源于手册、FEA 或试验数据。快速但需要先验知识。
2. **自动计算柔度**（勾选"自动计算"）：填几何（厚度 lK、等效外径 DA、基础实体类型）+ 材料（弹性模量 E_bolt、E_clamped），计算器按 VDI 2230 柔度公式族（螺栓分段 + 被夹件圆柱 / 锥台 / 套筒）估算。适合无试验数据、用标准几何做初步设计。

> 本工具的"多层被夹件"模型按多段串联近似，实现于 `core/bolt/compliance_model.py`。锥台 / 套筒模型仅支持对称夹紧体；非对称需要走手动输入。

## 输入 / 产出

**输入**：
- 基础实体类型（圆柱体 / 锥体 / 套筒）—— 自动柔度使用
- 被夹件数量（1 / 2 / 自定义 3–5）—— 自定义时启用分层输入
- 每层厚度、外径 DA、弹性模量、材料（自动填热膨胀系数）
- 接触面粗糙度等级（粗糙 / 中等 / 精细）—— 决定嵌入损失自动估算
- 总夹紧长度 lK（= Σ 各层厚度）、等效外径 DA
- **柔度 / 刚度**（二选一输入路径）：
  - 手动：δs、δp（mm/N）或刚度 cs = 1/δs、cp = 1/δp（N/mm）
  - 自动：留空 δ/c，勾选"自动计算"

**产出给下一步**：
- δs、δp → 载荷因数 ΦN 计算
- 接触面粗糙度等级 → 嵌入损失 FZ 自动估算（装配章节用到；见 `terms/bolt_embed_loss`）
- lK、DA → 自动柔度几何输入

## 常见坑

- **柔度和刚度不能同时填**：δ 与 c 互为倒数。勾"自动计算"后两套都留空；手动模式下只填一套（填 δ 或填 c，另一套留空）。
- **多层被夹件 E_clamped 不生效**：启用多层后计算器按每层自己的 E 计算，`stiffness.E_clamped` 只对单层模式有意义。
- **锥体/套筒模型需要对称夹紧体**：非对称布置（一侧法兰、一侧薄板）用锥体模型会偏乐观；建议走手动输入配合 FEA。
- **粗糙度等级"精细 (Ra≈1.6μm)"不等于零嵌入**：VDI 2230 嵌入参考值中精细面单界面仍有 ~1 μm 压实量，多界面累加仍然显著。
