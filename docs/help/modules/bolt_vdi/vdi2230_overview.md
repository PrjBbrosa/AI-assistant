## 解决什么问题

给定螺栓连接（几何 + 材料 + 外载），在**装配阶段**和**服役阶段**分别校核：

1. **装配阶段**：螺栓拧到最大预紧力 FM,max 时，会不会屈服？支承面会不会压溃？螺纹会不会滑牙？
2. **服役阶段**：外载作用后，螺栓会不会屈服？残余夹紧力够不够保持密封/防滑？疲劳循环会不会累积失效？

输出是一组**安全裕度**与 PASS/FAIL，由工程师对照目标拍板是否放行。

## 核心流程（按 VDI 2230 R 步）

VDI 2230-1 把完整校核拆成 R1–R13 共 13 步，本工具实现其中的核心链路。以下流程编号对齐 VDI 2230：

| 步骤 | 名称 | 本工具实现 |
|---|---|---|
| **R1** | 最小预紧力 FM,min 设计 | ✓ 由 FK,req、FA,max、ΦN、FZ、Fth 反推 |
| **R2** | 最大预紧力 FM,max 计算 | ✓ FM,max = αA · FM,min |
| **R3** | 残余夹紧力校核 | ✓ FK,work ≥ FK,req |
| **R4** | 装配等效应力校核 | ✓ σ_vm,asm = √(σax² + 3τ²) ≤ ν·Rp0.2 |
| **R5** | 服役等效应力校核 | ✓ σ_vm,work ≤ Rp0.2 / S_F；扭矩法保留 50% 装配扭矩 |
| **R6** | 交变应力（偏心弯矩） | ✗ **未实现**，偏心字段仅作记录占位 |
| **R7** | 支承面压强校核 | ✓ p_bear = FM,max / A_bearing ≤ p_G |
| **R8** | 螺纹脱扣校核 | ✓ 需填 m_eff + τBM；留空则**跳过该项**（overall_pass 仍可为 True，UI 不警告） |
| **R9** | 疲劳校核 | ✓ 简化 Goodman（非 FKN 法） |
| **R10** | 工作温度 / 热载荷 | ✓ 热损失 Fth 自动估算，参与 R3 / 疲劳 |
| R11–R13 | 工况等级、螺纹脱扣详细法、微动 | ✗ 未实现 |

## 关键公式骨架（均已在 `core/bolt/calculator.py` 实现）

```
ΦN = n · δp / (δs + δp)                              # 载荷因数（无量纲）
FM,max = αA · FM,min                                 # 预紧力上下限 [N]
FM,min ≥ FK,req + FA,max·(1−ΦN) + FZ + Fth          # R1 设计模式反推
σ_ax,asm = FM,max / As                               # 装配轴向应力 [MPa]
τ_asm    = 16 · M_thread / (π · d3³)                 # 装配扭转应力 [MPa]
σ_vm,asm = √(σ_ax,asm² + 3·τ_asm²)                   # 装配等效应力
σ_ax,work = (FM,max + ΦN·FA,max) / As                # 服役轴向应力
σ_vm,work = √(σ_ax,work² + 3·(k_τ·τ_asm)²)           # 服役等效（k_τ=0.5 扭矩法 / 0 其他）
σ_a = ΦN · FA,max / (2·As)                           # 疲劳交变幅值
σ_m = (FM,max + 0.5·ΦN·FA,max) / As                  # 疲劳平均应力
```

单位约定：力 N，长度 mm，应力 MPa，扭矩内部 N·mm。

## 本模块实现的范围 / 不实现的范围

**已实现（核心链路）**：
- R1/R2/R3/R4/R5/R7 核心应力与预紧力校核
- R8 螺纹脱扣（需手动输入 m_eff、τ_BM）
- R9 疲劳简化 Goodman（σ_ASV 按 d 线性插值 + 轧制/切削区分；内插表来自公开教科书整理）
- R10 热损失自动估算（基于温差与热膨胀系数差异）
- 嵌入损失 FZ 自动估算（基于粗糙度等级的本模块简化实现；详见 `terms/bolt_embed_loss`）
- 柔度 δs/δp 手动输入 + 自动计算（对称圆柱/锥体/套筒、单层和多层串联）

**未实现（未来扩展）**：
- R6 偏心弯矩校核（偏心夹紧 e_clamp、偏心载荷 e_load 当前仅记录）
- 完整疲劳 FKN 法（当前用简化 Goodman 折减代替，精度低于 VDI 2230 FKN 谱校核附录）
- 多层被夹件的**逐层锥台/套筒**刚度选择（多层目前按统一圆柱模型 + 每层自身 E）
- R11–R13（工况等级分类、螺纹脱扣详细法、微动磨损 / fretting）

## 常见误用

- **把 ΦN 当输入填入**：ΦN 是工具输出，不是用户输入；填 n 和 δ/c 即可。
- **为了 PASS 反复调 αA**：αA 由拧紧方式决定，不是调参数。扭矩法真实 αA ≥ 1.4，调小会得到虚高的 FM,min 余量。
- **期望 R9 疲劳很严格**：本工具 Goodman 简化法只能给初步判断；疲劳敏感场合（飞机、风电、高循环重载）必须走 FKN 法或试验。
- **误以为 R8 留空会标 "incomplete"**：实际 calculator 只是跳过 R8，`overall_pass = all(checks_out.values())` 对缺失项视为不存在，整体仍可能 PASS，**UI 不会警告你漏校**。铝壳体螺纹、浅孔螺纹**必须**主动填 m_eff 和 τ_BM。
- **把工具结论当设计合规依据**：VDI 2230 原文涵盖更多边界条件（偏心、动载、微动），本工具仅覆盖核心链。正式评审请按原文逐项核对。
