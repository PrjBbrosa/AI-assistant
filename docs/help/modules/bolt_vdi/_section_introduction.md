# 载荷导入章节

本章节用于**描述外载在螺栓-被夹件这个弹性组合体上的作用位置，以及服役阶段的安全系数目标**：载荷导入系数 n、载荷导入位置、服役屈服安全系数 S_F，以及两个偏心量字段（本版本尚未启用）。

## 为什么要填这些

### 载荷导入系数 n（0.2 – 1.0）

设想一颗螺栓把两块法兰夹在一起。外力 FA 作用时，它**从哪进入这个连接**决定螺栓分担多少：

- 外力直接作用在螺栓头下方（紧贴头部 / 螺母） → n ≈ 1.0
- 外力作用在被夹件中部（远离螺栓头） → n ≈ 0.5
- 外力通过薄壳传到螺栓 → n 可低至 0.2

n 直接进入载荷因数：

```
Φ_N = n · δp / (δs + δp)               [δ 单位 mm/N，Φ_N 无量纲]
```

n 越小 → Φ_N 越小 → 螺栓分担的外力越少 → 服役应力越低、疲劳越安全。但 n 选小的代价是"被夹件分担的外力增大"，残余夹紧力扣减也增大（ R3 更严）。

**工程取值**：不确定时取 n = 1.0（保守 —— 螺栓承担全部外力），等有 FEA/试验再细化。

### 服役屈服安全系数 S_F（≥ 1.0，常用 1.1–1.3）

R5 校核：

```
σ_vm,work = √(σ_ax,work² + 3·(k_tau·τ_assembly)²)
σ_allow,work = Rp0.2 / S_F
R5 通过: σ_vm,work ≤ σ_allow,work
```

S_F 是你希望服役应力与屈服之间留多大裕量：

- **S_F = 1.0**：允许服役应力达到屈服点。危险，仅用于极端重量敏感场合
- **S_F = 1.1**（默认）：10% 裕量。普通工业连接通用
- **S_F = 1.2–1.3**：重要连接（承压壳体、受冲击、长期运行）
- **S_F ≥ 1.5**：疲劳极端敏感 / 不可检修场合

### 偏心量（本版本暂未启用）

偏心夹紧 e_clamp、偏心载荷 e_load 本该进入 VDI 2230 的偏心弯矩修正流程（R6 步），但本工具**尚未实现**偏心弯矩校核，两字段当前仅作记录占位。正式偏心连接请参考 VDI 2230 原文或走 FEA 路径。

## 输入 / 产出

**输入**：
- 载荷导入系数 n（不确定取 1.0）
- 载荷导入位置（螺栓头 / 螺母 / 中间 / 分布式；仅用于记录）
- 服役屈服安全系数 S_F
- 夹紧偏心 e_clamp、载荷偏心 e_load（**当前不参与计算**）

**产出**：
- n → 载荷因数 Φ_N → R3 残余夹紧力、R5 服役应力、疲劳
- S_F → R5 服役等效应力允许值

## 常见坑

- **把 n 与 Φ_N 搞混**：n 是输入，Φ_N 是计算结果。n 是 "外力作用几何"，Φ_N 是 "螺栓分担比例"。工具里只填 n。
- **S_F 过低等于无校核**：S_F = 1.0 且服役应力刚好等于 Rp0.2 时，σ_vm,work = σ_allow,work 会判 R5 通过，但实际是临界状态，任何载荷波动都会屈服。工业用至少 1.1。
- **指望偏心弯矩校核**：本版本**不做**偏心弯矩。若你连接明显有偏心（薄臂法兰、法兰螺栓距离边缘过大），请转用 FEA 或等待偏心弯矩校核上线。
