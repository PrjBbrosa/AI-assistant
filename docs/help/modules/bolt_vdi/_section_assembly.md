# 装配属性章节

本章节用于**描述装配阶段的控制方式、散差程度、以及预紧力损失估算**：拧紧方式（扭矩/转角/液压/热装）、拧紧系数 αA、装配利用系数 ν、附加防松扭矩、嵌入损失 FZ、热预紧力损失 Fth，以及校核模式下的已知最小预紧力 FM,min。这些一起决定"设计阶段允许多大预紧力范围"。

## 为什么要填这些

VDI 2230 的核心逻辑是：

```
FM,max (装配最大预紧力) = αA · FM,min (装配最小预紧力)
                                                [α_A ≥ 1.0; FM,max / FM,min 都是 N]
```

不同拧紧方式控制精度不同 → αA 不同 → FM,max/FM,min 差距不同：

- **扭矩法**（MA 控制）：读扭矩扳手读数。αA 常见 **1.4–1.8**，散差大，因为扭矩与预紧力之间受摩擦系数波动强烈影响。
- **转角法**（屈服点 + 转角补偿）：旋入扭矩到预紧力接近屈服、再转一个角度。αA 常见 **1.15–1.35**，散差小。
- **液压拉伸法**：液压缸直接拉螺栓，旋紧螺母再卸液压。αA ≤ **1.15**，最精密。
- **热装法**：加热螺栓后旋入、冷却收缩产生预紧力。αA ~1.15，专用工艺。

为什么要知道 FM,max 和 FM,min？

- **FM,min** 是你必须保证的最低预紧力 —— 这么低还没松开，密封/防滑才有效。R3 残余夹紧力校核就以此为依据。
- **FM,max** 是装配时可能达到的最高预紧力 —— 这么高还没屈服，螺栓才安全。R4 装配应力校核以此为依据。

中间还要扣两项损失：

- **嵌入损失 FZ**（N）：接触面粗糙度压实造成的预紧力损失，装配后几小时到几天内发生。VDI 2230 嵌入损失参考表按粗糙度等级给出单界面典型值 1–3 μm，本工具在 "已选粗糙度等级 + FZ=0" 时自动估算（见 `terms/bolt_embed_loss` 了解本工具简化实现与 VDI 原表的差异）。
- **热预紧力损失 Fth**（N）：螺栓与被夹件热膨胀系数不同导致的预紧力变化。本工具在工况章节填温度/材料后自动估算（见该章节）。

## 装配利用系数 ν 的含义

ν（0.8–0.95）表示"装配时允许用到屈服强度的百分比"。工程实践：

- **ν = 0.9**（默认）：装配允许 σvm,assembly ≤ 0.9·Rp0.2，留 10% 屈服裕量。标准工业螺栓连接通用取值。
- **ν = 0.95**：更激进，适用于高强螺栓且控制精密（转角法、液压拉伸）。
- **ν = 0.8**：更保守，适用于疲劳敏感或反复拆装的连接。

本章数字都会直接进入 R4 校核：σ_allow,assembly = ν · Rp0.2。

## 输入 / 产出

**输入**：
- 拧紧方式（扭矩/转角/液压/热装） —— 用于 αA 建议提示；扭矩法还影响 R5 扭转残余
- 拧紧系数 αA（根据拧紧方式取值；手动可调）
- 装配利用系数 ν
- 附加防松扭矩 MA,prev（锁紧螺母等；无则 0）
- 嵌入损失 FZ（若填 0 且已选粗糙度，则自动估算）
- 热损失 Fth（从工况章节自动填入；手动可覆盖）
- 校核模式下可填"已知最小预紧力 FM,min"

**产出给下一步**：
- αA → 连接 FM,max 与 FM,min
- ν → R4 装配等效应力允许值 σ_allow,assembly
- FZ + Fth → 预紧力损失扣除
- 拧紧方式 → R5 扭转残余系数 k_tau（扭矩法 0.5，其他 0）

## 常见坑

- **αA 手动设成 1.0 是最危险的**：意味着你假设装配零散差，实际不存在这种工艺。扭矩法任何情况下 αA ≥ 1.4。
- **FZ 填一个很小的值（如 100 N）覆盖粗糙度自动估算**：这会绕过本工具的嵌入损失自动估算路径，UI 不会再警告。若是特意手动覆盖应该在"粗糙度等级"下拉里选，不是靠 FZ = 小正数。
- **MA,prev 与装配扭矩 MA 不是一回事**：MA,prev 是"额外"的防松扭矩，不包含在 MA 里。锁紧螺母典型 MA,prev = 3–6 N·m。
