# 工况数据章节

本章节用于**描述连接在现场实际承受的外力、温度、寿命要求**：最大轴向载荷 FA,max、最大横向载荷 FQ,max、密封所需残余夹紧力 FK,req、摩擦界面数 qF、防滑摩擦系数 μT、载荷循环次数 ND、螺栓/被夹件材料与温度（用于热损失 Fth 估算）、螺纹表面处理（影响疲劳极限 σASV）。

## 为什么要填这些

装配阶段的预紧力保护 R3/R4 校核；**工况阶段**的载荷驱动 R5（服役应力）、R3（残余夹紧力）、疲劳、防滑这些校核。这一章回答"连接装完之后每天被多大力拉、被多热、要干多少次循环"。

### 载荷分三类

1. **轴向载荷 FA,max**（N）—— 螺栓轴向拉力，进入 R5 服役应力与疲劳校核：
   ```
   F_bolt,work,max = FM,max + Φ_N · FA,max         # 螺栓承担的部分
   σ_ax,work = F_bolt,work,max / As                # 服役轴向应力
   ```
2. **横向载荷 FQ,max**（N）—— 剪切载荷，触发防滑校核 R1（要求夹紧力在 qF 个摩擦界面上能抵抗 FQ）：
   ```
   F_K,req_slip = FQ,max / (qF · μ_T)              # 防滑所需残余夹紧力 [N]
   ```
3. **密封夹紧力 FK,req**（N）—— 用户自定的最低残余夹紧力，用于密封、定位或刚度要求。

最终残余夹紧力要求 = max(防滑需求 FK,req,slip, 密封需求 FK,req)。

### 温度与热损失

热预紧力损失 Fth 不是你直接填的数，而是工具根据：
- 螺栓温度 T_bolt、被夹件温度 T_parts
- 螺栓热膨胀系数 α_bolt、被夹件热膨胀系数 α_parts
- 总夹紧长度 lK、螺栓柔度 δs、被夹件柔度 δp

按 VDI 2230 温差公式自动估算（`core/bolt/calculator.py:312`）。工具会把估算值回填到装配章节的 Fth 字段。

**关键**：热损失与装配损失分开计算，但都扣除在 FM,min 基础上：

```
FM,work,available = FM,min − FZ − Fth − (1 − Φ_N) · FA,max     [N]
R3 要求: FM,work,available ≥ FK,req
```

### 疲劳校核前提

勾选"考虑温度+疲劳"级别后，本章的 ND（载荷循环次数）和"螺纹表面处理"才起作用。简化 Goodman 校核：

```
σ_a = Φ_N · FA,max / (2·As)                       # 交变幅值 [MPa]
σ_m = (FM,max + 0.5·Φ_N·FA,max) / As              # 平均应力 [MPa]
σ_ASV = VDI 2230 表 A1 内插值（按 d）·（2×10⁶/ND）^0.08
σ_a_allow = σ_ASV · max(0.1, 1 − σ_m / (0.9·Rp0.2))   # Goodman 折减
R9 通过: σ_a ≤ σ_a_allow
```

表面处理：**轧制螺纹 σASV 明显高于切削螺纹**（VDI 2230 表 A1），轧制是工业螺栓标配；切削螺纹一般只见于自制件或特殊材质。

## 输入 / 产出

**输入**：
- 工况类型（轴向 / 横向 / 组合 / 自由）—— 仅用于记录场景，不改变计算
- FA,max（最大轴向工作载荷，N）、FQ,max（最大横向载荷，N）
- FK,req（密封所需残余夹紧力，N；无密封可填 0）
- qF（摩擦界面数，防滑校核用；通常 1 或 2）
- μT（防滑摩擦系数，钢-钢干 0.15–0.20，钢-钢油 0.10–0.12，典型默认 0.18）
- 载荷循环次数 ND（疲劳校核用；标准典型 ≥ 2×10⁶ 视为无限寿命）
- 螺纹表面处理（轧制 / 切削）
- 螺栓材料 + 被夹件材料 → 自动填 α_bolt、α_parts
- 螺栓温度 T_bolt、被夹件温度 T_parts

**产出**：
- FA,max → R5 σ_ax,work、疲劳 σ_a/σ_m
- FQ,max + qF + μT → 防滑所需残余夹紧力
- ND + 表面处理 → 疲劳 σ_ASV
- α 系数 + T → 热损失 Fth（自动填回装配章节）

## 常见坑

- **FA,max 与 FQ,max 需填"峰值"而非 RMS**：VDI 2230 校核按最恶劣工况，不是平均值。若外力有动态分量，FA,max 应含动载放大。
- **μT 的"默认 0.18" 只对钢-钢干净界面**：湿/油润滑界面应降至 0.10–0.12；防滑校核用偏低值更保守。
- **FK,req = 0 不代表不校核 R3**：R3 仍会要求残余夹紧力 ≥ 防滑需求；只有在"无密封 + FQ,max = 0"时 FK,req 的输入才完全脱钩。
- **载荷循环次数 ND = 1** 不等于"不做疲劳"：工具仍会算 σa，只是许用值变大。真不做疲劳的话选"常规校核"或"考虑温度"层级。
