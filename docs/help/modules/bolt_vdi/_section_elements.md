# 连接件参数章节

本章节用于**填写螺栓本身的几何、强度、摩擦参数**：公称直径 d、螺距 p、强度等级、螺纹/支承面摩擦系数、支承面内外径与许用压强。这些是后续所有校核（装配应力、服役应力、支承面压强、螺纹脱扣）的输入源头。

## 为什么要填这些

螺栓校核的第一步是确定"这颗螺栓到底是什么规格、由什么材料做的、和被夹件之间有多少摩擦"。每一项都独立影响一个或多个校核项：

- **公称直径 d + 螺距 p** → 派生中径 d2、小径 d3、应力截面积 As（VDI 2230 / ISO 898-1 公式）。选择标准规格（M6/M8/M10...）时，UI 从内置 GB/T 196 表查表填入；自定义规格时自行输入，计算器会按以下公式派生：
  ```
  As = π/4 · (d − 0.9382·p)²    [d,p: mm → As: mm²]
  d2 = d − 0.64952·p            [mm]
  d3 = d − 1.22687·p            [mm]
  ```
- **强度等级** → 自动填入屈服强度 Rp0.2（GB/T 3098.1 / ISO 898-1 表）。Rp0.2 决定装配允许应力 σ_allow,assembly = ν·Rp0.2 和服役允许应力 σ_allow,work = Rp0.2 / S_F
- **螺纹摩擦 μG、支承面摩擦 μK** → 决定拧紧扭矩 MA 分配给螺纹和头部的比例；服役阶段扭矩法下还保留 ~50% 装配扭矩（R5 扭转残余），直接影响等效应力 σvm,work
- **支承面内外径 DKm,i / DKm,o** → R7 支承面压强校核 p = FM,max / A_bearing 的计算面积
- **许用支承面压强 p_G** → R7 PASS/FAIL 的判据（钢约 700 MPa、铝约 300 MPa）

## 输入 / 产出

**输入**：
- 连接形式（螺纹孔 / 通孔） — 决定柔度模型使用 d 还是等效长度
- 公称直径 d、螺距 p（或"自定义"手动输入）
- 强度等级（或"自定义"手动输入 Rp0.2）
- 螺纹摩擦 μG（典型 0.10–0.14，干净干 0.14，有油 0.08–0.12）
- 支承面摩擦 μK（典型 0.10–0.14，与 μG 类似取值）
- 牙型角（公制 60°，几乎不变）
- 支承面内径 DKm,i（通孔直径）、外径 DKm,o（螺栓头对边宽度）
- 支承面材料 + 许用支承面压强 p_G

**产出给下一步**：
- 螺纹几何（d2、d3、As）→ R4/R5 应力计算的几何基础
- Rp0.2 → 全部应力型校核的允许值基准
- 摩擦链 → 装配扭矩 MA、服役扭转残余 k_tau
- 支承面几何 → R7 支承面压强校核

## 常见坑

- **强度等级写成 "8.8 级" 而非 "8.8"**：下拉选项严格匹配 "8.8"，手动填写时切勿带单位字。
- **摩擦系数默认值不一定适合你的工况**：默认 μG=0.12、μK=0.14 是 VDI 2230 表 A.5 的中间典型值。干燥、锈蚀的实际工况 μ 可达 0.18–0.20，会显著降低允许预紧力。若工况不明，**取偏高值更保守**（预紧力偏低）。
- **支承面许用压强按配对软侧取**：钢螺母压在铝法兰上，p_G 要取铝侧 ≈ 300 MPa，不是钢螺母的 700 MPa。
