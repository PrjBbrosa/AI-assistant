# 螺纹脱扣校核章节

本章节用于**校核螺栓拧到最大预紧力时，内外螺纹是否会被拉滑（脱扣 / 滑牙）**：输入有效旋合深度 m_eff、内螺纹剪切强度 τBM、外螺纹剪切强度 τBS、脱扣安全系数要求。VDI 2230 R8 校核。

## 为什么要填这些

螺栓拉力不光要传给被夹件，还要穿过螺纹这个"接力环"。如果螺纹啮合不够深，或内螺纹材料太软（常见于铝合金壳体），螺栓还没断，螺纹先被拉滑 —— 这是最难检测的失效模式之一，外观看不出来但预紧力早已失效。

本章直接校核：**螺栓最大拉力**（FM,max 或服役 F_bolt,work,max）是否小于内外螺纹剪切承载力。

## 公式（本工具实现）

```
A_SB = π · d3 · m_eff · C1        # 外螺纹（螺栓侧）剪切面积 [mm²]
A_SM = π · d  · m_eff · C3        # 内螺纹（螺母/壳体侧）剪切面积 [mm²]
F_strip_B = A_SB · τ_BS           # 外螺纹剪切承载力 [N]
F_strip_M = A_SM · τ_BM           # 内螺纹剪切承载力 [N]
S_strip = min(F_strip_B, F_strip_M) / F_bolt_max       # 脱扣安全系数
R8 通过: S_strip ≥ S_strip,req
```

其中：
- **d3**（小径，mm）：ISO 898-1 派生（d − 1.22687·p），从连接件章节自动取得
- **d**（公称直径，mm）：从连接件章节取得
- **m_eff**（有效旋合深度，mm）：螺栓实际旋入内螺纹的啮合长度
- **C1 ≈ 0.75**（外螺纹有效承载系数）、**C3 ≈ 0.58**（内螺纹有效承载系数）—— 本工具按 ISO 公制标准取固定值
- **τ_BS**（外螺纹/螺栓材料剪切强度，MPa）：默认 Rp0.2 × 0.6；可手动覆盖
- **τ_BM**（内螺纹/螺母或壳体材料剪切强度，MPa）：**必填**，无默认值

## 什么时候跳过脱扣校核

**m_eff 留空** → 本工具跳过 R8，整体结论判为 "incomplete"（不完整），不会给 PASS 绿灯。

什么时候留空？

- 螺栓用标准螺母（ISO 4032 / DIN 934），且螺母材料明确不低于螺栓强度 → 一般免校核
- 设计评审已另做 R8 或用产品标准认证过 → 不必重复
- 螺栓旋入已知 > 1.5d 深度的钢基体 → 风险极低

什么时候必须校核？

- 螺栓旋入铝 / 铸铁 / 塑料基体（τ_BM 显著低于钢）
- 螺纹孔深度接近或低于 1.0d（m_eff ≤ d）
- 重要连接需完整校核记录

## 输入 / 产出

**输入**：
- 有效旋合深度 m_eff（mm；标准螺母高度 0.8d–1.0d；螺纹孔需实测）
- 内螺纹剪切强度 τ_BM（MPa）—— 钢螺母 Rp0.2·0.6（8 级螺母约 380 MPa）；铝壳体 150–200；铸铁 200–250
- 外螺纹剪切强度 τ_BS（MPa；留空自动取 Rp0.2·0.6）
- 脱扣安全系数要求（典型 1.25）

**产出**：
- S_strip = min(F_strip_B, F_strip_M) / F_bolt_max
- PASS/FAIL vs S_strip,req

## 常见坑

- **m_eff 填螺母全高**：实际参与承载的只有"已啮合"部分。标准螺母 M10 高度 8 mm，但开头 1–2 扣承载最大、末尾几乎不承载，工程近似按 0.8–1.0 倍螺母高度取 m_eff。
- **τ_BM 填 Rp0.2 而非 Rp0.2·0.6**：剪切强度典型为抗拉的 0.6 倍。填成抗拉值会高估承载力 67%。
- **铝壳体螺纹没加深度警告**：M10 在 Al 6061（τ_BM ≈ 180 MPa）里需要 ≥ 1.5d 才能达到与钢螺母相当的承载；留默认 0.8d–1.0d 会判 FAIL。设计时需主动加深孔。
- **不填 τ_BM 但填了 m_eff**：计算器会抛 `InputError`。若启用 R8 就必须填 τ_BM。
