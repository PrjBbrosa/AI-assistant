# PySide6 本地桌面壳设计（Claude 风格）

## 1. 目标
- 构建本地运行的计算平台框架，先实现螺栓模块，其他模块保留入口。
- 保持计算内核与 UI 解耦，方便后续 `.exe` 打包和模块扩展。

## 2. 结构设计
- 左侧导航：模块切换（螺栓连接、轴连接、轴承、蜗轮、弹簧、材料与标准库）
- 右侧内容：`QStackedWidget` 承载每个模块页面
- 螺栓页：Chapter 14 式章节导航 + 全量参数表单 + 可读结果面板
- 占位页：统一“建设中”结构，保持后续扩展一致性

## 3. 视觉风格
- 主题方向：暖灰底 + 低饱和卡片 + 橙色强调
- 关键色：
  - 背景：`#F7F5F2`
  - 侧栏：`#EEE7DE`
  - 文本：`#1F1D1A`
  - 强调：`#D97757`
  - 边框：`#D9D3CA`
- 组件特征：圆角、细边框、弱阴影层次、清晰状态徽标

## 4. 技术要点
- `core/bolt/calculator.py` 提供纯计算函数
- `src/vdi2230_tool.py` 作为 CLI 包装层，避免已有命令失效
- `app/ui/pages/bolt_page.py` 调用同一内核，保证 CLI/GUI 结果一致
- `app/ui/widgets/clamping_diagram.py` 提供 FM/FA/FK 夹紧示意图
- `scripts/build_exe.bat` 使用 PyInstaller `--onedir` 打包

## 5. 风险控制
- 风险：GUI 输入自由度高，JSON 容易出错
- 控制：参数错误直接弹窗、样例一键加载、输出保留原始分项判定
