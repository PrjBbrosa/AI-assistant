# VDI 2230 螺栓校核工具设计文档

## 1. 目标
在空仓库中交付一个可复核、可运行的 VDI 2230 螺栓校核工具，并同步给出一份中文计算说明文档，确保“公式来源、单位、代码实现、示例结果”可追溯。

## 2. 备选方案

### 方案 A（推荐）：Python CLI + JSON 输入输出（核心链路）
- 特点：实现快、公式可读性高、便于对接后续 Web/桌面 UI。
- 范围：先覆盖 VDI 2230 核心链路（R1-R8 的工程常用部分）。
- 取舍：不在首版覆盖全部细分工况（如完整多级锥台刚度自动求解、全疲劳谱）。

### 方案 B：Excel 模板驱动
- 特点：工程师上手快，便于手工审阅。
- 取舍：版本管理差、测试自动化弱、扩展性一般。

### 方案 C：直接做 Web 前端工具
- 特点：交互友好。
- 取舍：空仓库从零起步成本高，先做会稀释计算内核验证精力。

## 3. 选型结论
采用方案 A。先确保计算核心正确，再根据需要扩展 UI。

## 4. 架构设计

### 4.1 模块
- `src/vdi2230_tool.py`
  - 输入读取与校验
  - 螺纹几何参数推导（`As`, `d2`, `d3`）
  - 核心公式计算（`FMmin/FMmax`, `phi`, 扭矩、应力、校核）
  - 结果输出（JSON）
- `examples/input_case_01.json`
  - 标准示例输入
- `docs/vdi2230-calculation-spec.md`
  - 公式说明、变量定义、单位与实现边界

### 4.2 数据流
1. 读入 JSON 输入参数
2. 标准化参数并补齐可推导几何量
3. 计算预紧力、扭矩、装配与服役校核量
4. 生成结构化结果和通过/不通过结论

## 5. 错误处理
- 缺失关键字段（强度等级、摩擦系数、刚度/顺从度）直接报错。
- 非法值（负数、零分母、摩擦系数越界）给出明确字段名和修复提示。
- 当输入导致物理不可解（例如 `phi_n <= 0`）时，返回失败状态并输出原因。

## 6. 测试策略
- 用示例输入执行一次端到端计算。
- 检查以下关键量是否合理：
  - `FMmax = alpha_A * FMmin`
  - `0 < phi_n < 1.2`（含载荷导入修正）
  - 装配等效应力与材料利用系数比较结果一致
  - 残余夹紧力满足防滑/密封需求

## 7. 范围边界（首版）
- 覆盖：轴向外载、横向防滑需求、嵌入损失、热损失（简化为等效力损失）、装配扭矩和装配/服役强度校核。
- 不覆盖：VDI 2230 全部特例（复杂载荷谱疲劳、详细锥台压缩体自动建模、多螺栓群耦合）。

## 8. 后续扩展
- 增加 ISO 材料与螺纹规格数据库
- 增加刚度自动计算（按几何推导 `delta_s`, `delta_p`）
- 增加 Web UI 与批量工况计算
