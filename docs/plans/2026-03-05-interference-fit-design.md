# 过盈配合模块设计（圆柱面首版）

## 1. 目标
- 在桌面端新增“过盈配合”模块，提供圆柱面过盈（实心轴 + 厚壁轮毂）的完整输入-计算-结果链路。
- 输出完整结果：接触压力、可传递扭矩、压入力、轴/轮毂应力、安全系数、调整建议。
- 提供压入力随过盈量变化曲线，支持装配窗口评估。

## 2. 参考流程（仅参考流程，不复刻界面）
- 参考 DIN 7190 相关流程组织：参数输入 → 载荷与安全系数 → 过盈/压力 → 应力校核 → 结果图表与消息。
- 对应实现章节：
  1. 校核目标
  2. 几何与过盈
  3. 材料参数
  4. 载荷与摩擦
  5. 工艺记录（首版记录项）
  6. 压入力曲线图
  7. 校核结果与消息

## 3. 架构
- 新增 `app/ui/pages/base_chapter_page.py`：抽象章节导航与页面公共骨架。
- 新增 `app/ui/pages/interference_fit_page.py`：过盈配合页面。
- 新增 `core/interference/calculator.py`：过盈配合计算内核。
- 新增 `app/ui/widgets/press_force_curve.py`：压入力曲线控件。

## 4. 公式边界（首版）
- 假设：
  - 线弹性
  - 均匀接触压力
  - 常摩擦系数
  - 忽略粗糙度/涂层/温度修正项对有效过盈的影响（作为后续扩展）
- 核心输出：
  - 由过盈量反算接触压力区间 `p_min/p_max`
  - 由接触压力计算 `T_cap`、`F_ax_cap`、`F_press`
  - 由压力计算轴/轮毂应力与安全系数
  - 由需求载荷反算最小需求过盈量 `delta_required`

## 5. 验证策略
- 单元测试覆盖：
  - 正常工况输出完整结果结构
  - 几何非法输入报错
  - 需求过盈量计算输出
- UI 验证：
  - 模块切换、样例加载、计算触发、曲线与结果更新
