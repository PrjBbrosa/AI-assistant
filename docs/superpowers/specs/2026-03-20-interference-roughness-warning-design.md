# Interference Roughness Warning Design Spec

## Goal
在过盈配合页面的“摩擦与粗糙度”章节中新增一个固定的特殊提醒框，提示批量生产时粗糙度与表面机加工状态会显著影响压入散差，并给出简要的分析方向。

## Scope

### In Scope
- 在 `InterferenceFitPage` 的“摩擦与粗糙度”章节内新增固定提醒框。
- 采用区别于普通输入卡片的样式，让提醒具备明显警示感。
- 文案明确指出批量生产压入散差的主要来源不是只看公差，还要看粗糙度相关因素。
- 文案提供简短、可执行的排查方向。
- 增加 UI 测试锁定该提醒框与核心文案。

### Out of Scope
- 不改动现有过盈、粗糙度或压入力计算公式。
- 不新增动态建议逻辑。
- 不把该提醒写入计算 verdict。
- 不新增报告导出段落。

## User Intent
用户希望在粗糙度页面直接看到一个醒目的工程提示，帮助使用者在批量生产现场快速意识到：

- 压入力突然偏大或偏小，很多时候与配合面微观形貌有关。
- 粗糙度影响的不只是表面“好不好看”，而是有效过盈、接触压力与装配摩擦的波动。
- 现场排查不应只盯尺寸和 `Ra`，还要看更具体的机加工与装配因素。

## UX Design

### Placement
- 页面：`过盈配合 · 圆柱面校核`
- 章节：`摩擦与粗糙度`
- 位置：粗糙度相关输入项之后、章节底部说明位置

### Visual Treatment
- 使用单独的提醒框而不是普通 `SubCard`
- 暖色浅底 + 左侧强调边框
- 标题简短直接，正文采用 4 条简要排查方向

## Copy

### Title
`批量生产提示：粗糙度会放大压入散差`

### Body
`批量生产中，压入力突然偏大或偏小，很多时候并不只是公差波动，而是配合面机加工状态变化导致有效过盈与装配摩擦同时变化。`

`可优先从以下方向排查：`

`1. 看 Rz 与压平量是否波动：粗糙峰压平会改变有效过盈，直接影响接触压力与压入力。`

`2. 不要只看 Ra：波纹度、圆柱度、刀纹方向和局部毛刺，往往比单一粗糙度数字更影响压入散差。`

`3. 区分是过盈变了还是摩擦变了：若压入力与脱出力一起变化，多半先看有效过盈；若只压入力异常，优先排查润滑、清洁度和表面擦伤。`

`4. 批量管控优先项：统一机加工参数、Rz 检测口径、润滑方式、压装速度、倒角去毛刺和清洁度。`

## Implementation Notes
- 通过 `InterferenceFitPage._create_chapter_page()` 在“摩擦与粗糙度”章节插入专用提醒框。
- 提醒框建议拆成小 helper，避免把特殊分支继续堆在通用章节生成逻辑里。
- 样式新增到 `app/ui/theme.py`，保持和现有 `Card/SubCard` 体系一致。

## Testing
- 断言页面存在提醒框 widget。
- 断言提醒框使用专用 object name。
- 断言提醒框文案包含：
  - `批量生产`
  - `压入散差`
  - `有效过盈`
  - 至少一个现场排查方向关键词，例如 `波纹度`、`刀纹方向`、`润滑`
