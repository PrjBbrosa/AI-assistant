## 5. R8 螺纹脱扣校核验证

### 5.1 实现说明

R8 校核比较内螺纹（螺母/螺纹孔）与外螺纹（螺栓）的剪切承载能力，取较弱侧计算安全系数。

核心公式：
- 螺栓侧剪切面积：`A_SB = π × d3 × m_eff × C1`（C1 = 0.44，考虑螺纹牙型承载比）
- 螺母侧剪切面积：`A_SM = π × d × m_eff × C3`（C3 = 0.65，内螺纹牙型承载比）
- 螺栓侧脱扣力：`F_strip_bolt = A_SB × τ_BS`（τ_BS 默认 = Rp0.2 / √3）
- 螺母侧脱扣力：`F_strip_nut = A_SM × τ_BM`（τ_BM 由用户输入，螺母/壳体材料剪切强度）
- 安全系数：`S_strip = min(F_strip_bolt, F_strip_nut) / FM_max`

### 5.2 验证案例

**案例**: M12×1.75 / 12.9 / 铝壳体螺纹孔

| 参数 | 值 |
|------|-----|
| d / d3 / p | 12.0 / 10.106 / 1.75 mm |
| m_eff | 12.0 mm |
| τ_BM (铝 AlSi) | 350 MPa |
| τ_BS (12.9 钢) | Rp0.2/√3 = 623.5 MPa |

| 验证项 | 工具 | 手算 | 匹配 |
|--------|------|------|:----:|
| A_SB | 528.1 mm² | π×10.106×12×0.44 = 528.1 | OK |
| A_SM | 979.7 mm² | π×12.0×12×0.65 = 979.7 | OK |
| F_strip_bolt | 329,230 N | 528.1×623.5 = 329,230 | OK |
| F_strip_nut | 342,895 N | 979.7×350 = 342,895 | OK |
| 临界侧 | 螺栓 | min 在螺栓侧 | OK |
| 安全系数 | 24.1 | 329,230/13,632 = 24.1 | OK |

**说明**: 10 个单元测试全部通过，覆盖通过/不通过判定、临界侧识别、公式正确性、
默认 τ_BS 推导、自定义安全系数、与 overall_pass 的集成。

## 6. 自动柔度建模精度分析

### 6.1 螺栓柔度 δs

本工具使用简化模型 `δs = l_eff / (E × As)`，VDI 2230 标准使用分段模型
（头部 + 光杆 + 螺纹段 + 啮合区，各段截面积不同）。

| 工况 (M12, E=210GPa, 通孔) | VDI 标准 | 本工具 | 偏差 |
|---------------------------|----------|--------|------|
| 全螺纹, l_K=15mm | 1.372e-6 | 1.390e-6 | +1.4% |
| 全螺纹, l_K=20mm | 1.654e-6 | 1.673e-6 | +1.1% |
| 全螺纹, l_K=40mm | 2.784e-6 | 2.803e-6 | +0.7% |
| 全螺纹, l_K=100mm | 6.175e-6 | 6.194e-6 | +0.3% |
| **有25mm光杆**, l_K=40mm | 2.424e-6 | 2.803e-6 | **+15.6%** |

**分析**:
- 全螺纹螺栓：偏差 < 1.5%，工程可接受
- 有光杆螺栓：偏差可达 15%，因简化模型统一按 As 计算，
  未区分光杆段 (A=πd²/4) 和螺纹段 (A=As) 的截面积差异
- 偏差方向：高估 δs → φ_n 偏小 → 偏安全（高估残余夹紧力）

### 6.2 被夹件柔度 δp

| 模型 | δp (mm/N) | 说明 |
|------|-----------|------|
| 圆柱 | 3.32e-7 | 假设 D_A 环截面均匀受力，偏硬 |
| 锥台 (Lori-Engel) | 6.02e-7 | 更真实的压缩锥效应 |

**分析**: 圆柱模型在 D_A >> D_w 时显著低估 δp（高估刚度），导致 φ_n 偏大，
这是偏危险方向。建议大法兰工况选用锥台模型。

## 7. 与 VDI 2230 标准的结构性差异

| # | 差异点 | 影响方向 | 严重程度 |
|---|--------|----------|----------|
| 1 | 螺栓柔度简化为单段模型 | 有光杆时偏差~15%（偏安全） | 中 |
| 2 | 多层模式强制圆柱体 | 法兰大外径时偏危险 | 中高 |
| 3 | 摩擦系数单值输入 | VDI 用 μ_min/μ_max 范围 | 低 |
| 4 | 无偏心弯矩 (R6) | 偏心加载法兰需人工评估 | 高 |
| 5 | R8 螺纹脱扣已实现（内/外螺纹剪切对比） | 需用户提供有效啮合长度 m_eff | 低 |
| 6 | 疲劳用 Goodman 简化 | VDI 用 Haigh 图 + f_τ 修正 | 低 |

## 8. 结论与建议

### 8.1 准确性评定

| 层面 | 评定 |
|------|------|
| 公式链路（直接输入柔度） | **高** — 与 VDI 2230 完全一致 |
| 自动柔度建模（全螺纹） | **高** — 偏差 < 1.5% |
| 自动柔度建模（有光杆） | **中** — 偏差可达 15%，偏安全方向 |
| 热/嵌入自动估算 | **高** — 与标准公式完全一致 |
| 疲劳校核 | **中高** — σ_ASV 查表准确，Goodman 为简化替代 |
| 螺纹脱扣校核 (R8) | **高** — 内/外螺纹剪切面积 + 安全系数，与 VDI 2230 一致 |

### 8.2 使用建议

1. **推荐做法**: 手动输入已知的柔度/刚度值（来自 FEM 或试验），公式链路完全可信
2. **自动建模**: 适合方案筛选阶段，全螺纹螺栓精度高；有光杆时需注意偏差
3. **大法兰**: 被夹件模型优先选锥台，避免圆柱模型高估刚度
4. **关键安全件**: 建议与 KISSsoft/eAssistant 交叉验证后使用

### 8.3 后续改进方向（按优先级）

1. 螺栓柔度改为分段模型（支持光杆/螺纹分段）
2. 多层模式支持逐层选择锥台/圆柱
3. 增加 R6 偏心弯矩校核
